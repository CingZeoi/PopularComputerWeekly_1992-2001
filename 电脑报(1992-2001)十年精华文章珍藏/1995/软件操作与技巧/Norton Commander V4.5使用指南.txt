冯成兴

        一、Norton Commander V4.5 简介
        (一)Norton Commander V4.5 的安装盘和程序文件
        Norton Commander V4.5 的安装盘共两张，第一张盘包括安装文件Install.exe及NC的主要文件：
        4372ansi.set	将CP437转换为ANSI标准	
        8502ansi.set 	将cp850转换为ANSI标准	
        ansi2437.set 	将ANSI 转换为437 标准	
        ansi2850.set 	将ANSI 转换为850 标准	
        nc.cfg       	针对过滤器和观察器的配置文件	
        nc.exe       	NC的加载文件	
        nc.hlp       	NC的帮助文件	
        nc.ico       	NC的图标文件	
        ncedit.exe   	NC的文本编辑器	
        ncmain.exe   	NC的文件管理器	
        nczip.exe    	NC的压缩文件	
        packer.exe   	NC的压缩解释程序	
        packer.set   	packer.exe的程序文件	
        readme.txt   	不包括在用户手册内的重要信息	
        term90.exe   	终端仿真程序	
        term90.hlp   	终端仿真程序的帮助文件	
        term90.res   	终端仿真的资源文件	
        testmdm.scx  	终端仿真器的字体文件	
        cmpsrv.scx   	终端仿真器的字体文件	
        genie.scx    	终端仿真器的字体文件

        如果希望节约你的磁盘空间，可只安装该盘即可。
        第二张盘只有一个压缩文件，nc_arc.exe  解压后为十余个文件观察器和格式转换程序。
        (二)Norton Commander V4.5特点和主要功能
        1.双面板显示：
        Norton Commander V4.5 用户界面的第一行是主菜单条，最后两行分别是 DOS命令行和热键行，中间部分平均分为左、右两个面板，两个面板可以如 Pctools一样左边显示磁盘树形目录，右边显示磁盘文件。也可反过来，右边显示树形目录，左边显示磁盘文件。或一边显示文件或者目录，另一边显示一些信息或成为快速文件观察窗口。此外，两边还可以同时显示磁盘的目录树，两边同时显示磁盘文件，两边同时显示重要信息。这种灵活多样的显示方式，使你在左右面板之间拷贝和移动文件等十分直观方便。左右面板还可以同时或分别关闭或快速的相互交换内容。
        2.方便的用户菜单：
        操作者可以根据自己的需要编辑一个用户菜单，在NC中可随时用热键调出该菜单，选择自己需要的菜单项。你还可以设置成一进入NC时就弹出菜单，这样你可以非常方便地进入菜单设置的某一应用程序。
        3.联机通讯和终端仿真：
        NC支持所有的串、并口通讯，如果两台PC之间已通过串并口线连结，NC可以以最快速度在两机之间传输数据。如果用户已入网，NC的终端仿真功能可以使你的机器仿真终端，享用网络资源和实现远距离通讯。
        4.各种格式文件观察器：
        Norton Command具有观察各种格式文件的功能，它可以察看十余种不同的格式文件，包括GIF、BMP等图形文件和数据库、电子表格等。
        5.灵活多样的操作方式：
        菜单、命令行、热键均可操作。
        6.独特的屏幕保护功能：
        按设定的时间关闭屏幕，以保护显示器。
        7.与中文平台极强的兼容能力：
        NC在许多著名的中文系统中都能很好的运行。

        (二)运行Norton Commander 4.5
        进入你在安装时指定的目录，键入NC回车后即可运行NC V4.5。此时，用Tab 或Ctrl+I键可将光条在两个面板之间移动，用Ins 键可以点亮面板中的目录或文件等结合下拉菜单、各功能键和热键，你将可以随心所欲的使用 Norton Commander 为你服务了。

        ├────────────────────┤
        │Command for this extension:             │
        │                                        │
        ├────────────────────┤
        │( ) Filename with extension      (!.!)│
        │( ) Filename without extension   (!)    │
        │( ) Drive letter                 (!:) │
        │( ) Pathname                     (!\)  │
        │( ) User Definde                        │
        ├────────────────────┤
        │          ［ Ok ］ ［ Cancel ］         │
        └────────────────────┘
        以上窗口中，第一项可填入你希望成为与执行程序相关联的文件的扩展名。第二项是你的命令项，可键入你的执行命令。再下一框中的五项分别是：
        1).文件名和扩展名，这是把相关文件的文件名和扩展名作为扩充命令的参数。
        2).扩展名之外的文件名，仅把相关文件的文件名作为执行命令的参数。
        3).驱动器字符，把驱动器字符作为执行命令的参数。
        4).路径名，以路径名为执行命令的参数。
        5).用户定义。
        你A一旦设置好后按［ OK ］，即可使你的文件功能得到扩展。
        例如：
        在File extension: 这一栏中填入bas 。
        在Command for this extension: 栏中键入qbasic。
        在下面五项中选择 Filename with extension (!.!)
        这样以后，你在NC介面下只要把光条移到某一扩展名为bas的basic文件，回车即可打开Qbasic，同时也把这一 bas文件装入Qbasic了。
        如果你设置了几个应用程序都与同一扩展名文件相关，届时NC将向你提问，执行哪一应用程序，你可根据意愿进行选择。
        (四)Option  选择菜单
        ┏━━━━━━━━━━━━━━━┓

        1.Configuration...  配置：
        该对话框可对五个项目进行配置。
        1).屏幕颜色：
        ()B & W 黑白方式        ()Laptop 便携机方式
        ()Color1第一种彩色      ()Color2 第二种彩色。
        2).屏幕中断保护的等待时间：
        ［］ On 打开或关上屏幕保护
        ［］ Miuntes  屏幕中断保护的时间设置。(可选择1-99分)
        3).打印口选择：
        该项选择你联接的打印口。
        4).面板选择：
        ［］ Show hidden files  选择是否显示隐含文件。
        ［］ Ins moves doun 是否允许插入模式：
        如果允许，在你用Ins 键选择文件时，光标将自动下移，否则光标将保持在原来的位置，除非你用光标键将其移动。
        ［］ Select directories 是否选择目录：
        如果在方括号里填上X，那么在你使用小键盘的*、＋、-号键选择文件时，目录将被当作文件也可被选，否则，只选文件不选目录。
        ［］ Auto change directory 自动改变目录：
        当你的NC面板一边是树形目录时，如果选择了该项，在你移动光条时， 当前目录立即随之改变，另一面板的文件表也随目录变化而变化。反之，则只有在你移动光条到某处目录回车后才有上述变化。
        5).其它选择：
        ［］ Menu bar alwaysvisible 菜单条是否经常出现：
        可选择屏幕顶部主菜单条经常出现或者必须点按f9才出现。
        ［］ Auto save setup 自动保存设置：
        在退出NC时是否自动保存当前NC的设置。
        ［］ Left-handed mouse 鼠标按钮管理：
        该项的设置与否，可使鼠标左右按钮功能互换，这将有利于左手使用者。
        ［］ fast mouse reset 快速鼠标重置。
        ［］ Copy newer files only 只拷贝新文件：
        如果对该项划X选择，当拷贝文件时不会拷贝那些与目标路径同名的文件，反之将以新文件覆盖目标路径中的同名文件。
        2.Editor 选择文本编辑器：
        选择该项回车后，弹出如下对话窗口：
        ┌────────────────┐

        │( ) Built-in                    │

        └────────────────┘
        这个窗口要求你选择在按下f4键时调入哪个编辑器，如果选择第一项，使用NC自己的编辑器，如果选择第二项，使用外部编辑器。一旦指定外部编辑器，将要求你在下面浅蓝色框内写入执行外部编辑器的命令。以后，击f4键，调入外部编辑器，如果要使用NC原来的编辑器击Alt+F4键。
        3.Confirmations...确认：

        以上五项是指在拷贝文件、移动文件、删除文件、删除子目录和退出NC时是否需要提示确认，如果需要确认，在［］内划上X，否则不予提示确认。提示时将弹出一醒目的窗口向你示警，要求你确认。
        4.Compression...压缩文件的设置：

        5.Auto menu 自动菜单：
        在选中该项后，进入NC时将自动弹出用户菜单。
        6.Path prompt 路径设置：
        选择该项，NC下部命令行的DOS 提示符内将显示当前路径。
        7.Key bar 键盘条，热键是Ctrl+B：
        在该项选择与否将决定进入NC后显不显示最后一行的热键条。
        8.Full screen 所有屏幕：
        该项将把NC的面板设置成屏幕的一半或全部屏幕。
        9.Mini status 小形状态条：
        如果选择该项，在面板的最一行会开劈一小窗口，显示你选中文件的个数和总字节数。

        在屏幕左上角显示或关闭时钟。

        保存你对NC的配置。

        四、其它
        (一)文件的压缩和释放：
        1.Compress 压缩文件，热键是Alt+F5：
        先将要压缩的文件或目录在目录表中点为高亮，然后按Alt+F5弹出以下对话框。
        ┌───────────────────┐
        │              Compress                │
        │Compress X file using XXX to          │

        │［ ］ Delete files afteruards         │

        第二框内的两个选项，第一个是是否包含子目录，第二个是压缩后，是否删除源文件。
        最后的三个选择按钮中如果选择了［ Select ］将弹出下一对话框：
        ┌───────────────────┐

        以上对话框中共有七个选项，其中第一个选项是NC自身已具备了的，若要用后面六个选项，必须将相关的应用程序拷贝到NC目录下，或者拷贝到DOS 的环境路径中。如果这个工作已完成，使用NC的文件压缩功能即可将文件压缩成以上各种模式。
        2).压缩文件的释放：
        在面板的文件表中点亮要释放的压缩文件，按Alt+F6键，一对话框弹出：
        ┌────────────────────────┐
        │                  Decompress                    │
        │Decompress "xxxxxxxx.xxx" using ARJ to          │
        │c:\                                            │

        2.新文件的建立和编辑：
        击Shift+F4弹出对话框
        ┌───────┐
        │     Edit     │
        │Edit the file:│
        └───────┘
        在对话框中键入新的文件名，回车即可建立一个新文件并且已调入编辑器供你编辑。
        3.随时可进行的屏幕中断保护：
        把鼠标光标移至屏幕右上角，可立即实现屏幕的中断保护。
        4.文件名和目录名的复制：
        在NC中执行应用程序，只需将光条放在要执行的文件上，回车即可，如若要在执行程序后加参数，你就必须在DOS 命令行上老老实实地键入执行程序名后跟参数再回车。这样做效率很低，利用复制功能可提高操作速度。
        在面板中将光条调至需要复制的文件名或目录名上，按Ctrl+Enter键，文件名或目录名将复制在DOS 命令行上，键入参数，回车即可执行应用程序。
        5.快速回到根目录：
        使用Ctrl+\组合键，无论你当前处在哪一级子目录，均可迅速退回到根目录。
