多媒体应用工具Video for Windows

曹国钧
1995-01-06

        VFW提供视频处理的软件主要有三个,一是VIDCAP,它将视频信号采样捕获后以.AVI文件存储在磁盘上;二是VIDEOEDIT,它将一段已经存储的视频图象文件进行编辑、处理等操作;还有一个媒体播放器Media player,主要用来对.AVI格式的图象文件进行播放。下面将分别介绍这三个软件的使用方法及特点。
          一、VIDCAP的视频采样捕获
        在Windows 3.1操作平台上运行VIDCAP可看到VIDCAP的界面窗口，若这时有视频信号输入,则在视频窗口中列出连续的动态图象,此时, 可根据功能菜单条中提供的五个功能选项对显示的视频信号进行采样和捕获。各功能选项命令介绍如下:
        1.File(文件菜单)
        在File上用鼠标器一点或按F键,则下拉出多个功能选项菜单,如下:
        File                            (文件菜单)
        Load Palette			(读入调色板文件)
        Set  Capture File		(设置捕获文件)
        Save Captured Video As		(存储捕获视频信息)
        Save Palette			(存储调色板信息)
        Save Single Frame		(存储单幅图象)
        Edit Captured Video  		(启动视频编辑器)
        Exit				(退出)
        下面分别对这些选项作简单的介绍。
        ● Load Palette           
        该选项可用于使用8位采样深度进行视频信息的采集,可利用用户指定的调色板对该视频信号进行着色处理。
        ● Set  Capture File
        该选项对将要把视频信号存入的文件进行设置,如设置该文件的文件名、大小等。    
        ● Save Captured Video As 
        该功能可以指定文件名或路径对视频信号进行存储。
        ● Save Palette
        该功能可将当前捕获文件中的调色板信息以调色板文件格式进行单独存储。
        ● Save Single Frame
        该功能可将视频信号中的某一幅画面以图象文件格式加以存储。
        ● Edit Captured Video
        它可调用VIDEOEDIT软件来对视频信号进行处理与编辑。
        ● Exit 
        退出VIDCAP软件操作环境。
        2.Edit(编辑菜单)
        该菜单有且仅有两个选项,即COPY(拷贝)和Paste Palette(粘贴调色板)。
        ● COPY
        将一幅图画及其附着的调色板信息拷贝到剪贴板中去。
        ● Paste Palette
        它可将剪贴板(ClipBoard)中的调色板信息引入到当前图象中。
        3.Options(配置菜单)
        在Options上用鼠标器一点或按O键,则下拉出多个功能选项菜单,如下:
        Options(配置菜单)
        Audio Format		(音频格式)
        Video Format		(视频格式)
        Video Source		(视频来源)
        Video Display		(视频显示)
        Preview Video		(预演视频捕获效果)
        Overlay Video		(从视频源直接在显示器上播放图象画面)
        ● Audio Format 
        该功能用于音频数据的采样格式(如采样频率、采样位数等)的设置。
        ● Video Format 
        它用来设置捕获视频信号的格式,如视频窗口大小、采样位数等。
        ● Video Source 
        常用的视频捕获卡由三个视频信号源构成,即可处理三路不同的视频信号。 本选择项提供用户可选择三路中的任意一路来捕获视频信号。
        ● Video Display
        它可对当前屏幕所显示的视频信号进行调整,如改变其亮度、对比度以及红、 绿、蓝三色的饱和度等。
        ● Preview Video
        通过该功能，用户在捕获一段视频信息之前可预先看到这段视频信息在被捕获后所得到的效果，以便及时修改设置情况。
        4.Capture(捕获菜单)
        Capture菜单有以下四个选项:
        ● Single Frame(单幅捕获)
        该功能可让用户将当前的一幅画面捕获并存盘。
        ● Frame(多幅捕获)
        该功能可使用户通过单击按扭有选择地挑选画面, 并以统一的文件形式进行存储。
        ● Video(多幅连续捕获)
        这个选择项使用户可动态地实时捕获一段连续的视频画面。
        ● Palette(捕获调色板信息)
        它可连续地将对某幅画面中的调色板信息进行捕获。
        5.HELP(帮助菜单)
        例子:如何利用Video for windows捕获一段采样深度8 bit的并有同步伴音的视频图象,并将其存储在ABC.AVI文件中。
        根据以上菜单的介绍,其操作步骤如下:
        (1) 选用File文件菜单中的Set Capture file选项来设定文件名和大小;
        (2) 在Options配置菜单中的Video Format中选择8bit 采样深度和在Audioformat中选择音频的采样频率和采样位数;
        (3) 用Capture捕获菜单中的Palette选项来采集一段待捕获的视频信号的调色板信息;
        (4) 选取Capture菜单中的Frame或Video选项来捕获多幅或连续幅的视频信号。
        二、VIDEDIT视频编辑工具
        运行VIDEDIT后即可选取VIDEDIT界面窗口中的功能菜单对上述已存储的视频图象文件进行修改、编辑与加工，并可通过窗口下面类似于录像机按键播放按扭对其进行播放、暂停、回放、逐幅进退等功能的操作，还可在显示器上任意缩放和移动视频图象窗口。下面介绍一下VIDEDIT各种功能选项。
        1.File(文件菜单)
        ① New(创建新的工作区)
        它可以帮助用户清除当前工作区中的内容,并将各种设置复位,以备下一个待编辑视频文件的调入。
        ② Save(保存视频图象文件)
        该选项可用以.AVI格式存储一个正在编辑的视频文件。
        ③ Open(打开一个视频图象文件)
        此功能可用来调入一个视频图象文件,其文件格式不仅包括.AVI,而且还支持如Autodesk公司的.FIC和.FLI等格式文件,并将其保存为.AVI格式,以便在MPC标准环境下使用。
        ④ Save AS(改名保存视频文件)
        它可让用户改变文件名称、路径等来保存正在编辑、修改与加工的视频文件。
        ⑤ Revert(重新装入)
        用户可在决定放弃当前修改,并想重新调入该文件时选择此命令选项。
        ⑥ Insert(插入)
        该选项可使得用户在当前视频画面的任意位置插入另一个视频画面或一段声音。
        ⑦ Extract(引出)
        该选项可从当前视频信息中抽出一段视频画面或一段声音并加以存储。
        2.EDIT(编辑菜单)
        ① Undo(取消)
        使上一次的修改操作作废。
        ② Delete(删除一段被选择的区域)
        ③ Set Selection(选择一段区域)
        ④ Go To(跳转)
        该选项可使得用户任意跳转到指定的一幅画面上。
        ⑤ Track(轨迹)
        它可使用户对.AVI文件中的声音、视频信息分别加以编辑。
        3.VIEW(显示菜单)
        该菜单可使用户在播放当前视频文件时,以一定尺寸或全幅播放、快速播放的方式来放映这段视频画面。
        4.Video(视频操作菜单)
        ① Compreeson Option(压缩选项)
        它可提供三种不同的视频压缩方法来对当前的视频信息进行压缩。
        ② Convert Frame Rate(改变幅的速率)
        本选项可用来改变视频播放的速度。
        ③ Synchronize(同步选择)
        当声音、视频在播放时不一致，如出现声音和画面的不同步，选取该项来进行调节,使声音、视频信息得以同步播放。
        ④ Audio Format(音频格式)
        该项可对音频格式进行调整,如采样频率、采样深度等。
        ⑤ Video Format(视频格式)
        该项可对视频格式进行调整,如采样频率、采样深度等。
        ⑥ Create Palette(创建一个调色板)
        该项可用来创建一段被选取的视频画面的调色板信息。
        ⑦ Drop(裁剪)
        通过选择该项来改变视频的显示大小和显示位置。
        ⑧ Resize(改变视频显示的尺寸)
        ⑨ Statistics(信息统计)
        可用该项来统计出当前视频、音频文件的有关信息,如视频、 音频数据所占用比率、采样位数等。
        10 Load File into Memory(将文件装入到内存中)
        选取本项,可增加视频画面的播放速度,以求得在播放中更好的连贯性。
        5.HELP(帮助菜单)
        三、媒体播放器Media Player
           VFW除了提供了VIDCAP和VIDEDIT两个软件外,   还提供了一个功能十分强大的Media player(媒体播放器)以支持.AVI格式图象文件的播放。
        只要键入.AVI格式的文件名后,就能对此文件进行播放,在播放过程中,可对图象中的声音、 视频信息进行放大、缩小等处理播放。该软件与Windos 3.1所提供的媒体播放机相似。
