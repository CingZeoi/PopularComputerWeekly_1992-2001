为你的E-mail加把锁

周少军
2000年 第41期 

　　E－mail和普通的邮件一样存在着很大的安全隐患，居心不良的人很容易窥视到你信中的秘密。如何能够保证你的信件不被人“拆看”呢？A－Lock为你提供了一个很好的解决方案。通过它，你能够很方便地为邮件加密，加密过的邮件有如天书，保证无人识得。而你的朋友则可以用你发给的密码让A－Lock轻松地把天书翻译出来。
#1    一、A－Lock的下载和运行
　　A－Lock为共享软件，最新版本v3.5，文件大小136KB，可以到该软件的主页http：∥www.a－lock.com/去下载。在这里需要说明的是即使你在其他的站点能够获得该软件，也最好去A－Lock的主页下载，因为在该主页下载A－Lock，用户能够免费享受到“Key－Exchange”服务，该项服务的内容是为用户提供安全性极高的密码。下载的是个自解压程序，解压缩后，直接运行其中的Alock.exe文件即可启动A－Lock。首先出现的是“A－Lock最终用户许可协议”，点击下端的“I Accept Its Terms”按钮，弹出的是一个“A－Lock Private Window”窗口。这实际上是一个文本编辑器，以后解读被加密的文件都是通过它来实现的。不过，A－Lock的加密、解密等功能并不能够在这里实现。A－Lock的所有的功能都需要在系统托盘中A－Lock快捷图标的控制菜单中调用（^41040302a^1）。
#1    二、A－Lock的参数设置
　　点击控制菜单中的“Advanced Options ...”项，弹出参数设置对话框（^41040302b^2）。下面介绍A－Lock的参数设置。
　　1.“One Click Encryptions/Decryption”项：如果你选中了该项下的复选框，当你准备对邮件进行加密或解密操作时，只须用鼠标左键点击系统托盘中的快捷图标，就可以直接调用默认的密码进行邮件的加密或解密了。
　　2.“After Decryption is Completed”项：如果你希望A－Lock解密时在“Private Window”中显示所有的信息，包括未加密的信息，请选择“Display entire message”项；如果你希望解密时在“Private Window”中仅显示加密过的信息，请选择“Display only portion of message that was encrypted”项。
　　3.“Start Up Options”项：如果你想让A－Lock随机启动，请选择“Load A－Lock automatically when computer starts”项，否则请选“Do not load A－Lock when computer starts”项。
　　4.“Password Book”项：在这里你可以设置如何访问密码本。如果你选择了“Request password each time  it’s accessed”项，当你每次访问密码本时都会被要求输入密码；如果你选择了“Request password only when first accessed”项，你只需要在第一次访问密码本时输入密码（限本次开机时间内）。
#1    三、A－Lock的使用
　　1.A－Lock的适用对象：A－Lock不但能够对通过各种E－mail客户端程序撰写的邮件进行加密，而且还可以对所有的应用程序窗口中正文的内容进行加密，前提条件是这些内容必须是可编辑的。
　　2.给邮件加密：
　　（1）打开一封需要加密的邮件，首先要保证需要加密正文所在的窗口已被激活，即该窗口中的内容是可编辑的。用鼠标点击窗口中的任意处，有光标在不断地闪烁即表示该窗口已被激活。
　　（2）选择控制菜单中的“Encrypt/Decrypt”项，这时会出现一个密码输入框，输入两次密码，点击“OK”按钮，邮件正文部分的加密工作就完成了。另外，你也可以点击密码框中的“Psssword Book”按钮，在密码本中选择预先设定好的密码或者点击控制菜单中“Encrypt with Default Password”项直接调用默认的密码进行加密。
　　3.给邮件解密：
　　（1）打开一封需要解密的邮件，同样要保证需要解密内容所在的窗口为活动状态。
　　（2）选择控制菜单中的“Encrypt/Decrypt”项，A－Lock将自动搜索密码本中的密码，如果发现有对应的密码将会自动进行解密，如果未发现对应的密码，就会弹出一个密码输入框，需要手工输入密码。解密完成后，会自动弹出“A－Lock Private Window”窗口，在其窗口中显示出解密的内容。如果密码本设置了密码保护，必须在输入了正确的密码之后，A－Lock才会为你自动解密。
　　4.建立密码本：建立一个密码本可以避免因遗失密码可能带来的麻烦。选择控制菜单中的“View/Edit Password Book”项，便可打开密码本。
　　（1）把一个新密码添加进密码本：点击密码本中的“New”按钮，弹出密码输入框，输入两次密码，点击“OK”按钮，新密码即被添加进密码本中。
　　（2）设定默认密码：在密码本中的密码列表框中选中一个密码，然后点击“Make Default on/off”按钮，这时密码条的图标变为黄色，该密码即成为默认密码；如果再次点击“Make Default on/off”按钮，该密码的默认密码资格即被取消，密码条的图标又恢复为原来的颜色。
　　（3）重新编辑密码条目：选中密码框中需要重新编辑的密码条目，然后点击“Edit/View”按钮即可对密码进行修改。
　　（4）为密码本加把锁：如果你使用的是多人共享的电脑，为你的密码本加个保护密码是非常必要的。点击控制菜单中的“Change Password Book Password ...”项，弹出密码输入框，这里有三个密码输入框，第一个是“Current Password”（当前密码框），由于是第一次输密码，该框不需要填，在第二、第三个密码框中输入相同的密码，点击“OK”按钮，密码本的的保护密码便设置好了。
　　5.享受“Key Exchange”服务：“Key Exchange”是A－Lock网站向A－Lock用户提供的一项免费服务，该项服务的内容是向A－Lock用户提供安全性极高的密码，由于这些密码是随意组合的，几乎没有破解的可能。当用户申请了该项服务后，网站会通过加密邮件的方式把密码寄到用户的手里。我们现在来看一下“Key Exchange”服务的整个过程。
　　（1）取得“Key Exchange”服务资格：当你在A－Lock站点请求下载A－Lock时，你就会被自动注册为“Key Exchange”的服务用户，并取得一个唯一的“Key Exchange”身份，你的“Key Exchange”信息会被自动加入到“the Key－Exchange”数据库中。
　　（2）把钥匙放进你的密码本：你第一次使用“Key－Exchange”时，按照网站的提示，在一个非常安全的页面找到你的钥匙（Exchange Key），按照提示把钥匙放进密码本后。你就可以很方便地向朋友传递密码了。
　　（3）让“Key Exchange”自动生成密码：点击控制菜单中的“Go to Key Exchange Site”项，登录到指定的页面，在指定的栏目中填入E－mail地址，可以同时填写多个地址。这样，A－Lock将会为每个地址自动生成一个安全性极高的密码，并会把密码通过加密的方式寄给你。
　　（4）把密码直接导入密码本：当你收到寄来的密码后，你首先需要通过“Encrypt/Decrypt”把密码解密，完成解密后，A－Lock会询问你是否把密码导入密码本，这时你只须点击“OK”按钮，密码就会被自动导入密码本。
#1    四、使用A－Lock可能遇到的问题
　　1.在解密的时候，有时候A－Lock可能会提示找不到“START_PC_Encrypt_DATA”，无法完成解密任务。
　　这是因为有些E－mail客户端软件不支持对收到的邮件进行编辑。解决的方法是点蓝邮件正文被加密的信息，然后再执行解密（Decryption）的操作。如果还不能解决问题，请先点蓝被加密的正文，然后打开“Private Window”窗口，点击“Edit”下拉菜单中的“Copy”，然后再执行解密操作。
　　2.在其它程序的文本编辑窗口中无法加密或解密的解决方法。
　　虽然A－Lock的帮助中说，它能够对所有活动的文本编辑窗口进行加密和解密。但笔者在实际使用过程中发现它对E－mail客户端软件以外的程序支持得并不好。如果碰到在其它程序窗口中的内容无法加密或解密的情况，你只有把它们复制到一个邮件的正文框中，然后再对它进行加密或解密的操作。
