完善的魅力——媒体播放器7.0

楠竹
2000年 第20期 

#1    一、媒体播放器——全能的多媒体播放中心
#1    ·集成的媒体播放器
　　当安装了Windows操作系统后，我们都可以在“开始”菜单的“附件”中发现媒体播放器（英文名称为Windows Media Player）这个程序（^20020201a^1），这个集成在Windows操作系统中的程序是专门播放各种音乐和影像的工具。媒体播放器随着操作系统的升级而不断推出新的版本，功能也不断地完善和强大，目前最新的Windows2000中集成的媒体播放器为6.4中文版，而Windows 98中集成的媒体播放器为5.0中文版。媒体播放器作为集成的工具，最大的好处就是如果安装了Windows操作系统，就可以免费使用，而不需要去注册或者购买。如果你想使用最新版的媒体播放器，还可以到网上去免费下载。
#1    ·使用方便的媒体播放器
　　当我们在Windows中打开媒体播放器，可以看到媒体播放器是标准的Windows界面窗口，具有中文菜单和中文按钮提示，操作上简单易懂。相对著名的WinAmp和XinMpeg，媒体播放器是比较容易上手的，这方面也只有超级解霸可以比拟。
#1    ·功能完善的媒体播放器
　　媒体播放器支持的播放格式是著名播放器中比较多的，从MP3到MIDI、从VCD的MPEG影像压缩格式到苹果公司的MOV动画格式，可以说包括了绝大多数的多媒体文件格式，换句话说，我们只需要媒体播放器这一个工具，就可以欣赏大多数的音乐和影像。另外，最早的媒体播放器功能不是很完善，特别是对播放的控制比较简单，比如声音的均衡调节很差、不能进行列表播放等等，但随着最新版本的7.0的推出，媒体播放器可以和其他任何播放器一较高低。
#1    ·家庭播放多媒体的首选
　　媒体播放器可以说是一个相当强大完善的多媒体播放工具，但仍然有一些引以为憾的缺点。首先，媒体播放器不支持当前流行的Real音乐和影像格式，也不支持VQF音乐格式，这对很多朋友来将是一种很大的遗憾。其次，媒体播放器对播放的控制仍然赶不上一些著名的播放软件。如果你只是立足于一般的家庭欣赏，则媒体播放器是完全足够的，但如果你是一个音乐欣赏方面的发烧友，则媒体播放器在功能方面可能不能满足你的某些要求。
#1    二、取长补短的跃进——新版媒体播放器7.0
#1    ·寄予厚望的媒体播放器7.0
　　日前，微软公司推出了媒体播放器最新版本7.0的测试版，这个媒体播放器是微软进军数字音乐市场计划的一部分，因此这个版本的媒体播放器较以前的版本有了革命性的改变，可以说集合了当前著名播放器的各种优秀设计，并且还具有许多先进的功能如网络点播、支持MP3随身听等，微软公司预计将媒体播放器7.0集成到下半年推出的新版本Windows 98中，以此来撼动RealNetworks公司在因特网数字音乐方面的支配地位。
#1    ·新的界面外观
　　媒体播放器7.0的外观将不再是标准的Windows窗口，而是一个带黑色背景的面板似的窗口，许多的操作功能按钮都集成在面板窗口中，我们可以在面版上直接点播音乐文件，均衡调节和视频调节也可以在面板上直接操作。另外，在面板上还有播放音乐的图形均衡显示窗口。新的媒体播放器一个重大的改进就是支持改变界面外观（也称为更换皮肤），在测试版中，我们可以看到媒体播放器八种自带的皮肤，如果你有兴趣，还可以到网上去下载自己喜欢的皮肤（^20020201b^2）。
#1    ·增强的播放功能
　　在播放方面，媒体播放器7.0的播放控制更多，比如增强的均衡调节、增强的视频调节、增强的CD播放调节等，更增加了播放音乐的可视化显示，在播放媒体文件时，将更容易控制和更容易操作。而且媒体播放器7.0还支持最新的DVD杀手MPEG4影像文件格式。
#1    ·支持播放列表和MP3随声听
　　很多人不用媒体播放器来听MP3音乐的一个重要原因，就是媒体播放器不支持列表播放，这一次，微软想将媒体播放器7.0做成一个家庭点播中心，因此这个最新版的媒体播放器不但支持音乐的列表播放，而且在自动搜索硬盘和网络音乐方面、音乐文件的管理方面、支持MP3随声听方面，都有非常大的创新。
#1    ·网络播放和浏览网上音乐信息
　　媒体播放器7.0现在有点像个浏览器，实际上，要使用媒体播放器7.0必须先安装好IE浏览器，这样，就可以使用媒体播放器方便地来收听网络音乐以及浏览网上的音乐信息。
#1    三、启动和播放
#1    ·安装
　　一般我们使用的是集成到Windows中的媒体播放器，如果我们想马上试用最新版的媒体播放器7.0，可以到网上去下载，下载的地址为http://www.mydown.com，下载文件大小为7MB，为一个安装程序文件。这里要说明的是，媒体播放器是作为附件集成在Windows操作系统中，因此安装最新版本的媒体播放器将替换原来旧的程序。因为目前媒体播放器7.0还只是英文测试版，在操作方便性和程序稳定性方面还有不足，一般的初学者还是建议先使用媒体播放器6.4中文版，等媒体播放器7.0中文正式版出来以后再下载安装。
　　运行下载的安装程序文件，安装程序提示将进行升级安装，然后开始解压文件，并给出安装向导，安装向导提示选择程序的功能附件、选择文件关联，一般按照默认设置，即可完成安装。安装完成后，可以在“开始”菜单的“程序/附件/娱乐”子菜单中看到“Windows Media Play”项，这就是新安装的媒体播放器7.0。另外，在桌面上和Win98中的任务栏里可以看到媒体播放器的快捷启动方式。
#1    ·启动
　　第一次启动媒体播放器，程序将试图连接http://windowsmedia.com/mediaguide/网站，这是一个介绍最新的歌曲专辑、音乐、电影动态的网站，可以在线点播和下载音乐和电影。这个时候，媒体播放器就有点像是一个浏览器（^20020201c^3）。
　　在媒体播放器7.0的缺省界面中（图3），我们可以看到左边是七个功能按钮，这七个按钮非常重要，当点击不同的功能按钮，媒体播放器将有不同的窗口形式以实现不同的功能。另外，在界面的右上部，有一个选择播放列表的下拉列表框。在界面的下部，则是一些常用的播放控制按钮。
　　注意，我们看到的窗口是缺省的界面窗口，这个窗口中进行播放操作比较方便，如果有朋友喜欢媒体播放器老的界面窗口，可以改换成老的界面窗口——不过很多操作将会不那么方便（参见图1）。
#1    ·播放
　　使用媒体播放器实现简单的音乐播放有两种方法，一种是直接双击已经关联的媒体文件，就可以启动媒体播放器播放它（参见^20020201d^4）。另外一种就是启动媒体播放器，选择菜单“File/Open”（文件/打开），在标准的Windows文件打开窗口选择要播放的媒体文件即可。注意，可以同时选择多个媒体文件。
　　注意，媒体播放器也支持拖拽，将文件窗口中的媒体文件拖拽到媒体播放器窗口中即可实现播放。
#1    四、列表播放
#1    ·列表播放
　　使用媒体播放器列表播放多个多媒体文件非常地方便，使用“File/Open”（文件/打开）菜单打开多个媒体文件，就可以方便地实现多个文件的连续播放，而且在播放的过程中，使用文件拖拽的方法，还可以继续添加播放的文件。我们可以在界面的右面看见播放文件的列表（图4）。在界面的右上方的播放列表选择框中，可以看到当前使用的播放列表名称。
　　注意，如果没有看到列表栏，选择菜单“View/Now Playing Tools/Show Playlist”即可。也可以点击界面上部的“Show Playlist”按钮。
#1    ·导入和导出播放列表
　　目前正在播放的文件列表，还可以导出为一个独立的播放列表文件，方法也是非常简单，选择菜单“File/Export Playlist to File”，然后设定文件名，就可以将目前正在使用的播放列表作为一个单独的文件保存到磁盘中，当要使用的时候，还可以再导入该播放列表文件（选择菜单“File/Import Playlist to Library”），从而实现播放自动化。
#1    ·点歌播放
　　在媒体播放器7.0中，点歌播放就非常方便了，先在界面右上方的播放列表选择框中选择某个播放列表，然后下面就列出了列表中的音乐文件，直接选择某个要播放的音乐文件，就可以实现点歌播放。当然，你也可以导入某个播放列表文件，然后进行点歌选择。在点歌播放中要注意的是，音乐文件所在的磁盘位置不要有任何改变，不然程序会找不到该音乐文件，从而点歌失败。
#1    ·播放列表的管理
　　媒体播放器具有非常强大的播放列表管理功能，这些功能强大到我们可以将媒体播放器作为家庭音乐的点播中心，这是其他播放工具很难望其项背的。
　　播放列表管理窗口：选择界面左面的“Media Library”按钮，就进入媒体播放器的播放列表管理窗口（^20020201e^5，当第一次打开该窗口时，程序将自动搜索硬盘，查找硬盘中的多媒体文件），在该窗口中的右边列出了程序内建的播放列表库，其中“Audio”表示音频列表库，内中列出了程序查找到的硬盘中所有音频媒体文件，并且还分了类别。而“Video”栏列出的是视频媒体播放列表，在“My Playlist”中则是自定义的播放列表。“Deleted Items”项中则是已经删除的播放列表。在播放列表管理窗口的右面，则列出了某个播放列表中的媒体文件，并可以看到该文件的具体信息。
　　新建程序内部播放列表：除了上面介绍的几种程序内建播放列表外，我们也可以自己建立程序内部使用的播放列表，这种列表使用起来非常方便，建立好后，直接在程序界面的右上部的播放列表选择框中选择就可以使用了。建立的方法也很简单，先在播放列表管理窗口中的点击“New Playlist”按钮，输入播放列表名称。然后将其他列表库中列出的媒体文件拖拽到新建的播放列表项中即可（从Windows文件窗口中拖拽也可）。另外，也可以使用添加按钮，不过没有拖拽的方法方便。
#1    五、播放的控制和状态显示
#1    ·媒体播放窗口
　　点击界面窗口左边的“Now Playing”，即可切换到通用的媒体播放窗口（参见图4），在这个窗口中，不但可以播放各种媒体文件，而且可以在窗口中完成大多数播放控制操作。
#1    ·播放过程中的控制
　　在播放音乐和影像的时候，控制播放的进度、音量等，可以点击界面下部的按钮，操作上也和一般的播放器操作类似，这里就不再详细解释。
#1    ·声音和视频的调节
　　媒体播放器7.0大大增强了声音和视频的操作调节，现在媒体播放器也可以在播放VCD的时候来调节画面的亮度、色彩的平衡，至于声音的均衡调节等更不在话下，而且这些操作都可以在媒体播放窗口中很方便地通过面板按钮来完成。
　　首先，切换到播放窗口（如图4），选择菜单“View/Now Playing Tools/Show Eguallzer Settings”，显示出声音和视频调节面板（在播放窗口的下部），声音和视频调节面板中有六个子面板，可以点击面板下部的箭头来切换，也可以在“View/Now Playing Tools/Show Eguallzer Settings”菜单中切换，这里我们简单介绍这六个子项目（^20020201f^6）：
　　SRS WOW Effects：声音效果设置。这一项主要调节声音的各种效果，如“TruBass”（左右声道的音量平衡）。
　　Graphic Equalizer：图形化均衡设置。这一项主要是调节声音的均衡，根据自己的喜好，调节高中音的比例。
　　Video Settings：视频设置。这一项主要设置播放视频媒体文件时调节视频影像，比如“Brightness”（亮度）、“Contrast”（对比度）、“Hue”（色彩平衡）等。
　　Media Information：查看媒体信息。
　　Caption：说明。
#1    ·音乐的可视化
　　当使用媒体播放器播放音乐的时候，我们可以让媒体播放器显示一些可视化的动画图像，以此来增强音乐的效果，而这些可视化的动画，可以根据自己的喜好来选择，选择的方法是打开菜单“View/Visuallzation”，然后再选择子菜单中的某一个项，就可以了。另外，比较简便的方法就是点击在动画显示窗口下面的箭头来切换动画（^20020201g^7）。
#1    六、播放CD、VCD和在线音乐
#1    ·播放CD
　　在媒体播放器中播放CD非常方便，当插入CD光碟到光驱，媒体播放器将自动启动并开始播放，我们也可以点击界面左边的“CD Audio”按钮，进入CD播放界面（^20020201h^8），在这个界面，我们可以选择播放CD上的音乐，也可以查看音乐信息。
　　另外，我们使用媒体播放器还可以转录CD上的音乐，转录成的音乐将以微软设计的压缩音乐格式WMA保存在“我的文档”文件夹中，而且录制的速度和质量都是上乘。录制的方法也非常简单，在“CD Audio”窗口中，选择某个要转录的CD音乐，然后点击窗口上部的Copy Music就可以了。
#1    ·播放VCD
　　媒体播放器播放VCD影碟的方法比较特殊，不像超级解霸中有一个“播放VCD”菜单，而是需要直接打开VCD中的数据文件，方法是选择“File/Open”，在文件打开窗口中选择VCD光碟所在的驱动器，然后在文件类型中选择“All files”（所有文件），然后再打开“Mpegav”目录，选择其中的dat格式的文件，打开即可（^20020201i^9）。注意，有些VCD碟片文件格式可能会有些出入，你可以试着一个文件一个文件地打开，看能不能播放。
　　注意，看VCD的时候，可能要全屏播放，选择菜单“View/Full Screen”即可。
#1    ·播放在线音乐
　　由于媒体播放器集成了IE的功能，使用媒体播放器收听Internet上的音乐和广播将非常简单和容易。点击“Radio Tuner”，媒体播放器将连接网上的音乐搜索网站（^20020201j^10），然后你就可以选择收听了。不过媒体播放器只是连接指定的一个音乐广播搜索网站，也许并不能适合所有的朋友。
　　另外，媒体播放器还支持便携式的MP3播放器，使用媒体播放器可以很方便地给MP3播放器灌装音乐文件。
#1    七、更换界面外观
　　媒体播放器7.0支持界面皮肤的更换，在测试版中，已经内置了八种皮肤，你可以在这几种皮肤中选择某个自己喜欢的程序界面形式，选择的方法也非常简单，先点击界面左边的“Skin Chooser”按钮，然后在右面窗口中就列出了可以更换的皮肤样式（^20020201k^11），双击选择的皮肤，即可切换到自己喜欢的界面形式中。
　　注意，除了缺省的外观形式，其他的程序界面都还外带有一个小的切换窗口，鼠标双击这个小窗口，就可以切换到缺省窗口中。
　　另外，如果需要更多样式的皮肤，可以到网址：http://windowsmedia.com/MediaGuide/gallery/skins.asp下载，然后将下载的文件拷贝到媒体播放器所在的“皮肤”目录（C:\Program Files\Windows Media Player\Skins）中去即可。
