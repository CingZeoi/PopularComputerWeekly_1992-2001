Windows Me安装全攻略

王寒晖
2000年 第37期 

　　微软的Windows Me（Windows Millennium Edition）或者叫Windows千禧版，于9月14日公开发售。Windows Me将是最后一个使用Win9x核心的家用操作系统，它实际上只是Windows 95/98的一个升级版，在许多方面它就像是改头换面的Windows 98。
#1    新增功能
　　Windows Me的设计主旨是充分利用目前数字媒体、Internet科技以及家庭网络中的先进技术。遵循以上设计思路，操作系统提供了一整套全新的应用工具，使用户能够更为轻松地保持计算机的最佳工作状态。Windows Me的新功能包括：IE 5.5版、Windows Media Player 7、恢复系统功能；此外，还有会自动下载更新软件的Auto Update（自动更新）功能、简化家用电脑与周边设备连接的Home Networking Wizard（家用网络向导），以及数位影像剪接软件Windows Movie Maker等。
#1    系统需求
　　Windows Me对机器配置的要求相对来说比较高，因此在安装Windows Me之前，你必须确认你的电脑能够安装和使用Windows Me。根据微软官方公布的安装Windows Me的系统需求，Windows Me所需的最低系统配置为VGA或者更高分辨率的显示器；奔腾150MHz以上处理器；32MB以上内存；最少320MB的剩余硬盘空间；CD－ROM驱动器；因为Windows Me强调网络功能，故要求有28.8K MODEM或更快的Internet连接；声卡；扬声器或者耳机。其典型安装需要约600MB空间，如果要升级安装则需要约400MB空间。笔者建议你最好用PⅡ300、64MB以上内存的电脑，一般较新的机器都能够满足这些需求。
#1    选择安装方式
　　与Windows 98类似，Windows Me同时提供了完全版和升级版。完全版针对那些从未安装过以前版本的Windows的用户，它适用于在空白的硬盘驱动器上进行安装。升级版则针对那些已经安装了旧版Windows的用户，该版本将在现有的Windows版本基础上进行升级。因此在安装之前，确定Windows Me的安装方式非常重要。
#1　　1.升级安装
　　升级安装就是系统中已经安装有Windows 95、Windows 98或Windows 98 SE，直接将Windows Me安装到Windows95/98的系统盘中（一般默认都是安装在C盘），安装程序将替换现有的Windows文件，保留现有设置和应用程序。注意：不能直接从Windows 3.x中升级，若Windows 3.x的用户想升级安装Windows Me，只能先升级到Windows 98，然后再升级安装Windows Me。
　　如果你计划将现有的Windows版本升级到Windows Me，那么你最好先作升级的准备工作，它将使你的升级过程更为顺利。首先，你应清除Internet临时文件缓存，并删除一些不必要的文件。如果你运行的是Windows 98，那么可以使用“磁盘清理”命令执行大部分的操作。此后你可以运行“磁盘清理”来检查和修复硬盘驱动器出现的问题。在删除了所有冗余的文件并使用“磁盘清理”命令检查完磁盘后，你便可以运行“磁盘碎片整理程序”，它能够将硬盘上四处分布的文件块收集起来并以毗邻的顺序进行排列。
#1　　2.全新安装
　　全新安装就是在一个没有Windows95/98操作系统的磁盘中安装Windows Me，这种安装比较简单，加载光驱的驱动程序后，直接运行光盘中的安装程序就可以开始了。如果你选择全新安装，那么你首先应做的就是备份你的数据文件。
#1    开始安装
　　下面，我们就介绍如何在Windows 98 SE下，来升级安装Windows Me中文版的过程。Windows Me的安装过程是非常简单的，与Windows 98很相似。启动安装程序后，安装模板将逐步引导你完成整个安装过程。在此过程中系统将提示你接受版权声明、输入序列号、创建启动盘、选择确认选项并再次重启系统。
#1    第一步：在CD－ROM中插入Windows Me光盘
　　如果你的系统支持光盘自动运行的话，系统会弹出一个对话框，提示你“CD－ROM包含比当前使用的版本更新的Windows版本，要将计算机升级到新版的Windows Millennium吗？”（^37020201a^1）。回答“是”后，就可以浏览光盘的内容。如果你的系统不支持光盘自动运行的话，可打开Windows 95/98的资源管理器，浏览光盘的内容。
　　提示：如果你有足够的硬盘空间，你可以将光盘中的文件全部拷贝到硬盘上，并由硬盘执行安装程序，硬盘驱动器的读取速度要远远高于CD－ROM驱动器，这将使Windows Me安装时间减少一半（一次典型安装往往需耗时30到60分钟）。
#1    第二步：运行setup.exe安装程序
　　用鼠标双击光盘中的“setup.exe”安装程序，开始安装过程。
#1    第三步：输入产品的序列号
　　Windows Me的安装程序提示输入该产品的序列号，填入后，按“OK”继续。
#1    第四步：检查驱动器
　　系统还是和以往的Windows安装程序一样，需检查所有的硬盘驱动器，为复制文件作准备（^37020201b^2）。
#1    第五步：许可协议
　　当然你必须同意它的许可协议才可继续安装过程，如果要查看协议可按Page Down翻页键。点击“下一步”继续（^37020201c^3）。
#1    第六步：输入产品密钥
　　此产品密钥在安装盘的背面可找到。在输入框内输入产品密钥后（不包括“－”），然后点击“下一步”（^37020201d^4）。
#1    第七步：检查计算机
　　这时，安装程序将检查硬盘是否有问题，并升级计算机的注册表数据库。完成后再点击“下一步”继续（^37020201e^5）。
#1    第八步：是否保存原有系统文件
　　如果你在现有的Windows版本上进行升级，系统将为你提供“保存原有系统文件”的选项。选择该选项可让你在以后卸载Windows Me，并恢复以前版本的Windows。根据你的系统配置，这可能需要125MB～175MB磁盘空间。如果你确信不需要保存以前的操作系统，可直接选择“否”，跳过下一步（^37020201f^6）。
#1    第九步：正在保存系统文件
　　上一步如果选择“是”，这时系统会查找完系统文件后，再保存系统文件。接下来再选择删除Windows Me所需文件的存储位置后，点击“确定”（^37020201g^7）。
#1    第十步：创建启动盘
　　跟Win98一样。这时请插入一张空白的软盘，制作一张系统启动盘。单击“确定”后继续安装过程。也可单击“取消”，留待以后制作（^37020201h^8）。
#1    第十一步：复制安装文件
　　点击“完成”，Windows Me将拷贝一些必要的安装文件到硬盘中，以后再安装的时候就直接从硬盘读取，安装程序将自动安排拷贝文件的储存空间（^37020201i^9）。
#1    第十二步：重新启动系统
　　文件复制完成后，即提示将重新启动。在这之前，程序提示将软驱中的软盘取走，按“确定”继续。
#1    第十三步：系统配置
　　系统再进行一些类似于Windows 95/98中初始化的系统配置工作，再通过两次的重新启动计算机，检测过硬件以后（Windows Me支持的硬件种类和型号更多），就可以一窥Windows Me的庐山真面目了。
#1    启动和退出Windows Me
　　安装成功后，启动系统，我们就可以顺利进入Windows Me了。第一次看见Windows Me时，会觉得它与Windows 98没有什么不同。Windows Me在界面方面最明显的改动是对“网上邻居”、“开始”菜单和“控制面板”的更改（^37020201j^10）。
　　退出Windows Me很简单，跟Windows 95/98差不多，界面有点像Windows 2000，只不过不可以退出到DOS状态中。
#1    其他安装方式
　　以上介绍的是比较典型的升级安装过程，安装的时候，随着系统和安装过程选择的不同，安装的步骤也有一定的差别，其实无非是多几个步骤和少几个步骤而已。如果在Windows里升级没有成功，可以试一试在DOS下进行Windows Me的安装，这样成功的几率大一些。步骤和前面介绍的差不多。只不过要用启动盘启动计算机，进入安装目录，执行Setup.exe安装程序，必须先加载磁盘高速缓存Smartdrv程序。其安装步骤多了“选择安装目录”、“选择安装类型”、“设置用户信息”和“设置网络标识”。
#1    实现双启动
　　Windows Me不支持双启动，但是如果我们想实现两个操作系统共存怎么办呢？安装前首先将原有操作系统启动分区根目录下的所有文件备份到一个目录中（包括隐藏文件）。然后按照前面介绍的方法安装Windows Me，但是切记，不要将新系统和原来的系统装在同一个分区内，比如原来安装在C盘，那么现在可以把Windows Me安装在D盘。再将安装好新系统的启动分区根目录下的所有文件复制到另一个目录内。你想进入哪个系统，只需将那个系统对应的目录下所有文件履盖到启动分区根目录下就行了。
#1    卸载Windows Me
　　卸载操作十分简单，你只需从“控制面板”上启动“添加/删除程序”，并在安装/卸载标签中选择“卸载Windows千年版”选项。如果你决定保留Windows Me，你可以在“安装/卸载”标签中选择“删除Windows千年版卸载信息”选项来删除保存的原有系统文件，此项操作可以释放部分磁盘空间。
#1    安装Windows 98、Windows Me 与Windows 2000的不同
　　安装Windows Me的过程与Windows 98基本相同，主要是界面比后者细腻多了。相比之下Windows 2000的安装过程就略显繁琐且安装的时间较长，其他的安装步骤是差不多的，最大的区别是Windows 2000安装程序还要进行磁盘分区设置和登录设置（Windows 2000中设置了一个有最高权限的登录账户“Administrator”，在安装的时候，要为该登录账户设置一个密码）。还有很重要的一点是Windows 2000默认的安装目录与Windows 98/Me是不同的。Windows 98/Me默认的安装目录是C:\Windows，而Windows 2000是C:\Winnt。
