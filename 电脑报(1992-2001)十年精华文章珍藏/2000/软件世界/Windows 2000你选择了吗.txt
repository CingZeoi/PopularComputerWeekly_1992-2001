Windows 2000你选择了吗

陈剑
2000年 第12期 

    2000年2月17日，发布日期被一再推迟的微软跨世纪操作系统——Windows 2000终于姗姗来迟，闪亮登场。这个构建在NT核心之上，继承了Windows 98天使般面孔友好的用户界面的全新操作系统，将成为微软巩固其桌面优势，全力进军高端市场的最新“王牌”。
　　那么，这个据称可以满足从桌面直到企业级关键应用需求的崭新的操作系统是否真的能够让所有人都心满意足，值不值得我们尽快升级呢？在颇费了一番思量之后，我们精心为你奉上“选择Windows 2000的五个理由”和“不选择Windows 2000的五个理由”。
#1    选择Windows 2000的五个理由
#1　　1.Windows 2000稳定性更高
　　和Windows 95、Windows 98（包括98第二版）以及Windows NT 4.0等微软以前的操作系统相比，稳定性是Windows 2000最大的特点和优点。从DOS时代起，层出不穷的系统稳定性Bug就一直是人们诟病的“众矢之的”。Windows 2000的问世很可能会结束这种让微软难堪的局面。Windows 2000没有使用Windows 9x的系统内核，而是继承了更为稳定的NT内核代码。不但如此，重新改写的内核代码对于非法内存访问的监控和限制更加严格，令Windows用户深恶痛绝的“蓝屏错误”已经成为极罕见的事件。在稳定性方面的长足进步是很多试用过该系统的用户的最大感受。
#1　　2.Windows 2000运行效率更佳
　　Windows 2000在运行速度和效率方面也有一定的提高，测试显示在配置了64M或更多内存的机器上，它的运行速度要比Windows 9x快25％左右。运行效率的提高主要得益于“全32位”的系统内核，它在带来高效率的同时，也降低了应用程序运行失败和系统意外重启的几率。这种“全32位”体系结构还带来了更好的多任务处理能力，和Windows 9x相比，Windows 2000的用户可以同时运行更多的程序，完成更多的任务。它的另一个显著变化是，以前在Windows 9x下让人恼火的系统频繁重启情况显著减少，现在安装很多新软件时，不再需要莫名其妙的系统重启了。
#1　　3.Windows 2000安全性更好
　　Windows 2000的另一个“卖点”在于安全性方面的进步，它采取了种种措施，用来确保对因特网时代来说性命攸关的安全性：
　　Windows文件保护机制可以防止核心的系统文件被有意或无意地删除或改写覆盖。这种机制可以在文件被改写覆盖的时候自动地用正确的文件来代替被改写的文件。加密文件系统（EFS）则使用随机生成的密钥来加密系统的每一个文件，加密和解密过程对用户来说都是完全透明的。驱动程序认证特性可以保证系统中最重要的驱动程序不被篡改，最大程度地减少用户安装那些来路不明的驱动程序可能带来的风险。Windows 2000继承了NT下比较成功的用户安全模式，只有获得了授权的用户才能访问特定的系统资源。在备受瞩目的网络安全方面，Windows 2000也着力不少。安全IP（IpSec）能够保证通过网络传输的数据的安全。IpSec是虚拟专用网络（VPN）的重要组成部分，VPN可以让商业机构的数据安全地通过Internet传输。Windows 2000还提供了对身份认证标准Kerberos的支持。Kerberos能够提供大强度、高速度的身份认证手段，允许用户一次登录即可安全方便地访问各种Windows 2000的系统资源。
#1　　4.Windows 2000管理使用更方便
　　方便的管理使用一直以来是Windows系列操作系统最大的特色，也是其能够牢牢统治桌面，觊觎高端市场的本钱。
　　Windows 2000中最显著的变化之一是“活动目录”（Active Directory）的引入。活动目录是一个存储有关网络用户、资源（比如打印机）以及网络配置信息的分布式数据库系统。对小型的公司来说活动目录的用处不是很明显，但是那些拥有上千用户、电脑和网络设备的机构则可以从中得到很大的好处。他们可以使用活动目录服务来组织和管理数量惊人的资源。全新的系统预设工具（System Preparation Tool）则能够帮助管理员完整地复制电脑上的配置信息、系统和应用软件，简化和加速大型机构中系统的部署工作。远程安装服务（RIS）则允许通过网络安装系统，降低了安装的成本、节约了时间。使用统一的Windows安装服务（Windows Installer Service），用户可以方便地安装、配置、跟踪、升级和卸载各种软件，改变Windows 95／98中那种软件安装各行其事，频繁安装卸载后系统中垃圾文件成堆的窘境。Windows 2000新增了很多智能化的向导程序，能够协助用户配置、优化和解决系统中可能出现的问题。比如问题解决向导（Troubleshooters）就能够帮助用户定位和解决系统故障。
#1　　5.Windows 2000硬件设备支持更好
　　在Windows 2000推出以前，作为定位中高端企业应用的NT 4.0对硬件的兼容性不敢让人恭维。很多在Windows 9x下运行良好的设备，在NT下要么根本无法识别，要么工作不正常，关键原因就是NT不支持Windows 9x下大受欢迎的“即插即用”（Plug ＆ Play）特性。现在，全新的Windows 2000对硬件的支持与NT相比可以说是上了一个台阶，设备的安装配置再也不是令人头痛欲裂的难事了。
　　Windows 2000专业版支持多达4G的内存和两个对称处理器，服务器版本则能够支持8G内存和八路对称处理器，提供更为强劲的计算和处理能力。它继承自Windows 9x的“即插即用”特性，使得很多硬件设备的安装工作自动化，手工配置的工作量被降低到最低限度。Windows 2000对各种流行外部设备接口标准的支持非常完善，对通用串行接口（USB）的支持可以让用户在无需更改配置和重启机器的情况下“热拔插”各种USB外设；流行的IrDA红外线传输协议可以为支持该协议的电脑和外设提供方便、安全的无线数据通讯服务；对IEEE 1394标准的支持，则可以为那些需要更快的数据传输服务的设备提供更高带宽的联结服务。此外，Windows 2000还为诸如DVD那样的可移动存贮设备提供良好的支持，对AGP标准、多显卡多显示器、OpenGL 1.2、DirectX 7.0的支持使得Windows 2000的图像显示和处理能力也异常强劲。
#1    不选择Windows 2000的五个理由
#1　　1.Windows 2000并非定位于一般用户
　　虽然Windows 2000在很多方面都极具吸引力，但是它并不一定能够满足任何应用需求，值得所有人尽快升级。Windows 2000不是Windows 98的升级版本，它的定位并不是针对一般用户的。相反，这个最初被称为NT 5.0的软件，其实是NT 4.0的直接后裔。4个不同版本的Windows 2000是微软市场细化策略的产物，专业版对应于原来的NT 4.0工作站版本，服务器、高级服务器版本则瞄准中高端企业级应用市场，数据中心服务器版本则是微软试图征战主机市场的王牌。对于一般的个人用户来说，Windows 98及其第二版仍然是现阶段较好的选择，它们在对流行硬件的兼容方面，在多媒体以及游戏的支持能力方面都是其它操作系统无法比拟的。更重要的是，微软正在紧张地开发针对一般用户的下一代视窗系统——Windows千禧版（Millennim Edition）。也许这个版本正式问世的那一天，才是Windows 9x的用户的最佳升级时机。
#1　　2.Windows 2000升级维护成本高昂
　　Windows 2000能够带来的诸多好处并不是近在咫尺唾手可得的。天下没有免费的午餐，为了获得这些好处，你可能为此付出很多。据国外的一些评测机构分析，要将现有的系统全部升级到Windows 2000，相关的升级、维护以及用于人员培训方面的成本将会相当高昂。和Windows 9x以及NT 4.0相比，Windows 2000对硬件的要求更加苛刻。几乎所有版本都要求2G以上的硬盘空间，专业版要求64M以上内存，服务器版则要求128M内存，推荐使用256M。这样光是硬件升级方面的费用就将相当可观。再加上用于维护其日常运转所需的费用以及用于人员技术培训方面的开支，升级到Windows 2000的总体费用就不可小觑了。对于那些预算紧张的中小公司来说，短期内暂缓升级是最明智的选择。
#1　　3.Windows 2000稳定性有待进一步提高
　　虽然微软为Windows 2000的稳定性的提高投入了大量的财力和精力，在这个方面的进展也是有目共睹的。然而，Windows 2000在稳定性方面并没有达到一劳永逸的程度，它还远远没有达到某些媒体所称的“坚如磐石”的地步，微软在这个方向上的努力仍然将是长期性的。据报道，目前的Windows 2000代码中尚存在6万多个已经知晓的Bug。根据以前对微软操作系统考查的经验，有专家建议即使需要尽早升级到Windows 2000的用户也最好等到微软公司推出一到两个服务包（Service Pack）以后。
#1　　4.Windows 2000面临兼容性考验
　　除了稳定性方面的困扰，Windows 2000在兼容性方面的问题也可能成为人们拒绝升级的重要原因之一。有国外专家指出，将操作系统升级到Windows 2000的用户中，将有四分之一的人会碰到诸如商业应用软件、非Windows系统以及网络体系结构等不兼容的问题。据估计，现有的Windows软件中至少有15％需要修改源代码，才能在Windows 2000上正常运行。至于年代更久远的DOS应用软件特别是绝大多数的游戏程序在Windows 2000上将遇到问题，甚至根本无法运行。在硬件方面，一些较老的硬件也可能无法在Windows 2000下被识别和正常使用。
#1　　5.Windows 2000竞争对手环峙
　　除了本身存在的一些问题之外，一些强有力的竞争对手也是用户选择Windows 2000的障碍。除了NT长久以来在高端市场的老对手UNIX以外，最近一两年来迅速崛起的免费操作系统Linux是Windows 2000最大的威胁。据统计，其市场份额已经由1998年的17％上升到了去年的25％，而且增长速度丝毫没有放慢的迹象。最近Linux走向桌面的进程也大大加快，部分专门针对桌面应用的版本已经上市，形成与Windows 9x和2000全面竞争的态势。与此同时，Sun公司也逐步放松了对原码的控制，公开了著名的Solaris系统的代码并且计划开放Java标准；老牌网络操作系统提供商Novell在推出Netware 5.0以后，凭借性能卓越的目录服务NDS也基本重新站稳了脚跟。除了这些重量级的竞争对手之外，包括Mac OS和BeOS这样的操作系统也并非毫无市场潜力。在强敌环峙的情况下，Windows 2000能否突围胜出目前确实还很难预料。
　　不管Windows 2000是好是坏，值不值得马上升级，我们可以确定的是微软拥有足够的和必要的财力、能力和用户忠诚度来保证Windows 2000的成功。虽然，用户将在何时完全抛弃NT 4.0和Windows 95／98转而选择Windows 2000还是未知之数，但是Windows 2000毫无疑问必将成为一个主流的网络操作系统。
