未来之窗——记Windows Whistler开发者预览版

佳杉 朱涟
2000年 第29期 

#1    Whistler 的背景
　　比尔·盖兹在微软开发者大会上透露Windows 2000后继版本研发成功发布的第二天，也就是2000年7月13日，微软公司为一组经挑选的测试者提供了Whistler的第一个演示版。同一天，微软公司在Windowsbeta.mirocsoft.com下载站点上公布了标号为2250的Whistler 个人版和专业版的演示版（访问者必须使用有权限的密码才能进入），这也是微软公司在奥兰多的专业开发者大会上所演示的版本。其实早在3月份，一个开发序号为2211.1的Whistler内部测试版本就出现在许多大学网站和一些互联网站上，据说这是由微软开发小组的内部成员泄露出来的（具体见Windows Whistler的时间^29020101a^表）。
　　微软不愧为微软，在一连串的打击中，面对强大的对手，身陷公司被拆解的境地，不但没有出现败乱之象，反而通过一系列的大规模技术研讨会、产品发布会，稳扎稳打沉着应战，给世人以深刻印象：微软公司超脱于困境之外，时刻保持对市场的敏锐感觉和对技术尽善尽美的追求。
　　Windows Whistler／Windows 2001早期测试版的发放显示了微软公司坚定的决心和长远的的市场策略：“电脑应用与互联网络必将紧密结合，‘PC增强’时代已经来临，消费者和商业用户需要的不仅仅是稳定、速度，更重要的是简化、简化、再简化的操作易用性。” 
#1    Whistler开发者预览版的介绍
　　微软的开发小组将推出两个版本的Whistler系统，一个是为商业用户优化的64位专业版本，另一个是针对消费者和初级用户的32位版本。第一个 Windows Whistler 开发者预览版已经在7月14日发放给一定范围内的测试者了，而第一个“beta”测试版可能会在9月份的时候发表。现在就让我们了解一下关于Whistler都有哪些方面的新内容。
#1　　新的概念，新的特性：
　　Windows Whistler是Windows 2000的后续版本，其易于使用的界面是微软公司最夸耀的卖点，微软官方强调 Windows Whistler的应用诉求点是成为下一代消费类个人电脑硬件的系统平台，它是针对未来的技术发展和市场需求应运而生的。
　　Windows Whistler操作系统本身 使用了Windows 2000的内核，比较于其他操作系统更多的基于HTML的应用，控制面板中的各个图标类似一个个网页上的链接。它包含游戏增强特性（但这只是针对可以在Windows 2000下面运行的游戏）。对网络应用进行了特别的优化，可以提升访问MSN以及网络的性能。目前内置的浏览器为IE5.5正式版，其网络浏览速度较以往更快，并修正了某些安全方面的漏洞。据测试者说在Windows Whistler中不再支持AppleTalk，NetBEUI以及DLC网络协议，但新添加了Ipv6网络协议支持。
　　同时，Windows Whistler对图形用户接口（GUI）作了较大的改变。用户在系统启动时会发现焕然一新的界面，微软称之为用户可选择的“可视化风格”（“visual styles”）。在目前开发者得到的版本中，Whistler使用的是Mars更换界面接口（Mars skinning interface），微软公司5月份曾在MSN发表了Mars更换界面接口。但至于Whistler正式版是否沿袭这一界面更换方式，目前还是个未知数。在目前的开发者得到的预览版中，用户可以为操作系统定制变化多端的外观和多种多样的色彩，还可以设置滚动条的形状和透明度。
　　Whistler还包含了一种崭新的服务功能称为：Windows数码影像体系（Windows Image Architecture）。消费者可以利用这一服务从数码相机内下载图片，进行编辑，然后发布到网络上，其制作过程一气呵成。
　　Whistler还将更进一步地整合电脑硬件和用户界面的协同性，未来运行Whistler的电脑可以配合小型液晶显示板“Digital Dashboard”（这是微软与合作伙伴共同开发的定制化用户界面），它扩展了现有的人机交互界面，以方便用户进行快速的数据输入。
　　除了易用性之外，Whistler具有更方便的硬件平台移植性，微软公司曾展示过一台运行Whistler系统的PCTV机顶盒。用户可以在该设备提供的类似网页链接的界面中选择喜欢的影视节目。
　　微软公司正在与Intel公司合作，在Whistler系统中支持新兴的蓝牙无线连接技术，同时该操作系统可以确保蓝牙设备与个人电脑相互通讯，目前其使用的是蓝牙1.0标准。蓝牙技术基于无线电技术，可以让各种设备在短距离进行无线通讯，而不需要直接的电缆连接和额外的通信协议。由于有可能通过这种技术轻易实现移动电话、PDA与个人电脑之间的无线连接，Whistler采用这种技术势必引起业内人士的普遍关注。
　　Whistler还支持P3P标准，可以为消费者提供更多的个人信息保护，并使消费者和企业用户能够对电子商务的应用树立起更大的信心。
#1　　紧密的集成.NET技术：
　　Whistler的某些特性显示了微软对未来操作系统的构想和长远目标：未来应用软件的发放和传播将更主要地依靠基于互联网络的服务。微软最近全力推动的.NET框架，其内容简单的说就是在系统级更紧密的捆绑各种各样的网络应用，增强的数据互操作性以及协同开发过程。在.NET框架下，微软将使用XML、SOAP以及其他标准语言与接口，将多个独立的网络服务器和站点联合起来提供服务。可以访问这个服务的设备包括传统型的PC、手持设备、移动电话以及带有因特网功能的电视以及PCTV机顶盒。Windows Whistler将是第一个完全适应.NET的Windows操作系统。.Net服务直接置入系统用户界面中，举例来说，用户可以利用该服务把文件存放到网络上的“微软社区站点”中，用户也可以利用在系 统中建立的个人身份信息，顺利的通过网络上的身份鉴别。
　　微软高层领导人称将会有步骤地帮助开发者们逐渐转向它的.NET开发平台，那时就是Whistler大显身手的时候了。
　　使用过Windows 2000的用户肯定对这个系统的稳定性大加褒奖，但对Windows 2000不支持大部分时下流行的游戏又感到非常的遗憾，那么作为Windows 2000的后续版本的Windows Whistler又如何呢，相信这是广大游戏玩家非常关心的问题之一，但到目前为止还不能确定在Windows Whistler系统中可以运行多少游戏。相信在使用NT 内核的 Whistler上运行目前流行的游戏很可能会产生这样或那样的问题，毕竟它还是“未来”的操作系统。
　　Windows“Whistler”正式版发表的日期虽然到目前为止还不十分明确，但相信是在2001的7月至年末这段时间内。
#1    业界人士对Whistler的评价
　　微软公司的目的十分明显，就是依靠操作系统向更广阔的范围伸展——从X－BOX游戏机、手持设备到高端服务器。依靠既有市场逐步占领发展中的新市场，在提升产品及服务品质的同时，制造新一轮的消费热点来扩张用户群。微软公司虽然在上个月的地方法院审判中输给了美国司法部，但依然我行我素，Whistler中不只是集成了浏览器，甚至在系统级捆绑了微软网络服务，看来美国司法部与微软又要龙争虎斗一番了。
　　微软在发布新的操作系统之前，要么是习惯性的延期，要么是为了按期发布而减少操作系统的某些特性，Windows 98和Windows 2000就是很好的例子，因此笔者认为用户在设定系统购买／升级计划时应该考虑到这一点。
#1    Windows的未来 
　　Windows千禧版带给用户最初的惊喜，在使用一段时间后，其稳定性难免令人失望，毕竟它还是基于Windows 98的系统内核进行一定的修改而成的。Windows 2000则稳定得多，但它作为NT 4.0的升级换代版，并没有太大的不同。而Whistler则代表了微软公司的对未来互联网络与电脑应用紧密结合的全新概念，就目前预览版的表现而言，这一操作系统值得期待，让我们在明年的圣诞前夕欢迎它的到来！
