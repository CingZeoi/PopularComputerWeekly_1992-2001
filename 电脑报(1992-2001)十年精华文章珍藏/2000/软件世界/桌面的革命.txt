桌面的革命

刘育军
2000年 第10期 

　　曾经有好几个朋友在一开始用我的计算机时总是会被眼前这与众不同的图标界面迷惑得呆好久，你不相信，那就首先看一看我的“开始”工具栏吧（^10020101a^1），是不是与你经常见到的工具栏大不一样？接下来再看一看我的硬盘中都装了一些什么，从下面的图中你可以通过五彩斑斓、各式各样的图标一下子就看出来每一个文件夹里到底是什么东东了（^10020101b^2）。别傻傻的光看了，这里每一个图标可都是代表一个文件夹，而不是驱动程序的图标。是不是太炫了，酷得不知道从哪儿下手了？在这样一个追求个性的时代里你还守着一成不变的Windows，每天看着它灰色的脸和那些平淡乏味的文件夹图标吗？你想你的Windows更漂亮，更具有个性化吗？该是来一回桌面革命的时候了。
#1    利器在手
　　在这里要向大家推荐的桌面革命所用到的利器是Microangelo 98。Microangelo 98是一个功能非常强大的图标和光标工具软件（http:／／www.impactsoft.com／muangelo／download／mike98b.exe），它可是集图标设计制作（包括动态光标）、图标浏览查看、图标更换等功能于一身，使你极为容易地更新、定制和创建Windows系统图标和光标方案（酷!），允许你的桌面图标和光标显示多达24位真彩色（炫!），且能搜索所有磁盘和文件夹中的文件图标资源，并能从中导出图标，进行修改，保存等操作，直接抓取图标进行编辑，它能检测和支持Windows95、Active Desktop of Internet Explorer 4.0、Windows98、Windows NT4.0及更高版本。以上大家看到的开始工具栏和文件夹图标都是在它的改动下得到的。
　　在安装完Microangelo 98（版本号：4.77）以后，从开始菜单中启动。从启动菜单中我们可以看到Microangelo 98由Explorer、Engineer、Librarian、Studio 和Animator五大部分组成。
#1    一起来动手
　　首先要介绍的是Microangelo 98的Engineer，它是一个功能强大的图标光标设置中心，通过这位“工程师”的帮助，你能对计算机系统图标的属性、系统桌面组件和Windows外壳的图标光标进行随心所欲地设置和更换，并能按主题备份来维护你的图标光标设置方案。
　　在系统图标属性的设置中，你可以设置图标的彩色格式，最多可以显示多达24位真彩色，当然这还要依赖系统显示的色彩数，一般的图标可只有16色；以及设置是否使用大图标和显示图标的快捷方式箭头；随意调整图标的水平和纵向的间隔；图标标题的字体、大小、粗体、斜体。
#1　　一、设置桌面组件和Windows外壳图标光标
　　1. 分别选择Desktop（桌面）、Start（启动）、Folder（文件夹）、Drives（驱动器）、Misc.（杂项），或者Cursor（光标），选中要更改图标或者光标。（见^10020101c^3）
　　2. 在选定图标或者光标后单击Change...（更改）按钮。
　　3. 将出现一个更改图标光标对话框，选择你要使用的图标光标，单击OK。
　　4. 如果你想更改其他图标或者光标，重复以上步骤1至4从而设置每个项目的图标。
　　5. 最后单击Apply（应用）按钮，就完成了系统的图标光标的更改。
　　当然，Microangelo 98的Engineer还能按主题备份来维护你的图标光标设置方案。方法是：
　　1. 在Config （系统配置）中选择Motif File Maintenance（主题文件维护）。
　　2. 在File菜单中选择Open或New，打开或新建一个图标光标主题文件，其扩展名为.motif。
　　3. 选择Desktop（桌面）、Start（启动）、Folder（文件夹）、Drives（驱动器）、Misc.（杂项）、或者Cursor（光标）、选中要更改图标或者光标。
　　4. 在选定图标或者光标后单击Change...（更改）按钮。
　　5. 将出现一个更改图标光标对话框，选择你要使用的图标或者光标，单击OK。
　　6. 如果你想更改其他图标或者光标，重复以上步骤3至5从而设置每个项目的图标。
　　7. 最后单击Save（保存）或Save As...（另存为）按钮，就把刚设置的图标光标以主题文件的形式保存下来。
　　8. 如果在Config （系统配置） 中选择local System Update （本地系统更新），这是系统的默认值，再在File菜单中选择Open打开图标光标主题文件，最后单击Apply（应用）按钮，就把图标光标的设置方案更新到你的系统了。
#1　　二、系统图标的更改
　　要想更改系统图标，这里我们首先要知道原有的系统图标的摆放位置和调用方法。原有的系统图标摆放在你的C:\WINDOWS\SYSTEM\shell32.dll文件中。ICO文件只有唯一的一个图标，用户可以通过ICO文件指定路径和文件名指定这种文件。而EXE、DLL、RES文件可能包含多个图标，用图标索引标记这类文件中的一个特定图标，索引号以0开始，第一个图标为0，第二个图标为1，依此类推。用户通过指定路径和文件名加图标索引号为EXE、DLL、RES文件指定图标。格式如下：path\filename，index。用户在开始菜单中看到的图标、系统文件夹和文件类型图标都是在Shell32.dll文件中定义的。如果你会修改注册表的话，你可以在\HKEY_LOCAL_MACHINE\Software\Microsoft\Windows\Current Version\Explorer\Shell Icon 中添加一个字符串型值项，用来替代任意的shell图标。而Microangelo 98的Engineer正是在C:\WINDOWS\Installed System Icons文件夹中记录了你所选择的图标，并通过改变注册表中所对应的缺省值改变图标，不信你可以自己找一找、看一看。
#1　　三、定制你的文件夹新图标
　　你的计算机中某个目录对你很重要吗？让它有个与众不同的图标吧，这样辨认起来也容易。首先我们在资源管理器中选择好要改变图标的文件夹，点击鼠标右键，选择“Appearance”项目，（注意，这可是4.77版新加的功能。）在出现的文件夹图标更改对话框中选择Custom（自定义）选项，选择你所喜爱的图标（.ico）文件，然后点按Change...（更改）按钮从而更改该文件夹的图标。同时你还可以在Info tip（文件夹提示信息）一栏中输入该文件夹的简要说明，这样当你选中该文件夹时，在资源管理器中将显示相应的文字说明（如^10020101d^4所示），文件夹里所包含的内容不就一目了然了么？
#1    修改的秘密
　　接下来让我们来看一看Microangelo 98是如何更改文件夹、驱动器盘符图标的。
　　首先是文件夹的图标更改，实际上在更改了图标的文件夹内已经增加了两个文件：desktop.ini文件和icon文件，属性均为隐藏，所以一般情况下我们没有察觉到在对应的文件夹中有什么变化。如果你将文件夹选项中的“显示所有文件”打开，你就明白为什么你的文件夹图标变得如此光彩照人了。其中desktop.ini文件内容为：
　　[.ShellClassInfo]
　　InfoTip=文件夹说明
　　IconFile=图标文件名
　　IconIndex=0
　　ConfirmFileOp=0
　　OriginalIcon=％
　　其中的InfoTip 表示这个文件夹的提示信息已经解释过了。IconFile 就是你所选择的图标文件，可以采用.ico、.exe、.dll等这类有图标的文件充当，并且通过图标索引号IconIndex指定，同时将这个目录设置为系统属性，这样一个全新的图标代表你的文件夹原有的图标了。
　　再来看驱动器盘符图标的更改。大家用我的电脑或资源管理器看过的一些游戏光盘或者多媒体教学光盘都有自己的图标吧，其实驱动器盘符图标的更改和它们的方法差不多，就是在硬盘的根目录下新建一个AutoRun.inf 文件并且打下下面两行字：
　　[AutoRun]
　　Icon=图标文件名（必须带有详细的路径）
　　这里的Icon 当然就是你选择的图标文件，设置硬盘的图标不需要加系统属性。
　　原理一讲透了就是如此地简单，而真正更改起来的话最好还是用Microangelo 98吧，毕竟它用起来实在是太方便快捷了（系统根本不须重新启动），而且相对亲自动手更改注册表更为安全可靠。
#1    另外的功能
　　Microangelo 98的Explorer界面与Windows的Explorer类似，是系统图标和光标的管理中心，通过Explorer你可以查看和管理磁盘、文件夹中所有的图标和光标。通过Explorer你也可以直接启动Engineer、Librarian、Studio 和Animator等各项功能。Microangelo 98的Librarian图标管理器、Studio图标制作室、Animator动画光标制作室都是功能很强、操作简便的工具，而且比较容易上手，相信经常自己制作图标或者动态光标的爱好者一定会喜欢上它们的，这里就不一一介绍了。
　　Microangelo 98可以说是现今为止功能最多、性能最好、操作最方便的图标和光标制作和管理工具软件，是Windows 95／98／NT用户设置和维护系统图标光标的利器，希望它能使你的Windows更炫更酷，更具有个性化。
