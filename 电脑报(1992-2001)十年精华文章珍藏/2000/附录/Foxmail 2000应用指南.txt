Foxmail 2000应用指南

胡锦承
2000年 第53期 

    FoxMail是国内目前最流行的客户端电子邮件软件，其无论是从功能的全面性、操作的简便性，还是从软件的兼容性来讲，FoxMail都足以“傲视群雄”，况且它本身又是免费的，因此赢得了众多用户的青睐。下面让我们一起来看看FoxMail的具体操作。
#1    一、下载与安装
    首先我们应到http://www.aerofox.com或http://www.bodafox.com下载FoxMail（目前的最新版为3.1，即Foxmail 2000。其文件名为fm31bch.exe），然后双击该文件，启动该软件的安装程序，一路“确定”即可完成FoxMail的安装。
#1    二、设置个人账户
    安装完毕，在正式使用FoxMail之前应对邮件账号进行设置（升级用户可跳过此步聚）：
    1. 第一次启动FoxMail时，系统将自动打开“用户向导”窗口，帮助用户对个人的邮件账号进行设置。单击“下一步”按钮。
    2. 系统将打开“建立新的用户账号”对话框，在“用户名”栏中键入自己的用户名，然后单击“下一步”按钮。
    3. 在“邮件身份标记”对话框的“邮件地址”栏中键入自己的E_mail地址，并在“发送者姓名”栏中键入自己的姓名（这里的姓名将加到你发送的邮件信息中，对方可以看到你的名字）。单击“下一步”按钮。
    4. 在“指定邮件服务器”对话框中，对POP3和SMTP服务器的地址进行设置。在“POP3服务器”栏中键入该邮箱的POP3地址；在“POP3账户名”栏中键入用户的电子邮件账号（即是电子邮件地址@之前的部分）；在“SMTP服务器”栏中键入SMTP地址。
    5. 单击“完成”按钮，个人账户的设置工作即告结束，不过此时我们仍然不能直接发送邮件，还应单击“选项”菜单的“当前账户属性”命令，打开“账户属性”对话框进行最后设置。
    6. 用户若拟设置专门的回信地址，单击左边的“个人信息”项，然后在“回复地址”栏中键入所需的回复地址。单击“邮件服务器”项，在“口令”栏中键入该邮箱的口令。单击“确定”按钮，关闭“账户属性”对话框，结束用户个人邮件账号的设置工作。至此我们就能利用FoxMail收发邮件了。
#1    三、邮件的常规操作
#1    1．撰写电子邮件
    （1）单击快捷工具栏上的“撰写”按钮，打开新邮件撰写窗口。
    （2）在“收件人”栏中键入一个或多个收件人的邮箱地址，每个收件人之间用逗号隔开。
    （3）在“抄送”栏中键入其它收件人的E-mail地址。并同时在“主题”栏中键入邮件的主题。
    （4）单击切换到正文撰写窗口，即可开始撰写邮件正文。
    （5）用户若拟在邮件中携带附件，则应单击“增加附件”按钮，然后在“打开”对话框中选择需要发送的附件。
#1    2．发送与接收电子邮件
    （1）邮件撰写完毕，单击“立即发送”按钮，即可将邮件按照指定的邮件地址发送出去；或者单击“保存到发件箱，等候发送”按钮，将撰写的邮件保存下来，待日集中发送；另外，我们也可单击“保存为草稿，以便下次编辑”按钮，将正在撰写的邮件保存为草稿，待以后编辑修改后再进行发送。
    （2）单击“接收”按钮，系统即会收取邮件服务器（POP3）上的所有信件。
    在邮件收发过程中，FoxMail将会打开一个状态窗口，向用户显示邮件收发的每一个过程，如网络连接、登录、发送（读取）邮箱的信息、发送（收取）每一封信的进度等。邮件收取完毕之后，FoxMail还会显示收到邮件的数量及其保存位置等信息，使用非常方便。
#1    3．阅读邮件
    通常情况下，FoxMail将收到的邮件保存到收件箱中，我们只需单击收件箱图标，系统即会将收件箱中的所有邮件列表显示出来，从中我们可以了解收到该邮件的时间、发件人、主题、大小、是否带有附件等信息，双击某个需要查看的邮件，打开专门的邮件浏览窗口，即可对邮件进行阅读。
#1    4．打开邮件附件
    在FoxMail中，有“回形针”标志的邮件即表示该邮件携带有附件，用户若拟对这些附件进行操作，则应双击该附件，打开“附件”对话框，然后利用“打开”按钮调用与附件文件相关联的应用程序打开该附件，或利用“另存为”按钮将附件保存到磁盘上。
#1    5．邮件的回复与转发
    收到别人的来信之后想回信又该怎么办呢？别着急，我们只需右击需要回复的邮件，然后从弹出的快捷菜单中执行“回复邮件”命令，系统即会打开一个回复邮件窗口，自动填上收件人的地址和主题，并将原来的邮件内容全部复制到该窗口中，接下来我们只需键入回信内容就可以了，这就省去了用户键入收件人、主题等操作的时间，使用非常方便。
    用户有时候还需要将收到的邮件转发给其他人阅读，这可不需要手工键入，我们只需右击需要转发的邮件，然后从弹出的快捷菜单中执行“转发邮件”命令，系统即会打开一个转发邮件窗口，自动将原有邮件的内容全部拷贝到该窗口中，并添加适当的转发主题，我们所要做的就是键入转发邮件的收件人并进行发送即可。
#1    6．删除邮件
    在收到的邮件中，必然有一部分是没有保存价值的邮件，对于这部分邮件，我们只需选中欲删除的邮件，然后单击“删除”按钮即可将其删除。或者将该邮件直接拖拽到“废件箱”也可达到同样的目的。
    需要注意的是，采用此方式并不能彻底删除该邮件，而只是将其转移到了“废件箱”邮箱中，用户若拟彻底删除该邮件必须在“废件箱”邮箱中再次将其删除。
#1    7．建立新邮箱
    除系统已有的收件箱、发件箱、已发送邮件箱、废件箱之外，FoxMail还允许建立自己特定的邮箱。选中需要建立新邮箱的位置，然后单击“邮箱”菜单中的“建新邮箱”命令，系统即会在用户指定邮箱下再建立一个新的邮箱，我们只需键入该邮箱的名字即可。
#1    8．使用邮件模板
    FoxMail 3.1提供了模板功能，利用它的撰写、回复、转发等模板，我们可输入一些常见的内容，如地址、姓名、电话、E-mail地址以及回复说明、转发说明等，这就可在很大程度方便用户撰写邮件。
    （1）单击“选项”菜单的“属性”命令，打开“账户属性”对话框。单击“模板”项，打开模板窗口。（如^53200022a^1所示）
    （2）单击选择需要修改的模板选项卡（如“写新邮件”），对模板原有的文字内容进行适当修改。将光标移至模板中的适当位置，然后单击“插入宏定义”按钮，并从菜单中选择来信时间、发信时间、发件人、收件人、原信件内容、发信地址、回复地址等可变内容（这些内容今后将根据邮件的实际情况发生不同的变化）。
    （3）采用相类的方法对回复邮件、转发邮件选项卡的模板进行适当修改。单击“确定”按钮，关闭“属性”对话框。此后，用户在撰写新邮件、回复或转发邮件的时候，FoxMail就会按照用户的定义对格式进行设置，并键入一些必备内容，从而简化了用户撰写邮件的手续。
#1    四、账号的管理
#1    1．多账号的建立
    一个用户基本上都同时拥有多个E-mail地址以处理不同的事务。有鉴于此，FoxMail特意提供了同时支持多个邮件账号的功能，当我们同时申请了多个电子邮箱之后，只需在FoxMail中分别为这些电子邮箱的邮件服务器建立连接即可实现同时收发所有邮箱的目的。具体步骤是：
    （1）单击“账户”菜单中的“属性”命令，打开“账户属性”对话框。
    （2）单击“其他POP3”项。点击“新建”按钮，打开“连接”对话框（如^53200022b^2所示）。
    （3）在“连接”对话框中依次键入新邮箱的显示名称、POP3服务器、POP3账户和口令等内容。
    （4）单击“确定”按钮，关闭“新建连接”对话框，完成创建。此后就可以同时收取多个邮箱中的邮件了，从而满足了用户的需要。
#1    2．接收指定账号中的邮件
    当设置了多个邮件账号之后，FoxMail在默认情况下将同时接收所有邮件账号中的邮件，用户若希望单独收取某一个账号中的邮件，只需单击“收取”按钮旁的下拉按钮，从中选择某个账号即可收发该账号中的邮件。
#1    五、多用户的管理
#1    1．多用户的创建
    FoxMail 3.1新增了对多用户的支持，它允许同时设置多个不同用户，并且每个用户都拥有独立的“收件箱”、“发件箱”等项，可以使用自己独立的身份对邮件进行收发操作。为此，我们只需单击“账户”菜单中的“新建”命令，启动“建立新用户”向导，在向导的指引下完成新用户的创建工作。
    新创建的用户与系统原有用户的功能完全一样，它同样包含了“收件箱”、“发件箱”、“已发送邮件箱”和“废件箱”等项，但用户的账号、密码以及邮件服务器却完全不同，不同用户之间的操作也完全独立，从而满足了多用户同时操作的需要。
#1    2．设置安全保护
    为保护邮件安全，防止非法用户“窃取”信件内容，FoxMail提供了访问口令和邮箱密码两级保护功能，完全保证用户数据的安全。
    （1）访问口令
    要使用FoxMail为用户设置登录密码，我们只需选择需要加密的用户，然后单击“账户”菜单的“访问口令”命令，打开“口令”对话框，键入口令，FoxMail就会自动对该用户进行加密。设置“访问口令”之后，相应用户在使用FoxMail收发邮件之前就必须键入正确的密码，否则系统将禁止使用，从而保护了用户邮件的安全。
    （2）邮箱密码
    除了访问口令功能之外，FoxMail还提供了邮箱密码功能。使用该功能，FoxMail会按照用户设置的密码对指定邮箱进行加密，并将该邮箱中的邮件全部隐藏起来，我们唯有键入正确的密码对该邮箱进行解密才能查看邮箱中的内容。
    用户若拟查看加密邮箱中的邮件，首先应右击该邮箱，然后从快捷菜单中选择“解密”命令，打开“解密”对话框并键入原来的密码，解除该邮箱的加密状态。注意，邮箱加密功能仅只对当前邮箱有效，即使是加密邮箱的子邮箱也得不到密码保护，我们必须分别为每个需要保护的邮箱设置各自的保护密码。
#1    六、邮件过滤
    FoxMail拥有一个非常优秀的邮件过滤器，它不仅可对邮件的收件人、发件人、主题、附件、邮件头及邮件正文等项目进行过滤，并且还提供了转移、拷贝、重定向、自动回复、发出声音、运行应用程序、显示消息等多种不同的过滤方式，完全可满足用户对邮件进行过滤时的需要。
#1    1．设置过滤器条件
    （1）单击“账户”菜单的“过滤器”命令，打开“过滤管理器”对话框。单击“新建”按钮，建立新的过滤条件（如^53200022c^3所示）。
    （2）分别选取“条件”、“动作”选项卡，设置相应的过滤条件。设置的条件可以是一个也可以是多个，以满足不同的需要。单击“关闭”按钮，完成设置。
    以后凡是符合过滤条件的邮件，FoxMail都会自动对其进行过滤，从而实现了防止垃圾邮件、自动分信、自动回复、自动查毒等功能。
#1    2．对已有邮件进行过滤
    FoxMail的邮件规则功能在默认情况下都只对新收到的邮件进行过滤，如果用户的邮件已经保存到收件箱中，那么系统就不能自动对这些邮件进行过滤了，用户若拟对它们进行过滤，则必须选择需要过滤的邮件（可同时选择多个邮件），然后单击“工具”菜单中的“过滤所选邮件”命令，系统即会按照用户设定的过滤条件对指定邮箱中的邮件进行过滤。
#1    七、地址簿的使用
#1    1．地址簿的建立
    （1）单击“地址簿”按钮，打开“地址簿”对话框。
    （2）单击地址簿对话框中“创建卡片”按钮，打开相应的对话框。
    （3）在对话框中键入亲朋好友的姓名、E-mail地址、家庭地址、工作单位、电话、个人主页等内容。单击“确定”按钮，该联系人的信息即被添加到地址簿中。
#1    2．创建组
    所谓“组”就是将多个联系人组合在一起，我们只需选择这个组，系统即会将邮件发送到该组中的所有成员。要在地址簿中创建组，应采取如下方法：
    （1）单击“地址簿”按钮，打开“地址簿”对话框。
    （2）单击地址簿对话框中“创建卡片”按钮，打开相应的对话框。
    （3）在“组名”栏中键入新组的名称。单击“增加”按钮，打开“选择地址”对话框。
    （4）利用“选择地址”对话框从原有地址簿中将已有的联系人添加到该组中。
    （5）单击“确定”按钮，即创建完相应的组。然后我们就可以利用它同时向多个用户发送邮件了。
#1    3．利用地址簿快速发送邮件
    建立地址簿之后，我们就能利用它快速键入收件人的电子邮件地址了，用户只需在新邮件撰写窗口中单击“收件人”、“抄送”等按钮，系统就会自动打开“选择地址”对话框，我们只需选择相应的联系人即可，从而简化了用户手工键入收件人电子邮件地址的步骤。
    此外，我们也可直接在地址簿中选择某个联系人，然后单击地址簿中的“发送邮件”按钮，系统即会打开一个新邮件撰写窗口，并自动将该联系人设置为收件人，然后我们就能快速向其发送邮件了。
#1    八、与Outlook Express的信息交流
    FoxMail 3.1对软件的兼容性进行了更新和扩充，实现了与Outlook Express的全面兼容，可以导入Outlook Express的账号、邮件，也可以将自己的邮件、地址簿导入到Outlook Express，极大的方便了用户的使用。
#1    1．导入Outlook Express账号
    所谓导入Outlook Express账号即是将用户设置在Outlook Express中的用户名、密码、POP3等信息导入到FoxMail 3.1中，以便直接利用FoxMail 3.1收取这些账号中的邮件。
    单击“账户”菜单中的“导入Internet账号”命令，FoxMail即会自动对用户原来保存在Outlook Express中的账号进行分析，并添加到自己的账号列表中，然后我们就能在FoxMail中直接收取这些邮箱中的邮件了。
#1    2．导入Outlook Express信件
    注意，这里的导入Outlook Express信件是指将Outlook Express中的所有邮件一次性全部导入到FoxMail中。
    单击“邮箱”菜单中的“导入Outlook Express邮箱”命令，系统即会将Outlook Express中的所有邮件全部导入到FoxMail中。
#1    3．导入Outlook Express邮件
    除采用上面的方法将Outlook Express的邮件一次性全部导入到FoxMail之外，我们也可以单独导入一封邮件。
    启动Outlook Express，然后将需要导入到FoxMail的邮件采用.eml的格式保存到磁盘上，接着单击“文件”菜单的“导入邮件”命令，弹出“打开”对话框，选择需要导入的.eml文件并单击“确定”按钮，系统即将其导入到FoxMail中。
#1    4．将邮件导出到Outlook Express
    要将FoxMail的邮件导出到Outlook Express中，我们首先应在FoxMail中选中需要导出的邮件，接着单击“文件”菜单中的“导出邮件”命令，在弹出的“另存为”对话框中键入导出邮件文件的名称，并在“保存类型”栏中选择“Outlook邮件 （＊.eml）文件”项，然后单击“保存”按钮，将所选邮件以.eml文件的形式保存到硬盘上。接下来，我们只需双击这个.eml文件，系统即会自动启动Outlook Express并打开该邮件，最后我们只需执行Outlook Express“文件”菜单的“转移到文件夹”或“复制到文件夹”命令即可将FoxMail邮件导出到Outlook Express。
#1    5．导入、导出Outlook Express地址簿
    在FoxMail的地址簿中对话框中单击“工具”菜单“导入”子菜单中的“Web文件”命令，即可将Outlook Express的地址簿信息导入到FoxMail的地址簿中；而单击“工具”菜单“导出”子菜单中的“Web文件”命令，可将FoxMail的地址簿信息保存成为.web格式，供Outlook Express等其它软件使用。
