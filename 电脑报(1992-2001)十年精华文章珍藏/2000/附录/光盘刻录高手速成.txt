光盘刻录高手速成

哥斯拉
2000年 第53期 

    面对着体积不断膨胀的电脑软件和海量的数据，电脑用户之间的数据、程序的交流，却成了一个问题。什么载体具有容量大且价格便宜呢？自然是CD－ROM!但是光盘的缺点在于它是只读的，由此光盘刻录机应运而生。而且，随着技术和制造工艺的不断进步，刻录机的价格不断下降，使之进入寻常百姓家已经成为了可能。本文将为大家介绍刻录的基本知识和刻录技巧，使你也能成为刻录的高手。
#1    一、基础篇
#1    1.目前刻录机的种类
    依刻录方式的不同，主要有CD-R和CD-RW。
    CD-R的全称为CD-Recordable，中文译作“可记录式CD”。能用CD-R盘片进行写入操作的驱动器则称之为CD-R Driver。
    CD-RW的全称为CD-ReWritable，中文译作“可擦写式光盘”。它与CD-R主要的区别在于盘片的可写入次数上。CD-R盘片上的介质只允许写入一次，写入的数据无法再修改，而CD-RW盘片上的介质则允许写入1200次左右，数据可以再修改。
    依接口的不同，有IDE、SCSI、Printer Port之分。相对来说，SCSI接口的刻录机的性能要好一些，但价格也相对较贵；IDE口的刻录机性能相对要差一些，但价格却便宜很多；Printer Port口的刻录机现在已不多见了。
    此外，还有内置式和外置式之分，不过在性能上相差不大。
#1    2.光盘的格式
    目前CD光盘格式主要有三种：音乐CD（CD-DA，CD-Digital Audio）、数据CD（即CD-ROM，又称CD-ROM Mode 1）和扩展体系CD（CD-ROM XA，CD-ROM eXtended Architecture，又称CD-ROM Mode 2）。
    对于前两者，我们都很熟悉。至于CD-ROM XA，则是VCD与Photo CD等光盘所采用的格式。这三种格式的最大区别就在于每扇区的字节数，CD-DA为2352字节/扇区、CD-ROM Mode 1为2048字节/扇区、CD-ROM Mode 2格式对数据可靠性的要求相对较低，为2336字节/扇区。由此，也使这三种格式光盘的数据容量也有很大不同。
    按标准的74分钟光盘计算：
    CD-DA格式的容量为：74×60×75×2352÷1024÷1024=746.93MB
    CD-ROM Mode 1格式的容量为：74×60×75×2048÷1024÷1024=650.39MB
    CD-ROM Mode 2格式的容量为：74×60×75×2336÷1024÷1024=741.85MB
    现在的CD-R、CD-RW都能进行这三种格式光盘的刻录。不过，格式化后的CD-RW盘片，其每个扇区的容量与硬盘相同，即512个字节，但扇区数则增加了4倍，因此在总容量上没有什么变化。
#1    3.光盘的类型
    CD光盘的类型目前共有5种，分别是音乐CD（Audio CD）、普通CD-ROM（Normal CD-ROM）、多区段CD-ROM（Multi-Session CD-ROM）、混合模式CD（Mixed-Mode CD）、特殊模式CD（CD Extra Mode）。下面就对大家较为陌生的后三种类型作一说明。
    首先要介绍一下两个有关光盘数据存储的概念：轨道（Track）和区段（Session）。无论什么类型的数据都是以轨道为单位记录到光盘中，区段则是一个独立而完整的记录区域，它可以包含若干条任意类型的数据轨，在一个区段刻录完成后（即Close Session），其中的数据就无法再更改了，因此只有一次操作机会，但在这个过程中可包含多次轨写入。在读取数据时，则要先找到相应的区段，然后再找相应的轨。
    普通CD-ROM就相当于将整张光盘设为单一区段（SS，Single Session），由于只有一次操作的机会，所以不管写入多少数据都等于将光盘写满。而多区段（MS，Multi Session）CD-ROM则把光盘划分成多个记录区域，这样就可以分多次写入（刻录）。由此所带来的好处相当明显，比如当需要刻录的的数据只有100MB时，用SS方式的话，光盘上的剩余空间就白白浪费了，但借助MS方式，你可以先刻录100MB，以后若想写入新的数据，再建立新的区段就可以了。而且，用MS方式刻录失败的盘片也可以通过这个方法使用余下的容量，所以MS技术非常有用，现在的CD-ROM也都支持多区段光盘的读取。
    混合模式与特殊模式CD就是将音乐与数据资料录制在一起，所不同的是，混合模式将数据资料写在光盘的开头位置（即从第1区段的第1轨开始），特殊模式则将音乐资料写在光盘的开头位置。我们现在所玩的带有CD音轨的游戏大都使用混合模式刻录，而特殊模式光盘在音响上播放时则与Audio CD一样（混合模式一般要前进至少1轨以跳过数据区）。这里需要强调的是，影音器材中的CD与VCD播放机都不支持多区段读取，所以若想制作通用且又含有其他资料的Audio CD或VCD光盘，相应的数据必须要从第1区段的第1轨开始刻录。
    目前市面上几乎所有的CD-R、CD-RW与大部分的刻录软件都支持以上5种类型光盘的制作。
#1    4.常见的刻录格式
    可以把它看作是光盘的文件系统，它与FAT、FAT32、NTFS等硬盘文件系统的作用类似。现在常用的光盘文件刻录格式主要有以下几种：
    ISO-9660：又简称ISO，是由国际标准化组织在1985年制定的，当前唯一通用的光盘文件系统，任何类型的计算机、刻录软件都支持它。ISO-9660目前有两个标准：Level 1和Level 2。Level 1与DOS兼容，文件名采用传统的8.3格式，而且所有字符只能是26个大写英文字母、10个阿拉伯数字及下划线。Level 2则在Level 1的基础上加以改进，允许使用长文件名，但不支持DOS。
    Rock Ridge：针对UNIX系统的ISO-9660文件系统，支持文件名字母大小写、符号字符以及长文件名。由于兼容ISO-9660，所以即使操作系统不支持Rock Ridge，也可以通过ISO-9660查看。
    HFS：苹果公司（Apple）的MAC机所使用的光盘文件系统，全称为混合文件系统（Hybrid File System）。
    Joliet：微软公司自定义的光盘文件系统，也是ISO-9660的一种扩展，支持Windows 9X/NT和DOS，在Windows 9X/NT下文件名可显示64个字符，并可使用中文。
    Romeo：这是由著名的接口厂商Adaptec公司自定义的文件系统，支持Windows 9X/NT，文件名最多可有128个字符，支持中文，但不支持DOS。
    UDF：UDF是统一光盘格式（Universal Disc Format）的缩写。它采用标准的封装写入技术（PW，Packet Writing）将CD-R当作硬盘来使用，用户可以在光盘上修改和删除文件。
    CD-RFS：是索尼公司自定义的一种与UDF类似的文件系统，也使用PW技术，其全称为可记录光盘文件系统（CD-Recordable File System）。
    在刻录时，对文件系统的支持一般取决于刻录软件，不过有些较为特殊的文件系统也需要硬件方面的配合，如UDF和CDRFS，用户可以在产品说明书中得到有关信息。
#1    5.光盘的刻录方式
    制作什么类型的光盘与采用什么样的刻录方式有非常密切的关系。刻录技术发展至今，主要的刻录方式有以下5种：
    整盘刻录（DAO，Disc At Once）：主要用于光盘的复制，一次将整张光盘刻录完成。其特点是能够复制出与源盘一模一样的光盘。不过，由于DAO方式就相当于将光盘当作一个区段，如果刻录失败这张光盘也就彻底报废，所以它对数据的传送和驱动器的性能要求较高。
    区段刻录（SAO，Session At Once）：这种方式一次只刻录一个区段而不是整张光盘，余下的空间可以继续使用，主要用于多区段CD-ROM的制作，非常适用于制作合集式光盘。
    轨道刻录（TAO，Track At Once）：以轨为单位的刻录方式。利用它，你可以向一个区段分多次刻录若干轨的数据，所以主要用于制作音乐合集或混合、特殊类型的光盘。
    飞盘（OTF，On The Fly）：可以说这是目前一种很常用的刻录方式，以前由于计算机处理速度不够，只能在刻录前将数据转换成使用ISO-9660格式的映像文件（Image File）然后再刻录，如今的电脑处理速度已允许进行即时转换，而这种将数据自动转换成ISO-9660格式然后进行刻录的方式就是OTF。
    封装刻录（PW，Packet Writing）：主要是制作UDF或CD-RFS光盘所使用的方式，对驱动器有一定的要求。
    目前绝大多数新出品的CD-R与CD-RW都支持这5种刻录方式。因此，至于采用哪一种就取决于刻录软件了。
#1    二、速成篇
    下面，以Easy CD Creator 4.0为例，来介绍刻录的一些方法和技巧。
    首先，在刻录以前，关闭屏幕保护，断开网络连接，因为刻录过程中要以硬盘作为缓存，对硬盘的任何读写操作都有可能造成缓存数据的中断，产生废盘。其次，硬盘上一定要有700-800M的自由空间，做为缓存和存放光盘映像用。
#1    1.刻录数据盘
    启动Easy CD Creator，选择“DATA”、“Data CD”。
    将欲刻录的数据由数据区用鼠标拖至光盘布局后，选择“Create CD”，在弹出的对话框中选择刻录速度（Write）选择２×（适当的降低刻录速度可以防止缓存数据的中断），刻录选择“Test and Create CD”（检测CD同样可以减少废盘率），刻录方式为“Close Session and Leave CD Open”（TAO，不封口，可以下次继续刻）。最后，单击“OK”按钮。
    其他的刻录方式还有：
    Leave Session Open：用来在刻录中发生意外而没有关闭Session的光盘，使之可以继续使用；Close CD：关闭光盘，使之无法继续追加数据；Session At Once：区段刻录；Disk At Once：整盘刻录。
    对于整盘的数据拷贝，除了用“Disk At Once”，还可以使用CD Copier。不过，CD Copier对源盘的质量要求较高，如果源盘有严重的划伤，会引起数据中断而导致刻废；而且，CD Copier会在刻完后Close Disk，以后再也没法追加数据了。所以，最好是先将源盘做成CD Image再进行刻录。
    至于追加新数据，启动CD Creator，放入未Close Disk的光盘，将新数据拖至光盘布局，选择“Create CD”即可。
#1    2.刻录CD唱片
    用Easy CD Creator刻录CD可谓简单。启动Easy CD Creator，选择“AUDIO”、“AUDIO CD”进入音频刻录窗口，在数据区选择欲刻录的音频文件，可以是CD音轨、MP3、WAV等等。单击“Play”可以试听。然后将欲刻录的音频文件拖到光盘布局中，单击“Create CD”即可。
#1    3.制作光盘启动盘
    进入Data CD，右击光盘布局，选择“New”、“Bootable CD”，然后在软驱中放入启动软盘；
    Easy CD Creator在刻录中会自动生成Bootcat.bin、Bootimg.bin文件，从而完成启动光盘的映像文件。
    其他的选项：Folder：创建新目录；Data CD：制作数据光盘；Audio CD：制作CD光盘；Mixed-Mode CD：制作混合模式光盘；CD Extra：制作特殊模式光盘。
#1    4.制作VCD
    Easy VCD Creator是Easy CD Creator中的一个模块。Easy CD Creator可以让你先把设计好的VCD试播看看，还有向导（Wizzard）来指导你一步一步操作，不过它制作出的VCD播放起来并不“Easy”，Simple video sequence（简单顺序型）VCD只能一段一段依次播放，没有选单；One Level Menu Structure （单层选单型）VCD有一个或多个选单，选单下只能有播放元件，不能有第二层选单。
#1    三、进阶篇——用Video Pack 4.0制作VCD
    Video Pack 4.0是目前最常用的VCD2.0刻录软件，它不但可以刻录顺序播放的VCD，还可以刻录利用遥控器点播的分支结构的VCD光盘，编辑刻录交互式多结尾的影片片断、教学光盘、菜单式选单音乐等。
    下面来介绍如何刻录利用Video Pack 4.0刻录选单式VCD：
    启动Video Pack，单击“File”、“New”创建一个空的VCD视频表，在观察框中通过鼠标右键快捷菜单，添加第一个主菜单选择表（New menu）。为主菜单增加两个按钮。用鼠标右键选中选择表，在快捷菜单中选中“Add seletion item”，增加两个选择条目。在观察框中，利用鼠标右键快捷菜单添加两个容器（New container），分别在容器快捷菜单“Properties”中起两个名字：Container1和Container2，此时容器的名字将显示在容器框内。利用鼠标右键快捷菜单在观察框内添加一个播放表（New playlist），作为VCD播放结束画面。将主结构的各个节点摆好后，画出节点之间的连线。用鼠标选中节点，在每个出口处按住鼠标左键，拖动到要连接的节点内，放开左键。按照事先设计好的播放顺序画出连接线。
    下面设置节点内容和属性：
    为选择表添加一个播放表，在源窗口将0.bmp文件拖入选择表，节点中将显示该图片文件。在选择表鼠标右键快捷菜单中选中“Button editor”进入按钮编辑器。将按钮拖至最终希望放置的位置上，通过右键快捷菜单“Align buttons”和“Make same size”设置两个按钮的对齐方式和尺寸大小。选择按钮1。设置按钮的属性可以通过执行快捷菜单中“Properties”命令，在弹出的打开对话框中将“Hotspot only”关闭，以便画按钮，如果回放设备具有不同于鼠标指示类型的遥控器时，“Auto number”应是开启的。使用的每个按钮将对应遥控器按键所激活的播放条目。在“Blackground”选项卡中选择“Background bitmap”并且调入相应的背景图片，单击“Apply”按钮设置背景图片。在“Text style”选项卡中设置好按钮字体和字号以及字体颜色。在“Text style”选项卡中，在“Button text”中输入按钮需要显示的文字：容器1，并在“Alignment”中设置按钮为“Center”（居中）属性。在“Border”选项卡中设置按钮的边界显示类型（Type）为“thick 3D border”。设置好以上属性后，单击“Apply”将设置属性加载在按钮1上。
    第二个按钮的设置与第一个相似，在快捷菜单中选择“Same style”项，这样就可以把第一个按钮的样式复制到第二个按钮上。不同的是在第二个按钮的“Text”文本属性中填写为“容器2”。设置好按钮属性后，关闭按钮编辑器窗口。
    双击容器1，在容器1中放置一个选择表和两个播放表。为选择表增添两个选择按钮，分别设置好的文本显示信息为节目1和节目2，并连好节点之间的连接线。将背景图片拖入选择表，并设置好按钮的不同属性。在源文件窗口中将Mpeg1.mpg和Mpeg2.mpg 拖入到两个播放表中，注意要与选择条目中的文本显示内容相吻合。完成设置后，双击容器1退出。双击容器2进入节点编辑，在其中放置好一个选择表和两个播放表，设置选择表的方法同容器1，设置好选择条目的文字显示信息，分别为节目3和幻灯片播放。将Mpeg3.mpg拖入第一个播放表。在源窗口中按住Crtl键对1.bmp，2.pcx，3.tif文件做标记，然后将它们拖入第二个播放表。下面要为这几个幻灯图片添加背景音乐。
    在快捷菜单中选择“Play item”编辑播放表中的播放条目，选中“Mpeg still picture with optional mpe”，单击“Properties”按钮进入“Still Mpeg with optional mpeg audio”编辑对话框，在“Mpeg audio”中“Filename”右侧单击“…”，调入音频文件exampl.wav，并选中“adjust time to Mpeg audio”，以使播放时间同伴音时间等长。
    返回顶级菜单，选择结尾画面的播放表，在快捷菜单“play item”中选定“Mpeg still picture with optional mpe”，进入“Still Mpeg with optional mpeg audio”编辑对话框，用“Add”按钮添加图片文件end.bmp，选中“adjust time to Mpeg audio”，设置影像格式为“576 lines pal”，调入“Mpeg audio file”文件end.wav做为背景音乐。完成设置后，返回顶级菜单，保存设置好的项目文件后，就可以刻录VCD了。
