        运行卸载程序：
        · 如果正在运行 Windows 95：
        (1) 单击“开始”菜单，指向“设置”，然后选定“控制面板”。
        (2) 双击“添加/删除程序”。
        (3) 在“添加/删除程序”对话框中，单击“安装/卸载”标签。
        (4) 在 Windows 可删除程序的列表中，单击 Windows 95。
        (5) 单击“添加/删除”，然后按屏幕提示操作。
        · 如果没有运行 Windows 95：
        (1) 如果在安装时生成了 Windows 95 急用修复盘，可以用它来引导Windows 95。
        (2) 在 A 提示符后键入 UNINSTALL。
        问2：如何从远程 CD-ROM 驱动器上安装 Windows 95 ?
        答： 如果带 CD- ROM  驱动器的计算机正在运行   Windows  for  Workgroups  或Windows 95,并且已将 CD-ROM 驱动器设置为共享，那么请按下列步骤执行：
        (1) 可以用“文件管理器”或在命令行中使用“net use”， 通过连接网络驱动器的方式来连接共享的 CD-ROM 驱动器。
        (2) 双击 Setup.exe ，或在命令行中键入 Setup 。
        问3：安装 Windows 95 之后，原有的程序是否要重新安装?
        答：当从原来的 Windows 版本或 Windows for Workgroups 升级时，Windows 95 将会获得的程序的设置信息。如果 Windows 95 安装在一个单独的目录下，所有Windows 下的程序都需要重新安装。
        问4：硬盘中有 25MB 可用空间，但当试图升级至 Windows 95 时， 计算机显示磁盘空间不够，升级时计算机中应有多大可用空间?
        答：升级到 Windows 95 时，需要 35～40MB 可用磁盘空间。如果进行完全安装，则需要 50～55MB 的可用磁盘空间。
        问5：能否在一台已安装了 OS/2、MS-DOS、Windows 的计算机上安装 Windows 95?
        答：Windows 95 的 Setup.exe 将不能在 OS/2 环境下运行。要安装 Windows 95，请先将计算机引导为 MS-DOS 方式，然后在 MS-DOS 提示符下运行 Setup.exe 。
        如果先从软盘启动 MS-DOS ，然后再运行 Setup ，那么在安装完 Windows 95之后不能启动 OS/2。在运行 Setup 之前应当先将 OS/2 使用的 Autoexec.bat 和 Config.sys删除掉。
        问6：如何复制原始安装磁盘?
        答：DMF 磁盘格式对 Diskcopy 或 Copy 命令不兼容，并且增加了在标准的 3.5英寸1.44MB 软盘中存储的数据总量。目前还没有办法直接复制这些磁盘。
        问7：能否将软件从 CD 复制到软盘?
        答：CD-ROM 中包含的 Cabinet 文件(*.cab)的大小均为 2MB ，因而无法复制到软盘上。
        问8：是否应该为 Windows 95 设置一个独立分区?
        答：和其它软件包一样，最好在安装之前进行备份。 Windows 95 已经在几千种计算机上、不同的系统配置下检测过，成功率非常高。因此我们认为：在安装 Windows 95过程中出现问题的可能性很少。
        在安装过程中可以使用“卸载”特性。 Uninstall 程序复制了原来版本的MS-DOS和Windows 3.x ，并且作了记录，因此可以删除   Windows  95  并将原来的文件恢复到Windows 95 安装之前的状态。Windows 95 并不需要自己的分区，但是， 如果需要在当前的 Windows 3.x 和 Windows 95 之间进行双重引导，可以将它安装在自己的分区里。
        问9：应当将 Windows 95 安装在现存 Windows 目录下还是在不同的目录下?
        答：建议保留原先安装的 Windows 并在其基础上升级， 因为安装程序将使用现存的配置文件(Config.sys、Autoexec.bat、System.ini、Win.ini、Protocol.ini) 来更加准确地检测和安装。
        问10：为保留 Windows 95 的设置应备份哪些文件?
        答：可以用 Windows 95 的备份程序来备份的系统以及用户和软件特别选定的所有文件。
        当运行“备份”程序时，请在第二屏选择“备份全部系统文件”。
        注意：把 Windows 95 安装到另一个目录之后，所有的程序( 以及可能的所有外设驱动程序)都必须重新安装。
        问11：已将 Windows 95 安装在与 Windows 3.x 或 Windows for Workgroups 3.1x不同的目录中，怎样才能在它们之间设置双重引导?
        答：如果已经将 Windows 95 安装在不同的目录下，默认状态为双重引导。双重引导要求现在使用的 MS-DOS 版本为 5.0 版或更高。如果满足这个条件， 执行下列操作即可实现对 MS-DOS 早期版本的双重引导。
        (1) 启动时看到“Starting Windows 95...”字样时请按下 F8 键。
        (2) 选择“Previous Version of MS-DOS.”。
        问12：能否在 MS-DOS 下运行 Windows 95 安装程序?
        答：如果正在运行 Windows 3.x 中文版，建议最好在现有版本的  Windows  下安装Windows 95 ，这是最安全、平稳的安装方式。如果升级 Windows 3.1英文版或 Windows for Workgroup 3.1x，则应在 MS-DOS 下安装。当在 MS-DOS 下安装， 安装过程将会变得较慢(特别是从软盘上安装的时候)。
        问13：能否从网络上安装 Windows 95?
        答：可以。从网络上安装 Windows 95 可以采用下面两种方法：
        (1) 对于 CD-ROM 版软件，请用 COPY 命令建立一个 Windows 95 目录；对于软盘装的软件，请用 Extract 命令建立 Windows 目录。与共享目录相连，运行安装软件，然后选择安装 Windows 95 的目录。
        (2) 进入 Nettools 目录(仅适于 CD-ROM 版)，用 Netsetup 安装。
        问14：第一次安装 Windows 95 时，在硬件检测阶段出现死机，怎么办?
        答：如果的计算机在安装过程中的硬件检测阶段死机，那么请先关机 10 秒钟，然后再开机。不要按 CTRL+ALT+DEL 键。当重新启动系统时，请用 Safe Recovery 模式重启Windows ，这样就会跳过引起问题的检测部分。如果的系统仍然死机， 那么可能是另一种模式造成的。 多次重复上述步骤，直到的系统检测完毕。
        注意：使用安全模式启动Windows 95时，无法进行系统设置等操作。
        问15：安装时没有检测到网卡，怎样才能设置网卡呢?
        答：添加一个网络适配器：
        在“控制面板”中， 双击“添加新硬件”，使Windows 能够检测网卡。如果Windows 不能检测网卡，可以按照以下步骤选择：
        (1) 在“控制面板”上，双击“网络”图标。
        (2) 单击“添加”，然后单击“适配器”。
        (3) 在制造商列表中找到制造商和型号。
        (4) 为了与网卡的配置相匹配，请改变显示设置。
        如果是表中没有的网络适配器，请检查适配器附带的磁盘。如果 Windows或Lanman(NDIS兼容)磁盘中带有驱动软件，那么选“从磁盘安装”并指向Oemsetup.inf文件的路径。
        如果只有 Novell ODI 驱动程序，请用 LSL.COM 和 〈nic-odi〉.com( 由 Novell 和 NetWare 卡制造商提供)安装驱动程序，然后选择“已检测到的网络驱动器”，从该驱动器安装“已有的 ODI 驱动程序”。
        问16：从 CD-ROM 上安装 Windows 95 ，第一次引导就死机，为什么?
        答：最可能的原因是实模式与 CD-ROM 驱动程序的保护模式相冲突。请在Config.sys中删掉引用 CD-ROM 的行。如果仍不能解决问题，请试试下面的几步：
        可以在“安全模式”下引导 Windows 吗?如果不能，请重新安装。如果能，则：
        产生 Bootlog.txt 文件，查看在何处死机。
        查找 Detcrash.log 文件。如果文件存在，问题肯定出在硬件检测上。
        检查 Ios.ini 文件，并将Config.sys 中调用驱动程序的行删除， 有时候这些保护模式驱动程序可能会引起问题。
        检查“设备管理”标签中的设置，看是否存在冲突。如果怀疑某个设备引起问题，请将这个设备驱动程序从“设备管理”中删除掉。
        将显示驱动程序设为标准 VGA。
        试着单步引导。
        将 Autoexec.bat 和 Config.sys 换名，这样在下次引导时将不会调用这两个文件。将 System.cb 复制到 System.ini 中，尽量减少加载的驱动程序(甚至不加载鼠标)。
        如果解决了问题，请检查System.ini中第三方应用程序的条目， 例如， AdobeType Manager 。
        问17：如何安装 Microsoft 传真程序?
        答：在安装 Windows 95 时就装入了 Microsoft 传真。显示“附件”屏幕时， 确认已选择了“Microsoft 传真”。如果没有选 Microsoft Exchange ，Windows 95  将会提示选择。
        在安装完 Windows 95 之后安装 Microsoft 传真程序。在“控制面板”上双击“添加/删除程序”图标，然后单击“安装 Windows”标签。在“附件”框中，翻到 Microsoft 传真那一页，确认已选中该组件。如果没有选 Microsoft Exchange，Windows 95 将会提示选择。
        注意：要安装 Microsoft Exchange 和 Microsoft 传真至少需要 1OMB 空间。
        问18：安装 Windows 95 之后。无法访问 CD-ROM，为什么?
        答：如果你的 CD-ROM 是 Sony 、 Mitsumi 或 Panasonic 的专用产品，Windows 95 将安装其驱动程序。如果没有检测到 CD-ROM，可能与设备或控制器发生了冲突， 应该在“控制面板”的“设备管理”中检查可能引起冲突的设置。在“控制面板”中双击“安装新硬件”图标可以添加新的设备。
        问19：监视器的刷新速率有问题，应该如何解决?
        答：要调整监视器的刷新速率，应该先确定监视器的类型， 这可以通过查看显示适配器和 PC 制造商提供的适配器安装程序来解决。一些实用程序必须在 Autoexec.bat 中运行。对于某些 PC 机，监视器类型在 BIOS 配置程序中设置。
        问20：使用Stac Electronics Stacker 软件压缩硬盘后，Windows 95能否正常工作?
        答：可以。Windows 95 与 Stacker 2.x 版及更高版本完全兼容，也和Superstore的各个版本兼容。第三方磁盘压缩实用程序的旧版压缩驱动软件只能在实模式下工作。建议升级For Windows 95 驱动程序。Stacker 4.1 for Windows 95已经发表了。
        Windows 95 提供了一个称为“磁盘空间管理”的内置磁盘压缩实用程序， 它在保护模式下运行。该软件的优点是压缩速度更快，并且是内置在操作系统中。
        问21： Windows 95 未能检测到 CD-ROM ，为什么?怎样才能使 CD-ROM 正常工作?
        答： CD-ROM 应当是基于 SCSI 、专用的或 IDE 控制器的。 Windows 95 只能检测到基于专用控制器的 CD-ROM(特指 Panasonic 、 Mitsumi 和 Sony 几种品牌)。
        Windows 95 也能检测到 SCSI 控制器， SCSI 总线的驱动程序将列举被支持的 CD-ROM 。
        问22：如何将 Windows 95 设置为使用 TCP/IP 网络?
        答：如果网络使用 DHCP 服务器来自动寻址 IP ，或者已经分配了一个特定的IP地址，请先找到系统管理器，然后按下列步骤操作：
        (1) 在“控制面板”上，双击“网络”图标。
        (2) 在“配置”标签上，单击“添加”，然后双击“协议”。
        (3) 单击“Microsoft ”，然后单击“ TCP/IP ”。
        安装完之后，单击“网络”属性，再单击“配置”标签上的“TCP/IP”，然后单击“属性”；并根据系统管理器的指导配置协议。
        问23：如何设置Netware 支持?
        答：在 Windows 95  中有三种方法可以访问   Novell  网络。 如果使用   Novell Netware3.x ，或者的服务器使用 4.x 版，可以使用 Microsoft Client for Netware软件。在“网络”属性中安装上述软件的步骤如下：单击“添加”，单击“客户”，单击“Microsoft”，然后再单击“ Netware 网络客户”。
        如果需要在注册文件中加载实模式 TSR ，必须使用实模式外壳软件 NETX 或VLM。如果使用 NDS 服务器 4.x 版，则必须安装 VLM客户外壳软件。
        如果使用 NETX ，应该安装 Novell Workstation Shell 3.x 。具体步骤为: 在“网络”属性中，单击“添加”，单击“客户”，然后单击“Novell”。
        如果使用 VLM 客户外壳软件，需安装“Novell Workstation Shell 4.x Client”。
        注意：必须有支持 Novell 文件的 Windows 版本才能使用 Workstation Shell选项。
        问24：如何设置 Windows 95 以支持 Lantastic 6.0 ?
        答：要配置 Windows 95 来支持 Artisoft Lantastic 网络，请与供应商联系或询问Windows 95 Lantastic *.inf 。
        注意：不能使用“网上邻居”来查阅 Lantastic 资源。必须使用 Lantastic 程序组中的 Lantastic 实用程序。
        问25：如何安装 PPP 拨号网络连接器?
        答：首先需要一个已装好的调制解调器以及正确的配置。如果没安装拨号网络选项，那么请按以下步骤添加：
        打开“控制面板”，然后双击“添加/删除程序”。单击“安装Windows”并按屏幕提示操作。安装完后，请在“我的电脑”中打开“拨号网络”文件夹，启动“连接向导”。默认情况下将安装 PPP 协议，因为它最具有灵活性，而且可以在 NetBeui.IPX、SPX 和 TCP/IP下工作。
        问26：如何安装拨号网络使之与 Novell 服务器相连?
        答：在“控制面板”的“网络”图标中安装下列组件：
        Microsoft Client for Netware Networks
        IPX/SPX 与 Microsoft 拨号适配器驱动程序协议兼容。
        要使用拨号网络连接 Netware Connect Server ，必须将“NetWare Connect ”中的服务器类型指定为“拨号网络”连接属性中的类型。
        问27：为什么在Windows 95注册时系统死机?
        答：如果与服务器的连接丢失， NetWare   的注册文件处理程序会停止工作，Microsoft Client for NetWare 超时并出现错误信息。
        问28：如何配置计算机，使它能响应拨号网络?
        答：如果有 Microsoft Plus for Windows 95中文版，就可以从报配置的计算机来响应拨号网络的呼叫。启动拨号网络；在“连接”菜单上单击“拨号服务器”；启用呼叫响应。
        如果没有 Microsoft Plus，只能使用拨号网络从所配置的计算机向外呼叫， 而不能接收拨号网络的呼叫。
        问29：在 Windows 95 中如何与 Internet 连接?
        答：Windows 95提供了多种连接 Internet的渠道，下面介绍其中两种：
        ·  可以向 Microsoft Network 联机服务注册。
        Microsoft Network (一种新的联机服务)是 Windows 95的新功能。使用 Microsoft Network，可以与世界各地的人们交流信息；阅读最新消息、体育新闻、天气预报和金融信息；查找解决技术问题的答案；下载数以万计的可用程序以及连接到Internet 等。
        要启动 Microsoft Network，请双击桌面上的 The Microsoft Network 图标。
        ·  通过拨号网络与 Internet 访问供应商连接。
        (1) 申请 Internet帐号(用户名、口令、电话号码、主机名和域名、域名系统( DNS)的服务器地址)。同样，Internet 访问代办处将提供下列信息：
        IP 地址和子网掩码；DNS 的搜索顺序(如果需要的话)； 网关地址；
        身份验证过程(不论是否需要终端窗口进行登录)。
        注意： PPP  如果访问代办处已提供   PPP  帐号， 请使用该帐号。 如果已直接与Internet 连接，可以跳过这一步。
        若已有了Internet 帐号，请转(2)。
        (2)：安装用于通讯的硬件
        需要一个调制解调器(速度为9600bps或更快，一般选为1440bps以上)。如果已直接与Internet 连接，可以使用网卡取代调制解调器。
        如果尚未安装调制解调器，请按下述步骤安装：
        单击控制面板中的“调制解调器”来安装调制解调器；
        然后按照屏幕上的提示操作。
        若调制解调器已安装完毕，请转(3)。
        (3) 确认已经安装了“拨号网络”
        在桌面上双击“我的电脑”。如果没有看到“拨号网络”图标，请继续执行；
        单击控制面板中的““添加/删除程序”；
        单击“通讯”，然后单击“详细资料”；       
        复选“拨号网络”，然后单击“确定”。
        若拨号网络已安装。请转(4)。
        (4) 安装 Windows 95 TCP/IP 协议
        单击 显示“网络”属性；
        单击“添加”。(如果已经安装了 Windows 95 TCP/IP 协议，请直接转(6)；
        单击“协议”，然后单击“添加”；
        单击“Microsoft”，然后单击“TCP/IP”；
        单击“确定”。
        单击“拨号适配器”，然后单击“属性”。如果贵单位已直接与 Internet连接，请单击网络适配器，然后单击“属性”。
        单击“绑定”标签，确认是否选中“TCP/IP”。
        如果出现关于重启动的提示，请重新启动计算机。
        注意：在安装  TCP/IP  时， 默认情况下它将绑定所有网卡。 如果某个网卡不使用TCP/IP，请转到该网卡的“属性”对话框，单击“绑定”，然后删除 TCP/IP。
        若该适配器已与 TCP/IP 绑定，请转(5)。
        (5) 输入 TCP/IP
        右击控制面板中的“网络”，然后选择“属性”，则显示出“网络”属性。
        单击 TCP/IP (如果有多个 TCP/IP 选项，则单击“TCP/IP-〉拨号适配器”)，然后单击“属性”。
        在“IP 地址”标签下，如果访问代办处提供了 IP地址，则单击“指定 IP地址”，然后键入 IP 地址和子网掩码。
        如果没有提供 IP 地址，则单击“自动获得一个 IP地址”。
        如果直接与 Internet 连接，请键入 IP 地址和子网掩码。如果网络使用 DCHP服务器分配 IP地址，请单击“自动获得一个 IP 地址”。
        在“绑定”标签中，建议清除“Microsoft 网络的文件和打印共享”复选框。
        输入 TCP/IP 信息，请转(6)。
        (6) 建立与访问代办处的连接
        在“拨号网络”中双击“新建连接”。(如果这是第一条“拨号网络”连接，系统将自动提示。)
        按照屏幕上的提示操作。
        使用鼠标右键单击该连接，然后单击“属性”。
        如果的访问代办处需要通过终端窗口进行登录，请单击“配置”。然后单击“选项”标签，确认是否选中“拨号后出现终端窗口”，然后单击“确定”。
        在该连接的“属性”中，单击“服务器类型”，然后确认是否将“拨号服务器类型”设置为 PPP。
        确认是否复选以下选项，这样可以减少建立连接的时间。
        在网络上登录；NetBEUI；IPX/SPX 兼容。
        注意：如果有一个 SLIP 帐号，则需要安装 SLIP 支持。
        SLIP 支持只能从 Windows 95 的光盘或  Windows  95 资源工具库中获得。 在安装SLIP 支持后，转到该连接的“属性”标签页，单击“服务器类型”， 并确认是否已正确安装 SLIP 。
        若连接已设置，请转(7)。
        (7) 拨通的 Internet 访问代办处
        双击该连接，输入名称和口令，然后单击“连接”；
        按访问代办处提供的指示浏览 Internet 、电子邮件以及要访问的信息。
        注意：Internet 的信息传递是双向的。这意味在连接后， 其他人仍可以访问您的计算机。因此建议您在连接之前禁用“文件和打印共享”选项。
        问30：如何在桌面上创建快捷方式?
        答：可以采用下面两种方式在桌面上创建快捷方式：
        使用鼠标右键单击桌面，单击“新建”，然后单击“快捷方式”。键入带特定参数的命令，或通过“浏览”按钮查找建立快捷方式的对象。单击“下一步”，按照屏幕提示给出快捷方式名称。输入名称后，单击“完成”。或者：打开“ Windows资源管理器”，找到想建立快捷方式的程序。按住鼠标左键将图标拖到桌面上，从而在桌面上创建一个名为“ 快捷方式  〈程序名〉”的快捷方式。
