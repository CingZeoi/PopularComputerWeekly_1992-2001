流媒体的应用──VCD转WMV

贾利萍 
2001年 46期

流媒体技术作为当今网络世界最流行的技术，可以广泛地应用在网络上的视频点播、多媒体新闻发布、网络演示、在线直播、远程教育、远程医疗、网络视频广告、网络电台、网络会议（Net-meeting）等互联网的信息服务领域。在现在最流行的视频点播上，流格式应用得最为广泛。现在我将和大家一起，通过把家用VCD影碟文件转换为网络上实时播放的流格式文件，来学习流格式文件的制作。
　　首先介绍要用到的工具：MPEGTOOLS、Microsoft Windows Media Encoder 7。
　　#1一、把VCD转成MPEG格式
　　用MPEGTOOLS把VCD上的视频文件转换成标准的MPEG格式。因为Microsoft Windows Media Encoder 7不能直接转换VCD的DAT文件。
　　1.打开VCD中的MPEGAV文件，选中其中的.dat文件。这就是我们将要放到网上播放的文件。
　　2.利用MPEGTOOL1.3工具。将.dat文件转为.mpg格式。
　　选择你的.dat文件文件所在路径，并在下面的选择中，选择你将要保存.mpg文件的路径，并点选“next”（^46020802a^）。
　　MPEGTOOL1.3工具还有截取部分片断的功能，只须在框中输入数值就可以完成。这里的数值是指该文件的总帧数，89700，它就显示为0～89700帧，你如果只要150000～1500帧的视频部分，就直接在框中输入精确的数值（^46020802b^）。
　　如果对自己刚才的设置不是很满意的话，就可以按“previous”（返回上一级菜单）。接下来就点“next”，接着MPEGTOOLS就开始转换了，转换文件的过程可能需要花上几分钟，不过，要视你给的视频文件大小而定。完成后，就可以点OK了。如果你在半途中，要做一下修改的话，就可以点“cannel”（取消）停止当前工作，这样就回到上一步的操作过程中。
　　这是非常实用的.datq转.mpg工具，很多视频的转换都少不了这一步！
　　#1二、把MPEG格式转换成WMV
　　用Microsoft Windows Media Encoder 7把MPEG文件转换成WMV流格式文件。
　　现在我们开始利用多媒体压缩技术的一个压缩软件Microsoft Windows Media Encoder 7。在使用这个工具之前，我们必须先了解它的一些基础知识。首先，可支持的操作平台有：Windows 95/98/NT/2000；支持多种文件格式：如AVI、MOV、MP3、WAV。
　　1.现在我们启动安装好的Windows Media编码器（不要忘了，先看看你的机器上是否有Windows Media Player 7以上版本，如^46020802c^。
　　2.启动后，会弹出欢迎使用Windows Media 编码器的对话框。在你想执行的操作中，其中提供了4种选项。
　　第一项为使用新建会话向导广播、捕获或转换文件
　　第二项为创建自定义编码会话
　　第三项为打开现有的编码会话
　　第四项为打开最近的编码会话
　　在这里我们很明显的选择了第一项，然后点确定。
　　3.此时，出现新建会话向导。会话选择中有说明：选择本页上的选项之一设置编码会话。然后在你想执行的操作中，选择其中的一项。
　　第一项：在附属设备或计算机屏幕广播实况事件
　　第二项：从附属设备或计算机屏幕捕获音频或视频
　　第三项：将音频文件或视频文件转换成Windows Media文件。
　　这里我们选择第三项（^46020802d^）。
　　4.点“下一步”，这里是文件选择对话框。源文件中的内容转换成Windows Media格式并保存到Windows Media文件中。在选项中，对要转换的文件，点击浏览，就可以选择你将要转换的文件的路径，并找到文件。
　　接着就要创建文件。同样点击浏览，也可以对将要转换成的文件选择存放的路径，并填入文件名（^46020802e^）。
　　提示：要得到最佳效果，请使用最高质量的源文件。对以前编码的文件进行重新编码不能得到最佳效果。
　　5.接下来会出现输出文件分发的对话框。分发编码内容有几个选项。你选择的选项决定在该对话中可以使用哪些配置文件及输出文件并进行分发。
　　第一项：文件将从Windows Media服务器传输
　　第二项：文件将从Web服务器传输或直接从计算机上播放。
　　选第一项，必须配置Microsoft Windows Media Server。该项主要用于配置音频或网络视频点播。
　　我们这里需要选择第二项，在网络上实时播放流媒体格式。
　　提示：请确保按照你选择的方法分发文件。如果使用其它方法，将导致在Windows Media Player中查看时质量降低。
　　6.再点“下一步”出现配置文件选择对话框。然后选择一个配置文件，使其配合你的源文件内容和目标网络环境。对话框下部提供了一些选择：里面的各个选项都有相应的说明，在网络上放映影片，我们要考虑到你将给哪些人看，他们是拨号上网的用户，还是宽带用户。这些问题都准备好了以后，我们就可以选择了（^46020802f^）。
　　其实在这里还可以自定义你的配置文件，只须点击创建和管理配置文件，就可以开始创建了。在管理配置文件中，你还得新建一个配置文件，在新建配置文件对话框中，写入你自己给配置文件取的名字。还可以任意选择源文件类型。如果只是音频的话，就不用把视频的选择选中了。其它的选项还有输出和描述。
　　再下一步给是听众选择要使用此配置文件支持的目标听众。选择多个听众可生成一个多比特率配置文件。可以从Windows Media服务器传输。选择后打上钩，再点“下一步”进行配置文件编辑。再打开公用流设置对话框。这里是配置文件生成的所有流选择公用设置。音频和脚本比特率会影响剩余可用于视频流的比特率。点“下一步”是单个视频流的配置，一般可以点完成了。如果你有兴趣的话，可以点选每个选项看一下。
　　我们还是继续前面的配置文件选择，选择完成之后，这里有两个选择，一个点“下一步”，一个是“完成”。先看一看，下一步会是什么？下一步是信息显示。键入查看编码内容时将显示的信息，是否提供该信息可选。具体内容可以在Windows Media Player中查看。这里有以下几个选项：标题、作者、版权、分级、描述。
　　7.再点“下一步”，就是设置检查。检查该会话信息。单击“完成”设置会话或单击“上一步”更换设置。
　　8.这时候弹出Windows Media编码器窗口，就可以看到视频在转换了。左边是音频，右边的上半部分就是预览窗口，有3个选项。“缩放”、“显示”、“输出预览”。
　　下半部分可以有3个分窗口。在“常规标签”中，可以看到输入、输出、编码、进度、系统；在显示信息中，可以看到你加入的标题、作者、版权、分级、描述；在连接中，主要是客户机、连接URL。
　　下面的横条，是显示转换的进度，有开始和停止的按钮，你可以随便的停止你要转换的视频文件，如^46020802g^所示。
　　9.完成之后，会出现编码结果的对话框。这里可以查看刚才转换视频文件的一些信息，其中有使用的源路径和名称，输出文件的路径和名称等。在右边的有“关闭”、“查看输出文件”（指输出视频文件的信息）、“新建会话”（可以新建一个会话，开始一个新视频文件的转换，就是重复以上的方法）、“帮助”，如^46020802h^所示。
　　现在我们已经大功告成。可以用Windows Media Player来观看一下我们制作的流文件了。选择你保存的.WMV。双击点开，就可以播放了。如果你觉得很满意，就可以放到网上或者给你朋友看一看了。
　　通过以上的操作，你已经完成了一个VCD文件向WMV流格式文件的转换，在制作过程中如遇到问题请大家来信与我交流。
