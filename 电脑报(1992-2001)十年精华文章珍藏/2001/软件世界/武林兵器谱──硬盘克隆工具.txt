武林兵器谱──硬盘克隆工具

魏俊岭 
2001年 19期

经常使用电脑，最怕的事情莫过于重装系统，不但要翻出一大堆光盘、磁盘，还要搭进去很多时间──Windows太脆弱了！有没有一种能快捷地恢复系统的简单方法？有，答案就是克隆。
　　当我们安装完操作系统、驱动程序和常用的软件后，就可启动硬盘克隆程序，将整个硬盘或硬盘的某个分区做成一个镜像文件备份到另一个硬盘或硬盘分区中。系统损坏时，只须将该镜像文件恢复到目标硬盘或分区中，系统就能完好如初。本文将以同类软件中的佼佼者Norton Ghost为例，介绍此类软件的功能及使用。（^19020201a^）
　　#1克隆精灵──Norton Ghost
　　在详细介绍如何使用Ghost之前，我们不妨首先一睹它的主要特色：1.支持FAT 16/32、NTFS及EXT2（Linux）等文件系统，可运行在Win 9X/Me/NT/2000平台下；2.除硬盘外，还可将镜像文件直接写入Zip等可移动驱动器及CD-R/RW；3.能够编辑镜像文件；4.支持高速并行、USB、TCP/IP等多种联机方式，加速计算机间的磁盘复制；5.强大的网络多机备份功能……本文将以Ghost 6.03版为例详细介绍。
　　●安装：双击文件Nge603trial.exe开始安装Ghost，在安装类型选择界面可以选择客户机（Client）或服务器（Server）两种类型，普通用户可选择默认的Server类型，如果你想体验Ghost的强大功能，不妨采取默认值全部安装，安装后重启计算机即可。
　　●启动和界面：Ghost是一款DOS软件，最好在纯DOS下运行，虽然在Windows下也能运行，但此时有些系统文件处于运行状态，这些文件被克隆后的状态将难以预料。启动Ghost有以下几种方法：一是制作一张带鼠标驱动程序的系统盘，将Ghost.exe拷贝到盘中；二是启动Windows时按F8选择命令行方式，并运行鼠标驱动程序（无法用于Windows Me）；三是从开始菜单执行（这是不被推荐的方法^_^）。Ghost启动后首先会弹出“关于”和“购买信息”窗口，然后出现仿Windows的界面（见图1）（^19020201b^）。
　　●准备工作:如果你要克隆操作系统所在分区，就应该保证克隆前该系统的完好和稳定性，以便以后能够恢复稳定的系统。
　　●克隆分区：启动Ghost，选择Local→Partition，从子菜单中可以看出有关分区的操作共有三种（详见图2）（^19020201c^）。
　　由于我们要将C区克隆为镜像文件，所以选择To Image，接着进入Select local source drive（选择源驱动器，见图3）（^19020201d^）。
　　    由于多数用户只有一块硬盘，所以点击OK直接进入Select source partition（选择源分区，见图4）（^19020201e^），此时显示出所选驱动器的分区情况，选择要克隆的分区，点击OK进入File name to copy image to（命名镜像文件），设置存放路径并命名（见图5）（^19020201f^），点击Open，会弹出Compress image（压缩镜像）窗口，有No（不压缩，占用空间最大，但速度最快）、Fast（一定程度压缩，速度较快）和High（最大压缩率，速度最慢）三种选择，根据需要进行选择后系统会询问是否继续，点击Yes系统便开始克隆，界面上会显示进度条、进程统计、细节等信息（见图6）（^19020201g^），不一会儿，镜像文件就生成了。
　　●恢复分区:当你的操作系统存在问题时，就可以让克隆好的镜像文件为你效力了，在纯DOS下，启动Ghost，选择Local→Partition→From Image，进入File name to load image from（选择镜像文件）界面，找到克隆的文件后，单击它便会立即进入Select source partition from image file（从镜像文件中选择源分区），确定无误后单击OK进入Select local destination drive by（单击选择本地目标驱动器），单击OK进入Select destination partition from drive（从当前驱动器分区中选择目标分区），选择后系统会询问是否继续，单击Yes，过不了多久，原来稳定的系统又回来了。
　　●分区的其它操作:在Local→Partition中还有Partition To Partition（从分区到分区）这种备份方式，用来将某个分区中的内容完全复制到另一个分区中，操作很简单，依次选择源驱动器、源分区、目标驱动器、目标分区，确认无误后点击Yes即可。
　　●硬盘的克隆和恢复:在Local 菜单中还有Disk这一重要选项，子菜单包括To Disk（从硬盘到硬盘）、To Image（从硬盘到镜像文件）和From Image（从镜像文件到硬盘）。这些功能可以实现整个硬盘的克隆和恢复，但前提是你的电脑中必须有两个或两个以上的硬盘，否则无法执行。以硬盘到硬盘为例，选择Local→Disk→To Disk，会显示本地电脑中所有硬盘状况，首先选择源驱动器，然后选择目的驱动器，确认无误后点击Yes，便开始硬盘间的克隆。Ghost会自动对目的驱动器进行分区、格式化、复制等操作，使得目的驱动器最终与源驱动器一模一样。To Image和From Image类似分区间的操作，不再赘述。
　　●利用LPT和NETBIOS进行异地克隆:对于用并口线连接起来的两台电脑，可以用LPT功能进行克隆，但缺点是速度太慢。Netbios应用于使用网卡连接的两台计算机间，用法与LPT相同，速度则快得多，试用版未开放Netbios功能。
　　●Check和Option在Local菜单中还有Check这一选项，下有Image File和Disk两个子菜单，前者用来检查镜像文件的完好性，后者用来扫描硬盘，以确定其是否存在问题。Option可以用来对Ghost进行一些细微的设置，对于普通用户采用默认即可。
　　● 管理镜像文件:Ghost Explorer是Ghost在Windows下的外壳程序，可以调用DOS版本的Ghost对硬盘或分区进行克隆操作，由于克隆前所有操作都是典型的Windows界面，所以使用起来更加直观，此外它可以像使用资源管理器一样管理镜像文件。打开一个镜像文件后，可以非常方便地对其中的文件进行打开、删除、复制、还原等操作，在右键菜单中执行相应选项即可。
　　●注意事项:使用Ghost时，每个操作步骤都要仔细确认，要是弄错了“目标”和“源”，没准硬盘中其它数据也会灰飞烟灭了。进行硬盘间克隆、恢复时，要注意驱动器的容量，不要试图将大硬盘复制到小硬盘中或将镜像文件恢复到比原来小的硬盘区域中。
　　#1克隆快手──Drive Image Pro
　　Drive Image Pro是PQ Magic的姊妹软件，它功能丰富、性能稳定、速度奇快，界面美观易用，而且有中文版本。
　　Drive Image Pro 的主要功能及特色有：1.克隆程序只能运行在纯DOS下，支持FAT16/32、NTFS、HPFS等磁盘格式，支持分区、全盘克隆及恢复；2.采用SmartSector技术，只复制包含数据的扇区，使得克隆速度成倍提高，镜像文件容量明显减少；3.可直接将镜像文件写入CD-R/RW及多种移动存储设备，如Zip等；4.集成PQ Magic的部分功能，能够有效地管理硬盘分区，确保数据无损；5.利用PowerCast Server可以将一个镜像文件同时发送到多个客户机上，实现一对多的克隆方式；6.利用Drive Image File Editor可以轻松管理镜像文件，包括查看内容、复制文件、恢复文件等；7.利用Task Builder可以在网络上实现自动创建和维护客户机，类似Ghost的控制台；8.可以创建应急盘。
　　Drive Image Pro的使用方法也很简单，从开始菜单中启动它后，计算机会自动重启动并引导至DOS下，然后Drive Image Pro自动运行。Drive Image Pro的界面上有创建映像、恢复映像和从磁盘到磁盘三个按钮，由于是中文界面，因而可以方便地按提示操作。
　　#1小 结
　　现今硬盘克隆工具不是很多。实际上我们可以使用这些软件的简化版本，而不必去追求最新的和过多的功能，因为对于普通用户，一个Ghost.exe足矣。
