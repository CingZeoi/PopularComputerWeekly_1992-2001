歌曲的发布

 赵航 
2001年 41期

经过前面的处理，我们的作品就要变为成品了，输出为什么样的格式好呢？现在的声音文件格式多种多样，Cool Edit Pro支持大多数常见的声音文件格式，包括Windows PCM Wave、MP3、RealPlay Media、YAMAHA的VQF等，下面我们就给大家介绍一下作品的制作和发布。
　　#1刻录成CD光盘
　　刻录成CD的过程实际上就是把*.WAV格式的声音文件转化成*.CDA格式文件的过程。先把处理好的声音文件另存为采样频率44100Hz、16bit、Stereo（立体声）的WAV文件，这就是标准CD音质的声音文件。我们推荐大家使用一种简捷、实用的刻录软件WINONCD3.8，大家可以在www.shaoba.com（烧吧）处下载这个软件。以下就以WINONCD3.8为例，讲讲CD光盘刻录的详细过程。
　　启动WINONCD3.8，程序会出现一个欢迎屏（新建方案），让我们选择刻录的类型，分VIDEO CD（也就是VCD影片）、DATA CD（数据光盘）、AUDIO CD（正是我们要的音频CD格式）等几种，选择AUDIO CD，随后进入程序的主界面（^41060104a^1），在文件的列表窗口找到硬盘上我们要刻录的WAV文件，在右面的类似资源管理器的窗口里我们能看到驱动器和文件的详细情况，选择要刻录的WAV文件并把它们用鼠标拖到下侧的窗口中。注意，如果文件的格式不合法，文件名前面的红灯就会亮起来，这个时候你需要看看声音文件的格式选项是否是正确无误。如果出错就需要重新输出正确格式的WAV声音文件。现在普通的CDR光盘的容量分185MB-21分钟和640MB-74分钟这两种，其中640MB-74分钟是标准容量，四分钟左右的一首WAV格式的歌曲都在40MB到50MB左右。这样的话，一张CDR光盘也就能刻录十四首左右，确认要刻录的整张光盘的数据量在允许范围之内，超出容量就会提示错误，然后点击“光盘”按钮进入刻录界面。在这里面的刻录设置选项很简单：
　　1.关闭此盘，还是允许以后继续进行刻录；
　　2.刻录模式，里面有三项
　　（1）仅仅是模拟测试；
　　（2）首先运行模拟测试，模拟测试正确无误后才进行刻录操作（推荐）；
　　（3）不进行模拟测试而直接进行光盘刻录（不推荐使用）；
　　注意：大家有时间、有精力的话还是尽量使用先模拟测试后刻录的方法使刻录准确而且安全，还能避免损坏CDR光盘的情况的发生。还需要注意的地方就是在刻录之前最好是先整理一下你的硬盘碎片，这样刻录出来的CD光盘在大多数CD机上很容易就能读出来。有条件的朋友可以准备一个600MB～800MB的小容量硬盘专门用来刻录CD光盘，免得每次硬盘整理都要等上很长的一段时间。
　　3.在进行刻录的时候，软件设置里注意选择与刻录机相匹配的速度（分2×、4×、8×……等等）。现在的刻录机速度是越来越快了，但刻录机的激光头功率却没有跟上刻录速度的增加，可以想像同功率的激光束在速度越快的刻录机上烧刻CDR的反射层就越浅，刻录好的CD光盘用的时间也就不会很长了，所以在时间允许的情况下最好选低速刻录。设置好了之后按“刻录”按钮开始刻录CD光盘，等候一段时间完成了的时候点对话框的“确定”按钮，刻好的盘就会自动的弹出刻录机。最后在退出程序的时候会有一个提示，是否要保存当前的项目文件？按你自己的需要决定，如果保存设置则以后可很方便地再次刻录与此相同的CD光盘。
　　注意：CDR光盘的选择，现在街面上有几种CDR，分金盘、银盘和彩色盘这么几种，其中金盘的质量最好，银盘就差一些了，彩色盘相比上两种就更差了。大家在购买CDR光盘的时候要注意看看光盘的正面有没有划痕，背面的反射层有没有破损、漏光的地方。如果有破损的地方就会影响光盘刻录。因为CD光盘经常携带，使用频繁，磨损的机会也就越多，为了长期使用，推荐使用金盘刻录，刻录的时候因为金盘的质地比较坚硬，需要用较慢的速度刻录，如果使用普通的彩盘刻录速度就不能太慢，以免彩盘质量不过关而烧穿盘面的反射层。
　　#1转换成MP3格式上传到网站
　　在Cool Edit Pro中进行音频文件的格式转换十分简单方便，我们在菜单栏中点击File→Open，打开需要转换的文件，在工作区内双击声音文件波形选择全部文件，在File菜单中选Save as（另存为..），在出现的对话框里点Options（选项）项，在设置对话框里有很多种压缩方式，为了保证声音文件的质量我们选择采样率160K-high quality（高品质）压缩方式，把WAV文件转换为MP3格式。如果文件太大，为了减小文件的尺寸，我们可以选择128K或64K这样低一些的比特率，这主要看你的音响器材的品质能否重现，其实128K比特率的压缩方式已经是接近于CD（Near CD）的品质，对于绝大多数的爱好者是足够的。文件转换完成之后就可以用FTP上传到网站了。
　　#1制作成流媒体格式在线欣赏
　　随着Internet的发展，我们在网络上的交流也是越来越多，QQ、BBS、查询资料、下载文件、收发Email等等，网络速度也是越来越快了，流式媒体（Streaming Media）也就越来越普及。流式媒体是通过网络传输的音频、视频等多媒体文件，流式媒体在播放前并不是下载整个文件，而是数据流一边传送一边播放，只是在开始时有一些延迟。当流式媒体文件传输到你的计算机时，在播放之前该文件的部分内容（存在缓冲区，缓冲区的大小可以自己根据网络速度的实际情况来设定）已存入内存。现在有很多的网站都提供在线音乐欣赏，这就需要我们把声音文件转换成流格式媒体文件。
　　先给大家介绍两种主流的流媒体格式，现在网络上流行的流格式音频主要这以下两种：微软的WMA（Windows Media Audio）格式和RealNetwork公司的RA格式
　　WMA（Windows Media Audio）格式是来自于微软的重量级选手，音质要强于MP3格式，更远胜于RA格式。它和日本YAMAHA公司开发的VQF格式一样，是以减少数据流量但保持音质的方法来达到比MP3压缩率更高的目的。WMA的压缩率一般都可以达到1:18左右，更深受各大厂商欢迎的是它内置了版权保护技术，可以限制播放时间和播放次数甚至于播放的机器等等。另外，还支持音频流（Stream）技术，适合在网络上在线播放，作为微软抢占网络音乐的开路先锋可以说是技术领先、风头强劲。更方便的是不用像MP3那样需要安装额外的播放器，而Windows ME/2000以上版本的操作系统和Windows Media Player的无缝捆绑让你只要安装了操作系统就可以直接播放WMA音乐。Windows Media Player7.0以上版本增加了直接把CD光盘转换为WMA声音格式的功能，虽然WMA声音格式现在在网络上还不是很流行，但是在微软的大规模推广下已经得到了越来越多站点的承认和大力支持。
　　RA（RealPlayer）是美国RealNetwork公司的一个元老级的产品，是目前网络上比较流行的流式媒体技术。许多Internet的音乐台、视频点播站点都采用该公司产品，在流媒体格式音频中RA格式是音质最差的，可是文件也是最小的。该公司的流式音频格式是.RA格式，特别适合低速网用户在线欣赏，它是根据用户的带宽来自动适应文件传输的速率的，从而保证在用户得到流畅的音质的情况下，尽可能的提高声音的质量。大家可以在网上下载最新版本的Realplayer8.5中文版，免除了英文不好的朋友们使用的后顾之忧。
　　#2（一）转换成RA格式
　　在Cool Edit Pro中已经集成了Realplay的*.RA格式，在FILE（文件）的SAVE AS..（文件另存）选项中大家可以看到*.RA格式，选择它并进入Options（设置）项（^41060104b^2），如果想获得高质量的音频文件就选择最上面的MUSIC，在下面的格式设置里可以看到根据不同的网络速度有不同的采样速率和单、双声道可供选择。这里声音文件素质最高的选项是REAL AUDIO DUAL ISDN（适应双信道ISDN的RA格式音频），大家可以根据网络的实际情况来选择，不流畅的时候大家可以自己设置缓冲区的大小。注意在线音乐的速率设置不要太高，否则网络速度不够快的地方欣赏起来就会断断续续的。
　　#2（二）转换成WMA格式
　　WMA格式的声音文件音质要明显强于RA格式，但是选择同样压缩方式的情况下文件也要比RA格式大。现在的网络速度我们可以以音质为主，一般来说不会有太大的影响。在Cool Edit Pro里不支持输出WMA格式的声音文件，我们要借助于其他的工具软件；WMA流媒体格式声音文件的制作也很方便，我们可以在微软的官方网站http://www.microsoft.com/windows/windowsmedia/en/download/default.asp下载WMA格式的制作利器Windows Media Encoding Utility v8.0，它是目前制作WMA最好的工具，唯一的缺点是Windows Media Audio 8是个用命令行方式来操作的DOS模式下运行的程序，在现在桌面PC上Windows视窗操作系统横行的情况下用起来不是很方便，开一个 DOS窗口进到软件安装目录，不带参数运行它的话会出现帮助信息，标准参数格式是：wm8eutil -input [路径]<输入.wav> -output [路径]<输出.wma> <参数>，参数的格式可以为 -a_setting+空格+<a_setting参数常量>或-profile+空格+<profile参数常量>，生成的文件默认格式是64Kbps、44KHz、Stereo（立体声）（^41060104c^3）。如果你想提高比特率和采样率可以改动相应的参数，具体可以在DOS窗口中键入wm8eutil -audio？ 以查看详细的使用帮助。
　　#2其他转换工具
　　可能有很多的人会觉得DOS界面的程序运行起来很不方便，没关系，牵上爱猫，我们可以到http://www.newhua.com.cn/down/encodeit.exe下载软件EncodeIT，最新版本是0.1，它是通过加入插件来增加可编码的格式，从而使EncodeIT可以利用Windows Media Audio 8的编码工具实现对WMA格式文件的支持。软件运行后左面共有三个标签，分别是SourceFile Selection（选择源文件）、Configuration（程序配置）和DestinationFile Selection（目标文件选择）。在这里主要介绍一下在Configuration（程序配置）中如何设置命令参数，切换到Configuration（程序配置）标签后，在WMA encoder（WMA 编码器）框的右边有个按钮，它是用来设置编码器的路径、生成文件的比特率和后缀名的。在编码器路径的后面按这个格式-input %sourcefile% -output %destfile% -profile a64设置命令参数就可以了。其中64对应的是生成文件的比特率。这时切换到SourceFile Selection标签选好原WAV文件，设置好输出文件的目录，再按下Start（开始）按钮就一切OK了。
　　另外给大家介绍一个使用更加简捷、方便的转换工具wmaencoder7，大家可以在http://www.newhua.com下载这个需要注册的共享软件，运行界面很简单，支持WAV、MP3和OGG等等声音文件格式，首先在File（文件）菜单选择Encode（编码）模式，在上面的Input file（输入文件）对话框中选择要进行转换的声音文件，在下面相同的目录就可以Output WMA file（输出WMA文件），在下面的选项中，Bitrate是比特率，正常的我们选择128Kbit/s，Sampling rate（采样频率）选择标准CD格式的44100Hz，Channels（声道）中选择双声道立体声，Delete source file after convert（完成后删除源文件）按实际需要而定，然后按“Encode”键就开始文件格式转换了，在Status选项中可以看到转换进度。这是一个试用版本，如果不注册的话只能输出源声音文件一半的容量，完成后我们就得到了需要的WMA格式的声音文件，然后我们就可以在网络上把地址的超级链接告诉给你的朋友们去欣赏了。
　　注意：做成在线音乐欣赏形式需要网络服务器端（Web Server）的支持，确定你的站点是否支持流格式媒体文件的播放。
　　希望经过以上我们精心制作，一张展示你个人风采的音乐CD、网络上那迷人的歌喉，都展现在你和你的朋友们的面前。相信大家对音频编辑也会有了更深一步的了解，同时也感叹电脑技术的神奇和便利。让我们赶紧行动起来，亲身感受一下新技术带给我们的享受，并创造出自己梦想中的神奇和体验制作成功后的成就感吧！
