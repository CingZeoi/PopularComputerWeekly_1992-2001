#1    声卡的全双工与半双工
　　    问：如何判断一声卡是全双工还是半双工？是不是在超级解霸下不能唱卡啦OK的声卡就是半双工的？还有intel信息需要时间倒退3-4年以上才能从主流市场上买到。Bonny Guo 810主板集成的声卡是不是都是半双工的？
　　    答：全双工是指能双向工作，即输出的同时能够输入。如果你唱卡啦OK的同时可以录音，你的声卡就是全双工的。半双工声卡几年前就退出市场，目前买不到半双工声卡，intel 810主板集成的声卡一定是全双工的，不过我没试过。
　　#1    学习3D MAX是否对CPU、显卡要求很高？
　　    答：就学习3D MAX而言，不需要配置很高的电脑，笔者曾在配置为VX主板、K7-75 CPU、32MB内存、S3765显示卡的最低档586级电脑上使用过3D MAX 2.5，能够完成绘制3D图形、贴图、着色、配音、动作设计、设置灯光和设置摄像机等工作，并能较好地进行一般的渲染制作，例如渲染制作单幅面3D图形。但在观看3D连续动作场面时有声音延时及脱色现象，渲染制作3D动画速度很慢，例如在渲染制作一幅50帧画面的简单3D动画时，居然用了4个小时，你可以在低档的电脑上学习、制作3D动画，在高档电脑上进行渲染制作成品。
　　#1    可否从硬盘安装Windows
　　    问：我有一台486电脑，没有光驱，请问怎样往硬盘里装Windows 3.2或95？可否将所有的Windows文件装到硬盘里，然后执行安装程序？
　　    答：可以呀！我从来是将Windows的所有安装文件COPY至硬盘的D分区（或其它分区）上，从硬盘上安装Windows。将Windows的所有安装文件COPY至硬盘上，从硬盘上安装Windows好处多多，例如，添加Windows应用程序、安装新硬件需要插入Windows安装光盘时，系统能自动去硬盘上的Windows安装目录中查找所需要的文件，省去了插入Windows安装光盘的麻烦，以后重新安装Windows也很方便。如果你的硬盘足够大，将你需要的所有硬件驱动程序（如显示卡、声卡、MODEM）全部COPY至硬盘的D分区，以后安装这些硬件就不再需要插入相应的驱动程序盘，即可以节省时间，又能延长驱动器的寿命。
　　#1    怎样升级不知名主板的BIOS
　　    问：很多读者来函问道，由于无法知道主板的具体型号，无法升级主板的BIOS。升级BIOS能够提升主板的性能，改善主板的兼容性，使主板适应于各种新型硬件。但在升级BIOS时，需要弄清BIOS是哪个厂家提供的，使用了错误的升级文件，有可能导致电脑无法启动。在国内大多数硬件网站（如http://www.newhardware.com.cn/）上，有一个称为CTBIOS 的实用程序，它能识别出大多数主板，检测出主板使用的芯片组、BIOS ID、BIOS版本号、BIOS生产日期、主板生产商和BIOS下载网址，帮你解决升级主板BIOS这一难题。CTBIOS的使用很简单，只需在资源管理器中双击CTBIOS图标即可运行（也可在DOS下运行）。
　　#1    无法启用Windows me的系统还原功能
　　    问：我的机器是组装的，主板是MSI-6153，CPU是P3 600EB，原来使用的是Windows 98SE，前一星期升级为Windows me，但是无法使用其中的系统还原功能，我只要一启用系统还原，计算机就告诉我系统还原在重启计算机后才能使用，但重启后情况还是一样，同样的Windows me在我办公室的计算机中系统还原使用很正常，在别人的机器上安装同一Windows me也很正常。请问这是什么原因造成的，如何解决？
　　    答：安装Windows me过程中，系统曾给出过提示界面，要求选择是否保留原有系统以备还原，如果你选择了不保留原有系统安装Windows me，则无法使用系统还原功能。
　　#1    为何只能以29帧/秒播放视频文件
　　    问：我的计算机配置为技嘉6VX7-4X主板、赛扬533 CPU、耕宇TNT2 M64 16MB显卡、希捷二代20.4GB硬盘， 128MB内存、索尼24倍速光驱。如此高配置的计算机在用XING 3.3版播放其自带的测试程序时，只有29帧/秒；用Windows 98媒体播放器播放时也一样。请赐教。
　　    答：NTSC制式规定播放速度为30帧/秒；PAL制式规定播放速度为25帧/秒；不管你用多好的设备和如何高档的播放程序，播放NTSC制式的文件或影碟速度均不会超过标准规定的播放速度30帧/秒。
　　#1    总是出现“Windows 在造访注册表时出现错误”对话框
　　    问：单位的“老机”换了一块CPU（6X86MX 300），原来只能工作在233MHz，经设置跳线使工作于266 MHz。但开机进入Windows 98时，总是出现“Windows在造访注册表时出现错误”的对话框，我反复好几次使用Windows SPEED或者SCANREG来恢复注册表才能让电脑正常工作。不知这是为什么，请尽快指点迷津。是不是因为跳线上266这个原因而使得电脑的稳定性下降呢?
　　    答：6X86MX 300标称工作频率为233MHz（3.5×66 MHz）或225MHz（3X75 MHz），使其工作于266 MHz属超频使用。而6X86系列CPU超频性能不好，会出现你所述的现象，建议还原设置，使其正常工作于233MHz。
　　#1    何处有ALG2064显卡驱动程序
　　    问：一台486电脑，Windows 32操作系统，显卡是ALG 2064.在Windows 32的情况下，在我手里的2064驱动程序中，没有SETUP这个文件，我从哪里才能得到该显卡的驱动程序？
　　    答：在电脑报配套光盘中，提供有大量的硬件驱动程序，你要的ALG 2064显卡驱动程序，在97年出版的电脑报第三期中就有。
　　#1    设置主板外频跳线
　　    问：技嘉586tx3主板，如何才能将外频跳至75MHz和83MHz。
　　    答：不同主板跳线的设置方法不尽相同，要设置主板外频跳线，请参见主板说明书进行，如果你没有主板使用说明书，请去“http://www.gigabyte.com.cn/home/technology.htm”下载主板使用说明的文件，不过有些主板使用说明文件未提供非标准外频（如83MHz）跳线的说明，主板是否提供有83MHz的隐藏跳线只好由你自己根据主板上的外频跳线设置情况组合试试。
　　#1    恢复光驱的自动读盘功能
　　    问：我的电脑原先放光盘时会自动读盘，近几天安装了一个优化内存的软件，不知道为什么就不会自动读盘了，请问有什么方法可以解决？
　　    答：估计是优化内存软件减少了光驱的缓存容量，请选择“控制面板/系统”打开系统属性窗口，在“性能”选项卡的“高级设置“栏目”中按“文件系统”按钮，打开“文件系统” 属性窗口，在“CD-ROM”选项卡中，将“追加的高速缓存大小”滑块移至最大，在“最佳访问方式”下拉选择框中选择“四倍速或更高”试试。
　　    如果不能解决问题，请选择“控制面板/系统”打开系统属性窗口，在“设备管理器”选项卡的设备表中选择你的光驱，后按“属性”按钮，打开光驱属性窗口，在“设置”选项卡中的“自动插入通知”前打钩。
　　#1    没有驱动程序如何重新安装Windows 98
　　    问：我想重新安装Windows 98，可是我所有硬件的驱动程序都没有。这还能进行重装吗?如果能的话，我该怎么做呢?是不是重新安装Windows 98以后，原先已经安装的软件也都要重新安装?
　　    答：复盖安装Windows 98不需要重新安装原有硬件的驱动程序，原先已经安装了的软件大多也不需要重新安装。格式化硬盘后安装Windows 98须重新安装所有的硬件驱动程序和应用程序软件。
　　#1    为何CPU工作频率与CMOS设置不符
　　    问：我的电脑是tcl718A，sis620的主板，64MB的内存（pc100），最近用一根128MB PC 100的HY内存条替换原来的内存条后，开机自检，系统显示CPU工作于450MHz，但在BIOS设置界面中，BIOS&CPU FEATURES SETUP一项中的CPU INTERNAL CORE SPEED显示却是350MHz。CPU CORE:BUS FREQ.MULTIPLE是3.5×。不知是什么原因，望在百忙中予以解答。
　　    答：有些主板的BIOS发现系统更新了硬件以后，会自动将CMOS参数自动降低为最安全的设置。对支持CPU工作状态设置的BIOS程序，会自动将CPU工作频率降为主板支持的最低主频CPU。你的系统外频为100 MHz，而100 MHz外频CPU的最低主频为350MHz，因而BIOS设置界面中应当显示350MHz（3.5×100 MHz），属正常现象。哪么为什么开机自检后系统显示CPU工作于450MHz（4.5×100 MHz），因为你的CPU被锁定于4.5×，也属正常现象。
　　#1    为何Windows 2000中无法安装内置猫的驱动程序
　　    问：我的联想射雕56kII型内置猫在Windows 98下跑得很好。但在Windows 2000下只能找到它，无法安装驱动程序。请问怎么办？
　　    答：在Windows 2000中不能使用Windows 98的驱动程序，请安装Windows 2000的驱动程序，也可以试试使用Windows NT的驱动程序。
　　#1   能否清除安装文件后多出的文件
　　    问：专家您好。我的机器为赛扬300A CPU，64兆内存，3.2GB硬盘。Win98第二版操作系统。前几天安装IE5.5浏览器后，认为不甚好用而卸载（从控制面板的“添加\删除”进行），同时清空了Windows\temp中的临时文件，但发现C盘中还是比安装前多出了7兆的内容，估计是安装IE时带进了部分文件，用Windows优化大师、Windows终极加速软件中的垃圾清理功能都不能清理，不知这部分文件能否清除？如何清除？还是只有格式化C盘这一种办法？盼指教。
　　    答：你发现的这个问题是比较常见的。出现这种现象的原因主要有两个：一是由于硬盘在反复地装载和卸载过程中会形成很多碎块，这些碎块可能很小，系统没有把它们登录到未分配空间队列中，因此这部分空间被认为是处于占用状态；二是由于文件在使用或者删除过程中，受到破坏，造成链接丢失造成的。在这种情况下，这些文件即使被删除，其丢失的链接块也无法被系统回收，当然使得可用空间减少。
　　    解决上述问题的最简单办法是用Windows系统自带的三个硬盘整理工具（开始/程序/附件/系统工具/磁盘清理程序、开始/程序/附件/系统工具/磁盘碎片整理程序、开始/程序/附件/系统工具/磁盘扫描程序），一般情况下都可以解决你的问题。不少的硬盘工具软件也能解决上述问题。采用格式化的办法当然也能够解决问题，但是需要备份文件并且重新安装系统，比较麻烦，而且经常格式化对于硬盘的寿命也是不利的。
　　#1    如何设置打开IE时的登录网站
　　    问：专家你好！每次打开IE上网，总是弹出登录微软网站的窗口并要求输入Password。如何解决？
　　    答：来信说“IE默认CPCW.COM，可每次打开IE上网，总是弹出登录微软网站的窗口”。实际上IE设定的默认网页正是微软网站。如果不作重新设置，在每次打开IE之后就会进入微软网站。解决的办法很简单：打开IE，在菜单栏中选择“工具/Internet选项”，在主页对话框中填入http://www.cpcw.com/，确定。在以后上网时只要打开IE即可进入本报的网站了。
　　#1    试用软件到期如何重新安装
　　    问：我的看图软件ACDS32（1208版）到期后，我把它删除了，但重新安装后仍然提示该软件已到期，反复试了几次均不行，请予帮助！谢谢！
　　    答：从网上下载的试用版软件，大多采用限期试用的办法，通常是从下载的时间起算，这个起算时间就在你下载的安装软件包中。试用期结束后，即使你重新安装，其起算时间仍然不变，因此要提示你该软件已经到期，不能使用。解决的办法是重新在网上下载。
　　#1    Ultra ATA/66是怎样提高数据带宽的
　　    问：专家你好，我的硬盘支持Ultra ATA/66，有一个问题一直没有搞清楚：Ultra ATA/33硬盘接口是通过在时钟脉冲的上升沿和下降沿都传送数据把数据带宽提高了一倍的，而Ultra ATA/66又是通过什么办法来提高数据带宽的，我看了不少报刊都没有说明这个问题。请大师指教。
　　    答：这个问题问得好。的确，在采用选通脉冲双沿传送数据之后，要进一步提高带宽就要想其它办法了。Ultra ATA/66把突发数据传输率提高了一倍，达到66MB/s的奥妙仍然是因为它采用了80芯线电缆并通过改进驱动器内部数据流管理来实现的。在以前的接口规范中，驱动控制器读一个信号之前，它必须先“看”一下数据线上的电平状态（表示1或者0的高低状态）是否发生了改变。因此，在上一次数据突发传送之后，下一次突发传送之前，控制器不能过早地检查数据线的状态，否则数据线的状态可能还没有来得及转换而仍然保持在原态。也就是说，控制器必须等待数据线稳定之后才能确定它的当前状态，这个等待周期称为“调整时间”。由于在Ultra ATA/66电缆中增加了40条地线之后，在相当程度上降低了数据线之间的道间干扰，使得数据线能够很快地进入稳定状态。使得这个调整时间减少了一半（实际上就是减少了选通脉冲的宽度）。这就使得Ultra ATA/66接口的突发时间传输率能够提高为Ultra ATA/33的两倍，而对于Ultra ATA规范和DMA协议则无须作任何其它实质上的改变。
　　#1    如何离线阅读邮件和回复邮件
　　    问：专家你好，我刚买了一台电脑，配了一个内猫，我想把我的263上的信直接下到我的Outlook中，然后再脱机浏览和回复，请问可以做到吗？如果可以，请问如何设置（不介意的话，请详细一点）?
　　    答：你的想法完全可以，这是一个基本操作方法。具体做法如下：
　　    1.在Outlook中设置263接收帐号：在主菜单中选择“工具/账号”，在打开的窗口中选择“邮件”标签页；
　　    2.在这个标签页中，选择“添加/邮件”；
　　    3.进入连接向导，填入姓名、电子邮件地址、接收邮件服务器和发送邮件服务器、账号名和密码，这样你就可以在Outlook中按照你的263地址接收邮件了。
　　    4.接收邮件之后，可以在离线的情况下打开Outlook阅读邮件，当然也可以选择“文件/另存为”把邮件保存在指定的位置。在离线状态下，打开Outlook，回复邮件，写完回信之后点击“发送”，就把回复邮件放入了Outlook的发件箱中，以后上线时再发送。
　　#1    如何区分精英P6LX-A+主板1MB和2MB BIOS
　　    问：专家同志，我的主板是精英P6LX-A+，REV:1.1A，现准备升级其BIOS，动手之前仍有一些疑问，请专家指点.
　　    1.我上网查阅了有关资料，我理解为该型号主板其FLASH ROM （PCB）有1MB 和 2MB的区别，如果是这样，该怎么确定我的主板是1MB 还是2MB ROM.
　　    2.现在主板商公布的BIOS版本是V3.3，我是否要从低版本一级一级往上升。
　　    答：你的主板是1.1A版本，其BIOS为2M bit，其BIOS版本是3.1或者3.2。1.0版和1.1版的主板则为1M bit BIOS，其BIOS版本为2.0到3.0A。BIOS的升级不必从低版本逐级上升，可以直接上升为最新版本。
　　#1    WORD能够保留备份吗？
　　    问：WORD和EXCEL有备份文件吗?我在用EXCEL制表时死机了，我能恢复我做的表格吗?请求指点，多谢!
　　    答：WORD可以有备份文件，但必须要预先设置。在WORD中设置保存备份文件的方法是：打开“工具/选项”窗口，选择“保存”标签页，再选择“保留备份”选项。这样你在保存文件时就可以得到一个DOC文件和一个备份文件，备份文件的文件名是“Backup of<原文件名>.wbk”。EXCEL则没有备份文件，只能人工作一个备份。即使不使用备份文件，只要文件被保存过，死机后重新启动也可以从保存的文件中调出原有的内容。
　　#1    如何安装DOS 622
　　    问：Expert，你好！我有一台老机，配置为：191MB硬盘，16MB内存，CPU的主频为66MHz。想装上DOS6.22 ，可是我只有DOS6.22的程序文件，不知如何下手，请专家指教。
　　    答：给你的机器安装DOS6.22的操作过程如下：
　　    1. 找一台安装有DOS6.22的机器，启动并进入DOS6.22；
　　    2. 在该机器中插入一张经格式化过的软盘；
　　    3. 键入命令“SYS a:”（这条命令的作用是把DOS系统文件拷贝到A盘中，应该注意的是SYS是一条外部命令，如果不在当前目录下，则必须在命令前面加上路径），然后把SYS命令也拷贝到这张软盘上；
　　    4. 把这张具有了DOS系统的软盘插入到你自己的机器中，上电从软盘启动机器；
　　    5. 再键入命令“SYS c:”，这样就把DOS系统装入硬盘中了；
　　    6. 最后，把你原有的DOS6.22外部文件全部拷贝到你机器的硬盘中即可。
