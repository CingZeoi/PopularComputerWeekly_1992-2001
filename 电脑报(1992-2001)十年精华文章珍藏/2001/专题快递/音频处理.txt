音频处理

赵航  
2001年 41期

演唱和伴奏声音我们都录制好了之后，就要进入我们的核心部分——音频效果处理了，Cool Edit Pro中给我们提供了强大的音效处理、编辑功能，可以对声音素材进行频谱分析、电平调整、去除噪音以及淡入淡出效果等等。对于人声，我们应该做的基础工作就是先进行前期的电平和音调调整，保证声音素材的电平和音调在合适的范围以内，不被削波，不跑调，然后才能进行去除噪声处理，删掉背景噪声和电噪声，去除噪声后还要进行动态压限处理，保证整个声音文件的动态范围都在合适的区域以内。最后是加入混响，使我们的声音通透圆润、音域饱满。
　　#11.电平调整
　　点击“File（文件）→Open（打开）”要编辑的文件，我们就可以在Cool Edit Pro的工作区里看到整个声音文件的波形，如果想详细地观看某一部分，我们可以选择下面的放大工具。
　　如果声音文件的电平过高，文件中超过零电平的部分就会被切割而造成声音文件的饱和失真，使声音听起来混浊、不清晰，那么我们怎么样来降低声音文件的电平呢？我们可以在工作区内双击选择声音文件的波形，选择全部文件，然后选择菜单栏的“Transform（转换）→Amplitude（振幅）→Amplify（增强）”选项（^41060103a^1），在“Constant amplification”（常数扩大）中调整前后声音文件的百分比就可以提高或是降低声音文件的电平，记住以总体不超过零电平为准。同时观看右面的振幅显示，保持平均振幅在25000左右为宜。
　　#12.音调调整
　　我们在调整的过程中要经常听听伴奏音乐和演唱声音的差距，这里有一个问题需要大家注意，就是音调的问题。我们在歌曲选择的时候应该尽量要选自己音域范围内的歌曲，演唱起来也不觉得吃力，也很容易控制歌曲节奏和演唱情绪。有的歌曲音调儿很高，像G大调的曲子，没有经过专业训练根本就唱不上去，可是歌曲很好听，歌词很有含意，就需要唱这个才能表达对MM们的心意，怎么办呢？
　　不用着急，在Cool Edit Pro里面集成了一个很方便、实用的升降调功能，这样即使歌曲很有难度我们也不用害怕唱不上去了。可以先用低一些的音调演唱，然后回到Cool Edit Pro里再把伴奏音乐和演唱声音调整成一个音调。操作起来也很简捷，首先我们选择“Transform（转换）”项中最下面的“Time/Pitch（时间/定位）→Stretch（时间延长）→Gliding Stretch（平滑延长）”，如果我们要想改变音调，而声音文件的总长度并不跟着改变的话，就需要在“Stretching Mode”（延长模式）中选择“Pitch Shift”（保持音乐节拍不变），而不能选择“Time Stretch”（时间延长）。如果选择时间延长选项，歌曲的音调是变化了，可是歌曲的总时间长度也随着变化了，会造成演唱与伴奏不匹配。注意中间的Precision（转换精度）要选择高精度（High Precision），虽然编辑起来速度是慢了，但是我们可以保证声音文件的质量。

　　#13.降噪处理
　　因为现在出版的伴奏光盘基本上都是在专业的录音棚里使用多轨机正规录制的，这样，在整个制作过程中噪声最大的部分就是我们自己录制的卡拉OK演唱部分，下面我们就给大家讲讲在Cool Edit Pro中怎么样仔细调整以减少噪声成分，还我们一个纯净的演唱声音。
　　（1） 去除“咔咔”声
　　在录音的过程中发现有“咔咔”的噪声，这种情况很好处理，Cool Edit Pro支持高达千分之一秒的编辑精度，有这样的噪声在声音文件的波形上就可以看到很整齐的噪音方波波形，我们可以先用鼠标选择噪音大致的位置，然后用放大工具（左下角有加号的按钮），仔细观看噪声部位声音文件的波形，找到准确的位置并用鼠标选择这部分，点击“Transform”（转换）项中的“Silence”（静默）选项，使噪音这部分静音，波形是一条直线。
　　如果只有一个声道有噪音，我们就需要利用人耳的听觉掩蔽效应，摘取其他声道相同的或近似部分的声音文件去替换它。如果发现噪音很多或是很难处理，那就只能重新录制！
　　（2） 去除喘气、换气声
　　接下来要处理演唱中的喘气声或是换气声。可能你在演唱的时候还不觉得有什么，可是在编辑的时候用监听音箱一听，这些噪音就很明显了，我们也需要对它们进行静默处理。我们先用放大工具在演唱声音里找到喘气、换气声这部分，仔细听听到底是在什么地方发声，注意不要把发声的尾音也计算在内，然后用鼠标选择这部分，点击“Transform”（转换）项中的“Silence”（静默）选项。
　　（3） 去除背景噪声
　　至于背景的电噪声就有些难度了，在Cool Edit Pro菜单栏中，选择“Transform”项中的“Noise Reduction”（减少噪音）选项。首先我们用鼠标选择声音文件开头录制的那段空白区域，这就是我们用来采样的一段纯噪声区域，按“Get profile from selection”（得到噪声样本）分析背景噪音的频率特性，分析结束后按“OK”按钮，这里需要注意的地方是采集背景噪音样本时应尽量采取声音波形振幅最小、最平直的一部分噪音，因为这部分包含着最基本的噪音要素，音乐部分很少，这样有利于编辑的准确性。得到噪音的数据以后就以它为样本，回到Cool Edit Pro的工作区里选择全部的声音文件，重新进入到“Noise Reduction”（减少噪音）选项，直接使用采集的背景噪音数据，按“OK”按钮，在整个声音文件中删除和采得的样品一致的部分。在这里面有一个选择项需要注意，那就是左下角的“Remove Noise”（删除噪声）项，在拉动滑动调节杆调节“Noise reduction level”（噪音减少水平）选项的时候，注意这个数值不能太高，如果鼠标调节的精确程度不高，我们也可以在后面的数值框里手工添上相应的数值，一般来说数值应限制在20以内，要不然Cool Edit Pro会不分青红皂白地把演唱成分也当成噪音删除的，那样声音文件中的成分损失过大，应该多试听几次感觉声音的变化。
　　我们再来看“Noise reduction”（噪音减少）选项中的第二项“Hiss reduction”（嘶嘶声删除），在Cool Edit Pro中为我们提供了对这种连续、平均而且有规律的DC噪声去除功能。首先在这里我们可以去掉有规律的连续噪声，同样的我们先点击“Get noise floor”分析噪声，这个功能叫“噪声门”，Cool Edit Pro在屏幕上会给我们一个返回报告，我们可以理解为一个值在连续变化的开关。给大家举个例子，在安静的房间里我们可以听得到别人的窃窃私语，但是如果这个时候门外呼啸而过一辆大客车我们就再也听不到别人的小声说话了。同样的问题，这个功能就是当声音的声压很低的时候（有背景噪声的时候）我们把门关上，禁止声音通过，我们得到的是静默声（无噪声状态）；当音乐响起来的时候，把门打开让声音通过。虽然音乐和噪声一起过去了，但是音乐声的声压很高，而背景噪声很小，利用人耳的掩蔽效应，对噪音也就没有多大的感觉了。我们可以很方便地用鼠标拖动那些节点来设置噪声门在不同位置的分贝值，需要注意的地方是右侧那些选项，上两项“Prenision factor”（精密因素）和“Transition width”（转换dB值）直接影响着我们的听感，数值设置太低声音听起来就会生硬、不顺耳。在下面的“Spectral Decay rate”（频谱衰减比率）采用默认值的65%就可以，也不能过高或过低，这选项对声音文件的影响很大，设置过低声音听起来干巴巴的，设置过高则声音显得拖尾严重，前面的演唱还没有结束而后面新的演唱部分又开始了，声音就会混浊不清楚，“Reduce hiss by”（减少噪音分贝）和人耳的听觉习惯有相当大的关系，很难说多大的数值是合适的，在这里需要一边试听一边调整。
　　接下来我们看 “Noise Reduction”（减少噪音）中的第三项“Click/POP Eliminator”（噪声消除），在这里Cool Edit Pro为我们提供了分析功能，左上角的图形视图中的绿色曲线代表查找阈值，红色曲线代表排除阈值，当你按下“Auto Find All Levels”（自动查找所有电平，这个选项速度比较慢但分析全面）后曲线的形状就会根据当前波形的情况改变。点击“Auto Find All Levels”，程序就开始自动分析声音文件。我们在“Presets”（预置）项中就可以看到“Constant hiss and crackle”（连续的嘶嘶声和爆破声）（^41060103b^2），再点击左面的“Find threshold levels only”（仅仅寻找开始电平值），Cool Edit Pro经过对声音文件的分析，在屏幕中间的参数框里返回了声音文件连续噪声的数值，按OK就可以消除了。
　　#14.均衡处理
　　因为每个人的声音不同，可能需要把高音或者低音加强一些才显得好听，所以我们也需要做一些均衡处理。选择“Transform”（转换）项中的“Filters”（过滤器）进入到“Graphic Equalizer”（图示均衡器）中，这里有十波段、二十波段和三十波段几种显示方法，简单的调节用十波段就可以了，要是想更细微地调整就用三十波段观看整个声音文件的频谱分布。如果你熟悉音乐调整，那么在这里试听并仔细地进行细微部分的调整也是一个好主意。我们建议一般的使用者就十波段均衡显示就可以，在十波段均衡器里相邻的频段刚好相差一个八度音，调整很方便。高级用户使用二十或三十波段均衡器（^41060103c^3），在这里二十波段的相邻频段则差半个八度音，分得越细调整的过程差别得就越小，对你的听力也是一个严格的考验，相关的问题可以参考笔者的一篇拙作《听觉的革命——频谱均衡器》。
　　#15.加入混响
　　接下来我们由“Transform→Delay Effects→Reverb”进入Cool Edit Pro程序自身的混响程序（^41060103d^4），从上至下的参数分别是：
　　Total Reverb Length：总的混响时间长度，如果过短则混响效果不明显，过长又会使声音混浊不清。
　　Attack Time：起始时间，即混响声从开始到达到它的最大振幅所需要的时间。
　　High Frequency Absorbtion Time：高频吸收时间，有点类似人声处理上经常用到的高频滚降混响效果。
　　Perception：感觉，调节到滑块左边表示Smooth（平滑，听不出一次一次的回声），右边表示Echoey（有明显的回声），总的来说平滑一些较好，但过分平滑声音会不自然。
　　Original Signal：原始声源比例，也就是直达声比例，一般100%就可以了，混响之后声音大小基本不变。
　　Reverb：混响声比例，一定要注意适量，合适地加入一些声音会很动听（这也是我们的目的），但是加入过量的话又会使声音失真。
　　这些效果设置总的来说要多实践，多听才能得到最好的效果，使整个音乐显得饱满，听感好，这样才是一个成功的作品。当然，程序本身也有一些预设值（右边窗口的Presets栏目），如果觉得对你合适，可以直接调用。
　　以上我们完成了演唱的录制和用Cool Edit Pro程序自身的功能对声音文件进行修饰及去噪声处理，听听自己的声音怎么样？还不错吧。不过要想达到更专业的效果，还得看一些专业插件。
　　#21.基础知识
　　我们这里要用到的主要就是动态限制器和专业的混响器，可能这样说有的朋友还不太明白，我们就先来谈谈为什么要进行动态处理。
　　动态处理和频谱调整可能很多朋友还很模糊，所谓的动态限制器就是在你演唱声音小的时候进行提升（增益）以适应伴奏音乐，而当你的演唱声音太大的时候进行衰减，免得影响整体的均衡。流行歌曲相信大家都是经常欣赏的吧，我们先给大家介绍一下流行歌曲和轻音乐的效果处理情况，相信大家就会明白为什么要进行动态处理。
　　首先让我们来分析一下流行歌曲的处理特点。
　　流行歌曲的录制过程中始终保持着伴奏音乐和演唱声音音量的一致，在伴奏声音轻的时候演唱声音也轻，我们能很清晰地听到歌手的演唱。以台湾歌手孟庭伟的《冬季到台北来看雨》为例，先来看看它的波形图（^41060103e^5），这是经过动态处理后的声音波形，我们可以看到整个曲子的前后波形相差不大，虽然歌曲旋律开始的时候音乐很轻柔，歌手演唱的音量也不大，但是我们一样可以很清晰地听到响板和沙锤；在后面歌曲的高潮部分，也不是各种乐器和演唱混成一团，还是可以清楚地分辨出演唱声是演唱声，伴奏音乐是伴奏音乐。在音乐的高潮部分和前面轻柔的部分音量起落相差不大，整首曲子听起来旋律轻柔、舒缓、饱满，很放松，感觉不到累。接下来我们看看另外一个声音文件，录制酒店大厅内演奏的理查德·克莱德曼钢琴曲《梦中的婚礼》，还是先看这个声音文件的波形图（^41060103f^6），可以看到，在这个文件里，声音波形的变化不大，也没有进行动态处理，细节部分要集中精力，要不然就听不清楚了。如果我们也对这个曲子进行了动态处理，处理后波形如图（^41060103g^7）我们可以看到同样内容的一个曲子，频率波形相差很悬殊，处理后在整个声音区域波形都接近于零电平，有细微的部分超出了，但是影响不大，处理后的声音文件听起来效果要好一些，声音饱满而充实。
　　经过比较我们就会发现，我们录制演唱声音文件的时候动态范围都是固定不变的，这样在音乐旋律起落很大的情况下，我们录制的演唱声音时大时小，很容易发生在平滑部分歌手演唱声音小而被伴奏音乐盖住演唱声音，后来歌手高潮起音的时候反而压住伴奏音乐的情况，这样对于以后的多音轨声音混合编辑工作是很不利的。怎样让我们的演唱声音能够从开始到结尾都保持相同的动态范围呢？Cool Edit Pro凭借自己本身的功能完成这样的工作需要很长的时间和烦琐的操作，而且效果还不一定能达到我们的要求，这就需要借助外界插件的力量了。
　　#22.Wave-C4
　　下面我们就给大家介绍著名的音频动态限幅器插件Waves C4，它同时集成了动态压限和均衡处理的功能，界面直观，非常好用。大家可以在公司的官方主页地址http://www.waves.com处下载Waves C4 Multiband Parametric Processor插件的十四天试用版本。下载后双击运行安装程序，在安装的时候注意要把安装目录定位在你的Cool Edit Pro的安装目录下，然后在Cool Edit Pro的“Transform”（转换）项下的“DirectX→Waves→C4”找到它。
　　接下来，我们详细介绍这个功能强大的插件的使用方法。这是一个48bit双精度 EQ和动态处理器的结合物，先看看它的界面，运行后图形界面如图8，中间是一个显示窗口，在这个类似EQ均衡器的窗口上随时随地体现着实时的增益变化，这样在调节声音文件的时候我们可以在窗口上很直观地看到调整后的结果，非常简捷、方便；在Crossover项有四个选择项LOW、MID、HIGH和Q值，分别定义低音区域、中音区域、高音区域和品质因数的数值，我们可以用鼠标拖动图示框屏幕下面的小按钮调节各自区域的范围，当鼠标精度不够或是拖动不准确的时候也可以双击窗口里的数值手工输入。
　　屏幕下面那些滑动调节杆可以调整我们的演唱声音文件的音调；下面来看看滑动调节杆旁边的设置：
　　⑴Threshold：阈值，相当于开关位置，到了设定值的位置效果器才开始进行调整。我们可以手工设置它的大小，单位是分贝，比如我们把阈值设定为Threshold =-15dB，那么只有当信号超过了-15dB的时候才开始压缩，不超过-15dB的时候开关不动作，效果器没有打开，也就不会进行任何的处理。
　　⑵Range：当前频段的最大增益变化值，如果我们把它设置为一个负值，那它的作用就相当于一个压限器，对超出部分进行压缩；如果值为正数，那它就相当一个扩展器，对低于设定值的部分进行提升；如果值等于零，就不会有任何动态处理，也就是说相当于一个静态的EQ使用（当然这个EQ的精度也是很高的）。
　　⑶Gain：当前频段的输出值，我们可以理解为一个输出端前的音量控制器，改变它的值大小就可以提升或是降低某一频率范围的音量。
　　⑷ATTK：意思是Attack time（起始时间），它的作用是压限器工作的时候延时的时间，在规定的时间里发挥到全功率状态。比如你设置成100毫秒，到达限制值压限器并不是马上就满功率工作，是在100毫秒的时间里从不压缩到全功率压缩，这是个不断变化的过程。
　　⑸REL：Release time（释放时间），点击右面的Release按钮我们可以在自动调整和手动调整之间切换。自动调整状态也就是启用ARC功能（Auto-release control technology——自动释放时间控制，也就是自动调节释放时间），这个时间就是压限器从全功率压缩到不压缩的时间，也就是说压限器并不是很生硬地在限制位置马上就停止了，而是经过一段时间才到达完全放松状态，可以使我们以最小的失真获得最大的RMS值。
　　⑹Behavior：（行为控制）这里面有两个选项Opto和Electro，Opto指的是在深度压缩的时候，压缩值越大释放时间就越快，压缩值越小释放时间越慢，Electro和Opto 模式正好相反，它适用于普通程度的压缩情况。（^41060103h^8）
　　⑺Knee：调整部分区域转折点的尖锐程度，这个参数的数值越大就越尖锐，数值越小越平滑，它直接影响着声音的听感，为了声音的圆滑一般来说数值设置小一点为好。
　　在屏幕上方的菜单栏的LOAD（读取预置）项，Waves C4中同样的预置了很多种参数调节（^41060103i^9），我们可能对Waves C4预置选项并不满意，这时候就需要我们自己手工调整声音文件的细节了。
　　点击右面的“Preview”（预览），在图示框的屏幕上我们可以看到一条跳动的线，这就是在显示我们演唱声音的频率范围，我们可以看到在某一频段的具体情况，很直观地看到是不是在进行压缩以及压缩了多少。
　　如果我们某一中音区域低于其他音域，就可以在图示框屏幕的上方用鼠标拖动那几个各种颜色的十字调节按钮，直接设置各个频段的增益情况，来提升或降低这一频段以适应整体旋律（^41060103j^10），在显示屏幕中间位置那个浅紫色的宽带子表示的是控制范围，一旦声音超过这个值就会被等比例压下去。调整的时候注意要细微地拖动屏幕下方的滑动调节杆，速度不能太快，以免听不准确声音的变化。
　　图形界面中，中间的黄色线条实时显示压缩的状况，也就是是否在压缩以及压缩了多少，这条线是时刻变化的；紫色飘带上边缘表示增益，就是说将这个频率的声音提升多少。紫色飘带的下边缘表示能够允许压掉的最大范围，也就是说压缩过程中最多也就压到此处。三条竖线分别将四个频段切割开来。
　　这种音频编辑操作是非常占用系统资源的，机器的反应有可能不是像想象中那么快，以笔者的机器为例，PⅢ866的处理器，PC-133外频的384MB内存，再加上两块IBM-46G（7200转、2M缓存）做成的RAID硬盘（磁盘阵列），就是这种配置的机器在进行声音编辑、修饰操作的时候有时都还会发生停顿，系统的资源占用就可见一斑了。
　　#23.TC Native Reverb
　　TC Native Reverb是一个专业级的混响效果器，公司网址http://www.tcworks.de，据说效果可以和专业硬件设备媲美。由于可以调节的参数比较多，混响的好坏和适合与否大部分是由这些参数决定，下面我们就来详细了解一下各个选项：
　　首先让我们看看图11第1部分的那三个按钮，它们的调整方法是先点击按钮选择，红灯亮表示选中，然后上下拖动VALUE圆形按钮来调整数值。
　　ROM：是一些预设值，笔者感觉没有太大意义，不如直接分别调整其他各值。
　　Decay：衰减时间，设置大一点的话可以使混响时间更长，混响声衰退减慢，但如果值过大会使声音模糊不清。
　　Size：混响房间的尺寸大小，一般说房间较大混响较长，小房间声音则比较清晰、干涩。
　　然后是下面左边的LEVELS（^41060103k^11第2部分），它是指干湿混合度，IN为输入的原声，OUT为加了效果后输出的声音，MIX为加效果的量。
　　IN和OUT没有特别需要不用调整，0dB就是指不改变声音进/出的大小，而MIX则需要自己调整一下。如果太小，会感到混响力度不够；如果太大则会导致声音失真。要根据自己实际听到的效果调整。
　　至于右边的三个图标（图11第3部分）：
　　SHAPE：这里可以调整房间的形状，有三个选项Round、Curved、Square分别为圆形、椭圆形和方形空间，对听音感觉会有一些影响。
　　DIFFUSE：指扩散度，它控制着反射声的“密度”，也就是Echo的密度（混响可以看成一系列Echo的组合，一个混响对应的Echo的数量十分巨大，它们通过不同方位、不同反射距离和不同材质的反射壁反馈回来）。
　　从效果上来说，DIFFUSE值越低，混响声音越清晰，比如在比较小的房间里，墙壁比较平整光滑的声音。不过清晰并不一定意味着好听哦，据笔者使用经验看，往往模糊一些好。
　　调整的时候，看滑动条上面的图就知道“-”号是越来越清晰，“+”号是越来越模糊。
　　COLOR：这个选项很有意思，它把混响度对声音的影响形象地比作“冷”和“暖”，左下为最暖，右上为最冷。笔者比较喜欢暖一点的感觉，会抹掉不少生硬的感觉。
　　PREDELAY：预延迟的时间，以毫秒计算。
　　如果你点击图11第4部分的黑条，还可以弹出（^41060103l^12）的菜单，里边有不少预设值，可以直接调听效果。如果对某种效果比较满意，可以先装载预设值，再根据自己的需要做一些微调。Med Room Dark（中等大小的声音温暖的房间）和Small Hall（小厅）是我比较爱用的。
　　这里我要再次声明，每个人声音不同，听觉感受也不同，我们这里无法给出一个准确的值，只能尽量说明每一个值的含义方便大家调整。当然，如果你使用了本插件，就不必再使用Cool Edit Pro里边自带的混响器了。
　　#1    声音的合成
　　#21.基本操作
　　经过以上漫长而细致的工作，我们的伴奏音乐和演唱声音文件就都准备好了。下面我们进行最后的合成工作，把伴奏音乐和演唱声音文件用Cool Edit Pro里面的多音轨编辑功能混合录制成一个WAV文件，在通俗演唱方法或是卡拉OK声乐录音中，遵循的基本原则是应该保持人的演唱声音和伴奏音乐的响度比例在5:4左右，也就是演唱声音要稍稍大于伴奏音乐；先点击屏幕左上角的多音轨编辑按钮进入多音轨编辑模式（^41060103m^13），在第一音轨上（TRACK 1）单击鼠标右键，在弹出菜单上选择Insert-Wave From File插入一个WAV文件，我们先选择把伴奏音乐插入到Cool Edit Pro中。然后在第二音轨上同样单击鼠标右键插入我们的演唱声音文件（^41060103n^14），用鼠标右键单击演唱声音文件选择“VOLUME”（音量）我们可以用鼠标拖动调节杆或是手工输入数值来确定两路声音的大小，按播放键试听，一直到合适为止。这种调整是对音量的调整，并不会影响到输出电平的高低。
　　#22.声音的同步
　　声音的同步是很重要的，否则就会出现不合拍。接下来看看伴奏音乐和演唱声音能不能同步，导入两路或是多路声音文件后，如果发现不能同步的话，我们可以对伴奏音乐单击右键选择Edit Waveform（WAV编辑）退回到原来的编辑窗口，先播放伴奏音乐并仔细听应该是在什么时候演唱，按下暂停键，用放大工具观看第一句开始的位置并记录屏幕右下角的时间。如果还是听不准什么时候开始的话那就打开摘取伴奏音乐的原文件，就是有演唱有音乐的那种，这样就可以看准确了吧。
　　单击多轨编辑按钮返回到多轨编辑界面，在第二音轨对我们的演唱声音单击右键选择Edit Waveform，回到演唱声音文件的编辑状态，同样的用放大工具观看我们的演唱文件的第一句是在什么时间开始的，然后删掉第一句之前的部分，双击工作区选择全部声音文件并保存。然后，关闭这个修改后的文件，这时候会有一个警告对话框，告诉你多音轨编辑器正在使用这个文件。如果你选择关闭的话，在多音轨编辑器里就会删掉这个文件，不用理会Cool Edit Pro的警告，按YES关闭这个文件。这时候多轨编辑窗口就会关闭返回到普通模式下，我们再次点击多轨编辑按钮重新进入多音轨编辑状态。在第一音轨TRACK 1上点击那个小m锁定第一音轨，然后双击屏幕右下角的“Begin”（开始）处的时间框，我们可以手工输入刚才记录的时间数值。这个时间就是第一句开始演唱的时间，输入后按回车，第二音轨的光标线就会停留在这个时间的位置（^41060103o^15）。我们再重新单击鼠标右键选择“Insert→Wave From File”插入一个我们编辑好的演唱声音文件文件，怎么样？这回我们的伴奏音乐和演唱声音同步了吧。
　　需要注意的地方是：如果只有个别的地方不同步，那么我们先另存这个基本合格的文件，然后在回到单轨编辑模式，重新打开这个文件，这样在Cool Edit Pro里就有两个演唱文件。如果有两个以上的声音文件，在Window（窗口）项里就会出现Waveforms  List（WAV声音文件列表）在那里切换可以分别对这两个声音文件进行编辑。先选择最新打开的声音文件（也就是未被编辑的声音源文件），用放大工具准确地选择不同步位置的单句演唱，并复制下来，然后切换回正在编辑的演唱文件里，在不同步的地方用放大镜仔细观看正确的发声位置，把你复制的单句演唱粘贴在正确的位置上，最后在新粘贴部分的结尾处删掉多余的部分。注意新加入的这部分和下一句的结合处要圆滑过渡，操作后返回到前面试听，整段音乐要连贯，不能有断续的地方。如果效果不理想就要撤销重做，直到满意为止。
　　3.淡入淡出效果
　　很多的歌曲在开始或是结尾的部分采用淡入淡出效果。淡入效果就是声音在开始的时候无声，然后声音以线性方式慢慢增大起来，淡出效果则是在演唱的结尾部分，声音缓缓低下去（也是线性变化过程），直到消失。
　　在电视上的MTV节目中这是经常使用的编辑手法，运用好了有很强的感染力。我们先来看看淡入淡出效果的波形图（^41060103p^16），看到这里很多朋友就会明白，淡入淡出效果就是简单的振幅调整，让声音文件的开头或是结尾的一部分文件的振幅以线性变化增加到平均振幅或减少到零振幅。
　　首先我们启动程序打开被编辑的声音文件，先确定音乐从无声状态到正常振幅的时间，一般的开头淡入效果声音不要太长，控制在六七秒左右为宜，用鼠标选择这部分。在选择的时候看下面的时间栏，确定选择部分的时间长短，然后在“Transform”（转换）项中选择“Amplitude（振幅）→Amplify（增强）”选项，在右面的“Presets”（预置）选项里选择“Fade In”（淡入效果）。按OK按钮，声音文件开始部分的波形就会从零慢慢增大到平均的振幅。完成后试听一下，如果效果不满意可以更改淡入部分的文件长短；至于淡出效果，同样的我们在声音的结尾部分选取合适的长度，然后在右面的“Presets”（预置）选项里选择“Fade Out”（淡出效果）选项。这样在结尾部分声音文件的振幅就会以线性方式降低到零。听一听，演唱的声音慢慢低沉下去，空气中还回荡着演唱的意境，有余音绕梁的感觉，很不错吧。
　　整个声音文件都处理好之后，我们就可以选择“Edit”（编辑菜单）中的“Mix Down（混合）→All Waves（把全部音轨的WAV文件混合成一个WAV文件）”输出成一个新的WAV文件，这就是我们最后的成品了。呵呵，就要大功告成了，赶紧取个名字另存起来吧！
　　注意！我们在声音合成的时候，每一个音轨里（Track）可以是单声道，也可以是双声道，为了保证声音文件的质量，在前期编辑的时候，我们可以都选择双声道立体声，保留高素质的源文件，后来输出的时候选择的余地就大。如果开始声音素材质量不好，最后的质量也就不会高了。在最后文件混合输出、转存的时候不会大幅度影响文件的尺寸。
