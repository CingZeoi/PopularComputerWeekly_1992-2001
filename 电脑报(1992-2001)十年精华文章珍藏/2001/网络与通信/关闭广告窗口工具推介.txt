关闭广告窗口工具推介

周建军 
2001年 11期

    在网上冲浪时，我们经常会遇到Pop-Up窗口，也就是随着主页的打开，会自己弹出的一些广告窗口。这些窗口的打开，不但影响正常网页的下载速度，还会耗费系统资源，严重时甚至会造成系统崩溃、死机。而有的网页虽然没有Pop-Up窗口，但是网页的作者会自作聪明地加上一些背景音乐，而且更多的网站上还存在一些图片格式的广告条，这都会影响网页的下载速度，对于中国的网速来说，这些都是奢侈品。这些讨厌的东西，如果靠单纯修改浏览器的属性效果并不好，而且也会影响到正常网页的浏览。如一般对付Pop-Up窗口，都是设定浏览器关闭Java功能，但对于需要使用Java的网页来说，关闭了这个功能就无法正常访问。那么有没有两全其美的办法呢？当然有，网络上很多大虾们早就编制了一些专门对付这类东西的小程序，有的能关闭Pop-Up窗口，有的能关闭广告条，有的还能屏蔽网上的不良信息，这里给大家介绍四款。
　　#1    一、Pop-Up Stoper
　　    软件大小：373KB
　　    功能：自动禁止Pop-Up窗口的弹出。
　　    软件安装后，运行时会在任务栏托盘里生成一个图标。第一次启动时需要进行设置，第一个页面里显示软件的功能，按“下一步”后可以在“Start Option”里设置是否在启动系统时同时启动（Start Pop-Up stoper When Windows Starts），按“下一步”后设置不允许Pop-Up窗口弹出（Do Not Allow Pop-up windows to appear while surfing with Internet Explorer）页面，接下来设置关闭窗口后的效果，选中“Play a sounds when a Pop-up is prevented”可以在关闭成功后发声，选中“Flash System try icon when a Pop-up is prevented”则会在阻止成功后，图标会由灰色闪烁变为黄色。
　　    双击托盘里的图标可以快速禁止或开启软件，当在图标上出现一个红色带斜杠的禁止符号时，软件停止工作。右击图标选择“Preferences”则可以对我们前面的设置进行修改，选择“Exit”可以退出。
　　#1    二、广告杀手Advertising Killer
　　    软件大小：336KB
　　    功能：自动阻止弹出式窗口的弹出，在窗口弹出后能够马上关闭。
　　    这个软件不用安装，只须直接执行即可。程序执行后，会在任务栏托盘内生成一个小图标，右击该图标选择“Show Form”即可对软件进行设置。在Banned Windows可以对软件不能发现的弹出窗口进行设定（^11040401a^1）。按下“Add”按钮，可以发现所有打开的窗口标题都在这里，我们只须选中想要自动关闭的窗口，选中“Kill windows who starts with this title too”就可以手工把这个窗口加到自动关闭的范围中。如果添加错了，还可以在下面的列表中选中添错的项，按下“Remove”按钮将它清除。按下“Import”或“Export”按钮可以将规则导入或导出。在“Option”里可以选择软件是否随系统同时启动、能够看见本次软件共自动关闭了哪些窗口，还可以把这些记录到文件中。在“Intelligent”里可以自动检测机器上安装了哪些浏览器，并可以把其他类型的浏览器添加到软件作用的范围之内。
　　#1    三、AD Closed
　　    软件大小：1099KB
　　    功能：Ad Closed可以通过设置自动关闭指定标题的Pop-Up窗口，同时还有防止小孩浏览不健康网页或其他我们指定的网站的功能。
　　    第一次启动时，会提示我们输入防止孩子更改设置的密码，密码至少四位，然后进行设置。可以看到，软件共分为两个页面，一个是“Default”，一个是“Children protection keywords”。
　　    在“Default”里，可以设置哪些窗口需要自动关闭，按下“Select”按钮，可以看到当前IE或者网景浏览器都打开了哪些窗口。我们可以双击列表中需要自动关闭的窗口标题，然后按下“Add It”将它添加到自动关闭范围中。如果加错了，可以在列表中将它选中，然后按下“Delete”按钮将它删除。也可以直接在下面的文字框内输入需要关闭的窗口标题，然后按“Add”按钮添加（^11040401b^2）。
　　    在“Children protection keywords”可以设置小孩不能浏览的内容或者网站，我们可以在下面的输入框内输入需要屏蔽的网页中含有的关键字，然后按“Add”进行添加，或者按下“Select”按钮将已经打开的网页标题添加进去。
　　    最下面还有几个设置选项，如果选中“Minimize to try”，则可以在最小化时缩小到任务栏托盘里，如果选中“Children protection”则每次在修改“Children protection keywords”规则或者退出软件时需要键入密码。按下“Set password”可以修改密码等。
　　    设定完毕之后，按下“Hide”按钮将它缩小到任务栏托盘里，软件即可开始工作。
　　#1    四、ADfilter
　　    软件大小：520KB
　　    功能：ADfilter能智能地移除网页上的广告、自动关闭弹出式窗口、禁止背景音乐的播放、过滤成人站点、保护个人隐私。
　　    软件在安装完成之后会马上进入设置。我们可以看到软件界面有五个标签页。
　　    在Main标签页里，软件主要提供了三种保护，选中“Setup adFilter”可以自动关闭Pop-Up窗口和网页上的MIDI音乐。选中“Setup adultFilter”可以保护小孩免受侵害。选中“Setup privacyFilter”可以保护个人隐私（^11040401c^3）。
　　    按下“Setup adFilter”按钮可以对是否显示Pop-Up窗口、播放MIDI音乐、显示网页动画等进行设置。软件默认的设置我们不用改变，其他选项的含意如下：“Remove ALL banner-sized graphics no matter what!”是删除所有广告条格式的图片，“Remove ALL Popup windows no matter what!”是删除所有弹出的广告窗口，“Remove ALL graphics Pointing Off-site!”和“Remove ALL graphics Stored Off-site!”是删除所有非本网站提供的图片（这类图片一般情况下都是广告），这些选项我们可以视自己的需要进行选择。按下“Setup adultFilter”按钮可以选择是否屏蔽含有色情文字（Pornography）、脏话（Bad Language）和暴力（Violence）的网页。我们还可以设置是否屏蔽搜索引擎中的色情链接等。按下“Change the passwords to access adultFilter”可以更改密码，防止别人修改选项。按下“Setup privacyFilter”按钮可以选择是否隐藏浏览过的网址、是否屏蔽Cookies等。如果没有什么特殊的要求，所有的选项可以使用默认设置。
　　    在Update标签页里可以设置软件更新；在Statistics标签页里是一些有趣的统计数字：每下载一个广告需要多长时间、每个小时对我们来说值多少钱、这个软件可为你节约多少金钱；在Exceptions标签页里可以设置在自动屏蔽之内但不需要屏蔽的内容，我们可以在文字框里输入例外的网站，然后按下“Add”将它添加在例外的范围里；在Setup标签页里可以设置软件是否在任务栏托盘里工作和是否随系统同时启动，同时还可以从这里退出或卸载该软件。
　　    需要注意的是，我们首先要把浏览器里的历史记录及临时文件清空，然后软件才能正常工作。这可以在IE中主菜单“工具”下“Internet选项”中进行，在“常规”页里按下“删除文件”和“清除历史记录”两个按钮即可。
　　    如果你感兴趣，不妨一试。这四款软件都可从华军软件园：www.newhua.com下载。
