黄昏与黎明──《轩辕伏魔录》场景攻略

Lastsun 
2001年 8期

    一些攻略要点：碰到需要掏钱的地方，如果有足够的钱，就不要吝啬，一般都能获得不错的物品，但因为物品是随机出现，所以在攻略中不再特别注明。吃药是升级的根本，在经济情况允许的前提下，能吃药就不要使用特技补血，因为吃下去的全部都是经验值。地图上有一些隐藏宝物，限于时间关系，没有全部发掘出，请各位见谅。
　　#1    第一话：纵妖
　　    剧情：墨门大师兄辅子洵奉师命下山剿妖，掌门望其顺利通过这次试练，增长经验以便继承师门。
　　    要点：实在没有太多需要解释的，显然本关是用来给大家熟悉图形系统和战斗系统。值得注意的是，每一个角色都有自己的五行属性，当屏幕左上角外环套住内环文字时，便代表时辰进入该文字所代表的五行时刻，此时拥有该五行属性的角色各方面的属性都将有一定程度的提升。
　　    宝物：未发现。
　　    进展：主角轻松消灭两个敌人后，小师妹赶来，使用炼妖壶收服其余妖魔，却因为操作不慎，误纵被炼妖壶困三百年的狐妖小九。小九占据小师妹身体，打伤主角，随即去灭了墨门。
　　#1    第二话：灭门（丁芸加入）
　　    剧情：主角养伤完毕，回到墨门，发现师门已遭灭门之祸。
　　    要点：本关开始前可以在村庄采购，不要吝啬金钱，多多购买疗伤药品。村中有一口井，可供扔钱。由于是2对3，无任何难度可言，此时玩家可以发现，游戏中最易获得经验值的方法是吃药，恢复多少点气或者命就直接获得多少点经验值。吃药能加经验的游戏，笔者见得不多，而吃药能加如此多经验值的游戏，则前所未有。
　　    宝物：金创药1个、100两。
　　    进展：埋葬师傅与师弟尸骨后返回村庄。（^08070301a^）
　　#1    第三话：葬父
　　    剧情：当两人回到村庄时，已经是一片火海，丁芸之父和众多村民已被盗匪杀害。
　　    要点：此关开始前无法恢复及采购。依旧是2对3，只要注意不被盗匪头头用绝技连续打中，则无大碍。
　　    宝物：还灵丹；消灭盗匪小兵得药草。
　　    进展：由于丁芸无依无靠，即与主角一起踏上寻找小九与炼妖壶的征途。
　　#1    第四话：豪门娇女（江无双加入）
　　    剧情：两人在途中碰到一可爱小女孩打劫，正在斗嘴之际，阴风寨盗匪却准备来个一网打尽。
　　    要点：此关开始前无法恢复及采购。由于本关有两拨敌人，所以要注意保留实力，比较远的宝箱没有太大意义，多杀敌挣得经验为上。第二拨敌人出现后，可以控制江无双进攻。
　　    宝物：绸缎衣，100两、布鞋；消灭僵尸得解毒草。
　　    进展：原来江无双是在此地占山为王，碰到主角后，便决定离开山寨，一起闯荡。
　　#1    第五话：振威镖局
　　    剧情：江无双这样好动的女孩子，一定是惹是生非的好角色，在途中竟然一定要看别人镖车中运的货物，自然战事难免。
　　    要点：此关开始前可恢复及采购。本关稍微有一点难度，因为左震河不遵守“镖师不移动”的话，镖师会出手攻击。小心不要被几个镖师使用特技围攻即可逢凶化吉。由于打倒左震河也是过关条件之一，所以最好先打倒其他镖师获取经验。
　　    宝物：还灵丹、入梦铃、100两；我方桥墩左方有隐藏物品鹿茸膏；打倒左震河得金创药。
　　    进展：一场恶战下来，发现镖车少了一个，众人急忙分头寻找。
　　#1    第六话：江北神偷（裘千里加入）
　　    剧情：原来偷窃物品是江北神偷裘千里，动手在所难免
　　    要点：此关开始前无法恢复及采购。本关有回合限制，虽然裘千里移动很快，但10个回合还是足够让玩家轻松拣完地上的宝箱并打倒裘千里。
　　    宝物：辟邪戒指、100两；打倒裘千里得灵神天药。
　　    进展：在裘千里决定痛改前非与丁芸的苦苦哀求下，左震河决定不带裘千里见官，裘千里与主角等人一并上路前往狐仙居所──仙狐谷。
　　#1    第七话：仙狐谷之一
　　    剧情：4人到达仙狐谷，请求看门狐仙容许他们进入谷中询问小九之事，看门狐仙自然不放几位进谷，言语不合，双方大打出手。
　　    要点：此关要求玩家顶住10个回合不全灭，由于第六回合后方出现野兽，所以打法有二。第一是攻击型打法，所有角色突入中央，以裘千里为中心结“丁”字阵，每回合由丁芸负责恢复命，而其余众人攻击敌人，这样可获得大量经验值。第二是保守型打法，一开始便退入地图右方的峡谷中，这样只会受到移动速度较快的敌人攻击，无太大机会受到法术和野兽的攻击。
　　    宝物：鹿茸膏、止言符；消灭狐仙术士得还灵丹，消灭狐精战士得药草。
　　    进展：10回合已过，狐仙小七出现，战斗停止，众人被带到谷中见老谷主。老谷主与众人交谈后，叫出小九，痛斥其为非作歹。小九假意认错，劫持小七，威逼老谷主将内丹予以交换。众人猝不及防，老谷主只得交出内丹交换小七，小九得内丹后飞奔出谷，裘千里与江无双尾追而去。老谷主在告知主角轩辕剑之位置后便去世了。
　　#1    第七话：仙狐谷之二（冷仲秋与七色加入）
　　    剧情：裘千里与江无双未能追到小九，却在仙狐谷碰到了寻求内丹解救亲人的冷仲秋，冷仲秋错以为两人是狐仙，便抢先出手。
　　    要点：本关开始前可以在仙狐谷采购，同时在谷中有一只名为包打听的狐仙可以告知关于此谷的一些信息，而与另一只名为败家狐的狐仙对话，有机会送300两给她换取一件饰物。战场进入第五回合便出现3对1的局面，补给充分且已经会恢复魔法的主角显然能顺利打倒冷仲秋。
　　    宝物：五鬼戒指、100两。
　　    进展：冷仲秋为救亲人性命，死命寻求狐仙内丹，而七色要为老谷主讨一个公道，双双加入我方阵营，一起讨伐九天（小九的本名）。
　　#1    第八话：天涯孤客（左震河加入）
　　    剧情：左震河的镖师队伍遇到九色投毒暗算，仅剩其一人，欲与九色一决生死。九天逐渐熟悉使用炼妖壶，且因为吞下的老谷主的内丹开始发挥作用，所以放出了五将山的魔道修罗之白虎，以还青龙自由为条件，命白虎阻击左震河，正当左震河以一敌众的时候，主角一行人赶到，双方展开混战。
　　    要点：此关开始前无法恢复及采购。至此游戏开始逐渐进入正题，难度开始展现出来。本关有两大难点，白虎修罗拥有极高的攻击力和让物理攻击失效的法术“天英星”。地图上两个宝箱都会引发5个隐藏的敌人。由于我方7人中，有两人新加入，一人是NPC，所以要不损一人过关较难，建议多存盘，尽量保持每人状态完好。
　　    宝物：鹿茸膏、琥珀项链；打倒白虎修罗得到灵枢玉液。
　　    进展：白虎修罗被迫撤退回五将山，九天为了掩饰自己的野心，必恭必敬地放出被困在炼妖壶中的青龙。五将山妖王朱雀重得昔日手下，声威大振，紧锣密鼓地进行着带领妖魔侵占中原的计划。
　　#1    第九话：妖集
　　    剧情：大家根据老谷主提供的线索，继续前往洛水寻找轩辕剑，途中经过一集市，却见到官兵杀伤百姓。待众人齐上阵打退官兵后，青龙突然出现，将百姓变成妖魔，为了拯救百姓，必须先制服已经变身的妖魔。（^08070301b^）
　　    要点：又是两连战，其间无法补给。好在敌方无强力角色支撑，很快即可结束战斗。
　　    宝物：九节鞭、100两、归神香、还灵丹；消灭官兵将领得金项链，消灭赣巨人得迷烟岚。
　　    进展：青龙通过此役看出主角一行人绝非等闲，通报朱雀知晓，九天说服朱雀让她去取得轩辕剑，以免此剑影响朱雀统一大计。
　　#1    第十话：轩辕剑（梁错加入）
　　    剧情：主角一行人在森林内迷路，误打误撞进入轩辕剑守护人梁错的家，碰巧九天也带青龙、白虎赶到，血战在所难免。好不容易退敌，却中了九天计策，使得轩辕剑仙被炼妖壶所收，主角急忙追赶，却被由玄武率领的第二队敌人围困，痛失轩辕剑。
　　    要点：本关继续两连战，但中途可以在梁错家休息恢复体力。第一场战斗比较简单，唯一需要注意是不能让青龙白虎相互靠近，因为青龙可以恢复，而白虎的天英星可以让物理攻击失效，实在让人头痛。第二场战斗相对困难，因为恶疫鬼会不断重生，虽然它的攻击力有限，但可以让人麻痹，最好配备大量解麻药品，且速战速决。
　　    宝物：罂粟散、200两、天仙酒、银丝战袍；打死玄武得贯灵针。
　　    进展：憨厚的梁错最后发现自己一直看守东西实在不如和众人一并除妖伏魔有趣，决定与众人一起上路，找回轩辕剑。
　　#1    第十一话：独龙岗破贼
　　    剧情：一行人继续赶路，进入清水镇，因为长期有土匪骚扰，镇上的人已经四处逃难而去。在客栈中得知客栈老板的女儿被土匪强抢去，众人义愤填膺，决定为老板夺回女儿。
　　    要点：如果在平原村与书生翔元对过话，则与本镇莫亭对话可获得昂宿七星戒。土匪实在不堪一击，不再浪费版面。
　　    宝物：150两、祥惠之书。
　　    剧情进展：众人接受客栈老板答谢后便一起出发过洛水。
　　#1    第十二话：渡河
　　    剧情：为了阻挠众人过河，一干妖魔杀了船家，并乘船在江中埋伏，人魔不共戴天，只能杀过河去。
　　    要点：本关敌人看上去较多，实则非常分散，各个击破即可。敌方头领冯夷的特技水漫神州威力甚大，需要特别提防。
　　    宝物：止言符、150两、鹿茸膏；打败冯夷得冯夷拓印。
　　    进展：全歼敌人后顺利渡河，一路无话。
　　#1    第十三话：水草马明王
　　    剧情：民不聊生，供奉神明更无从谈起，水草马明王受朱雀言语蛊惑，堕入妖道。为了维护正义，众人决定打醒马明王，一场恶战在马明王庙外展开。
　　    要点：本关所有敌人皆会特技攻击，要避免被敌人围攻，最好分批歼灭。马明王有很强的恢复能力，可以用来练功。在地图一个角上有个宝箱，开启后会出现一个隐藏的地牛精，需要注意。
　　    宝物：归魂香、灵枢玉液、神木精元；打倒马明王得天仙酒。
　　    进展：马明王被打败后，听从七色教导，决定上天庭面玉帝请罪。
　　#1    第十四话：地底墓穴
　　    剧情：在马王庙休息后，众人来到夏后皋墓，好动的江无双前去探墓，狐仙七色感到妖气凝重，大家只得进入墓中寻找江无双。
　　    要点：古墓中应该藏有一些隐藏宝物，玩家可以利用有两个敌人不会移动而搜索每一个位置找寻这些物品；对付敌人时，切忌过于分散。
　　    宝物：300两、活骨灵药，祭坛后方靠墙有隐藏宝物玄雪丝衣；消灭红翼鬼得灵神天药。
　　    进展：大家因为江无双而虚惊一场，教训无双几句便继续上路。
　　#1    第十五话：崤山遇险
　　    剧情：崤山一带有黑店以及变身为西夏军队的魔物，众人披荆斩棘快速行进。
　　    要点：本关为两联战，不过中途会自动恢复一次，所以无大碍。唯一需要注意的是第一拨攻击的第二回合会在客栈门口出现30级的武罗。
　　    宝物：归魂香、鹿茸膏、封穴针、龙纹佩、500两、鹿茸膏。
　　    进展：战斗结束后进入太原城。
　　#1    第十六话：六指伽蓝
　　    剧情：太原城守将惧怕城外妖魔六指伽蓝只得紧闭城门，众人为了赶路，自告奋勇消灭妖魔。
　　    要点：太原城武器铺的招牌后有隐藏宝物五彩石项链。六指伽蓝和小妖实力并不强，各个击破即可。
　　    宝物：鹿茸膏，白玉龙纹佩；消灭六指伽蓝得蟠桃。
　　    进展：距离决战越来越近，丁芸为前景担忧，不知该如何对待九天，是杀还是释？
　　#1    第十七话：现踪
　　    剧情：江无双和众人斗气，结果被野兽围困，众人解围后，进入高塘镇稍作休息便赶往百人冢。九天与青龙、白虎内讧，结果失去炼妖壶，九天换醒百人冢的妖魔抗拒众人，自己却逃之夭夭。
　　    要点：营救江无双时只需要让无双逃遁四个回合等待援军来临即可解围。需要注意的是第四回合敌我双方援军会同时到达。百人冢之役，敌方无强力角色，我方因在高塘镇得到补充，可轻易克敌制胜。
　　    宝物：金创药、归神香、仙炉灰、2000两、天珠武铠；战斗胜利得雷神拓印。
　　    进展：匆匆解决挡路妖魔后，众人追上青龙与白虎。
　　#1    第十八话：剑仙归来
　　    剧情：青龙与白虎两人准备销毁炼妖壶，却不慎被裘千里所盗。为了立刻营救轩辕剑仙，七色冒险进入炼妖壶寻找剑仙，而丁芸与主角两人也因为对炼妖壶施法而脱不了身。余下五人只得，面对强大的青龙与白虎背水一战。
　　    要点：我方会强力恢复的两人皆无法参加战斗，本关全靠药物维持。应集合力量，个个击破，尽量不要给敌方出手攻击的机会。最后歼灭青龙与白虎时，还是不能让两人紧靠在一起，天英星和青龙的龙鳞初返实在厉害。
　　    宝物：白玉龙纹佩、朱红丝带、冰石玉镯；消灭白虎得蜂王露，消灭青龙得天树之叶。
　　    进展：九天在主角救出剑仙后，又偷走炼妖壶并回到五将山。九天在五将山串通玄武，一起反了朱雀。朱雀与青龙、白虎三人一并沦为喽  为九天卖命。
　　#2    第十九话：野心
　　    剧情：四修罗被一同派出阻挡众人，前阵由朱雀统领，后阵由剩余三修罗统领，一时间战云密布。
　　    要点：第一阵敌方过于分散，我方可分两队出击，逐个消灭即可。第二阵较为困难，应该将防守敌人一个个引出集群予以消灭，青龙与白虎两人可采用群殴的方式在一个回合内消灭。玄武除了生命较多，其它一无是处，可轻松围而歼之。
　　    宝物：带火风轮、镜舞、凤羽靴、天难、五行法靴、鬼云；消灭朱雀得苍莽毒牙，消灭青龙得祥慧之书，消灭玄武得神木精元，全歼敌人得轩辕剑。
　　    进展：了结四修罗后，众人在五将山下养精蓄锐准备决战，由于决战在即，应将各种道具调整至最佳状态。
　　    最终话：黎明与黄昏
　　    剧情：九天放出炼妖壶中大量强力魔物，利用修罗宫做最后的顽抗。
　　    要点：我方角色修炼到40级以上的，多半都会究级攻击特级，只需分成两队一起突进即可。由于是最后一关，不要节约道具，尽快杀敌即可。
　　    宝物：踏风奇鞋、跃千重、地赦、delete键、轩辕庖刀、帝牙。
　　    大结局：九天最终仍不服输，暗算丁芸后，毁了内丹，变回原形；七色将内丹赠予冷仲秋解救亲人，决定继续陪伴妹妹；丁芸的魂魄附到主角师妹身上与主角一同重整墨门，其余众人皆有美满结局。
