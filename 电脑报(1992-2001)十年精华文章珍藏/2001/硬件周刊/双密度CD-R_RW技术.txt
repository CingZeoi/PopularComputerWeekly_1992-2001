双密度CD-R/RW技术

刺客 
2001年 17期

海量、安全、快速一直是众多存储设备厂商苦苦追求的技术目标。如今，人们在CD-R/RW领域又一次取得了技术上的重大突破：日本索尼（SONY）公司日前宣布，该公司成功地开发出了容量为1.3GB的双密度CD-R/RW盘片，其中CD-R的型号为CDQ13G1，CD-RW的型号为CDRW13G1。
　　#1    双密度CD-R/RW技术的诞生
　　早在去年下半年，日本索尼公司和荷兰飞利浦电子公司便悄悄酝酿着推出一种全新的大容量光盘格式。据说采用这种格式的光盘存储系统将为用户提供更大的存储空间，这无疑将引发光盘存储领域的一次大革命。换句话说，新一代存储光盘格式的出现将使得未来的光盘存储器市场变得更加扑朔迷离。为了让这种双密度CD-R/RW技术尽快地进入应用阶段，索尼公司在此项技术的前期宣传中花费了不少力气。其日本本部的发言人在去年曾经透露，这种新的双密度CD-R/RW光盘格式有望于2001年上半年确定下来，其相应产品随后也将正式进入量产阶段。采用新一代双密度CD-R/RW技术制造出来的光盘存储容量可以扩展到1.3GB。尽管这一容量仍然不到同属光存储介质的DVD（Digital Versatile Disc，多功能数字光盘）盘片容量的1/2，但作为一种在旧有格式下的更新与提高，用户可以在不淘汰原有介质光盘及设备的基础上拥有更多的容量，其意义恐怕比使用一种新设备更为实在。正如索尼公司的发言人Masami Kato所说，这种新的双密度CD-R/RW光盘格式的诞生，将会给用户们带来多方面的好处，其中最大的好处就在于，这种光盘的存储容量将比现有的光盘更大，而其价格又比即将问世的DVD-RW可刻写光盘更便宜。由于此种系统是基于现有的光盘技术的，对于硬件设备制造商而言，大家只要对原来的生产设备作较少的改造，便可以适应新技术的需要。
　　#1    双密度CD-R/RW技术的特点
　　就在索尼公司和飞利浦公司发布开发消息半年多之后，索尼公司终于正式对外宣布了将很快全面推向市场的双密度CD-R/RW，第一批双密度CD-R/RW盘片采用的是SONY和Philips联合开发的 DDCD标准，容量达到1.3GB，也就是现有普通CD-R/RW盘片容量的两倍（650MB）。为了得到双倍的存储容量，设计人员针对于普通的CD格式的特点，对光盘本身做了一些修改，通过把物理磁道和最小磁道的长度都改小，把数据存储量从650MB增加到了1.3GB。另外，其校验参数设置（CIRC）也有了变化，地址格式（ATIP）做了进一步的扩展。其余的地方都继承了普通CD的基本格式，以便与现有的CD兼容。这种新的光盘格式由于盘片上光道间隔更小，凹坑的长度也更短，因此换来了更大的数据密度，而且纠错方式也有所改进。另外，这种光盘系统还有一个特点就是设计了一个版权控制系统，以防止对光盘数据的非法拷贝。在发布新的双密度CD-R/RW光盘的同时，索尼也推出了配套的刻录机产品──CRX200E。该产品的刻录速度是12倍速，复写速度是8倍速，最大数据读取速度是32倍速，内置8MB缓存，无缓存欠载（Run Under Buffer）防止系统。此外，双密度的CD-R/RW驱动器既可以支持标准的CD-R/RW盘片，也可以正常读取普通CD光盘上的数据，并向下全面兼容其他格式的CD盘片。同时，双密度CD-R/RW驱动器将提供给用户一个界面友好的软件包，这样可以为那些需要快速、轻松管理大容量数据的家庭和商业用户提供良好的、低廉的存储方案。此外，随双密度CD-R/RW驱动器附带有高性能的刻录软件，如Primo CD、abCD和Retrospect Express等，它们均兼容于Windows 95/98/Me /2000和Windows NT 4.0等操作系统。（^17030201e^）
　　目前，双密度CD-R的价格为16元，CD-RW的价格为30元，双密度的CD-R/RW刻录机的价格为2400元。
　　#1    双密度CD-R/RW技术的未来
　　作为CD规范的倡导者，索尼和飞利浦凭借它们在业界举足轻重的地位和强大的影响力，新一代双密度CD-R/RW技术必将得到进一步的推广和普及。目前，已有多家数码设备制造商表示出对这一新技术的浓厚兴趣，索尼决定将继续加快对CD-R/RW技术的普及速度，以便让相应产品迅速进入市场。
　　#1Logitec上市新型液晶显示器
　　日本Logitec公司将于近日上市一款采用Silver Metalic机箱的14.1英寸TFT液晶彩色显示器“LCM-T143AS”。该产品支持0.279mm的点距和最高为1024×768的分辨率，可以显示1677万种颜色，并具有在1024×768分辨率下实现平滑画面的缩放/全屏显示功能。LCM-T143AS外形尺寸为383mm×363mm×165mm，重量为4.5kg，背面内置两个扬声器。（^17030201a^）
　　#1Iomega推出支持多种接口的CD-RW
　　Iomega（艾美加）公司继发布外置4倍速CD-RW刻录机后，日前又推出了一款造型极酷的“血滴子”（Predator）外置刻录机，该产品最大的特点是可支持USB 1.1、USB 2.0、IEEE 1394等不同传输接口，最大刻录速度为8×，最大复写速度为4×，最大数据读取速度为32×。（^17030201b^）
　　#1惠普推出特殊用途服务器
　　日前，日本惠普推出了该公司的首批面向特定用途的服务器产品：Web专用服务器“hp web hosting server appliance”和SSL（Secure Sockets Layer）加密处理专用服务器“hp e-commerce server accelerator”。hp web hosting server appliance是在配备了Red Hat Linux系统的PC服务器上预装了Web服务器软件“Apache”、邮件服务器软件“Sendmail”，其特点是不需要进行特别设定就可以作为Web服务器和邮件服务器来使用。（^17030201c^）另外，它还配备有两套支持10BASE-T/100BASE-TX的LAN接口系统。（^17030201d^）
　　hp e-commerce server accelerator是通过配备专用处理器、用硬件来完成SSL加密/解密处理的产品，它主要连接在Web服务器和路由器之间使用。
　　#1RadeonⅡ即将问世
　　近日，ATI公布了Radoen（镭）系列显示卡的下一代产品──RadeonⅡ的部分细节。这款定位于高端市场的显卡采用了研发代号为R200的图形芯片，采用完全可编程架构和改进型的Hyper-Z技术，可有效地节约显存带宽，并完全支持DirectX 8。此外ATI也将在RadeonⅡ中采用新的图形核心引擎，力求使多边形处理能力达到一个新的高度。
　　#1LaCie推出高速IEEE1394硬盘
　　日前，法国LaCie公司推出了最高数据传输速度可达45MB/s的IEEE1394接口硬盘。该硬盘盘片尺寸为3.5英寸，转速为7200rpm，缓存容量为2MB，平均寻道时间只有8.5ms。由于采用了6针的连接器，因此它可以通过IEEE1394接口获得电源。
　　#1USB新标准即将出台
　　最近，USB-IF（USB Implementers Forum，USB执行者论坛）宣布将在今年年内制订允许外部设备之间相互连接的USB追加标准“USB On-the-go”。这种标准可使外部设备之间不必通过处理器即可交换数据、或者对设备进行控制。换句话说，USB On-the-go标准就是一种为了方便小型嵌入设备，而在USB2.0标准中增加了去除部分主控器（host controller）功能的附属设备标准。
　　#1Diamond Modem重现“江湖”
　　日前，SONICblue发布了由下属Access公司生产的Diamond SupraMax拨号上网Modem。该产品采用USB接口，支持最新的V.92和V.44协议，具备快速连接、呼叫等待、PCM高速上传（48Kbps）等特性，数据压缩可以达到30%-40%的比例，并且支持从Win98/Me/2000等操作系统。
　　#1Power Book G4用512MB SDRAM现身
　　近日，美国Trans Intl.公司推出了用于苹果公司“Power Book G4”电脑的512MB内存模块“SDRAM SO-DIMM”。该内存模块数据宽度为64bit，采用144针脚DIMM封装形式，支持3.3V电压和PC100内存规范。
　　#1Synergy 2000显卡技术细节曝光
　　最近，德国ELSA（艾尔莎）公司决定在5月底推出新的入门级3D专业显示卡──Synergy 2000。该显卡将采用NVIDIA的Quadro2 MXR图形芯片和32MB SDRAM，核心频率/显存频率分别为175MHz/166MHz，每秒可处理2000万个多边形和3.5亿个单纹理像素，具有350MHz RAMDAC和2.7GB/s显存带宽，并支持OpenGL。但Synergy 2000不具备双头显示输出功能。
　　#11.7GHz奔腾4正式上市
　　近日，英特尔正式上市1.7GHz的奔腾4处理器，这也是目前业界主频速度最高的处理器了。该处理器采用0.18微米工艺制造，其工作电压由原来的1.56V-1.70V提高到1.75V，功耗为64W（1.5GHz奔腾4的功耗为54.7W）。1.7GHz奔腾4的每千颗售价仅为352美元，目前已进入量产阶段。
　　#1VIA发布集成网卡功能的南桥芯片
　　近日，VIA发布了南桥芯片VT8233的增强版本VT8233C。该芯片的主要改进在于支持3COM的10/100MBps以太网MAC技术，它能够和支持V-Link技术的任何一款北桥芯片协同工作，包括支持DDR内存的VIA VT8366 KT266和VIA VT8633 Pro266芯片。在其他功能方面，VT8233C和VT8233基本相同。
　　#1三星显示器全新登场
　　日前，韩国三星公司决定推出755DFG、955DF、1200NF、151MP、171MP、170T和240T共7款全新显示器。其中22英寸纯平显示器1200NF采用最新的“NaturalFlat”显像管，带宽为360MHz，最大分辨率可达2048×1536；240T是目前最大的液晶显示器之一，可接收数字、模拟以及S-Video信号，还支持画中画和并置画功能；755DFG为17英寸纯平显示器，是专为家庭量身制造的家用多媒体显示器。
