真“芯”告白──五款CPU信息检测软件大比拼

引火虫 
2001年 9期

     DIYer们最害怕的就是买到Remark过的产品，偏偏CPU经常被可恶的JS打磨，令人真假难辨。因此，大家在购买CPU时除了请一位懂行的朋友“坐镇”之外，再随身携带一些实用的工具软件就更加妥当了。今天笔者向各位推荐几款常用的CPU信息检测软件，相信有了它们几位“老兄”的帮忙，你就再不用怕JS宰人啦。
　　#1    WCPUID
　　#2    一、WCPUID概述
　　    WCPUID是一款日本人编写的免费软件，你可以从华军软件园（http://www.newhua.com）或驱动之家（http://www.mydrivers.com）下载最新版本。下载后得到一个大小为163KB的文件wcpu30b103.exe，它是一个自解压程序，只要直接双击该文件，将它解压缩到相应目录下就可以了（^09030401a^1）。假如你的“E”文水平不高，也可以下载WCPUID的汉化版本，不过汉化版是RAR格式的压缩文件。最新版本的WCPUID支持几乎所有已经面市的CPU（当然，假如你的CPU太古老的话，可能得不到它的支持），包括英特尔的奔腾系列、AMD的K5/K6/K7系列、VIA的Cyrix系列，而且它还支持Rise的mP6微处理器（^09030401b^2）。
　　#2    二、如何看懂WCPUID的信息
　　    WCPUID的操作比较简单，只要在Windows 9x/Me/2000下直接运行wcpuid.exe主程序文件就可以看到如图1所示的“CPU Status”主窗口，该窗口的菜单栏中共有“File”（文件）、“Edit”（编辑）、“View”（查看）、“Help”（帮助）四项，里面包含了WCPUID的所有功能。菜单栏下面是工具栏，上面有“Exit”（退出）、“Save”（保存）、“Copy”（复制）、“Analyze”（分析）、“Refresh”（刷新）、“Standard Feature Flags”（标准特征标记）、“Cache Info”（缓存信息）、“Chipset Info”（芯片组信息）、“System Info”（系统信息）等9个工具按钮，工具栏往下就是WCPUID的信息显示区了。信息显示区的内容可以分为以下四部分：
　　    1.CPU自身信息。即主要显示CPU的型号、封装方式、芯片制造厂商等信息，根据这些信息你可在网上查询更详细的资料。
　　    2.系统时钟信息。这里的数据比较实用，你可以清楚地看到CPU最重要的信息。如“Internal Clock”（内部时钟）、“System Bus”（系统总线）、“System Clock”（系统时钟）、“Multiplier”（倍频）等。
　　    3.缓存信息。在这里，我们可以看到CPU的“L1 I-Cache”（L1指令缓存）、“L1 D-Cache”（L1数据缓存）、“L2 Cache”（二级缓存）、“L2 Speed”（二级缓存速度）等多项重要数据。
　　    4.扩展信息。在这里，你能看到CPU是否支持MMX、SSE、SSE2、3DNow!等多媒体指令集，假如看到某项功能后面是“Supported”，则说明系统具有该项功能，假如是“No Supported”则说明系统不支持该项功能（^09030401c^3）。
　　#2    三、WCPUID的扩展功能
　　    刚才我们看到的仅仅是CPU的一些最基本的信息，其实，通过WCPUID我们还能了解到更详细、更实用的信息。
　　    1.CPU更深层的特性。从“View”菜单下执行“Feature Flg/Standard Feature Flags”命令或直接从工具栏上点击“Standard Feature Flags”按钮，就可以打开“WCPUID/Standard Feature Flags”窗口，从中你能看到CPU支持哪些功能，如内置浮点运算器（FPU）、虚拟模式扩展（VME）、调试扩展（DE）、内置APIC硬件（APIC）、页面属性表（PAT）、ACPI功能（ACPI）等是否得到支持。假如该项后面显示“Supported”说明得到了支持，假如是“No Supported”则说明系统不支持。
　　    2.芯片组信息。WCPUID不仅可以识别你的CPU信息，而且还可以告诉我们许多有关主板芯片组的情况。从“View”菜单下执行“Chipset Info”命令或直接点击工具栏上的“Chipset Info”按钮，可以看到如图2所示的“Chipset Information”窗口，其中“Host Bridge”表示北桥芯片的型号、版本等；“South Bridge”表示南桥芯片的型号（笔者的主板采用的是VIA的“VT82C686A”超级南桥芯片）；“VGE Device（显示设备）”显示了显卡所用芯片的型号。特别值得一提的是，从图2中还可以看到AGP插槽信息，这里显示了AGP插槽的版本以及传输速率（1×、2×、4×等），以及是否支持“Fast Write”（快速写入）、“Side Band Addressing”（边带寻址）等先进特性，同样，若显示为“Supported”说明得到支持，而显示Disabled”则说明系统不支持。
　　    3.缓存信息。从“View”菜单中执行“Cache Info”命令或点击工具栏上的“Cache Info”按钮，就可看到L1、L2缓存的有关资料及缓存校验的信息。
　　    4.系统信息。这里的“系统信息”可不是Windows系统工具中的“系统信息”，从“View”菜单下执行“System Info”命令或直接点击工具栏上最右边的按钮后，程序会让你选择一种存取类型（Select access type），其实无论你选择哪一种存取类型，得到的结果都是一样的，选择后便可看到“System Info”窗口了。这里显示了系统BIOS信息（制造商、版本、发布日期、BIOS文件大小）、主板信息（主要是芯片组信息）（^09030401d^4）。
　　    5.在查看上述信息时，我们可以通过各个窗口内“File”菜单中的“Save”命令或直接点击工具栏上的“Save”按钮将显示的信息保存到一个位图文件或剪贴板中。
　　    假如你的主板是双CPU结构，那么还可以通过工具栏最右侧的下拉列表进行CPU的选择；另外，点击图1窗口最右侧的“>”按钮，将会增加“Date/Time”（日期/时间）、“Chipset”（芯片组的主桥、南桥信息）、“VGA”（VGA信息）、“Memory”（内存，可以看到内存的总容量和内存工作频率）等诸多信息。
　　#1    SysID
　　    SysID可以详细地显示出关于CPU的每一条信息，包括电压、CPU的运行频率、CPU的图片、 L1 L2 Cache以及各项技术资料。目前SysID的最新版本是1.8d版，文件大小为668KB，而且这一版修正了SysID运行在Windows Me下出现的一些问题。从SysID中看到的CPU信息还是颇为详细的（如图3所示），如CPU的型号、生产工艺、产品系列、一、二级缓存大小、时钟频率、FSB频率等都一清二楚，注意到“CPU Serial number”后面的数字“00000673000019DDD5E5CDC2”了吗？这就是PⅢ的序列号，不管你的CPU序列号是否被关闭，SysID都可以读出来；此外，点击“Refresh”按钮是刷新当前信息；点击“CPU Flags”按钮后可以看到你的CPU是否支持MMX、SSE、SSE2、3DNow！等功能或指令集，判断的方法很简单，只要看该项目后面显示的是“Yes”或“No”就行了。不过，笔者更感兴趣的还是点击图3窗口右下角的“CPU Spec”按钮后，立刻能看到CPU的图片、型号、生产日期、原始频率、封装方式、插槽类型、工作电压等技术参数，继续点击“Next”或“Previous”按钮还可以看到其它的CPU图片，当然这里看到的并不仅仅是英特尔公司的CPU系列，也有Cyrix、AMD公司的K系列、最新的钻龙、雷鸟等，当然也可以看到古老的486、586系列CPU图片（^09030401e^5）。
　　#1    Hot CPU Test
　　    这是CPU信息检测工具中为数不多的需要安装的几款软件之一，最新的版本是0.96版，文件大小为569KB。Hot CPU Test的安装过程并不复杂，按照提示一步步做下去就行了。图4就是我们所看到的主程序运行后的界面（由于它的测试时间比较长，笔者并未给出它的截图），它一共可以测试浮点运算、每秒可执行多少条指令等，同时也能够让你检查CPU的各种功能、显示CPU相关信息，如CPU型号与频率、序列号、CPU本身内建的Cache、是否支持MMX、3DNow！技术等，该软件支持各种牌子的大多数CPU。除了测试功能，该软件另内置了各种牌子及系列的CPU信息，从早期的Intel i386DX、Cyrix Cx486SLC/DLC、AMDAm486DX，到最新的Intel Pentium Ⅲ等，而且同样附有相关CPU的图片。
　　#1    WhatCPUIs
　　    与前面的CPU信息检测工具不同，WhatCPUIs在安装时会自动集成到“控制面板”中（出现一个“WhatCPUIs”的图标），因此我们必须在“控制面板”中运行该程序。如图5所示，这里一共有“Processor”、“Advanced”、“System”、“Setup”、“Links”五个标签页，从中看到的信息是非常丰富的，它可以测出你的CPU类型、生产厂家及其网址、是否支持MMX、内部处理器时钟以及CPU其他一些功能支持等，如果你的CPU是Intel的PⅢ处理器，那它还可显示处理器内部序列号（依次选择“Advanced”→“Page2”→“Processor Serial Number”即可）。点击“System”标签你还能看到操作系统和网络的大致信息。而且该软件使用绿、红两种颜色来表示CPU是否具有某项功能，显得别具一格（^09030401f^6）。
　　#1    CPUCool
　　    这是一款来自德国的共享软件，试用版本可以免费使用21天，试用期满后它会提示你进行注册。可能有些朋友只知道CPUCool是一个温度监控软件，其实它也可以同时用来检测CPU，最新的6.04版本的CPUCool是集冷却与优化于一身的工具软件，支持在Win95/98/NT/2000系统下优化CPU，此外，它还能监测、显示主板温度、风扇转速及电压 （支持Intel、SiS、VIA、ALI等厂商的芯片组）。在这一版里还加入了对Athlon处理器的冷却支持。第一次运行时，CPUCool会要求你对主板型号、主板芯片组类型进行确认，在弹出窗口的下拉列表框中你可以找到目前市场上绝大多数主板，不过毕竟现在Intel、AMD、VIA、SiS各款芯片组争奇斗艳，各主板厂商推出新品的速度也越来越快，因此经常会发生可以找到主板厂商的名称，却无法在下拉列表框中发现主板型号，因此笔者建议各位还是选择“I don’t know”选项，由CPUCool自己来识别吧。
　　    CPUCool主界面上显示了当前CPU的主频、系统的外频（如图6所示），除此之外，它还提供了当前CPU和系统总线的实时信息、系统PCI插槽工作时钟频率、系统内存的大小、工作频率、当前内存的实时状态等，应当说功能十分强大。不过，笔者要提醒大家的是，CPUCool还具有硬件优化功能，假如你是一个菜鸟的话，还请手下留情，否则有可能将主板和CPU烧坏。
　　#1    结  论
　　    本文所介绍的工具软件都可以在华军软件园（http://www.newhua.com）或驱动之家（http://www.mydrivers.com）找到并下载，有一些还提供了汉化版。除了文中提到的五款测试工具之外，CPU-Z、MyCPU、Intel Processor Frequency ID Utility和AMD CPU Information Display Utility都是非常有名的CPU测试软件，其中CPU-Z对检测CPU是否被Remark有独到之处；上述检测CPU信息的工具软件，每一款的个头都不大，有些仅有50K左右，一张软盘可以放下好几个。你购机时如果能尽量多带上一些这样的软件，一定可以减少你不少麻烦！假如朋友们对这些工具软件的用法有什么问题，欢迎与我联系，笔者的E-mail是：wzj2001@sina.com。
　　    编后：关于CPU-Z、MyCPU、Intel Processor Frequency ID Utility和AMD CPU Information Display Utility几款软件的用法，大家可分别参阅《电脑报》（“软硬兼施”栏目）2000年第40期中《让假冒CPU现原形──CPU-Z简介》、第47期中《借你一双火眼金睛──PC主要信息软件大赏》及2001年第1期中《CPU的照妖镜──MyCPU》诸文。
