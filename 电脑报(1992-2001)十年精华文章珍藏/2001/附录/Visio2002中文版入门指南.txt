#1    第一篇：了解Visio2002中文版
　　#2　　第一节 Visio能做什么？
　　　　项目经理拿着厚厚的一堆项目计划书和管理流程站在我的面前，哇！密密麻麻的小字，让我头晕脑胀。很明显，经理也看出了我的感受，语重心长的说：“根据人类文化的产生过程来说，人类对图形的接收率比对文字要快上10倍，我也想图形化这些文字，但是我不是个画家，我知道有个Visio软件可以搞定这些内容，这个艰巨的任务就交给你了”。“嘿嘿，您还真找对人了，虽然我从小画乌龟象一条穿着马甲的蛇，但是用Visio可还真是高手”。（^60090109a^）
　　    Visio的用途通俗的说，就是把我们要表达和阐述的内容，结合图形更加形象的描绘出来。Visio是一个图形化管理软件，它可以将管理活动以图形的方式展示出来，便于跟踪管理。然而，传统方式在处理图形化的工作过程中，既费时，又费力，且重复劳动影响工作效率与质量。 Visio不但将绘图过程简化，而且融入了图形化管理的规则。所以，它越来越成为办公管理活动中不可缺少的工具。Visio专门提供给工程技术人员或一般商业人士使用，是一种快速的绘图软件，能够让你轻松做出专业化、高质量的图形或图表。Visio为您的日常工作带来明确、直观地沟通的能力。无论您管理的是员工、项目还是网络，Visio都能很好的解决。
　　    Visio确实能带给我们很多的方便，这里就列出一些好处给读者：
　　    1．你不必成为专业美工或艺术家，就可绘出专业水平的图表。
　　    2．只要学会Windows的图形操作，就会使用Visio。
　　    3．提供专业图形模板，由用户自行绘制图形或图表。
　　    4．不同形式的绘图工具软件，使得学习与操作更方便。
　　#2　　第二节 Visio的使用领域
　　　　Visio长期以来在商业绘图领域占有领先的地位，它丰富的图形库、简便的操作得到了全球60多个国家500多万行业和个人用户的广泛认可。自从微软公司从2000年开始接管这个软件以来，经过一年的研发工作，不仅使得Visio软件在Windows平台下更加方便实用，而且以‘让用户快速轻易的绘制各种图表，进行有效沟通’为核心设计理念，在Visio 2000的基础上增加了很多精彩的新功能，并终于将全中文化的Microsoft Visio呈现给了我们。以前在使用Visio2000的时候还是英文版，而中文版2002的出现无疑是广大国内用户的福音，Visio2002中文版真正让我们做到快速简便的完成各种商业图表的绘制，清晰简洁地表达复杂的信息，使商业流程图表化，管理沟通可视化。
　　    Visio适合很多领域的商业应用，这也许是人类文明是从象形文字起源的缘故。为了能让读者了解到Visio在各个领域中应用的情况，在这里主要简单介绍以下两种领域的应用：适合商务人士的Visio应用和适合专业人士的Visio应用。
　　    A． 商务人士的Office商务图表解决方案：如果您作为一个商务代表，要向客户展示您所要谈论的内容。使用Visio就可以依据商务图表功能帮助您利用足以抓住受众注意力的流程图、组织结构图、办公室布局等演示手段，围绕您的想法展开形象化处理、文档编制及信息共享。而您则可通过对这些熟悉的Office界面与工具加以应用，迅速创建出美观实用的视觉效果。您不需要为了创建一幅动人的图表而去成为一个艺术家。任何人都可以使用Visio对信息及观点进行可视化和共享，并使其具有最佳的效果。只要使用Visio的拖放功能就可以让您轻松自如的制作出希望的内容，同时一些诸如快速建立图表，搜索完美图形等功能可以为您更快的制作出解决方案。
　　    B． 面向商务与技术专家提供的综合性图表制作与绘图功能：Visio产品为IT专家、工程师及开发人员提供了能以简便快捷方式对信息进行创建和共享、且最具综合性的图表制作与绘图工具。
　　    同样我们还可以听听传统行业对该软件的反映。
　　    来自制造行业的用户反馈显示：如何有效地与客户进行沟通，迅速的对瞬息万变的市场做出反映是企业所面临的一个重要课题。在工作过程中用户经常被复杂冗长的进度流程报告占用掉很多阅读时间，甚至还会因表述不清造成理解上的偏差而做出错误的判断或操作。用户非常需要Visio这样的绘图软件绘制网络与目录图、行销图、组织结构图等图表，使工作和沟通能够更加清晰生动，从而提高工作效率和决策的正确性。Microsoft Visio增加了许多有效的新功能，而全中文版本的推出则极大的方便了这一绘图软件在国内的普及和应用。 
　　#2　　第三节：Visio2002功能特性介绍
　　　　（笔者注：在没有拿到Visio2002中文版之前，我一直用的是英文版的Visio，打开Visio2002中文版时，感觉界面特别熟悉，使用起来确实方便了很多，呵呵，我可不是在为微软做广告，没有特殊说明，以下设计的Visio2002皆为Viso2002中文版）
　　　　关于Visio2002：
　　　　全中文简体版Microsoft Visio分为两个版本：标准版—为商业人士设计的直观的图表绘制解决方案；专业版—为商业用户及技术类用户设计的全面图表图纸绘制解决方案。另外，Visio 企业网络工具附件—为IT专业人士设计的高级网络图绘制解决方案。 　
　　　　Microsoft Visio是收购以后微软推出的第一个Visio产品。Microsoft Visio加入了一些新功能，包括增加了对8种语言的支持，基于XML的文件格式等，此外还跟微软其他软件如Office等保持紧密对应关系。
　　　　Visio 现在包含了 Microsoft Office 功能，便于 Visio 与 Office 产品系列中的其他程序一同使用。下面介绍几个Microsoft Visio新增的一些功能特性： 
　　    1．为与 Microsoft Office 兼容而新增的 Visio 功能
　　　　新的 Office XP 的外观：这些菜单包含与 Office XP 应用程序相同的个性化选项。Visio 还包括任务窗格和 Office 搜索功能。
　　    一般的 Office 功能：Visio 提供了一般的 Office 功能，如内置自动更正功能、Office 拼写检查器、键盘快捷方式、打开、保存和另存为对话框。
　　    操作向导帮助：通过在“操作向导”的提问框中键入自然语言问题，您可以迅速找到有关如何使用 Visio 的信息。您会看到一个与每个问题最相关的主题列表。如果看起来每个主题都不相关，您可以单击链接，直接转到 Web 去查找更多的帮助。
　　可靠性和数据恢复工具，自动恢复：自动备份系统可防止工作丢失，并方便您在发生意外情况时恢复图表。
　　    在Office应用程序中实时编辑：当您在 Office 应用程序中编辑 Visio 文档时，可以使用更多的 Visio 工具，它们能让您在 Office 文档中更方便地更新和修改图表。
　　    2．有效的建筑设计图
　　    Microsoft Visio 中改进的建筑设计图解决方案提供了一些自定义功能，使您可以将数据和 CAD 绘图加入到图表中。
　　　　简化的界面：您既可以通过直接调整大小或定位，还可以通过在自定义属性窗口中指定属性来更新形状。
　　　　弯曲墙：使用表示弯曲墙的形状。
　　　　改进的墙壁相交部分的清除工作：Visio 将自动调整墙壁之间相交的部分，减少了创建建筑设计图时您需要执行的清除工作。
　　　　平面布置图数据填充：可以使用存储在 Active Directory、Exchange、Excel 工作表、Microsoft Access 数据库或其他 ODBC 兼容数据源中的数据来填充平面布置图。
　　　　按值配色：根据自定义属性值给形状配色。
　　　　改进的 Autodesk AutoCAD 兼容性：查看由图纸空间支持的所有模型空间视图。
　　    3．简化的工艺工程
　　    Microsoft Visio 简化了处理工艺工程组件信息的方法，并提供了一些选项，使标记组件更加灵活。
　　　　改进的组件属性管理：工艺工程解决方案现今可在自定义属性窗口中显示工艺工程组件信息，而不再需要外部数据库。
　　　　更灵活的标记：在保留自动分配组件标记选项的同时，可以手动标记工艺工程组件并有选择地重新标记这些组件。
　　　　更佳的管道排列：Visio 提供了增强的管道布局控制，支持接合点以及用阀门和其他工艺工程组件拆分或修复管道段。
　　    4．强大的IT绘图功能
　　    网站映射和报告功能具有更多的形状并已得到改进，使 IT 图表的绘制更简单、更完整。Microsoft Exchange Server 2000 形状??使用 Active Directory 解决方案中的 Exchange 模具形状可定义并表达 Microsoft Exchange 2000 规划和配置。
　　　　改进的网站映射：控制网站的搜索和布局。通过在树视图中显示网站元素，列表窗口使浏览变得很简便。
　　　　网站报告：生成有关网站中所有链接（包括已断开的链接）的完整报告。报告工具甚至还可以识别自最后一次绘制图表以来网站的变化。
　　    5．开发人员工具和增强性
　　    使用 Microsoft Visio，对 Microsoft 开发人员工具的互用性和支持将更完善。Visio 还提供了用于创建数据库图表和软件图的生产率增强工具。
　　　　界面生产率：您可以向数据库表或软件类连续添加新的字段，使数据输入更快捷。对话框大小可调整，使您可随意显示或多或少的信息。窗口（输出、代码、数据库属性、表和视图）可随意放置，这样可灵活调整工作空间，且使访问模型不受阻碍。
　　　　数据库建模：除了 DBMS 的完整列表外，Visio 还支持 SQL Server 2000。可在 Visio 中从反向工程向导自动生成完整的数据库图表，而无需从表窗口中拖动表格。
　　　　软件建模：Microsoft Visio 支持 Microsoft Visual Basic 6 和 Microsoft Visual C++ 6。
　　    6．更简便的项目日程安排
　　    Microsoft Visio 项目日程安排解决方案的增强实现了与其他工具的集成，从而提高了生产率。
　　　　与Microsoft Project集成：您可以将 Microsoft Project 文件中的数据和任务导入甘特图和时间线，然后将它们发布或显示。 
　　　　时间线/甘特图的互用性：您可以从时间线中导出信息以创建甘特图，反之亦然。 
　　　　实时编辑任务：无需重新键入任务，您就可以对任务进行更改。
　　　　新的进度栏：由于 Visio 会在导入数据以及布置甘特图时显示进程的状态，这样您就可以更直观地查看工作的进度。
　　　　改进的时间线性能：Visio的时间线解决方案更加快捷并提供了更直观的里程碑方式，这样您就可以更轻松地安排您的计划了。
　　    7．更灵活的组织结构图
　　    Microsoft Visio 让您可以更自由地控制组织结构图的外观，并可更加灵活地处理自定义属性。
　　　　直观的布局方式：将组织结构图形状放到上级形状上时，它们将自动定位；但是添加更多的形状时，它们将保留在最初的位置。 
　　　　同步性增强：如果您对组织结构图文本和自定义属性的同步副本进行更改，更改的方式将更为直观，这让您在更改复制实体格式的同时避免了意外的删除。 
　　　　多个自定义属性：您可以存储和显示的自定义属性不受限制。还可以在图表生成后切换需要显示的属性。
　　    8．增强的搜索能力以及优化的数据库向导
　　　　属性报告向导：提供针对图表的报告，并取代了一些属性报告工具。使用报告格式可将范围广泛的图表信息导出成XML 文件、网页、Visio 表形状、Microsoft Excel 工作表，或是嵌入的 Excel 形状。
　　　　增强的搜索能力：用户可以在自定义属性、形状名和用户定义单元格中搜索。
　　　　简化的数据库连接性：除了分步的数据库向导外，Visio 现在提供一个单独的对话框，通过其中明确的选项，您可以在形状和 ODBC 兼容数据源之间建立连接。
　　　　固定自动隐藏窗口：通过将多个自动隐藏窗口（如扫视和缩放以及绘图资源管理器）固定在模具区域中，可以灵活调整工作空间，且使查看图表不受阻碍。
　　    9．与Web的紧密集成
　　    Microsoft 网站提供了更多的信息，帮助您更有效的使用 Visio。您还可以将 Microsoft Visio 绘图转换为网页。 
　　　　查找形状功能：您可以扩展您的形状库，而无需等到下一次产品发布。查找形状功能提供了在本地和基于 Web 的形状存储功能。 
　　　　另存为网页：您可以将 Visio 图表作为网页进行发布，此网页带有属性视图，可显示形状的自定义属性数据。形状可以显示多个超级链接，以提高浏览能力。 
　　　　自动程序更新：自动更新实用程序可以分析所安装的 Visio 产品，提出更新建议，并使您能够立即下载并安装更新软件。您可以从 Visio 网站链接到该实用程序。 
　　    10．更精确的图表
　　    Microsoft Visio 具有更流畅的线条和文本、更丰富的色彩，以及新的图形导入工具和增强的工具，这些功能使您能够创建外观更为专业的图表。
　　　　更流畅的图形和文本：绘制更流畅的线条和文本。
　　　　更丰富的色彩：每个文档可从多于 1600 万种颜色中进行选择，向图表添加丰富的色彩。?

　　　　扫描仪/数码相机接口：直接从扫描仪或数码相机中导入图像或照片。

　　#2　　第一节：基本知识和简单术语
　　    1. 为什么要费心思绘制图表？
　　　　还是重复这句话，千言不及一幅图。合适的图表可以简明扼要地将您的想法、概念、系统、数据和计划传达给任何受众。使用视觉提示、符号和关系来增进理解。提供普通的形象化的语言，跨越了文化交流和技术交流的障碍。
　　    2. 在Visio中创建图表是多么简单的事情
　　    专门用于创建所需图表的现成的模板和形状。要创建图表，您只需将想要的形状拖到绘图页上即可。您只要熟悉Microsoft Office环境，使用熟悉的工具栏和菜单，就可以迅速地创建图表。为组织中每个人（从基本业务用户到IT、开发和工程领域中的技术用户）准备的工具都包含在一个程序中。组织中的任何人都可以使用Visio产品来创建图表。
　　    3. 创建图表的基本步骤
　　    a） 通过打开模板并向图表添加形状来开始创建图表。
　　    B） 在图表中移动形状并调整形状的大小。
　　    C） 向图表添加文本。
　　    D） 连接图表中的形状。
　　    E） 设置图表中形状的格式。
　　    F） 保存图表以示完成，并向观众演示图表。
　　　　要记住，创建一个你的应用方案其实就是不断的重复着“创建图表的基本步骤”这么一个过程。

　　　　在流程图文件夹中可找到用于创建各类流程图（如审计图、原因和效果图以及工作流程图）的模板和模具。什么人需要使用流程图以及如何使用？通过使用流程图，着重测试新来的应聘人如何处理棘手的任务，经理人员可在面试过程中筛选出胜任的人选。保险公司可以使用流程图来记录风险评估流程。会计师可以使用流程图来描述财政管理、资金管理和财务库存流程。
　　　　物理网络图：（^60090109d^）
　　    物理网络图表示组织的网络，包括电缆线路和物理硬件。
　　　　在网络文件夹中可找到基本网络模板，通过它打开的模具可用来创建各种物理网络图。什么人需要使用物理网络图以及如何使用？设备经理可将物理网络图并入灾难恢复计划、布线图以及包含有关公司资产及其相关成本信息的文档。网络经理可使用物理网络图来显示产品在组织内的使用分布状况。组织中的任何人都可通过参考网络图来找到他们能够连接的设备，例如打印机和复印机。
　　　　组织结构图：（^60090109e^）
　　    组织结构图显示了汇报关系以及雇员分属的部门。
　　　　在组织结构图文件夹中可找到用来创建组织结构图的模板和模具。什么人需要使用组织结构图以及如何使用？人力资源专员可创建组织结构图，并将其张贴到公司的Intranet中。项目经理可使用组织结构图来显示团队结构和任务细分状况，以便生成项目日程安排。经理人员可使用组织结构图来直观地表示如何进行部门重组或估计人员配置的需要。
　　　　提示：你还可以查找 Visio 中自带的图表示例。在文件菜单上指向新建，单击浏览绘图示例，双击一个文件夹，然后打开绘图示例，里面有很多做好的图表事例，非常适合初学者使用，模仿。
　　#2　　第三节：一个创建图表的实例
　　　　还是举流程图的例子吧。现在请跟着我的操作步骤来进行操作。
　　　　具体过程如下：选择一个模板-->移动形状和调整形状的大小-->添加文本-->连接形状-->设置形状格式-->完成和使用图表。
　　    A） 选择一个模板
　　　　打开Visio界面以后选择一个流程模板，如图：（^60090109f^）
　　　　用模板开始创建 Microsoft Visio 图表。模板是一种文件，它打开一个或多个模具，模具中包含创建图表所需的形状。打开模板后，将形状从模具中拖至绘图页，添加到图表中。如果需要其他形状，您可以打开其他模具或使用查找形状功能查找。
　　    B） 移动形状和调整形状的大小
　　　　很少有人在创建图表时能一次性正确地定位各个形状。如果您属于大多数Microsoft Visio用户，您会向图表添加一些形状，在工作时按照需要移动形状并调整它们的大小。当您移动形状并调整它们的大小时，可能还需要放大形状以便操作。
　　    C） 添加文本
　　　　您可以向 Microsoft Visio 图表添加两种类型的文本—独立文本和与形状相关的文本。独立文本，例如标题或带项目符号的列表，是独立的文本块，与任何形状都不相关。与形状相关的文本或在形状中输入的文本会随形状移动。这两种类型的文本的添加方式都很简单，只需单击并输入。移动文本也只需单击并拖动。
　　    D） 连接形状
　　　　流程图、组织结构图、框图、网络图和Web图这些图表都有一个共同的形状，即称为连接线的一维形状。连接线连接了图表中的形状，当您移动一个具有连接的形状时，它仍保持连接状态。例如，当您移动流程图形状时，连接线将随之一起移动。
　　　　连接线还可以代表形状之间的关系。例如，流程图中的连接线代表工序流程。组织结构图中的连接线代表组织层次结构。数据库模型图中的连接线代表数据流。
　　　　其他情况下，连接线不代表关系；它只是用来连接两个形状。例如，您可以将网络设备连接到网络支架上。当您移动网络支架时，所有连接的设备将随之一起移动，无需逐个移动设备。
　　    E） 设置形状格式
　　　　在Microsoft Visio中设置形状的格式十分快捷，只需用鼠标单击即可。先单击形状，然后单击工具更改其属性，例如颜色、图案、线条粗细等等。您可以使用颜色来突出图表中的特定形状，或者用线条图案或线条粗细来表示信息流的特定类型。
　　　　您还可以使用控制手柄更改形状的外观。例如，可以在椅子形状上拖动控制手柄来旋转椅子，在网络形状上拖动控制手柄来添加分支，或者在饼图上拖动控制手柄来添加扇区。
　　    F） 完成和使用图表
　　　　现在您已经差不多掌握了创建任何Visio图表的基本技巧，现在需要了解如何处理已完成的图表。首先当然要保存它，然后可以打印、以其他文件格式保存，或将它包含到Microsoft Word文档中，等等，就像您以前使用Office软件一样，使用这些功能就可以了。
　　
　　#1    第四篇．应用方案实例分析------网站规划与管理
　　　　利用Microsoft Visio专业版内建的“Internet图”解决方案，我们可以： 
　　    * 建立高级、集中式的网站图表，并明白显示其目标、内容及整体架构组织。
　　    * 产生已存在网站的图形组织以便来分析架构及将内容分类。
　　    * 利用网站地图及事件查看器来检查断掉的链接和排错。
　　　　接着我们便一同来探讨有关Visio在设计网站及管理上所能够带来的便利。
　　#2　　第一节：设计全新的网站——网站总体设计
　　　　虽然网站配置的工程都是交由信息人员设计的，然而在配置之前进行的系统需求、网站整体功能规划等内容，却并非全由信息项目人员来进行。借助Visio所提供的网站总体设计模板，非信息专业人士可以通过拖曳页面图形等操作，创建有层次的模组图。同时，与信息项目人员的沟通变得更加简单。
　　　　借助网站总体设计功能，非专业人士也可以轻易把网站所需功能绘制出来。方法如下：
　　    1. 开启Microsoft Visio专业版。
　　　　2. 此时便会出现“选择绘图类型”对话框。如果您已经打开Visio 则由菜单中选择“文件/新建/选择绘图类型/Web图表/网站总体设计”。（^60090109g^）
　　　  3. 自“网站总体设计形状”样版中拖曳“主页”形状到模块图上。（^60090109h^）
　　　　4. 接着再拖曳其他的形状到模块图上，利用 “动态连接线”进行两个形状间的连接工作。
　　    5. 从“标注”模板中拖曳标注形状到模块图上。
　　    6. 用鼠标选择“标注”模板，再由标注形状上的连接点，拖曳到被批注形状上的连接点，即可建立连接。利用各类标注样式可为各个形状加入说明。（^60090109i^）
　　#2    第二节 制作网站图
　　　　Visio强大的工具及设计功能可以为信息管理人员提供相当完整的文件，使他们只需要把注意力集中在数据库和网站管理和维护上，节省了以前花费在文件准备和确认数据正确性等事务上的时间，从而提高了工作效率。使用Visio创建Web图表的优势在于：不但可以提供可视化的网站说明文件及图标，而且可以使网站的管理轻松，沟通与开发更加容易！
　　    现在我们就来看看网站地图的操作步骤：
　　　　1. 启动Visio之后会直接打开“选择绘图类型”对话框。如果您已经启动Visio，亦可由菜单中选择“文件/新建/选择绘图类型”。
　　　　2. 在打开的“选择绘图类型”对话框中，选择类别为“Web图表”。
　　　　3. 再从“Web图表”中选择“网站图”。（^60090109j^）
　　　　4. 选择“网站图”类型后，就会打开“生成站点图”对话框。
　　　　5. 接着，在“地址”区输入（或单击“浏览”按钮），确定产生网站图的网址及位置。（^60090109k^）
　　    6. 为了顺利完成本练习，各位读者可以利用“浏览”方式选择光盘范例路径中“INFO2”目录的“default.htm”文件，也可以直接选择您想要分析的网站网址。
　　    7. 当用户设计网站地图时，最好能够限制搜寻条件以便提高性能。例如，只显示网站里一部分的超级链接形状，因此我们接着选择“设置”。
　　    8. 此时便会打开“网站图设置”对话框。
　　    9. 在“布局”选项卡中的“搜索”，选择“一级”以便加速我们产生网站图的速度。
　　    10. 另外，我们也可以单击“修改布局”按钮，修改网站地图的形状排列方式。（^60090109l^）
　　　　11. 在“排放形状”对话框中可以设置形状排列的样式、方向以及深度等。
　　　　12. 单击“间距”按钮，则可以设置形状之间的空格、版面布局和形状间的间距等。（^60090109m^）
　　　　13. 在这个画面中您可以设置形状之间的空格大小、连接线到连接线的距离、平均形状大小以及连接线到形状的距离，以mm为单位。
　　　　14. 设置完成后，单击“确定”按钮直到回到“网站图设置”对话框中。（^60090109n^）
　　    15. 接着单击“扩展”选项卡，则可以设置在绘制网站地图时要包含哪些扩展名的文件类型，默认情况下包含ASP、Archive（压缩文件）、Audio（声音）、Document（文件）、FrontPage、Generic（标准）、Graphic（图形）、HTML以及最新的XML等十余种名称及对应形状。
　　　　16. 在此，我们将扩展名 “ASP”、“Document”、“FrontPage”、“HTML”、以及“XML”勾选，其余的扩展名则不勾选。
　　　　17. 如果这些还不够，例如我们想要Visio能绘制PHP的文件格式，可以单击“添加”按钮增加其他类型。（^60090109o^）
　　    18. 接着，可以在“名称”区中输入“php”，并且在“形状”中选择一个代表的图标．单击“新建”按钮以便添加这个新的类型到扩展列表中。（^60090109p^）
　　    19. 在“协议”选项卡中我们可以设置要包含哪些协议，像FTP、文件、Gopher、Mailto、Telnet等。（^60090109q^）
　　    20. 而在“属性”选项卡中则可以设置在网页中程序代码有那些内容要获取进来。（^60090109r^）
　　    21. 最后在“高级”选项卡中您可以设置搜寻文件的条件，是全部文件、或是指定域、或是指定子树状结构，以及哪些链接要包含进来，默认是包含与搜索条件之外的文件的链接。
　　    22. 单击“确定”按钮开始产生网站图。（^60090109s^）
　　    23. 之后便会产生一层的网站图。（^60090109t^）
　　    24. 放大后我们还可以发现不同的文件类型会用不同的图标来表示。
　　    25. 您可能会注意到有些图标下方有个加号，例如：“Products.htm”，表示此网页有超级链接。选择“Products.htm”图标后单击鼠标右键，选择快捷菜单中的“展开Products.htm”。（^60090109u^）
　　　　26. 即可展开下一层。（^60090109v^）
　　    27. 我们也可以选择“收合Products.htm” 将下一层的网页隐藏起来。如果画面显得过于拥挤，您还可以选择Product.htm后单击鼠标右键，再选择快捷菜单中的“创建子页”。（^60090109w^）
　　    28. Visio会自动产生另外一位新的工作页面，同时将Products.htm次层的组件展开建立在此绘图页面中。（^60090109x^）
　　    29. 在新的工作页面中我们可以注意到在Products.htm上方有个“换页连接线”图标。选择后单击鼠标右键，发现Visio已帮我们自动建立好相对应的超链接，同时放在快捷菜单中。（^60090109y^）
　　    30. 有些文件或是程序代码等的超级链接断掉了，或是移动原来的位置，会使得图标上呈现红色大叉，表示此处无法建立找到链接的内容。这时，我们可以试着选择呈现红色大叉的文件图标，单击鼠标右键，选择快捷菜单中的：“配置超链接”。
　　    31. 此时便会出现“超链接”对话框。（^60090109z^）
　　    32. 依照流程图中插入超级链接的方式便可重新设置超级链接。
　　    33. 选择“Web/窗口/列表窗口”。
　　    34. 在屏幕上会出现列表窗口，里面会列出所有已显示在画面上组件的列表。
　　    35. 选择“Web/窗口/过滤器窗口”。
　　    36. 在屏幕上会出现过滤器窗口，在里面会将显示在画面上的组件作个分类，同时在每一个类别中详细列出有那些，若您不希望查看出现图标，可以将打勾的取消即会重新排列形状。（^60090109a1^）
　　    37. 同时我们还可以查看不同的报表以知道现在状态。由菜单中选择“Web/报表/含有链接错误的形状”。
　　    38. 打开浏览器，以报表方式显示包含超级链接错误的页面。（^60090109b1^）
　　    39. 而当我们由菜单中选择“Web/报表/所有形状，依类型”则可以利用浏览器分类查看。（^60090109c1^）
