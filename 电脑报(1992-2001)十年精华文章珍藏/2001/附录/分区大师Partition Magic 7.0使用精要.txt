分区大师Partition Magic 7.0使用精要

张熙平 
2001年 60期

    Partition Magic，魔法分区大师，是由PowerQuest公司推出的，目前市面上最好的磁盘分区管理工具，是你实现硬盘动态分区和无损分区的最佳选择。该工具最突出的特点是可以实现真正意义上的无损硬盘分区，你可以轻松地在已存有数据的硬盘中自由划分磁盘分区空间，并可以自动地扩大或缩小存在的硬盘分区大小，这是目前其他磁盘分区管理工具所不能提供的。Partition Magic 7.0是该工具的最新版本。
　　    Partition Magic 7.0具有以下新的功能：
　　    （1）全面支持Windows 9x/Me/NT/2000/XP等操作系统。它可以根据所安装的操作系统安装不同的支持程序。即使是在没有DOS实模式的Windows 2000/XP下，也可以完整地使用Partition Magic管理硬盘分区。
　　    （2）Partition Magic 7.0提供“Undelete Partition”功能，可以帮助你轻松恢复由于格式化分区所导致的磁盘分区文件丢失。并且，通过这项功能你只需简单的操作就可以完全恢复被格式化的硬盘数据。
　　    （3）新版本支持直接将FAT32格式的硬盘分区转换为NTFS的硬盘分区格式，以及更好地支持对Windows XP提供的NTFS格式的分区。加强了对 Linux 分区的支持。
　　    （4）Partition Magic 7.0新提供了无损分割硬盘分区功能“Split Partition”。现在你可以直接将一个分区中的任意空间和这个分区中的任意文件划分并创建为一个新的硬盘分区，在分区过程中不会有任何的数据丢失！ 
　　    （5）Partition Magic 7.0还提供了一项超强的无损硬盘分区合并功能，使用这项功能，在合并分区过程中程序会将其中一个分区中的全部数据自动复制到合并后的分区中，并生成一个文件夹，而不会删除任何数据！
　　#1    一、Partition Magic 7.0的安装与启动
　　    Partition Magic采用标准的Windows应用程序安装模式，只需要回答几个简单的问题，如：是否允许所有工具使用安装的程序（Windows NT/2000/XP系统）、用户注册信息、安装路径等，之后连续单击“Next”按钮即可轻松完成软件的安装。
　　    如果你得到的是CD版本，插入CD后安装程序会自动启动，然后执行如上所述的安装。在安装过程中程序会询问是只安装Partition Magic，还是需要同时安装PowerQuest提供的其他实用处理工具（主要视具体的安装版本不同），你可根据自己的使用环境及需求选择。安装过程中会提示创建程序的DOS应急盘。它需要两张1.44MB的空白软盘：第一张是引导盘，第二张是程序的DOS模式运行程序。
　　    安装完的成后，程序会在开始菜单中建立程序组，从程序组中直接选择Partition Magic，便可启动Partition Magic主程序。
　　#1    二、Partition Magic 7.0基本操作
　　#2    1.创新的硬盘分区
　　    （1）单击“Create new partition”图标，首先会出现一个提示窗口，提示你这项操作可能会改变程序中的盘符引用，不必理会，单击“Next”按钮进入硬盘选择窗口，如果只有一块硬盘，直接单击“Next”按钮。
　　    （2）进入分区使用说明窗口，如果想安装多操作系统，就应该建立主分区，选择“Yes”，但我们大多数情况下并不想这么做，所以选择默认值“No”，单击“Next”按钮进入分区文件格式选择窗口，默认的是FAT格式，建议选择FAT32。
　　    （3）接着是分区类型窗口，其中包括“Primary”和“Logical”，这里选择“Logical”（逻辑分区）。紧接进入新分区位置确定窗口，可选择在某分区前或在某分区后。
　　    如需要在原来C：和D：盘之间创建一个新的分区，则新的分区会被命名为D：，原来的D：会自动变为E：。如果后面还有其他的分区，则盘符顺延。在此需要说明的是，盘符顺延后可能导致系统中安装在不同分区中的应用程序由于路径发生变化不能运行。如果没有特殊要求，最好在当前分区的最后创建新的分区，这也是程序的默认选项。
　　    （4）随后进入“Size of new partition”（分区大小）、“Enter the label for the new partition （optional）”卷标设置窗口，输入要建立的分区大小和卷标，窗口中会提示分区容量的最小值和最大值，只要在这个范围之间即可，卷标可随意。单击“Next”按钮进入结果预览窗口，在此给出了转换前后的对比图示，审核无误后单击“Finish”返回主窗口。
　　    （5）此时，在程序界面的右下角多了一个“Apply changes”按钮，单击该按钮，并在打开的对话框中确认，程序开始分区创建操作。硬盘分区完成后，给出一个提示框，单击“OK”按钮回到程序主界面。单击“Exit”按钮，程序给出“Warning”对话框，提示用户系统需要重新启动，以使以上操作生效。
　　    （6）单击“OK”按钮确定，系统重新启动，分区创建结束。
　　#2    2.格式化分区
　　    利用Partition Magic的格式化功能，可以在格式化的同时进行分区文件系统格式的转换，格式化前请做好数据的备份。
　　    选择要格式化的分区，单击“Fromat partition”按钮，或通过菜单和鼠标右键单击需要格式化的分区，在快捷菜单中选择格式化命令，在打开的格式化对话框中选择格式化后的分区格式，然后输入卷标，单击“OK”按钮返回主界面后，执行确认操作即可。
　　#2    3.调整已有分区的大小
　　    实际上在前面的操作中我们已经涉及到了调整分区的容量问题，也就是在程序界面中选择需要缩小的分区，单击“Resize/Move partition”按钮，在打开的对话框的“New Size”框中设置该分区的新容量。此时剩余的空间将会转化为只有Partition Magic才能识别的“Free Space”空间，而且程序默认“Free Space”空间在当前分区之后。设置完成后返回程序主界面，单击“Apply”按钮添加设置即可。
　　    以上给出的是缩小分区的方法，如果你想要扩大分区，硬盘中必须具有没有被使用的“Free Space”空间，之后才能扩大分区容量。一般来讲“Free Space”空间最好由Partition Magic自己创建。如果你使用新的硬盘，硬盘中没有划分的硬盘空间就是“Free Space”空间，这与Fdisk是一样的。此外，需要注意的是：对于扩刁分区容量，硬盘中的“Free Space”必须紧挨着这个分区，如果中间隔着其他的分区，就不能将“Free Space”空间添加到这个分区中了。所以如果需要扩大的分区在需要收缩的分区之前，你创建的“Free Space”必须在需要收缩的分区之前，也就是在创建Free Space空间时，需要在“Free Space Before”文本框中设置收缩容量。如果在其之后，“Free Space”也要在后面。
　　#2    4.合并硬盘分区
　　    对于合并硬盘分区，程序提供了两种方式，一种是直接合并硬盘分区，如果这两个分区是紧挨着的，后一个分区中的文件在合并时会被删除，前一个分区中的数据保持不变。如果是不相连的，也需要将其中一个分区首先转化为“Free Space”空间，之后使用分区扩充功能实现，这样也可以保持一个分区中的数据不丢失。另一种方法是Partition Magic 7.0新提供的 “Marge partition”，使用这项功能可以保留被合并的两个分区中的所有数据，其中后一个分区中的全部数据会被放到新合并后的分区中的一个由程序创建的文件夹中，你可在合并完成后重新调整其中的文件位置。
　　    显然使用后一种方式效果好，但是它也会花费很多的转换时间，而如果使用前者，只需首先将删除分区中的数据拷贝出来，之后在合并后重新拷入即可，速度会很快。该项功能的具体用法如下：
　　    （1）首先确定需要合并的分区中哪个分区中的数据需要删除（做好备份）。然后使用鼠标选择该分区（可以是多个分区），单击“Delete partition”按钮，在打开的对话框中输入该分区的卷标，如果删除分区没有设置卷标，需要输入“No Name”。
　　    （2）单击“OK”按钮确定，返回程序主界面，该分区删除结束。此时可以看到这个分区的空间已经变为“Free Space”空间了。选择需要合并的主分区，然后可以按照前面提供的扩大硬盘分区的操作方法进行操作就可以了。
　　    对于直接合并分区的操作，可以单击“Marge Partitions”图标，然后使用程序提供的向导选择需要合并的分区，并进行确定就可以了。具体步骤在此不再给出。
　　#2    5.转换分区格式
　　    对于分区格式的转换，Partition Magic 7.0提供了两种类型，一种是在FAT、FAT32、NTFS、HPFS格式之间进行转换，一种是在主引导分区“Primary partition”和逻辑分区“Logical partition”之间转换。对于一般用户而言，前一种转换使用得多一些。以下给出具体的操作步骤：
　　    在程序界面中选择需要转换分区格式的硬盘分区，然后单击“Convert partition”按钮，在打开的对话框中的“Convert to”项中选择需要转换成的分区格式和类型，单击“OK”按钮返回程序主界面，单击“Apply”按钮添加设置即可。该项设置需要重新引导系统启动后才能实现。
　　#2    6.复制硬盘分区
　　    对于硬盘分区的复制，可能是我们经常接触的功能。Partition Magic 7.0提供了两种复制硬盘分区的方法，一种是直接执行复制操作，一种是使用程序提供的向导（Wizard）操作。
　　    首先我们介绍直接操作的方法：使用程序提供的“Move/Resize partition”功能在硬盘中创建不少于需要复制的硬盘分区容量的“Free Space”空间。使用鼠标右键单击需要复制的硬盘分区，并从快捷菜单中选择“Copy…”命令，单击“OK”按钮，返回程序主界面。单击“Apply”按钮添加设置即可。
　　    如果你想使用向导（Wizard）方式操作，可单击“Copy partition”图标，打开复制分区向导设置界面；切换到下一界面，在其中选择需要复制的原分区；切换到下一界面，在其中选择目的分区；切换到下一界面，在该界面中设置分区复制方式，其中包括了设置所复制的分区类型，如为引导分区还是逻辑分区，以及是否隐藏当前的复制的分区，以免发生分区盘符更改。而且使用隐含分区还有一个好处，就是可以完全的保护这个分区中的数据不被任何的操作系统识别，也就保证了其中的数据安全。按照向导提示完成后续的操作。
　　#2    7.无损分割分区
　　    分割硬盘分区，也就是重新定义分区大小，你可以借助“Free Space”调整分区，或者是创建新的分区。而这里所说的是Partition Magic 7.0新提供的无损分区功能，你可以将一个含有数据的分区分割为两个分区，而且可以自己定义每个分区中可以存放的数据，使用效果非常好。其唯一的缺点就是不能支持NTFS的分区格式。
　　    首先选择需要分割的硬盘分区，然后单击“Split partition into two partitions”按钮，在打开的对话框中的“New partition”项中设置新创建的分割分区的卷标、分区类型、分区盘符。在“Root Entries”列表窗口中选择需要移动到新的分区中的文件和文件夹，然后单击“>”、“>>”按钮添加到右边的新的分区中列表中。单击“OK”按钮即可。
　　#1    三、Partition Magic 7.0实用工具
　　    Partition Magic 7.0提供了许多实用的磁盘整理工具，作为其基本功能的扩展。
　　#2    1.快速修复应用程序盘符
　　    由于创建或删除了分区，导致硬盘盘符的改变，一些应用程序在注册表中的注册将会因盘符的改变而不能运行，CD 自动播放将会失效等。使用 “DriverMapper”实用工具将会快捷地解决这一问题。具体操作如下：
　　    运行“DriverMapper”，程序需要搜集信息，单击“Next”继续，“DriverMapper”需要知道你增加、删除了几个分区，导致变化了几个盘符，比如原有C盘，D盘为光驱，现在增加了D、E盘，光驱变为F盘，改变了两个盘符。在出现的对话框中有两个选项：“Only One Drive Letter Change“和”More Than One Drive Letter Change”，应根据实际情况选择。一般地，如果改变两个以上的盘符，如在 D 盘和 E 盘前增加了一个逻辑分区，“DriverMapper”将一步一步作处理，先将E盘转换为F，再将D盘转换为E。删除时的盘符处理与此相反。单击“Next”按钮继续后，在出现的对话框中选择盘符。比如原先的软件安装在E盘，现在由于增加分区，E盘变成了F盘，原始盘符应选E，新盘符应选F；同理，要改变光驱盘符，则原始盘应选F，新盘符应选G（假设F为原光驱盘符）。选择完毕，“DriverMapper”将会扫描注册表和所有逻辑盘路径，在出现的对话框中，选择“Change All Path Found”则同意对所找到的结果作改变，选择“View Changes To Be Made”则可以查看所有要改变的信息。确认无误后，“DriverMapper”将把信息写入，重新启动计算机后改变生效，你会发现光驱又能自动播放音乐了！再运行一次“DriverMapper”，选择应用程序所在的盘符，所有安装的程序又都“活”了！由于应用程序盘符和光驱盘符的改变，要想使二者都变为可用，一般应运行“DriverMapper”两次或多次。
　　#2    2.移动应用程序
　　    在应用Partition Magic 7.0进行分区重组以后，你也可以将一部分应用程序移动到相应的分区中，而不需要重新安装。“MagicMover”就可以方便地将某一硬盘中的Windows应用程序完整地移动到另一个分区中。在移动的同时，自动修改该程序在Windows中的有关盘符配置，免去重新安装、配置运行环境之苦！
　　    运行“MagicMover”实用工具，它将对所有分区进行扫描。选择要移动的应用程序，包括在桌面上的和开始程序组中的程序，可以用鼠标单击“+”来展开。选择一个程序后，单击“Next”按钮，在出现的对话框中的“Select the destination folder”中输入要将程序移动到的目的文件夹，单击“Browse”按钮可以对目的文件夹进行选择。单击“Details”按钮将给出此次操作要建立的文件夹、移动的文件、删除的文件夹等详细内容。单击“Move”按钮即开始对文件夹的移动。如果文件夹不存在，将提示你建立一个。一个任务完成后，单击“Continue”按钮将会进行新的挪移任务。
　　    注意：在运行“MagicMover”工具之前，最好不要运行其他程序，应关闭任何磁盘高速缓存程序和一切驻留内存的TSR程序。
　　#2    3.快速引导工具
　　    安装了多个操作系统，如何使用不同的操作系统呢？“PQBoot”作为指定分区快速引导工具，能够方便地让你选择哪一分区作为引导磁盘。下面给出具体的操作：
　　    启动“PQBoot”实用工具（会进入DOS状态）。目前处于活动状态的分区显示为“Primary，Bootable”；选择需要启动的分区，输入ID号，如3，回车后“PQBoot”将启动机器，引导的操作系统就是你选择的了。
　　    以上介绍了Partition Magic基本功能的使用方法。此外，Partition Magic还提供了“Bad Sector Reset”（重定义硬盘中的坏扇区）、“Resize Clusters”（重定义硬盘的簇）、“Unformat Partition”（反格式化分区）、“Undo”（恢复最后一次操作）、“Check Partition”（检测硬盘分区错误）、“Boot Magic”（设置多重引导菜单）、“Boot Disk Builder”（创建系统启动盘）、“Partition Info”（查看硬盘分区信息）、“Create Rescue Diskettes”（创建程序应急盘）等功能，限于篇幅，在此就不作详细介绍了。
　　    最后提醒一点：如果你发现执行了错误的操作，千万不能中途终止操作，退出程序，甚至重新启动计算机。可以等当前的操作完成后使用程序提供的恢复功能恢复操作。
