流行多媒体播放软件操作详解

黄智 
2001年 60期

#1    一、Windows Media Player 7.1
　　    最新版本：7.1简体中文版
　　    软件大小：9.97MB
　　    使用平台：Windows 9x/NT/2000
　　    下载地址：http://www.microsoft.com/windows/windowsmedia/en/download/default.asp
　　    Windows Media Player可支持目前Windows平台下的所有音频、视频格式，并提供几乎所有目前常见的播放控制功能，而且程序提供了更加简单的操作，使用效果极好。
　　    （一）播放多媒体文件
　　    1.按文件播放
　　    如果是播放单个或同一路径下的多个文件，可直接单击“文件→打开”菜单命令，并在打开的系统文件列表窗口中选择需要播放的文件即可。此外，由于程序安装时会自动创建其可支持的文件关联，所以你可以在资源管理器窗口中直接运行需要播放的文件图标即可。
　　    2.播放CD唱片
　　    如果是播放音乐CD，在光驱中插入唱片后，单击程序界面右上角的下拉列表菜单，从中选择“请插入音频CD”命令，Windows Media Player会自动扫描，并给出其中的音轨列表，单击“播放”按钮即可开始播放操作。
　　    3.播放VCD
　　    如果是播放DAT格式的VCD视频文件，需要单击“文件→打开”菜单命令，在打开的播放任务列表窗口中选择显示所有文件（*.*），然后选择DAT文件播放即可。
　　    对于播放的文件，你可以将其添加到程序提供的某个播放任务列表或者是自己创建的播放任务列表中，这样日后播放时会给你提供很多的方便。添加方法非常简单：打开相应的文件后，单击“文件→添加到库中”命令即可。
　　    （二）复制音乐CD到硬盘中播放
　　    Windows Media Player提供了将音乐CD的音轨复制到硬盘中播放功能，你甚至可以一边复制一边欣赏CD音乐。这项功能与抓取CD音轨为WAV文件或MP3文件的作用是一样的。程序提供的输出音频格式为WMA，这是微软提供的压缩音频格式。同MP3格式一样，WMA也是转换质量与生成的文件占用空间成反比的，具体见下表。

　　    （1）启动程序，将音乐CD放入光驱中，单击程序界面右上角的下拉列表菜单，从中选择“请插入音频CD”命令，程序会自动读取CD音轨。
　　    （2）切换到“CD音频”选项卡，在界面中选择需要复制的音轨列表项，单击“复制音乐”按钮即可。当所有的文件复制完成后，程序会自动创建相应的播放任务列表，以供日后直接调用。
　　    此外，如果你觉得程序的抓取质量不高，也可以修改，方法是：单击“工具→选项”菜单命令，打开程序的运行环境设置界面。切换到“CD音频”标签，在“复制设置”项中的“以此质量复制音乐”项中设置抓取音轨输出文件的质量即可。
　　    （三）使用Media Player媒体库
　　    Windows Media Player的媒体库是一个非常重要的保存所有播放文件列表的地方，在该项功能的选项卡中你可以创建新的播放任务列表或者是编辑已有的播放列表。如果需要创建播放列表，可按以下方法操作：
　　    （1）切换到“媒体库”选项卡，在该界面中给出了所有的播放列表项，以及相互的所属关系。单击“新建播放列表”按钮。
　　    （2）打开“新建播放列表”界面，在其中设置新建播放列表的名称，然后确定返回，此时可以看到新建的播放列表已经显示在界面窗口中了。
　　    （3）选择该列表项，单击界面按钮条最右边的“＋”号按钮，程序会给出一个下拉菜单，其中包括“添加文件”和“添加URL”两个命令项，根据自己的需要选择相应的命令，并添加需要播放的文件即可。
　　    对于已经存在的播放列表，程序提供了编辑功能，方法是：选择需要编辑的播放列表，程序会给出该列表中所有的播放文件列表，如果需要添加新的播放对象，可按照上面提供的方法操作，如果需要删除播放对象，可使用鼠标右键单击相应的列表项，并从快捷菜单中选择“从库中删除”命令即可。所有删除的项目将会保存到“已删除的项目”项中，这就像Windows中的回收站，你可以从其中将被删除的列表恢复到删除前的位置。
　　    （四）收听Internet广播
　　    Windows Media Player提供了直接收听来自Internet的广播功能，该项功能的使用方法非常简单，首先切换到“收音机调谐器”选项卡，在界面左边给出的是程序默认提供的Internet电台列表，在其中选择需要收听的电台，程序会自动连接到相应的电台网站，并下载音频数据，之后就可以收听了。此外，对于收听的电台，是可以自由添加的，具体方法是：
　　    切换到“媒体库”选项卡，选择其中的“收音机调谐器预置”列表项，单击“添加到库中”按钮，并在打开的菜单中选择“添加URL”命令，设置需要添加的电台URL地址即可。
　　    （五）更换界面外观
　　    Windows Media Player的一个显著特点是程序提供了丰富的界面外观更换功能。此外，程序在其支持站点提供了附加的界面外观文件下载，这样你可以选择下载自己喜欢的外观文件，以丰富程序界面。如果需要更换界面外观文件，可首先切换到“外观选择器”选项卡，程序会给出相应的界面，直接在界面中选择需要使用的界面外观列表项即可。如果使用鼠标双击界面外观列表，即可快速应用。
　　    与其他的工具不同，当使用界面外观文件后，在系统桌面的右下角会有一个小窗口，其中只有一个按钮，单击该按钮会出现一个快捷菜单，其中包括返回程序的主界面（完整模式）、选择新的界面外观、打开需要播放的文件或站点URL、退出程序等，根据自己的需要直接选择即可。
　　#1    二、Winamp 3.0
　　    最新版本：3.0 beta
　　    软件大小：1.29MB
　　    使用平台：Windows 9x/NT/2000/XP
　　    下载地址：http://www.winamp.com
　　    Winamp是目前非常好的音频文件播放平台，该工具可以支持目前常见的各种音频文件格式，尤其是对压缩音频如MP3等有很好的支持。Winamp 3.0支持MID、MP3、MP2、VOC、WAV、CDA、XM、STM、MOD、ULT、MTM和CD共20种文件格式。
　　    （一）播放音频文件
　　    （1）启动程序，如果需要播放音频文件，如MP3文件，可按下“L”键，程序给出“打开”对话框，在其中的文件列表窗口中选择需要播放的文件，如果需要播放多个文件，可按住“Shift”或“Ctrl”再配合鼠标左键进行选择。
　　    （2）单击“打开”按钮回到程序界面，此时程序开始播放所选择的第一个文件。在程序界面中提供了多个播放控制按钮，使用鼠标直接点击即可。
　　    （二）使用播放控制列表
　　    （1）使用鼠标单击程序界面中的“PL”按钮，打开“Play List”列表框。
　　    （2）如果需要添加文件到播放列表中，可单击最左边的“Add”按钮，在弹出的菜单中选择“Add files”菜单来添加来自指定路径下的音频文件。
　　    （3）程序给出“打开”窗口，在其中的文件夹列表窗口中选择需要播放的文件所在驱动器或文件夹，之后单击确定按钮回到程序界面。Winamp会自动扫描该文件夹中的所有文件，并找出其所支持的音频文件，添加到界面中的播放列表窗口中。
　　    （4）单击程序界面中的播放按钮，程序开始按照列表窗口中文件的排列顺序播放文件。
　　    （5）对于设置的播放列表，程序提供了保存和载入功能，如果需要保存当前的播放列表，可单击界面中的“playlist”按钮，并从弹出的菜单中选择“Save this Playlist”，程序给出“另存为”窗口，在其中设置保存文件的名称和路径即可。程序提供保存为“*.bpl”格式。当需要使用保存的播放清单文件时，可使用打开单个文件的方法调入。
　　    （三）使用音频均衡器
　　    （1）使用鼠标单程序界面中的“EQ”按钮，程序会在界面旁边给出一个EQ（Equalizer）均衡器窗口。
　　    （2）单击“ON”按钮，然后调节“频谱设置滑块”进行调整即可。如果需要使用预设的频谱方案，可单击界面中的“PRESETS”按钮，程序会给出调整菜单，在其中提供了载入保存的预设方案、保存当前频谱设置方案、使用默认设置、从音频方案文件EQF中载入等相关命令。
　　    （四）使用外观修饰插件
　　    将下载的外观插件文件复制到程序安装路径下的“Skins”路径下的相关文件夹中，在程序界面中单击鼠标右键，程序给出关联菜单，在该菜单中的“Skins”项关联菜单中给出了所有保存在“Skins”路径下的Winamp界面外观文件，直接选择即可。
　　    （五）使用效果修饰插件
　　    Winamp的效果插件包括播放视觉外观插件和播放插件、音效处理插件等。如果需要安装并使用新的插件程序，可首先将下载的文件复制到程序安装路径下的“Plugins”路径下，然后在程序界面中按下“Ctrl+P”键，在弹出界面中切换到“插件”标签项，其他提供了五个分类插件项目，选择相应的标签，程序会在界面右边窗口中给出安装的插件列表。对于程序默认的插件，其中只提供“设置”功能按钮，在其关联界面中可以设置该插件的运行方式，对于外观插件，在界面中包括“开始”、“停止”、“设置”三个功能按钮，单击“开始”按钮表示使用当前插件程序，“停止”表示关闭当前使用的插件、“设置”表示设置当前插件的运行方式，直接选择即可。
　　    此外，对于视觉外观插件，也可以通过鼠标右键菜单快速控制，方法是在程序界面中单击鼠标右键，在关联界面中的“视觉外观”项关联菜单中提供了视觉外观设置、启动、选择等控制命令，直接选择即可。
　　#1    三、RealPlayer 9.0
　　    最新版本： V9.0
　　    文件大小： 8.63MB
　　    使用平台： Windows 9x/NT/2000/XP
　　    下载地址： ftp://ftp1.mydown.com/home2/soft30/rp9.zip
　　    Realplayer是美国RealNetworks公司出品的一款多媒体播放软件，因为它独有的流媒体RM格式在现有的网络带宽下，能够比较流畅的进行在线视频和音频播放，所以许多网友将Realplay列为多媒体播放软件的首选。当然Realplay具有强大的多媒体播放功能，不但可以播放独有的流媒体Rm格式文件，还可以播放包括MP3、AVI、WAV等多达十几种的多媒体格式。
　　    （一）使用播放功能
　　    启动RealPlayer。在主界面中单击“Option”按钮或者直接点击鼠标右键，从弹出的菜单中选择“Open”命令后，然后用“Browse…”功能来查找并打开要播放的本机媒体文件或者直接在浏览窗口中输入网上流媒体文件链接即可播放。
　　    （二）使用媒体浏览器的管理功能
　　    1．制作播放列表
　　    （1）在主界面的右上角点击“Show/Hide Clip List”图标，媒体播放列表将在“Media Browser”窗口中的左侧显示出来。
　　    （2）点击左下角的“添加”按钮后，你可以选择“Add File…”、“Add Directory…”或“Add URL…”三种添加媒体文件的方式。需要删除列表中的文件，请点击“添加”按钮旁边的“删除”按钮，便可以有选择地，或者单个地删除列表中的文件。
　　    （3）在选择列表中的文件，点击鼠标右键或者点击下面的“Edit”按钮便可以对列表中的文件进行有效的管理。其中包括对列表文件的剪切、复制、移动、复制到媒体库等，你可以根据自己的需要进行选择。
　　    （4）为了管理好这些媒体文件，我们可以把产生的列表加入到RealPlayer的媒体库中进行管理。点击“删除”按钮旁边的“保存”按钮，在出现的菜单命令中直接选取“Save Clip List My Library”进行管理。点击该命令后，需要输入文件列表的名称，输入完毕点击“OK”便可。
　　    2．用媒体库（My Library）管理文件
　　    （1）点击“My Library”按钮，在ReaklPlayer的右边窗口便会出现媒体库的管理界面。界面分两个部分，左边是管理项目树型结构，右边便是媒体文件的基本信息。
　　    （2）“Playlist”项目里存放的便是当前我们制作的播放列表，你可以在里面任意选择要播放的列表文件。此时在右边媒体文件显示窗口点击右键，选择相应的菜单命令可以进行创建新列表或者导入列表文件等操作。
　　    同时，在这里可以有选择性地查看RealPlayer所管理的媒体文件和文件列表的信息，包括音频、视频文件的属性等。包括我们制作的所有音频、视频的播放列表。
　　    （3）设置完成，若要找到你想要的媒体文件，可点击“Search”按钮进行搜索，在弹出的窗口中输入关键字即可查找。
　　    （三）个性化功能
　　    1．选择可视化效果
　　    在RealPlayer主界面里选择“Options”→“View”→“Choose Visualization”命令，RealPlayer共提供了五种漂亮的可视化效果以供选择。
　　    如果选择“Choose Visualizations”菜单下的“Visualization Settings”命令，还可以分别对上述五种可视化效果进行设置：
　　    由用户自行设置变化程度的Fade（淡入淡出）、Rotate（旋转）、Zoom（缩放）、Slide（滑动）、Blur Type（拖影）等。
　　    复选框“Randomize Effects During Playback”（随机播放效果）用来选择是否在播放的时候随机选择效果。
　　    如果你的CPU主频够高，还可以在最下面的滑动条“Frame Rate”中设置最高显示速度，使可视化效果得到更加流畅的变幻画面。
　　    2．更换皮肤（Skin）
　　    在RealPlayer 9.0主界面里选择“Options”→“View”→“Choose Skin”来更换软件外观，如果连接了因特网，还可以利用“Choose Skin”菜单下的“Get More Skin”命令来自动下载更丰富多彩的Skin。
　　    3．在线新闻
　　    RealPlayer 9.0特设了接收RealONE信息中心的功能，使用方法如下：
　　    在RealPlayer 9.0主界面里选择“Options”→“View”→“Real.com Messages”命令，然后在“Messages Center”窗口右侧选择你所感兴趣的新闻类型（包括热门头条新闻、电影片断、音乐下载、体育新闻等），日后一旦有了新的快讯媒体信息到达，软件会给出提示。
　　    （四）音频的转换
　　    1．CD→WAV、MP3或RMJ文件
　　    （1）首先在你的光驱里放进一个带有CD音轨的光盘。
　　    （2）在主界面中点击“CD”图标，然后点击“Preferences…”按钮，进入设置窗口。在弹出的设置窗口中，先选中CD设置项目，然后在右边的设置选项里进行设置。
　　    注意：其中提供了RMJ（RealPlayer的专用播放格式）、MP3和WAV格式。选中要转换的格式后，可以在下面的“音质选项”里选择压缩的质量，其中你可以明显地看到每种质量的解释和压缩率。
　　    （3）设置完成，点击“OK”按钮退出设置窗口。然后在RealPlayer 9.0主界面中点击“Add CD to Media”按钮，在弹出来的对话框中选择你要压缩的CD歌曲后，RealPlayer便自动地按照你刚才的设置，把歌曲压缩在系统目录的“My Music”（默认）文件夹下面了。
　　    2．转换成CD音轨
　　    如果你的电脑配置了刻录机，点击CD列表中的“Burn CD”按键，只要按照软件提示到指定的网站下载音乐CD刻录引擎“CD Creator Burn Engine” ，就可以刻录自己的音乐CD。
