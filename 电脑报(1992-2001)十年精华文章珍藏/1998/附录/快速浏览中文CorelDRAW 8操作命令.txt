快速浏览中文CorelDRAW 8操作命令

曾刚
1998年 第51期 38版

    中文CorelDRAW 8的许多功能都需要通过下拉菜单来引用，下面介绍各下拉菜单所包含的命令的功能和用途，以便于读者查询。
#1    一、“文件”下拉菜单
    这一组下拉菜单所提供的命令，主要用于建立新文件、保存文件、导入与导出、打印输出及其有关设置。
    1.新建
    建立新图形。如果当前已有打开的图形，新图形将在当前绘图页面之上打开，有关的操作选项将延用先前的设置。 
    2.从模板新建
    启动CorelDRAW的“模板”向导，选择模板并基于它来创建新的图形。
    3.打开
    打开“打开绘图”对话框。在此对话框中可以装入指定的图形或者样式模板，或者搜索打开某一份图形文件，以便于对它进行编辑操作。如果用户想观察一下所要打开的图形内容，可以在此对话框中进行预览。预览功能所显示的将是图形文件的缩略图，以及相应的版本信息、上一次打开的时候、文件的长度。
    4.关闭
    关闭当前图形。 
    5.保存
    保存当前图形，并且使用“标题栏”中的名称作为文件名。如果当前图形还未保存过，屏幕上将显示“保存绘图”对话框，并提示用户输入一个文件名。
    6.另存为
    保存当前图形至另一份文件中。 
#1    注：如果既要修改当前图形，又要保留其原始版本，就可以使用“另存为”命令，使用不同的文件名称与格式来保存它。这样一方面可以建立当前图形的副本，另一方面又可以使原始版本不受改动的影响。
    7.还原
    将图形还原为上次保存的版本。如果用户想撤消自上次保存以来对绘图所做的全部更改，就可以使用此命令。
    8.导入
    打开“导入”对话框，将其它应用程序建立的图形导入中文CorelDRAW 8中。 使用该命令还可以将其它CorelDRAW的CDR或CMX类型文件与当前绘图合并。
#1    注：如果用户知道要导入文件的格式，就可以从“文件类型”列表框选择它，以便只显示某一扩展名的文件。
    9.从CorelSCAN中获取
    启动CorelSCAN向导。CorelSCAN是中文CorelDRAW 8提供的一个扫描实用程序，此向导将引导用户通过制作高品质扫描图像的操作。
    10.导出
    打开“导出”对话框，为其它程序输出当前图形。
#1    注：在多页文档中，只有当前显示页面或首页上的对象可以被此命令导出。
    11.打印
    打开“打印”对话框，通过该对话框可以打印输出图形、设置或者修改打印选项，更改打印机及其属性。
    12.打印预览
    打开“打印预览”对话框，以便在打印之前查看打印的效果。
    13.打印设置
    打开“打印设置”对话框，以便更改设置打印机及其属性。可以设置的打印属性包括纸张大小、打印方向等。
    14.为彩色输出中心做准备
    打开“配备彩色输出中心”向导，完成为在彩色输出中心进行专业输出而做的准备，以及收集文件的所有操作。
    15.转为因特网文件格式
    将用于Web文档的用户图形转换为“超文本标记语言（HTML）”格式、Corel Barista 格式或单个图像格式。 用户可以选择使用“转为因特网文件格式”向导或“转为因特网文件格式”对话框来转换 Web 文档。前者将指导用户逐步完成转换过程，而后者则允许用户设置自定义的转换参数。
    16.文档信息
    显示关于文档内容和文档所包含对象的详细信息。如果需要还可以打印和保存这些信息。
    17.版本控制
    “版本控制”命令包含有三条子命令，通过它们可以完成下列操作：
    （1）存档文档
    打开新建的“存档属性”对话框，设置有关存档的参数。
    （2）从当前档案检索
    打开“选择检索版本”对话框，指定要检索的文件版本，以及比较文件的不同版本。
    （3）从保存的档案检索
    打开“打开”对话框，选择文件的备份版本。
    18.最近使用的文件
    这是一份列表，将列出最近编辑过的四份文件，单击其中的某一项，或者键入某项前面的序号，即可以打开此文件。
    19.退出
    结束当前在中文CorelDRAW 8中的操作，并且退出该程序。如果当前有没有保存的图形，此命令将在退出之前提示用户保存它。
#1    注：按下键盘上的 Alt + F4功能键，或者单击“标题栏”右上角的“关闭”按钮也可以退出CorelDRAW。
#1    二、编辑
    这一组下拉菜单主要用于撤消与恢复操作、使用剪贴板、复制、查找与替换、插入因特网控件与条形码等操作。
    1.撤消
    撤消刚执行过的操作。每当用户做了一种操作后，该命令名的后面都将显示此操作。例如，如果上一个操作是填充，此命令将显示为“撤消填充”，再如上一次做的是旋转操作，此命令就显示为“撤消旋转”。 如果无法撤消操作或没有可撤消的操作，则“撤消”命令将显示为“不能撤消”或变为灰色。 
    2.重做
    恢复被“撤消”命令所撤消的更改。当单击“撤消”命令之后，“重做”命令就会变得可用。在下拉菜单中“重做”命令名称后将紧跟上一次撤消的操作。例如，如果上一个撤消的操作填充，此命令就显示为“重做填充”命令，再如上一次撤消的是旋转，则显示为“重做旋转”。
    3.重复
    重复上一条命令或所做的操作。此命令的具体名称也将取决于最近执行过的那一次操作。例如，如果上一次操作是填充，就会显示为“重复填充”命令，再如上一次操作是旋转，则显示为“重复旋转”。如果没有可以重复的操作，此命令将显示为灰色。
    4.剪切
    从图形中移除当前选定的内容，并将它放到Windows的“剪贴板”上，以便随后从“剪贴板”中将这些内容粘贴到另一个Windows 应用程序中，或者是CorelDRAW文件中。该命令只有在选定对象或文本之后才能使用。
    注：在剪切或复制其它对象到“剪贴板”之前，“剪贴板”上的内容将一直保留。
    5.复制
    将当前的选定内容复制到“剪贴板”中。 与上述剪切不同是它不从图形中移除选定的内容。用户可以从“剪贴板”中将这些内容粘贴到另一个 Windows 应用程序或 CorelDRAW 文件中。 该命令也只有在选定对象或文本之后才能使用。
    6.粘贴
    将“剪贴板”中的内容粘贴到图形中。
#1    注：用户可以使用“剪切”或“复制”命令将对象放到“剪贴板”上，在复制或剪切其它对象到“剪贴板”之前，或者在结束当前 Windows 会话之前，这些对象都将一直保留着，直到下一次“剪切”或者“复制”了新的内容。
    7.选择性粘贴
    将“剪贴板”中的内容副本放到“绘图页面”的中心。与上述“粘贴”命令不同的是，它可以指定这些内容所用的格式，甚至可以创建与其源文件的链接。 链接是一种在应用程序中放置OLE（对象连接嵌入）对象的方法。 
    8.删除
    移去选定的对象或者节点。
#1    注：与上述“剪切”命令不同的是，此命令将移去选定的对象或者节点，而不会在“剪贴板”上放置其副本。因此，对于一个已删除的对象或者节点，只能在尚未执行其它命令之前使用“撤消”命令来恢复。相比之下，使用“剪切”命令来删除对象更为有用，因为它可以在“剪贴板”上放置对象的副本。这样，当需要该对象时就可以使用“粘贴”命令来恢复它。
    9.再制
    将当前选定范围的副本添加到图形中。在默认情况下，副本会出现在原始对象之上，并略有偏移，以便于观察与分辨。 同时，副本也会被自动选定。
#1    注：按下键盘上的加号（+）键也可以再制对象，副本将位于原始对象之上，但没有偏移。
    10.克隆
    将当前选取范围内的副本添加到图形中。“克隆”是一种特殊的复制手段，在默认情况下，克隆对象将出现在原始对象之上，并略有偏移，并且将延续和继承源对象的一切属性。因此，大多数随后应用到原始对象（主对象）的更改将自动地应用到副本（克隆对象）上。    11.全选
    此命令所包含的子命令与功能如下：
    （1）对象
    选择图形中当前页面上的所有对象，包括那些当前看不到的对象，但不包括辅助线。选定了之后，就可以让它们接受随后所做的有关编辑操作。 
#1    注：双击“挑选”工具也可以选择当前页面上的所有对象。辅助线是一些用于辅助操作的直线，不属于图形的范围，也不会被打印输出。用户可以控制使用辅助线。
    （2）文本
    选择图形中当前页面上的所有文本对象，包括那些当前看不到的文本对象。
    （3）辅助线
    选择绘图中当前页面上的所有辅助线。 
    12.属性
    打开“对象属性” 泊坞窗，更改选定对象的属性。在执行该命令之前必须有选定的对象，否则此命令无用。选定对象的类型将决定对话框要显示的页，如选定矩形，则有关矩形的页将显示。
    注：“泊坞窗”是一种CorelDRAW 8的对话框，与其它的对话框一样，它包含有命令按钮、选项和列表框等控件。不同的是在操作此软件时可以一直打开“泊坞窗”，以便于访问常用的操作，或者试验不同的效果。“泊坞窗”可以泊于应用程序窗口的边缘，或者可以使其出泊。“泊坞窗”入泊后，可以将它最小化，这样它就不会占用宝贵的屏幕空间。
    13.复制属性
    打开“复制属性”对话框，将一个对象的属性复制到其它对象上。这是一条用途广泛的命令，如在填充了某一个对象之后，就可以用它将同样的填充复制到其它的对象上，从而让不同的对象使用相同的填充，而不必每次都重新创建填充。
    14.查找和替换
    此命令所包含的各子命令与功能如下：
    （1）查找对象
    打开“查找”向导。此向导提供有识别那些符合指定搜索条件的对象，以便在图形中搜索与选定对象的条件相匹配的对象。完成搜索后，还可以保存搜索条件，以便于随后再次使用它。
    （2）替换对象
    打开“替换”向导。此向导用于查找与指定的搜索条件相匹配的对象，并用其它具有相似属性的对象替换之。用户还可以搜索颜色模型或调色板、轮廓笔属性以及文本属性。完成搜索后，也可以保存搜索条件。
    （3） 查找文本
    打开“查找文本”对话框。通过这个对话框可以在当前图形中查找文本，而且可以搜索整个图形或者任意数量的单独文本项。打开“区分大小写”复选框还可以区分大小写字符。    （4） 替换文本
    打开“替换文本”对话框。通过这个对话框，可以在当前图形中查找并替换文本，而且可以搜索整个图形或者任意数量的单独文本项。此对话框也提供有“区分大小写”复选框。    15.插入因特网对象
    此命令所包含的子命令与功能如下：
    （1）Java Applet小程序
    在图形中插入Java Applet小程序。
#1    注：如果需要，用户可以通过右击对象，然后从快捷菜单中选择“属性”，打开“对象属性”对话框，为“Java applet 小程序”页上的因特网对象设置自定义属性。
    （2）内嵌文件
    在图形中插入内嵌文件。
    （3）简单按钮
    在图形中插入简单按钮。
    （4）提交按钮
    在图形中插入提交按钮。
    （5）重置按钮
    在图形中插入重置按钮。
    （6）单选按钮
    在图形中插入单选按钮。
    （7）复选框
    在图形中插入复选框。
    （8）文本编辑区
    在绘图中插入文本编辑域。
#1    注：如果需要，用户可以在“对象属性”对话框中为“文本和口令域”页上的因特网对象设置自定义属性。
    （9）文本编辑框
    在图形中插入文本编辑框。
    （10）弹出式菜单
    在图形中插入弹出式菜单。
    （11）选项列表
    在图形中插入选项列表。
    16.插入新对象
    打开“插入新对象”对话框。该对话框允许链接或嵌入在其它程序中创建的对象。这种内嵌或者链接对象的形式主要用于插入其它文件的内容。
    17.插入条形码
    打开“条形码”向导。这个向导用于创建符合工业标记符号标准的条形码。为了使用此功能，用户在安装中文CorelDRAW 8时，需要在安装向导中指定安装它，否则此命令将呈现为灰色。
    18.对象
    用于编辑选定的 OLE 对象。
    19.链接
    列出当前图形中的所有链接。 该列表包括与其它 CorelDRAW 文件的内部链接和与其它 Windows 应用程序的外部链接。
#1    三、查看
    这一组下拉菜单中的命令主要用于各种显示控制。
    1.简化框架
    打开“简单线框”视图方式。在此方式下将只显示对象的轮廓，因而可以较快地编辑绘图，中是在每次编辑后刷新显示对象轮廓。
    注：改变视图质量不会影响图形的内容，只影响在计算机屏幕上的显示方式。
    2.框架
    打开“框架视图”方式。在此方式下，将隐藏填充和显示单色位图、立体图、轮廓线和中间调和形状。 
    3.草稿
    打开“草稿视图”方式。在此方式下将显示均匀填充和低分辨率位图，并以纯色显示透镜和喷泉式填充。
#1    注：在“草稿视图”中，填充将使用唯一的图样来表示。
    4.正常
    打开“正常视图”方式。在此方式下将显示除PostScript以外的所有填充、所有对象和高分辨率位图。这是中文CorelDRAW 8默认的视图方式。
    5.增强
    打开“增强视图”方式。如果计算机的速度比较快或者是想看到最接近于打印结果的绘图显示效果，就可以使用此方式。
    6.全屏预览
    显示图形的全部细节，而不中文CorelDRAW 8的任何界面。此命令将根据当前“选项”对话框中设置的视图质量，在预览时使用“普通”或“增强”方式视图。如果选择“增强”视图，“全屏预览”还将显示渲染后的PostScript填充。通过键盘上的F9功能键可以快速切换到该预览屏幕，此后若按下键盘上的任意一个键即可返回“绘图窗口”。
    7.只预览选定的对象
    若打开该命令，那么在需要涮新屏幕上显示的图形时，就会仅预览选定的对象。此功能在处理复杂的绘图时特别有用，它对于减少图形的刷新次数和识别重叠的对象很有帮助，这将有利于提高操作速度。当关闭此命令时，“绘图窗口”中当前可以看到的所有对象都被刷新，这是默认的设置。
    8.调色板
    此命令所包含的各子命令与功能如下： 
    （1）无
    隐藏“调色板”，使它在CorelDRAW的操作窗口中不可见。
#1    注：调色板是多个纯色的集合。在CorelDRAW和Corel PHOTO-PAINT中，可以使用常驻屏幕的“调色板”为填充、轮廓、纸张等选取颜色。
    （2）自定义颜色
    显示当前的自定义调色板。用户可以创建新的自定义调色板，并将其保存起来以创建调色板库。
    （3）标准颜色
    显示“均匀颜色调色板”。
    （4）装入调色板
    打开“打开调色板”对话框，以便选择其它的“调色板”。
    9.集锦簿
    此命令所包含的各子命令与功能如下： 
    （1）浏览
    浏览并搜索用户计算机中的文件夹和文件。
    （2）剪贴画
    在文件夹中搜索并导入剪贴画文件。剪贴画是由中文CorelDRAW 8提供给用户使用的一些图像素材，此命令为了帮助用户查找正确的图像，还将显示每一份文件内容的缩略图、草图以及文件名。
#1    注：要使用“剪贴画”页，CD-ROM 驱动器中必须有“CorelDRAW 剪贴画”CD-ROM。
    （3）相片
    在文件夹中搜索并导入相片文件。相片是由中文CorelDRAW 8提供给用户使用的一些图像素材，此命令为了帮助用户查找正确的上片，还将显示每一份文件内容的缩略图、草图以及文件名。
#1    注：要使用“相片”页，CD-ROM 驱动器中必须有“CorelDRAW 相片”CD-ROM。
    （4）填充与轮廓
    显示预设的填充集和轮廓集，将它们添加到在CorelDRAW 中创建的对象上是使用该软件的一个重要操作。在应用这些填充和轮廓时，可以从“集锦簿”中将它们拖动到文档中的指定对象上。在“填充和轮廓”页上还允许保存用户自己最喜欢的填充和轮廓，以便此后使用它们。要保存所创建的填充或轮廓，也只需将具有该填充或轮廓的对象拖动到“填充和轮廓”页上并指定要保存的属性即可。
    （5）三维模型
    通过在文件夹中搜索要导入文档的三维模型。此命令能够便捷地访问“相片”CD-ROM上的三维模型集。为了帮助用户查找正确的模型，此命令还将显示每一份文件内容的缩略图草图以及文件名。
    注：要使用“三维模型”页，CD-ROM 驱动器中必须有“CorelDRAW 相片”CD-ROM。
    （6）Ftp站点
    通过匿名方式或提供用户名和口令的方式连接到任意“文件传送协议”（FTP）站点。 用户可以直接从这些 FTP 站点中将文件导入文档或将文件副本下载到本地驱动器中。
    10.卷帘窗

#1    六、 效果

    7.颜色调整
    通过下列子命令控制使用颜色：
    （1）亮度-对比度-强度
    使用HSB（色度、饱和度和亮度）值调整图像中的颜色。该命令主要对于改变颜色的深度，也能够完全改变颜色的色度，功能非常的强大，也很容易掌握。
#1    注：色度、饱和度和亮度的是相互作用的，改变它们中的任何一个，颜色都会产生变化。
    （2）颜色平衡
    按CMYK颜色值或者RGB值校正偏色，以及改变整个图形或选定区域内的色度值，并且可以在产同的方式下观察效果。
    （3）伽玛值
    在中文CorelDRAW 8，应用伽玛值可以对色调进行校正。此值与人眼对相邻颜色值的感觉有着直接的联系。例如，将10%灰色值的圆形放在黑色背景上，再将同样的另一个灰色的圆放在白色背景上，那么对于人眼来说，黑色背景下的圆形要比白色背景下的圆形亮，而事实上这两上圆的亮度值是相同的。这里的“伽玛值”可以对这种对效果进行校正，增强低对比度图形的细节，而且不会明显地影响阴影和高光部分的图形，也不会影响绘图的所有颜色值。
#1    注：由于此命令将基于曲线进行处理，因此所产生的变化将主要作用于中间色调。
    （4）色度-饱和度和光度
    使用 HSL值调整图形中的颜色，而且可以改变整个图形的颜色强度，以及完全改变颜色的色度。
    （5）反显
    将图形所有的颜色值转换为其相反值，以制作图形的负片。这种反转将基于各颜色的颜色号为进行，如0号颜色（黑色）将变为255号颜色（白色）。
#1    注：“负片”指的是图像中颜色原始数值被反转，如黑色区域变成白色，而白色区域变成黑色，其它所有的颜色都用其互补色。
    （6）极色化
    将图像中在一定范围内变化的颜色转换为纯色块。

#1    七、位图
    “位图”是由像素网格或点网格组成的图像，可以来自扫描仪和别的绘图程序，在中文CorelDRAW 8中将使用矢量对象来创建图像。用户可以将矢量图形转换成位图，或者导入一幅外部的位图文件。这一组下拉菜单主要用于对位图进行操作。
    1.转换为位图
    将在中文CorelDRAW 8中创建的矢量对象转换为位图。
    2.编辑位图
    打开创建位图的应用程序来修改位图。此应用程序是中文CorelDRAW 8所附带，名为 Corel PHOTO-PAINT。当完成修改并关闭该应用程序之后，修改过的对象就会被合并到中文CorelDRAW 8的文件中。
    3.重新取样
    打开“重新取样”对话框。在该对话框中可以增大或减小图像的尺度和分辨率。操作时，可以此对话框中指定使用绝对值或百分比值来调整图像的大小、更改图像的水平和垂直分辨率（dpi），还可以选择重新取样图像的处理质量，以及校正在重新取样时可能出现的图像变形。
    4.转换为
    使用下列子命令转换位图：
    （1）黑白（1位）
    将当前位图转换为1位的黑白位图。用户可以选择的转换方式有：线条图、有序、偏差扩散和半色调，每一种选择都能生成一种黑白位图。
    （2）灰度（8位）
    将位图转换为灰阶位图。灰阶位图是转换到 0-255（256）级灰度范围的位图，基于这种灰度范围产生的位图效果类似于传统的黑白相片。
    （3）双色调（8位）
    将位图转换为双色调。双色调颜色模式的位图是由1~4种附加颜色增强后的灰阶图像。使用双色调颜色模式可以给灰阶图像增加色彩感。
#1    注：双色调图像可以是单色调、双色调、三色调或四色调的。
    （4）调色板色（8位）
    将位图转换为8位调色板色位图。用户可以选择的转换方式有：均匀的、标准VGA、自适应的、优化的。用户还可以执行专业化的位图转换功能，包括对“抵色”的处理和位图调色板的转换。在常用的彩色打印机上打印时，请选择256色来创建不同于相片效果的位图和优化用户的系统内存。
#1    注：“抵色”是设备或图像像素的随机化过程。在此过程中将使用有限的调色板来模拟连续的色调渐变。屏幕的抵色处理是增强监视器显示能力，使其具有16位左右的颜色。图像的“抵色”是增强摄影图像外观的一种方法。
    （5）RGB色（24位）
    将位图转换为24位的RGB颜色的位图。
#1    注：RGB颜色模型将使用三种颜色：红、绿、蓝的百分比来创建颜色，是一种最常用的颜色模型，在RGB或CMY（参阅下面的内容）打印机上打印时，选择RGB颜色通常能够创建高品质的效果。
    （6）Lab色（24位）
    将位图转换为24位Lab颜色模型的位图。
#1    注：Lab（CIE L*a*b）颜色模型是由Commission Internationale de l＇Eclairage（CIE）创建的一种颜色模式。它包含一个亮度组件（L）和两个颜色组件：a（由绿色到红色）和b（由蓝色到黄色）。所创建的位图与设备无关，并包含CMYK和RGB颜色模型的色谱。
    （7）CMYK色（32位）
    将位图转换为32位的CMYK颜色的位图。
#1    注：CMYK颜色模型包括四种颜色，这是在四色印刷中使用的四种油墨的颜色，它通过将各种百分比的青蓝、品红、黄色和黑色进行组合来表现色彩。青蓝、品红、黄色构成CMY颜色模型，加上黑色后即构成了CMYK颜色模型，它可以用于产生全色分色片的。不过，由于CMYK颜色模型的位图在输出时将依赖于设备的颜色空间，也就是说它要使用CMYK输出设备的信息来创建适合于该设备的位图颜色。但是当在印制设备或CMYK打印机（如HP彩喷打印机）上打印时，使用CMYK颜色模型可以创建具有专业品质的位图。
    5.位图边框扩充
    使用下列两条子命令扩充位图：
    （1）自动位图扩充
    自动扩充位图边框，并让用户以此来应用各种效果。
    （2）手工位图边框扩充
    打开“扩充位图边框”对话框。在此对话框中，用户指定为扩充位图边框的准确尺寸，并且控制纵横比。
    6.位图颜色遮罩
    打开“位图颜色标记”卷帘窗。在该窗口中可以指定在位图隐藏的内容，此内容可多达十种，其中隐藏颜色可以使位图后面的对象或背景显示出来，从而改变位图的外观。 该卷帘窗的控件还可以调整颜色容限设置，定义颜色剪贴敏感度，以及保存位图颜色遮罩以供将来使用等。
    7.解除链接
    移除图形中导入图像的链接。
    8.从链换更新
    更新图形中具有链接的位图。
    9.二维效果
    使用下拉列子命令控制位图的二维效果：
    （1）描边
    给位图上的颜色添加轮廓。该命令在处理时将查找位图边界像素，并将它们转换为单色背景上的线条，此线条即可成为位图上各颜色像素区域的轮廓，按不同的颜色像素，将会有不同的效果。
#1    注：在包含文本的高对比度位图中可以取得最佳的效果。
    （2）偏移
    根据指定的值移动位图。移动位图后，它以前所在的位置将成为空白区，该命令允许用户对该空白区，或者位图的其它部分进行填充，并且可以指定填充的颜色。
    （3）像素化
    将图像划分成若干个方形或者矩形、圆形单元。可以使用方形或矩形来为像素创建放大的、块状的数字外观，使其看上去更加平滑，或者使用圆形选项来创建如蜘蛛网那样的效果。
    （4）漩涡
    根据选定的方向和角度使位图变形。此命令能够让图像围绕一个固定的中心点，沿顺时针方向或逆时针方向旋转。并且可以指定旋转的角度值。
#1    注：用户可以控制“完全旋转”角度值。此值越小，漩涡效果越明显；值越大，同轴的旋涡效果越明显。
    （5）湿笔画
    创建使位图看起来象湿的绘画作品那样的效果。 
    10.三维效果
    使用下列子命令给位图添加三维效果：
    （1）三维旋转
    水平和垂直地旋转位图，让位图看上去就象是三维盒子的一个面。用户可以设置水平与垂直旋转的角度值，并且可以预览旋转的结果。
    （2）浮雕
    创建三维浮雕效果。此效果将使位图的细节具有平面上的三维凸起和凹陷的外观。
#1    注：在具有较高对比度的位图上应用“浮雕”效果，可以产生最显著的效果。
    （3）卷页
    创建位图边角回卷的效果。操作时，用户可以指定边角、方向、卷页的大小和透明度。    （4）透视点
    使位图具有三维透视下的纵深感。此效果有两种模式：“透视”和“修剪”，前者将根据位图四个节点位置计算出一段的移动值，以此为位图创建三维外观；后者也应用透视效果，但可以保持位图的原始大小和形状。
    （5）挤远挤近
    通过“挤远”或“挤近”的方式来使位图产生三维效果。
    （6）映射到对象
    创建将位图包裹在球体，或者垂直与水平圆柱体上的效果。
    11.模糊
    使用下列子命令对联位图进行模糊处理：
    （1）高斯式
    根据高斯分布来模糊位图。该效果可以改善具有尖锐边缘的位图的质量。
#1    注：高斯分布使用的是一条钟形曲线，此命令能按该曲线向外扩展像素信息。     
    （2）动态
    在位图中创建运动效果。用户可以指定运动方向，以及运动的距离。运动的距离越远，位图就会越模糊。
    （3）平滑
    平滑位图。此命令能够减小相邻像素的差别，使得在对位图进行平滑处理时只有轻微的细部损失。
    12.杂点
    “杂点”是位图编辑中所加入的一些随机像素点，类似于电视机屏幕上的静电干扰，使用下列子命令可以控制它们。
    （1）添加杂点
    加入“杂点”，创建粒状效果。
    （2）移除杂点
    柔化位图并减少在扫描或视频捕获过程中可能产生的杂点。此命令会将把每一个像素点与其周围的像素进行比较，并计算出平均值，如果像素的亮度值超过使用滑块设置的阈值，该像素就将被移除。
    13.鲜明度
    “鲜明度”控制着位图的像素显示效果，通常“鲜明”可以突出像素的边缘，使用下列子命令可以控制鲜明化效果：
    （1）鲜明化
    查找位图边缘像素，并增强它与相邻或者“背景”像素之间的对比度，以此来突出位图边缘。
    （2）非鲜明化遮罩
    突出位图的边缘细节，并全使得一些模糊区域变得清晰。
    14.艺术效果
    用户可以为位图增加一些额外的艺术效果，通过下列子命令可以完成此操作：
    （1）玻璃块
    建立透过一些玻璃块来查看图像的效果。用户可以设置每一块玻璃的尺寸，这是以像素为单位的高度与宽度值，较小的值会产生轻微的像素化效果，较大的值则可以产生钻石般的玻璃图样。 
#1    注：中文CorelDRAW 8推荐使用25到75之间的值来取得最佳的效果。
    （2）印象派
    将位图转换为小的纯色块，使得具有印象派绘画的外观。
    （3）虚光
    使用一种“虚光”来创建位图四周的边框，进而产生梦幻般的、怀旧的效果，或者给旧照片添加椭圆的边框。
    15.颜色转换
    使用下列子命令转换位图的颜色：
    （1）梦幻色调
    将位图的颜色改变为明亮的、绚丽的颜色，比如桔黄、鲜艳的粉红、青蓝、橙绿等。使用较小的数值可以产生一些有趣的效果。
    （2）曝光
    将位图的颜色变换为类似于摄影负片的颜色，将这种效果应用到彩色位图时将更为明显。
    （3）编辑双色调
    打开“双色调”对话框，编辑有关的设置，使位图转换为双色调。
    16.外挂式过滤器
    在中文CorelDRAW 8中包含有数十个过滤器中，其中有些是其他软件开发者提供的的，从此命令中可以访问它们。一旦用户安装了这些过滤器，它们就会出现在此。
#1    八、文本
    这一组下拉菜单用于控制使用文本。
    1.格化式文本
    打开“文本格式”对话框，定义文本的格式，或者编排其属性。
    2.编辑文本
    打开“文本编辑”对话框，修改或者编辑文本，以及访问各种格式编排选项。
    3.立体化文字 
    创建三维文本。
#1    注：只有在选择“美术字文本”时才可以使用此命令。
    4.使文本适合路径
    沿选定的非文本对象的轮廓成形美术字文本。这样的文本和路径将成为动态链接的群组，以便于编辑文本或者改变路径的形状，并且自动使文本重新嵌合。
    5.使文本适合框架
    增大或减小段落文本框中文本的磅值，即文字的大小尺寸，以便让文本恰好适合于文本框的尺寸。
    6.对齐基线
    文字的基线位于文字的底部，此命令能够使选定文本中的字符与基线对齐，同时保持各字符的角度与水平间距。
    7.使文本对齐基线
    将移动了的文字对齐基线。
    8.书写工具
    使用下列子命令辅助书字文字：
    （1）拼写
    打开“拼写检查器”对话框，检查图形中拼写错误的文字、重复的文字和大小写不规范的字。
    （2）语法
    打开“语法”对话框，以检查图形中拼写的语法和标点符号是否有错误，并且帮助用户修改它们。
    （3）同义词
    打开“同义词”对话框，显示同义词、反义词和定义列表，以便帮助用户进行替代操作。    
    （4）键入辅助
    打开“选项”对话框中的“键入助手”页，指定替换文本。
    9.更改大小写
    打开“更改大小写”对话框，控制文字串中字母的大小写形式。
    10.使文本与HTML兼容
    将标准的“段落”文本转换为与“超文本标记语言（HTML）”兼容的文本，以便于直接在因特网浏览器中编辑已转换的文档中的文本。如果在将文档转换为因特网文件格式之前没有将“段落”文本转换为HTML文本，那么文本将被转换为位图进行发布，并且不能在浏览器中进行编辑。
    11.转换
    将“美术字”文本转换为“段落”文本，或者将“段落”文本转换为“美术字”。
    12.显示非打印字符
    显示文本模式下的非打印字符。这些符号如制表符、段落标记和空格符等。
    13.文本统计信息
    打开“统计信息”对话框，统计图形中出现的文本元素的行数、字数、字符数以及使用的字体数等信息。 
#1    九、 工具
    这一组下拉菜单提供有一些辅助操作的工具。
    1.选项
    打开“选项”对话框，设置运行中文CorelDRAW 8时一些默义选项，以及复制对象的位置、备份的时间间隔、是否要备份、可以撤消的操作数等操作方式。
    2.创建
    使用下列子命令建立新箭头、图样、符号：
    （1）箭头
    打开“创建箭头”对话框，使用选定的线条来创建箭头和线条端头形状。
#1    注：新创建的箭头将被添加到“轮廓笔”对话框和“笔”卷帘窗中。
    （2）图样
    打开“创建图样”对话框，创建自定义的双色和全色位图图样填充。
#1    注：新创建的图样将被添加到的对话框的列表中，通过“填充”工具展开式按钮中的各种图样填充图标可以访问它。
    （3）符号
    打开“创建符号”对话框，将选定的对象添加到指定的“符号类别”中。
#1    注：新创建的符号将被添加到“符号”卷帘窗中。
    3.从选择新建调色板
    创建并保存新的调色板。
    4.从文档新建调色板
    创建并保存包含当前文档中所用的新调色板。
    5.调色板编辑器
    建立新的调色板、修改编辑已有的调色板。
    6.脚本
    此命令适用于中文CorelDRAW 8的高级用户，对初、中级用户无用。该命令通过下列子命令控制使用“脚本”：
    （1）脚本和预设管理器
    打开“Script 和预设管理器”，记录、编辑并执行 Script 和预设。
    （2） 运行脚本
    打开“运行 Script”对话框，执行Corel SCRIPT script文件。
    （3） Corel SCRIPT 编辑器
    打开“Corel SCRIPT 编辑器”，它是用来创建和编辑 Corel SCRIPT script 文件的工具。
#1    十、窗口
    这一组拉菜单用于控制操作窗口。
    1.新建窗口
    建立一个新的图形窗口，而且是一个与当前“绘图窗口”内容相同的新窗口，以便于从不同的角度来查看图形。
    2.层叠
    以层叠方式排列打开的所有窗口，以便于看到每一个窗口上方的“标题栏”。由于单击“标题栏”可以激活绘图窗口，拖动它可以移动绘图窗，因此层叠的用途是很大的。
#1    注；最小化后的窗口将被排列在“应用程序窗口”的底部。
    3.水平平铺
    显示打开的所有窗口，并以平行的方式排列它们。
    4.垂直平铺
    显示打开的所有窗口，并以垂直列的方式排列它们。
    5.排列图标
    关闭当前窗口。
    6.关闭
    关闭所有的打开的绘图窗口。
    7.全部关闭
    关闭所有的打开的绘图窗口，包括那些最小化窗口。
    8.刷新窗口
    刷新屏幕，清除以前操作时标记。
#1    注：在此命令的下方会列出了当前所有绘图窗口的名称，单击其中的某一项，可以将此绘图窗口置为当前窗口，以便对它进行操作。
#1    十一、帮助
    这一组下拉菜单用于进入中文CorelDRAW 8的帮助功能。
    1.帮助主题
    显示CorelDRAW “联机帮助”系统的内容。
    2.这是什么
    选择此命令后，当前光标的右旁将出现一个问号，使用它单击屏幕上的某一个控件时，一条有关的“帮助”信息将显示出来。
    注：按下键盘上的Shift + F1功能键，也将进入此功能。
    3.CorelTUTOR
    进入CorelTUTOR教程笔记。
    4.提示
    进入“CorelDRAW 提示”功能，以便查阅有关当前所执行操作的有用提示。此功能提供了一些关于使用中文CorelDRAW 8的有用信息，对初学者是很有用。
    5.技术支持
    打开“CorelDRAW 技术支持帮助”文件。此文件提供了有关Corel应用程序的产品支持详细资料，包括可得到的技术服务、“导入”和“导出”过滤器信息、错误信息和故障排除提示等。
    6.网上Corel
    提供了关于在因特网网上获取帮助的有关信息与网站，并且让用户通过因特网网的浏览器获取它们。 
    7.关于Corel
    显示中文CorelDRAW 8软件的注册系统和版权信息。
