Windows 98 新增功能总汇

金伟
1998年 第51期 09版

#1    一、增加的系统工具
    1.磁盘整理优化向导
    该功能用于减少频繁使用程序的装入时间，它通过使用注册文件，并重新将其写回连续磁盘空间。这样可消除对硬盘的频繁访问，试验证明该功能对转换为FAT32的系统尤其有效。
    2.Windows系统升级
    该功能使用ActiveX技术，通过Internet下载新文件来自动更新用户的系统文件。
    Windows系统升级能尽可能及时更新用户的系统，它通过和站点联系，检查需要更新的文件并做更新。
    具体的原理是先把所有的升级程序、修正版本、新的驱动程序、补丁程序等放在站点上，然后再利用ActiveX控件为每个用户筛选。
    要使用该功能，首先运行HelpDesk，单击“升级程序连接”，系统自动让你连接Internet，然后到升级站点。首先它会检查计算机中的软件和硬件（检查注册表和配置），找到对应的数据库，然后会提供最近建立的一些文件的名单，解释它们的功能，并询问是否下载和安装它们。
    一旦使用过了升级站点，就会出现第三个站点，历史详细记录曾经作过的线上操作。也可以改变所做的更新，恢复原来的配置。 
    3.系统文件检查功能
    该程序跟踪对重要系统文件的修改，允许用户恢复以前的设置文件。
    计算机用户都必须经常备份，尤其是系统文件。经常要安装一些软件，在安装的过程中不可避免要修改系统文件尤其是注册表，如果一旦安装出现问题或所安装的软件发生冲突，没有系统备份会有重大损失。
    “System File Checker （SFC，系统文件检查功能）” 自动保存关键的Windows 98系统文件的版本信息，监视.DLL、.COM、.VXD、.DRV、.OCX、.INF、.HLP、.SCR、.EXE、.SYS 和386 文件——在一个SFC 文件中，这样就可以通过单击“恢复缺省”按钮恢复系统了。
    还可以添加其它需要的文件扩展名，或者选择要监视的文件夹，缺省时只检查 “＼WINDOWS＼ACCESSORIES” 和 “＼WINDOWS＼SYSTEM folders”。
    该程序还可以自动解开安装文件压缩包，可以选择“Extract one file from installation disk”（从安装盘解压缩文件），然后单击“编辑”查找相应文件。
    当在检查系统文件时，应用程序会根据数据文件检查当前文件夹的路径，以便保存所做的修改或者升级新版本。如果SFC能显示文件怎样被修改就更好，这样可更方便地决定是否恢复原状。应用程序还提供了两个小方法：在审查SFC文件之前先备份配置文件；可以保留注册文件，看看是哪里出了问题 。
    4.Windows调整向导
    该功能规划日常的维护工作（例如扫描硬盘、整理硬盘）。
    Windows调整向导的任务是规划日常的存储维护工作。它会根据制定好的计划自动运行SCANDISK 、去除无关的文件、整理硬盘。例如可以规定是每天、每周还是每月运行一次SCANDISK，此外，还可以根据不同的时间制定启动顺序，在高级选项中，可以在不同的时间运行不同的启动顺序。
    更好的是，还可以选择禁止某种操作的时间，例如假设硬盘碎片化很明显了，例行的硬盘整理要花费很长的时间，这时可以规定所有程序都只能在8：30以后运行。
    5.Dr.Watson功能
    该功能用于观察Windows 98和应用程序运行的情况，记录对WLG文件的修改，当程序发生冲突后，能够截断软件所产生的错误，并显示产生错误的原因、过程等详细资料。
    6.新的备份功能
    该功能现在支持SCSI类设备。
    Windows 98 附带的备份程序是Seagate Software希捷软件的备份程序的简化版，在1997年曾经获奖。
    该程序较以前的备份程序最大的改进之处在于兼容性增强了。根据测试报告，新程序支持并行、IDE/ATAPI 和SCSI 设备， 包括 QIC-80， QIC-80 Wide， QIC-3010， QIC-3010 Wide， QIC-3020， QIC-3020 Wide， TR1， TR2， TR3， TR4， DAT （DDS1， 2， & 3）， DC 6000， 8mm and DLT。 此外，报告特别说明支持下列厂商: Conner， Exabyte， HP/Colorado， Iomega， Micro Solutions， Seagate， Tandberg， WangDAT and Wangtek。 
    运行备份程序，接着出现的对话框种有五个选择：使用备份向导；运行备份程序；打开一个已存在的任务；运行恢复向导；开始恢复。
    在“高级”栏可以设置“文件确认”、“压缩选项”和“口令”等。
    有趣的是，还可以在这里备份注册表，单击“选项”，选择“高级”，如果将“备份注册表”设为默认值，那么每次备份文件时都会备份注册表。
    7.新的可访问的工具
    这个工具为高级用户设计，便于它们修改Windows以适合他们的要求。
    Windows 98 中的这个工具比Windows 95中的更容易访问，它现在已经是Windows 98“开始/程序”菜单中的默认选项了。它有两个性能：第一个是 Windows Accessibility Wizard，知道如何使用 that steps you through the process of turning on the Accessibility features you want.
    它提供五个选项： “Items are too small in Windows”，“Seeing colors in Windows”，“Hearing sounds in Windows”，“Using the keyboard” and “Using or seeing the mouse”。你可以选择其中一些或所有选项。
    除了使用“开始”菜单，还可以通过以前的方法使用该功能：通过控制面板，选项中有键盘、声音系统、显示和鼠标。
    唯一的一个全新的选项是在“开始”菜单中的修改器，它可以将光标所在的屏幕放大2到12倍。
    8.FAT32
    新的操作系统所带的改进的FAT文件系统，能更有效地利用超过2GB容量硬盘空间，系统还包括了一个图形界面的转换程序用于新旧格式的转换。
    9.远程访问服务器
    Windows 98 直接在操作系统上建立远程访问服务器
#1    二、帮助及技术支持
    1.Windows 98的帮助
    这个增强的帮助压缩包可以使用户不但和本地机相连，还能连接网络资源。
    这个新的 Windows 98 HelpDesk 直接在操作系统中建立技术支持。
    HelpDesk 加入进了“开始”菜单，它所连接的HTML文件提供三个水平的支持：本地帮助、网络资源技术支持。
    2.系统调解工具“System Troubleshooter”
    该工具自动执行一系列的调解步骤解决设置文件的问题。
    以前使用AUTOEXEC.BAT， CONFIG.SYS， SYSTEM.INI and WIN.INI 进行系统设置。System Troubleshooter 可以在WINDOWS 98启动时选择击活那一个进程，让你单独解决某一个检查出的问题。
    用户还可以选择装入/不装入启动程序组中的程序，这对于因为故障要反复启动的用户非常有用。
    用户可以在这里禁止ScanDisk运行， 或限制可访问内存仅为4MB。 
    System Troubleshooter 提供编辑器用于编辑 AUTOEXEC.BAT， CONFIG.SYS， SYSTEM.INI and WIN.INI 文件。选择其中一个，可以编辑每一行并调整它们的位置，或通过选框禁止，此外还可以确定执行顺序。
    3.系统信息功能System Troubleshooter
    该功能通过搜集系统信息帮助个人技术支持。
    系统信息功能 System Information （MSI）utility version 4.1 初看起来有些像某些Microsoft应用程序中系统信息功能的老版本，但是从工具栏上看，多出了许多内容，包括ScanDisk， the Registry， ScanReg 和 Dr.Watson。
    Windows 98的 MSI ScanDisk 增加了许多选项，例如检查由于文件名长度产生的错误和报告重复文件，它现在不只是在磁盘发生问题后让用户选择是否检查硬盘，只要用户存在硬盘错误或不正确关机，它会自动运行。如果用户想禁止WINDOWS 98中SCANDIAK，就必须进入MSI的另一部分System Troubleshooter。
    MSI报告的屏幕范围超出了 Windows 95的控制面板，增加了报告设备冲突、定义各种不同资源占用的空间等。 
    MSI 的工具菜单可以显示和排序注册表中的信息，特别的是显示每个OLE元素正在被哪些应用程序使用，例如，它包含了一个按钮，用于将进入另一个编辑器所经过的路径拷贝到剪贴板，同时还可以将路径输出到一个文本文件中。 
    MSI的ScanReg tool 可以对注册表做快速备份和扫描，显示潜在的问题。
    即使其它的都失败了，Dr.Watson 还在。
#1    三、所支持的硬件
    1.广播结构
    Windows 98允许用户使用装有TV调节器的主板从广播网中接收电视和其它数据信号，此外还提供了一个不断更新的程序，用户可以从中直接访问程序，Windows 98的用户可以直接从广播网中接收Internet内容。
    2.Misc Hardware支持
    Windows 98同样支持USB、图形加速卡AGP、ACPI（对笔记本用户尤其有用）。
    3.支持多显示器
    Windows 98允许在一台计算机接八个显示器，在一个桌面中使用它们，这意味着在显示器之间可以象在窗口、文件夹之间一样拖动程序、重新定义桌面元素的大小等。
    使用多显示器不需要特殊的显示卡，只要每台显示器都配有一个标准显卡，类型不必完全相同。
    有了Windows 98的支持，每台显示器可以设置为不同的分辨率和颜色深度。
    打开的窗口（但不是最大化）可以在显示器之间移动，甚至可以跨越中间的显示器，但如果窗口最大化了，就只能在一个屏幕中，不能在所有的显示器中显示最大化的程序和文件夹。
    Windows 98会把其中一个显示器作为主显示器，一些较老的文件只能在主显示器中运行，在默认状态下，任务栏和桌面只能在主显示器中显示。
    4.支持PCMCIA
    Windows 95增加了对PC Card32，3.3V长和多功能卡（例如网卡）的支持。

#1    四、多媒体
    1.Active Movie
    该格式大大增强了Windows 98处理常见多媒体文件的功能。
    2.支持MMX
    Windows 98加入了Intel MMX技术。
    3.Misc.Enhancement
    安装更方便。微软试图通过把Windows 98中包含的安装选择自动化，建立安装流水线，用户只有在安装后通过访问控制面板以修改系统。
    4.更快的关机
    Windows 98通过消减不必要的程序提高了关机速度。
#1    五、Internet连接
    1.集成Shell
    Windows 98中的IE4.0安装后包括了一个选项可开启活动桌面，使IE成为Windows 98桌面的一个集成部分，通过选择活动桌面，用户可以将桌面的墙纸变为可以直接从网上获取信息的活动的HTML页面。从更深的意义上看，通过将传统的Windows资源管理器变换为Internet浏览器改变了Windows界面。任务栏发生变化，包括了地址栏和链接、收藏加入到了“开始”菜单，以便访问更方便。
    2.改进的拨号网络
    Windows 98改进了拨号网络，可以使用script，支持混合ISDN或Modem线路提高了传输速度。
    3.改进的编辑器
    Windows 98中的IE提供了许多对新的因特网标准的支持，包括动态的HTML，还提供了略有不同的界面，功能有一些改变，例如改进了历史性能后，允许用户修改缓存中的文件。
    4.频道
    IE4.0使用“推”技术允许用户能够通过离线浏览自动接收Internet信息。
    5.附加的Internet工具
    Windows 98还带了几个附加的Internet工具，包括Outlook Express（一个改进的邮件程序，支持HTML格式和多个POP3帐号），Net Meeting（网上会议系统）、Front Pad（Front Page 97的简化版）和个人网络服务器。
    6.支持PPTP
    PPTP（点对点通道协议）允许用户秘密地通过网络连接服务器。
