上网技巧荟萃

无忧
1998年 第52期 17版

#1    一、上网省钱妙招
#1    1.养一只好“猫”
    用户最好选择速度块、功能强的Modem上网。由于用户端Modem和ISP端Modem二者中最低速度是用户上网的最高速度，故用户端的Modem应选择速度高的。
    功能强的Modem可以自动检测线路的好坏，且可以根据线路情况降低和升高传输速率。反之，在使用时会经常断线。因此用户端的Modem还应选择功能强的。
#1    2.让MODEM保持最佳状态
    每天上网的朋友，是否调整过调制解调器，也让它正常运作呢？
    查询电话线状况：在买了一个全新的调制解调器、更新驱动程序、完美地设定浏览器的情况下，还是没办法拥有好的连线状况的话，请试着问问电话公司目前的电话线状况。其实电话线路状况好坏对连线有着深远的影响。如果这个问题一直持续的话，就可能在保护传真或文件传送时线路防干扰的设施上花心思。
    考虑装设ISDN：在电脑里装上ISDN介面卡，会比一般外接连线配备更为迅速。
    尽量选择合适的时间上网。上网速度慢并不是Modem速率的问题，而是与要访问的网络站点是否繁忙或使用同一出口的人数的多少有关。能在网络上玩在线游戏或是在非高峰时段在网络上冲浪，真的是件非常高兴的事。想躲开网络塞车惨状的话，国内站点就要在深夜或凌晨上网，最糟糕的时段就是下午，国外站点（如美国）可以选择在白天上网。但各个地方不一样，只有靠网友们自己摸索。
#1    3.计算机硬件和软件环境
    原则上讲，计算机性能越高，浏览器软件版本越高，用户浏览速度越快。用户最好选择高档次的计算机和高版本的浏览器，这样浏览速度将是较快的。
#1    4.拨号电话线的质量
    正常情况下，接一个普通的Modem，通信速度应能达到Modem本身的速度。但可能由于通信线路老化或局与局之间的转换设备已老化，串音、噪音干扰等问题严重影响线路的质量，从而限制了用户拨号上Internet网的速度和通信速率。用户应选择线路质量比较好的电话上网。
    一般选择出口带宽最大、线路速度最高和费用合理的ISP（网络服务商）。
#1    5.加快Windows 95上网速度
    （1）对系统的端口参数进行设置：进入Windows 95，单击“开始”按钮，选择设置中的“控制面扳”，双击“系统”图标，选择“设备管理”标签，单击“端口”左边的“+”号，双击连接MOdem的端口COM1或COM2，选择“端口设置”标签项，将波特率设为“921600”，数据位设为“8位”，无校验，停止位设为“1位”，控制流改为“硬件”，单击“高级”按钮，选择“使用先进先出缓冲方式”复选框，然后将将接收缓冲区和发送缓冲区条拖到最大，确认后返回即可。
    （2）对端口缓冲区进行设置：可以用编辑器（如Edit）修改Windows 95系统目录下的SYSTEM.INI文件，在［386ENH］段下加入ComXBuffer=1024（X=1或2，由MODEM连接的端口而定）。
    （3）执行Regedit.exe修改注册表SYSTEM.DAT文件的MTU值为576。
    （4）执行Regedit.exe修改注册表Maxsockets的max值提高到1020。
    以上设置，可以手工完成，也可以利用一个名为“MtuSpeed”的软件实现。注意：在修改之前，请记下原来的值，万一出现问题，请恢复原来的值。
    设置完毕，重新开机，你可连上网络享受高速的连接。
#1    6.同时下载多个软件
    Getright下载软件稳定且速度快。可以利用GetRight设置中的“Limits 标签”，将允许同时下载的软件数设定为一个恰当的数值。但不要把参数设定太大，否则会占用大量的内存，导致系统速度下降。一般将此参数设为10足以满足需要，也不会浪费太多的资源。这样，你可以用GetRight同时下载多个软件。
#1    7.离线撰写和回复电子邮件
    在使用IE的Outlook Express撰写和回复电子邮件时，不用拨号上网，撰写好邮件后，不是立即“发送”，而是选择“文件（F）”下的“以后发送（L）”命令，将邮件存放在“收件箱”中，直到下一次选择“发送和接收”命令时才发送出去。这样可以节省上网费用。
#1    8.下载心得
    对个人用户来说上网直接下载是获取最新软件最快捷的途径，但同时也会碰到一些问题，诸如下载过程中的意外断线，选择连接的站点速度太慢等，要改善这些情况，你可以按如下办法：（1）充分利用WWW上的软件搜索引擎（如http:／／www.softseek.com、http:／／www.tucows.com、http:／／www.shareware.com、http:／／www.download.com等）；（2）浏览软件制作公司的主页，这样你才能了解到关于这个软件最新的专业信息，并保证可以下载到它的最新版本；（3）挑选最优秀的下载软件（如GetRight、Go!zilla等）；（4）选择合适的时间段；（5）下载和浏览同时进行。
#1    9.使用代理服务器
    代理服务器，即Proxy服务器，在互联网上的完成跑腿服务。当你在浏览器中设置了某个Proxy服务器之后，由你的浏览器所发出的任何要求，都会被送到Proxy服务器上去，由这台Proxy服务器代为处理。
    设置Proxy服务器的最大好处是可以加速网络的浏览速度。这是因为当许多人都把Proxy服务器指向同一台时，所有的需求都会经由这台Proxy服务器来代为处理，当有人在网址上看过某一个Web页面时，这些内容都会被记录在Proxy服务器的硬盘缓冲区（Cache）中。等到下一次你要浏览相同的网页时，这些文件直接由Proxy服务器送到你的电脑。
#1    注意：硬盘缓冲区（Cache）不能太大，否则影响服务。
    设置Proxy服务器的另一个好处是：若你没有指定DNS（域名服务器）时，它也能解决IP地址的对应问题。
#1    10.最新版本的浏览器
    使用最新版本的浏览器（测试版除外），可提高浏览速度，有意想不到的收获。
#1    二、微软探索者（IE）浏览技巧
#1    1.调整工具栏
    Internet Explorer提供了对自己屏幕最大的可调整性，菜单拦和工具栏都可以用鼠标的拖放操作来组合在一行中，当多个工具栏位于同一行时，可以通过左右拖动鼠标指针来显示隐藏的工具栏。
    移动工具栏的方法是：将鼠标指针移到工具栏左侧单条处，直到鼠标指针形状变为左右两个尖头壮；按住鼠标的左键，此时可一上下（或左右）拖动工具栏调整其位置和大小。
#1    2.更多Web画面
    你可以单击“查看（V）”菜单中的“全屏（F）”命令，浏览方式变为全屏浏览。或通过Internet Explorer “查看（V）”菜单中的“工具栏（T）”和“状态栏（S）”命令，关闭文本标签、工具栏和状态栏的显示，以获得更多Web画面。
#1    3.翻出过去历史 
    Explorer会记录使用者最近曾浏览过的网站，但数量不只是“地址”下拉清单里面那几个。想一次调出所有的，请单击工具栏上的“历史”按钮，开启使用“历史记录”列表。
    当再按一次工具栏上的“历史”按钮时则关闭“历史记录”显示列表。
#1    4.查看历史细节
    在前述的“历史记录”列表中，单击某一天浏览的某个网址。则可进入访问过的历史网站。
#1    5.改变历史
    Explorer默认只储20天内浏览过的网站。要变更这个时限，请单击菜单栏中的“查看（V）”，选择“Internet 选项（O）…”进入设置，在“常规”选项卡的“历史记录”框中改变保存多少天内的访问记录。但注意：过大的历史文件夹个过长的历史记录，不但会消耗大量的磁盘空间，而且严重地影响浏览速度。
#1    6.抹去历史 
    要删除“历史记录”，在“改变历史”的设置框中按“清除历史记录（H）”即可。此时连“地址”下拉清单中的历史记录也会一并消除。
#1    7.好站共享
    假如你发现一个好的网站，欲告诉你的朋友，只须先进入这个网站，再执行菜单栏的“文件（F）”／“发送（E）”／“电子邮件页面（P）…”，进入编写和发送电子邮件视窗，填写好“收件人”的E-mail地址，按“发送”按钮寄出电子邮件即可。
#1    8.好的开始
    没有谁规定每次浏览都要先从微软的网站开始，除了对Explorer进行设置外，你可以先进入最想做为起点的网站，再执行菜单栏的“查看（V）”／“Inetrnet 选项（O）…”，在“Inetrnet 选项”视窗的“常规”选项卡的“主页”框中，单击“使用当前页（C）”即可。
#1    9.自订搜寻引擎
    在 Internet Explorer 中单击工具栏中的“搜寻”按钮，打开“搜索”浏览器栏，会带领你到Microsoft所预设的搜寻引擎，在文本框中输入关键词，选择一个搜寻引擎，单击“搜索”按钮开始搜索。假如有其他更好的搜寻引擎，请先进入该更中意的搜寻引擎网站，再执行菜单栏的“查看（V）”／“浏览器栏（E）／搜索（S）”。
#1    10.画面不完整
    无法看到Web画面的文字时，问题可能是出在字型上。可以到“Internet 选项（O）…”中进行修改，或可单击Exporer工具栏上的“字体”按钮进行字体调整。
#1    11.大字报
    想要将文字放大以便阅读时，可单击Exporer工具栏上的“字体”按钮进行字体调整。
#1    12.清除快取（临时）空间
    遇到硬盘爆满而无法进入网站，此时可能是临时文件太多了。消除临时文件可以回收几十兆的硬盘空间，只须执行“查看（V）”菜单中的“Internet 选项（O）…”命令，在“Internet 选项”视窗的“常规”选项卡下的“Internet 临时文件”框中单击“删除文件（F）”按钮即可清除快取空间。
#1    13.增加快取（临时）空间
    假如你的硬盘空间足够大，可以拿一部分来做为Internet Explorer 4 的快取空间。执行“查看（V）”菜单中的“Internet 选项（O）…”命令，在“Internet 选项”视窗的“常规”选项卡下的“Internet 临时文件”框中选择“设值（S）…”，在“设置”视窗中再选择“移动文件夹（M）”，此时便可决定新的快取（临时）文件放置位置。
#1    14.降低快取（临时）空间
    Internet Explorer 4 的临时文件会膨胀到占满硬盘的10％或是更多，假如你愿意牺牲一点连线效率以换取更多储存空间，可以到上述（即13）“设置”视窗中将“Inetrnet 临时文件框”中的“可用的磁盘空间（A）”的百分比滑杆调整为自己可忍受的限度。
#1    15.善用快取（临时）空间
    假如硬盘空间足够，扩大快取空间可增加浏览Web的效率。此时你可考虑在“14”的设置中加大快取使用硬盘的百分比。
#1    16.等得不耐烦
    等待网站慢慢载入而难以忍受，请按下工具栏上的“停止”按钮即可。使用键盘上的Esc键更方便。
#1    17.使用收藏夹
    要想使用“收藏夹”，只须执行“收藏（A）”功能菜单表中的各个命令。
    将自己喜爱的网站加到“收藏夹”中，只须执行“收藏（A）”功能菜单表中的“添加到收藏夹（A）…”命令。但更快的方法是在你中意的网页空白处按下鼠标右键，再选择“添加到收藏夹（F）…”命令。想放入其他文件夹时，只须在“添加到收藏夹”视窗中选择“创建到（C）>>”按钮，再选择文件夹后按“确定”即可。
    增加收藏夹类别：想在收藏夹中增加类别时，只须按前述的“创建到（C）>>”按钮后，再单击“新建文件夹（W）…”按钮，通过文件夹分类。
    管理收藏夹：想管理“收藏夹”清单，请执行“收藏夹（A）”菜单下的“整理收藏夹（O）…”命令，进入“整理收藏夹”视窗后，你可以进行“删除”、“移动”、“重命名”等操作。在选择某个资料夹或类别后，再按“打开”，此时如同在Windows的桌面一样，你可在各文件夹之间进行各种操作。
    对于经常访问的主页，在桌面上创建其快捷方式也是一个好办法。用鼠标右键单击网页，然后单击“创建快捷方式（T）”，当出现提示“指向当前页的快捷方式将放置在桌面上”时，按“确定”即可。
#1    18.快速查看目录
    无需离开Internet Explorer 4 便可查看电脑内的文件，只要在“地址”栏空格内输入如C:＼即可列出目录清单。同样也可以使用Explorer的脱机访问功能。
#1    19.加速Java
    要让Java程序性能更高，只须启动JIT编译器。执行“查看（V）”菜单中的“Internet 选项（O）…”命令，在“Internet 选项”视窗的“高级”选项卡下，选中“Java VM”下的“启动Java JIT 编译器”复选框。
#1    20.彻底加速
    当你要享受极速浏览，可以关闭所有多媒体功能，如图形、音效、视讯等。执行“查看（V）”菜单中的“Internet 选项（O）…”命令，在“Internet 选项”视窗的“高级”选项卡下，取消“多媒体”下的“显示图片”、“播放声音”、“播放视频”、“播放动画”等复选框。
#1    21.不再等待载入
    不想等待网页中的图片慢慢载入，只须按方向键中的向下键，即可以看到其他文字的内容，或用拖动滚动条查看Web页面内容。
#1    22.查看用不同语言编写的网页
    假如你想查看其他国家的网站，可以向微软索取各国字集。添加语言的过程为：在浏览器的“查看（V）”菜单下，单击“Internet 选项（O）…”进入“Internet 选项”视窗；选择“常规”选项卡，单击“语言（L）…”按钮，进入“语言首选项”视窗；单击“添加（A）…”，选定要添加的语言，然后按“确定”。
    安装之后，可以按Internet Explorer 4 工具栏中的“字体”按钮来选择“语言”；或执行“查看（V）”菜单下的“字体（N）”命令；或在网页空白处按下鼠标右键，再选择“语言（L）”命令。
#1    23.编辑当前网页
    可以按Internet Explorer 4 工具栏中的“编辑”按钮，启动FrontPage Express，编辑当前的Web页面。
#1    24.发掘网页秘密（查看当前网页的HTML源文件）
    假如你想知道某个网页如何撰写而将其复制下来，可以在该网页空白处按鼠标右键，再选择“查看源文件（V）”选项，或执行Internet Explorer 4“查看（V）”菜单下的“源文件（C）”命令。你可以将该页存储，然后在FrontPage Express中进行编辑和修改，编辑完后，还可以在Internet Explorer 4 中打开，查看所做的改动。
#1    25.保存主页信息
    网上有许多非常有用的信息，用户必须将它存储，以便日后参考或与他人共享。你可以保存整个网页，也可以保存其中的部分内容（如文本、图形或链接）。
    保存当前页面（文本）：在Internet Explorer 4的“文件（F）”菜单上，单击“另存为（A）…”命令，然后双击用于保存网页的文件夹。在“保存HTML文档”视窗的“文件名（N）”框中，键入该页的名称，然后单击“保存（S）”就可以将其保存下来。
    好图大家用：在网站中看到的任何图形，都可以将它存储或变成Windows的桌面背景，只须在图形上按鼠标右键，若选择“图片另存为（S）…”选项，然后双击用于保存图片的文件夹。在“保存图片”视窗的“文件名（N）”框中，键入图片的名称，然后单击“保存（S）”，则可以将其保存下来；若选择“设置为墙纸（W）…”，则将该图片设成桌面墙纸。
    不打开网页或图片而直接保存：用鼠标右键单击所需项目的链接，在快捷菜单中单击“目标另存为（A）…”。在“另存为”视窗的“文件名（N）”框中，键入该项的名称，然后单击“保存（S）”就可以不必亲自前往储存该网站中文件。
#1    26.安全至上
    鼠标右键还可以查询各网页的安全设施，在Internet Explorer 4中的“Internet 选项”设置视窗“安全”选项卡进行设置。
#1    27.同时使用多个窗口
    在Internet Explorer 4 中，可执行“文件（F）”菜单下的“新建（N）／窗口（W）”命令打开一个新的浏览器窗口，同时新窗口显示当前网页的复件。这样若在进行网站搜索时，就可一边观看查询结果，而利用另一个浏览器视窗进入新的网站。
    在新的窗口中打开主页中的超级链接：将鼠标移动到准备访问页面的超级链接上；单击鼠标右键，从快捷菜单中选择“在新窗口中打开（N）…”命令选项。该超级链接在新窗口中打开。
#1    28.多头进行
    不要因为有大型文件正在下载而让所有工作停下来等待。执行“文件（F）”菜单下的“新建（N）／窗口（W）”命令打开一个新的浏览器窗口，同时由于新窗口显示当前网页的复件，就可在新的视窗中继续浏览，原先的下载工作并不会中断。
#1    29.脱机浏览
    通过Internet Explorer 4新提供的“脱机浏览”功能，用户不必连接到Internet就可以查看万维网。只要在“文件（F）”菜单上，单击“脱机工作（W）”，就可以进入脱机浏览工作状态。但在进行脱机浏览前必须将相关内容下载到本机上。
#1    30.打印主页信息
    打开准备打印的页面：执行“文件（F）”菜单下的“打印（P）…”命令，或直接单击工具栏中的“打印”按钮，进入“打印”视窗。如果用户只准备打印主页中的某个框架，则在单击“打印”前，应先在框架内单击一下。
    如果该网页使用框架，在“打印”视窗中的“打印框架”区域选择框架的打印方式。如用户希望按屏幕上显示方式进行打印，就应该选择“按屏幕所列布局打印（I）”。
    如果用户希望同时打印链接到该主页的所有主页，选择“打印所有链接的文档（K）”复选框。该选项在已选择了“按屏幕所列布局打印（I）”选项时则不可用。
    如果希望见到该页左右链接的列表，则选择“打印链接列表（B）”复选框。同样该选项在已选择了“按屏幕所列布局打印（I）”选项后不可用。
    最快地打印一个框架：在该框架内单击鼠标右键，然后选择“打印（I）”命令。
#1    31.安全上网
    Internet上的病毒进入用户计算机的主要途径是用户下载的软件或文件，因此用户下载的软件或电子邮件中的附加文件在使用时必须进行杀毒处理。防止下载软件带有病毒的最好办法是安装网络防杀毒软件，使用防杀毒软件清杀所下载的软件的病毒，做到防患于未然。
    另外，千万不要使用Beta版本的浏览器，因为这些版本的游览器往往有缺陷，或有安全的漏洞，如果你使用Beta版本的游览器，正等于给了骇客们搞破坏的大好机会。
#1    32.处理乱码 
    使用Microsoft公司的IE4浏览器中自带的Outlook Express收发邮件，有时收到的邮件打开一看全是乱码，即使加挂了Richwin for internet或南极星，仍对它无计可施，你可以试以下方法：
    先用鼠标选中乱码邮件，再打开Outlook Express的“文件（F）”菜单，选中“属性（R）”，进入属性视窗，点击“详细资料”标签，再点击“邮件的源文件（M）”按钮，可以看到邮件原文，而不是先前看到的乱码了。对还不能解决的邮件，可用下述办法。
    对于大多数人来说不会只用一个邮箱，可以在Netaddress、Bigfoot、163.net上申请一个免费E-mail帐号，再在你的计算机上装入Netscape Communicator，把申请到的免费E-mail加入到Netscape Communicator中，这样你的免费E-mail就可以收发邮件了。启动Outlook Express选中收到的乱码邮件，点击工具条中的“转发邮件”按钮，进入邮件视窗，在“收件人”处输入你的免费E-mail邮箱地址，按“发送”，再退出Outlook Express，启动Netscape Communicator，打开Mailbox，你收到的转发邮件就不会是乱码了。
    在WWW浏览中出现乱码 ，你可以通过安装Richwin for internet或南极星解决如BIG5码等问题，或浏览改变字体，或在Microsoft公司注册下载字符集等。
#1    三、网景通讯家浏览技巧 
#1    1.抓图（包括抓背景图）
    想要抓WWW上的图，只要把鼠标移到图上，按下鼠标右键，就会有储存图档的对话框出现。
#1    2.秘密鱼缸 
    在Netscape中按下“CTRL+ALT+F“，就可以看到在 Netscape公司中的鱼缸。
#1    3.拜访美国总统 
    在Mac版的Netscape中按下“CTRL+OPTION+W”就可以连到白宫！
#1    4.发生什么事了？ 
    按下“CTRL+ALT+T”就可以看到正在下传页的状态。
#1    5.快速储存
    在Netscape下储存Page最快的方式是，先按下SHTFT键不放，将游标指在你想要储存的连接点，再按下滑鼠左键，即可储存在你指定的磁盘中。
#1    6.URL地址的省略 
    在Netscape中，如果要开启一个WWW 站台时，可以不用输入“http:／／”，而直接输入后的“www.XXX.XXX”就行了。FTP与 Gopher站也一样。
#1    7.状态栏 
    按下“CTRL+ALT+S”可以将Netscape最底端的状态栏删除。 
#1    8.再开一个浏览视窗 
    只要在Netscape中按下“CTRL+N”，就可以再开一个浏览视窗。如果需要在Netscape下长时间FTP，或某个画面传输时间太长时，就可以利用这个功能继续在WWW漫游。
#1    9.Netscape热键 
    “CTRL+R”重新下载（Reload）；“CTRL+B”显示书签（Bookmark）；“CTRL+H”显示浏览历史（ History）。
#1    10.最快的书签（Bookmark）方式 
    如果屏幕画面够大，可以同时开启 Netscape与 Bookmark窗口，那么当要记录某个站台时，就可以用拖曳的方式，将某个站点直接放到书签（Bookmark）中的任何一个位置。
#1    11.离线使用Netscape
    想在没有连线的环境下使用Netscape看HTML档吗？把一个叫作 mozock.dll的文件放 到Netscape的目录下就可以了。 
#1    12.在Netscape下FTP
    在Netscape下可以直接连上Anonymous的FTP站，但需要“User ID／密码”的站台也是可以直接连的。方法是在URL前加你的ID与@符号，如 xxxx@ftp:／／ftp.xxx.xxx／，接下来 Netscape就会问密码了。
