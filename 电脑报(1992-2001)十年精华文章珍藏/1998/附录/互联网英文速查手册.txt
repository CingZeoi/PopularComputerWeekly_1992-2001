互联网英文速查手册

张霄军
1998年 第52期 17版

#1    一、互联网专有名词
#1    1.access连接
    （1）与互联网的一种连接；（2）访问互联网的方式（网络访问、拨号访问等）。执行某些活动或读取特权信息的能力。
#1    2.adapter适配器
    插入计算机扩展槽中的一种电路板，为计算机提供额外功能。个人计算机上使用的通用适配器包括提供视频输出的视频适配器、存储器扩充板、内部调制解调器和声卡等。
#1    3.address地址
    （1）表示互联网上一台计算机或现场的一种特殊标识符。它可以是数字IP地址或文本哉名地址；（2）一个非常具体的电子邮件地址。
#1    4.address bus地址总线
    从微处理器至随机存取存储器（RAM）的一种内部电通道，用于传送存储器存储单元的地址。
#1    5.anonymous FTP匿名文件传送协议
    最通用的互联网文件传送协议。允许用户使用FTP和ANONYMOUS作为用户名进行登录，通常可使用任何口令或只按回车键。
#1    6.Archie网络查询工具
    互联网提供的一种为FTP资源服务的检查工具，具有自动采集、索引、分发信息的功能。
#1    7.attached document附加文档
    在电子邮件中，一个附加到某电子邮件消息上的二进制文件（如一个程序或一个压缩的字处理文档）。要包含一个带电子邮件消息的附加文档，发送方和接收方都必须拥有可处理MIME的电子邮件程序。
#1    8.backbone中枢链路、供给中心
    连接其他网络的一种大型调整网络，用于传送数百甚至数千里之外的数据。
#1    9.bandwidth带宽
    指一个频带的上下界频率之差，其单位以赫兹（Hz）或位/秒（bps）表示。
#1    10.bootstrap protocol引导协议
    用于网络节点的引导启动的一种协议，这些节点在计算机硬盘中没有存值得引导指令。
#1    11.breakpoint断点
    程序中的一个位置，系统在此暂停执行，让用户决定了一步该做什么。
#1    12.bridge网桥
    是一个存储转发设备，用来连接不同类型网路拓扑介质的局域网。它能完成数据链层的功能。
#1    13.Browser浏览器、浏览程序
    一种可在互联网上任何地方查找和访问文件的程序。
#1    14.buffer缓冲区、缓冲器
    存储器的一部分，用于储存一定数量的数据或文本，直到其被处理或显示。
#1    15.bus总线、总线配置、导线、导线拓扑
    （1）处理机的一种物理部件；（2）一种计算机配置，以串联方式将处理机连接起来；（3）数据通信技术中一种网络拓扑结构；（4）计算机系统中各功能逻辑模块之间传送信息的公共通路。
#1    16.cache高速缓存
    一种保持经常存取数据或随时调用程序指令的存储区域，通过在内存的快速部分存存储数据或指令，以及采用有效的设计来提高高速缓存的性能。
#1    17.cell信元
    由信头和信息段组成，信头中装有控制信息，信息段中装入被分解成数据块的用户信息或其它管理信息。
#1    18.channel通道、信道
    （1）信号从一个设备发送到另一个设备的通路；（2）数据通信中一种单向传送数据的信道。
#1    19.chrominance色度
    多媒体中组合视频信号的一部分，它包含有关色彩的信息。
#1    20.Client客机户程序
    它是用户访问服务系统的界面程序，运行在网络用户的本地主机上。
#1    21.client/server客户/服务器
    （1）分配信息或文件的一种方法，中央服务器应用程序按这种方法存储文件，并在客户应用程序申请时供它们使用；（2）一种LAN结构，在该结构中，文件和其他资源保存在一台中央服务器计算机中，每个人通过客户计算机与网络交互，获取这些资源。
#1    22.Common Agent Technology共用代理技术
    共用代理技术允许收集的大量硬件和软件的管理信息得到统一和合成。它为计算机系统使能部件以及现有代理标准已经支持的部件提供了一种以集成方式管理的途径。
#1    23.convergence会聚
    包交换网络中自动的网络映射过程，它在路由器接通后出现，只在几分钟内就使路由器的网络图“会聚”于现实（网络的当前状态）。
#1    24.core主存
    （1）UNIX和某些IBM机器上的RAM（随机存取存储器）；（2）主存信息转储后出现在UNIX上当中的文件名。
#1    25.daisy chain菊花链
    一种网络结构，亦称总线拓扑结构。在这种结构中，所有的设备沿一条总线连接在一起，并管理了每个设备的信号。
#1    26.digital monitor数字监视器、数字监控器、监控程序
    接收来自显示适配器的数字输出并将数字信号交换成模拟信号的一种监控器。
#1    27.distribution-list分配表、分配清单
    发送邮件清单的一种简单形式，在这种形式中将一个别名分配给一个电子邮件（e-mail）地址表。指向该别名的邮件被发送到该表的每一地址。
#1    28.domain name域名
    任何一台需要连接互联网的计算机服务器，都必须向互联网的“网络信息中心”申请一个唯一的域名。
#1    29.dot点
    域名、newsgroup名及其它面向文件的分隔符。
#1    30.download下载
    从中心源将文件拷贝传送到外设或计算机上，或通过调制解调器和电话线从另一台计算机上将文件传送到自己的计算机上。
#1    31.dynamic-link动态链接、动态连接
    一种链接数据的方法。这样数据可由两个程序共享。当一个程序中数据被改变则你使用更新命令时在另一个程序中该数据同样被改变。
#1    32.echoplex同声（方式）
    接收站把消息送回到发送站的方式来确认和证实接收的一种通售协议。
#1    33.E-mail电子邮件
    E-mail是利用互联网进行信息交换的一种方式，即网络上的邮政。电子邮件是计算机应用软件的通用术语，可使用户在不同的计算机上发送信息。
#1    34.emoticon情感标点、情感图标
    使人洞察编写者的感情状态的标点或图标。按顺时针方向旋转90度，立刻就会明白该图标所表明的含义。下面是某些常用的情感图标：

#1    35.filter过滤器、滤波器、滤光器、筛选过程
    （1）将一种文件格式转换成另一种格式的程序；（2）电子邮件中，在取消另一些消息到达用户的一种程序；（3）能自动操作屏幕数据的任一程序。
#1    36.Finger指定服务
    查询互联网上某用户的一些公开信息，包括用户标识、全名是否登录、最后一次登录时间、邮件是否阅读以及其他一些公开信息。
#1    37.Firewall防火墙
    （1）通过筛选经由给定节点的全部IP（Internet Protocol）包来限制与网络某一特定区域通信的软件；（2）一种互联网网上的安全性保护软件。
#1    38.frame relay帧中继、帆分段传输
    是为计算机局域网络或主机与终端间提供的一种快速分组交换技术。
#1    39.free-net免费网
    对某一社团的成员提供互联网访问的免费公共网，一种其于团体的布告栏（BBS）系统。
#1    40.gateway网关
    它是高层互连，用于传输层以及传输屋以上的连接器。
#1    41.Gopher菜单查询、Gopher工具
    专门用于浏览互联网资源的菜单式查询软件，它以菜单方式显示各种资源，用户通过菜单选择就可实现对互联网资源的远程访问。
#1    42.guest客户
    在局域网（LAN）中允许用户无需提供口令便可访问其它计算机的一种特权。
#1    43.grur专家
    一位有经验的专家是用户遇到计算机或网络的问题时通常可以讨教的人。
#1    44.handshaking信号交换
    两台相接的设备，在准备接收数据时，交换发送信号，以表明数据可以进行可靠传送。
#1    45.heap堆栈
    Microsoft Windows的早期版本，存储器中用于存储关键资源的特定存储区。
#1    46.homepage启始页（面）、主页面
    进入一个主机后用户最先看到的屏幕信息，旨在向用户介绍该站点的概貌。
#1    47.host主机
    网络上允许众多用户同时访问的计算机。
#1    48.hub集线器
    类似于多端口网桥，把高速多端口桥接器功能集成于其中为分段的提供交换连接功能，向网络的终端站提供所需的专业带宽。
#1    49.hyperlink超级链接
    指在一个计算机文档的特定区域能引入其他文档或程序。
#1    50.icon图符
    图形用户接口（GUI）中的一种屏幕符号，可以表示程序、数据文件或其他一些计算机实体或功能。
#1    51.inport进口
    把某程序生成的文件装入不同的程序中。
#1    52.inbox邮箱
    邮件程序将收到的电子邮件消息存入文件中。
#1    53.info-server信息服务器
    激发邮件服务器的电子邮件地址，借发送存储的信息以响应含关键词的消息。
#1    54.intel英特尔公司
    世界上最大的微处理器及其他半导体的生产厂家。公司位于美国加利福尼亚洲的圣克拉拉。
#1    55.Internet国际互联网络
    是由那些使用公用语言相互通信的计算机连接而成的全球网络。一旦连接到Web节点，就意味着计算机已进入互联网。
#1    56.Java（爪哇）语言
    由SUN公司开发的一种跨平台的、适合分布式计算和向对象的新型编程语言。
#1    57.layer层
    在网络体系结构中，一组概念上完整的服务、操作和协议。
#1    58.leaf叶
    在网络休系结构中，从远方中央计算机接收信号的计算机，但该计算机不能把信号传送给更远的计算机。
#1    59.list表、列表
    一种有序的数据集合；查看目录中的文件名；显示文件的内容；互联网的一种邮件名。
#1    60.log运行记录
    在通信程序中，为记帐或收集数据，存储在辅存上的一些信息或信息段。
#1    61.match匹配
    在搜索中满足特定标准的记录或文件，也称为命中（hit）。
#1    62.message邮件、报文
    （1）一封电子邮件信；（2）发给上某指定用户的评论；（3）一个信息包。
#1    63.mirror镜像
    在另外一个档案库存储文件的精确拷贝。
#1    64.modem调制解调器
    是一种通信设备，其作用实在模拟通信信道上传输数字信号，它由调制（modulation）和解调demodulation）组合而成。
#1    65.multiplex多路传输
    一个传输媒体同时在多个逻辑信道上传送。
#1    66.negotiation磋商
    通信网中，通信的双方协商通信阶段中必须一致遵守的一组通信规则的过程。
#1    67.neighbor邻域
    对于图象上的任一像素，它周围的适当像素的集合。
#1    68.Netware网件
    由Novell公司研制，适用于局域网的网络操作系统。操作方便，命令格式类似于DOS命令，应用程序几乎不加修改即可在网络环境下工作。
#1    69.networking网络化、联网
    将多个独立分散的系统相互连接组成更大规模、能力更强的整体系统的过程。
#1    70.node节点、结点、网点
    网络中的任何计算机或其他设备，能产生、接收或转发消息。节点包括中继器、文件服务器和共享外设。
#1    71.object对象、目标
    对象连接和嵌入粘贴到另一文档中的文档或文档的一部分。
#1    72.offline脱机、脱线
    （1）当前未与网络接通；（2）对网络请求不响应；（3）形容一个人不再参加闲谈（chat）。
#1    73.online联机、在线
    （1） 当前与网络接通；（2）能得到网络请求服务；（3）形容一个人正在参加闲谈（chat）。
#1    74.open开放的
    指一个协议使用的算法与技术任何人都可得到，与远程主机进行连接。
#1    75.priginate产生、起源
    发送电话呼叫而不接收。该术语经常用在计算机经调制器互连的场合。
#1    76.packet（报文）全组、（信息）包
    在网络上发送的数据的单位。分组中除了数据外，还包括发送者和接收者的地址以及差错控制信息。
#1    77.page分页、页面、页
    在基于字符的界面中一次显示一屏文档。
#1    78.password口令
    一个密码，用来限制访问权限，只有知道口令的授权用户才能访问帐户、信道、文件等。
#1    79.path路径、通路
    （1）在发送者和接收者之间E-mail消息所经过的路线；（2）信号通过的信道；（3）对一个目录或文件完全指定的从根目录开始的位置。
#1    80.penetration渗透
    计算机安全学中非法侵入自动数据处理系统的行为。
#1    81.polling轮询
    在局域网中，中央计算机连续地查询或轮询工作站，确定它们是否有信息需要传送的一种控制信道访问的方法。
#1    82.primary主站
    在HDLC（高级数据链路控制）协议中的一种数据站，它支持数据链路的主要控制功能、产生传输命令、解释收到的应答信息等。主站负责控制信号交换的初始化，数据流的组织以及涉及错误恢复的处理。
#1    83.primary group基群
    基本信号的复用。
#1    84.privacy保密性、秘密、加密
    网络上预先假定的权限：未经用户许可，任何人不能查看用户的磁盘储存区、电子邮件和文本等。
#1    85.protocol协议
    允许不同的计算机设备或软件通信的一种协定规则。
#1    86.range区域
    电子报表过程中的一个单元或一组长方形的相邻单元。
#1    87.raster光栅
    在监视器或电视屏幕上形成图形的扫描线的水平模式。
#1    88.refile接力传送
    从租用线路网络上的一个站，将报表文件送到一个未经租用线路网络服务的另一个站的过程。
#1    89.remote远程
    指在其他地方的主机或网络上的计算机或网络资源。它与本地主机或资源相对。
#1    90.repeater中继器、转发器
    它是物理层上的互连，这种连接只涉及物理硬件，此方式适用于完全相同的两类网络的互连。它是通过对信号的重发转发，扩大了网络传输距离。
#1    91.response响应
    在数据通信的问答过程中，从站对主站询问的回答。
#1    92.router路由器
    它是网络互连，它能完成（1）流量控制；（2）差错控制；（3）路由选择；（4）数据包交换等功能。
#1    93.segment段、区段
    （1）指网络中一段逻辑电缆；（2）根据协议，对送给的数据进行打包的单位。
#1    94.server服务器系统
    它是支持服务系统资源管理和执行用户请求的系统程序，运行在提供网络服务的主机上。    
#1    95.session会话、对话
    指要求主机（host）之间建立连接，并进行交互通信的过程。
#1    96.subdomain子域
    一个互联网域的一部分，有自己的名字（子域名），统称指由一所大学或公司的主机组成的子网。
#1    97.subnet子网
    网络的一部分，它与网络的其他部分共用一个网络地址，并通过子网地址互相区别。
#1    98.tag标记
    由浏览器读入的用以格式化被访问文件元素的HTML代码。
#1    99.talk talk协议
    Talk协议能使远程计算机上的两个用户以实时方式进行通信。
#1    100.telex用户电报
    一个能发送和接收数据的终端组成的国际通信系统。
#1    101.telnet远程登录
    本协议能使终端和基于终端的进程在运行TCP/IP的网络上进行通信。
#1    102.topology拓扑（结构）
    指任何一种类型的物理网络布局，即如何组织网络设备及设备间连线。
#1    103.transceiver收发器
    既能发送信号也能接收信号的设备，它可以在节点和网络间交换帧。
#1    104.unsubscribe注销
    （1）在向多个用户发讯时，在用户注册名表中注销一个用户名；（2）从已注册的小组中注销一个专题小组的名字。
#1    105.Usenet网络新闻
    一种主要的分布公告牌，是世界上最大电子公告牌，用户可在其上发布或阅读信息。
#1    106.user用户
    任何一位登入计算机系统和网络的人。
#1    107.Vacation休假、度假
    一个UNIX程序，它可以设置一个返回信息给任何一个向你发送电子邮件的人，告知他们正在度假。
#1    108.weather天气、气象
    互联网上有许多有关天气信息的服务器。
#1    109.Web环球（万维、全球）网
    它把寄存在服务器的超文本文档和其他一些文件、菜单和数据库通过超级链接连接到一起，构成一个整体。这些文件均可通过URL获取。
#1    110.wetware活动件、活跃件

#1    112.workstation工作站

#1    113.zone区域、逻辑网
