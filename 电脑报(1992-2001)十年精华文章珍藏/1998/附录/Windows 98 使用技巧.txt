Windows 98 使用技巧

金伟
1998年 第51期 09版

    1.自动完成URL
    对于以前参观过的网址，你无需在IE的地址栏中输入完整的地址，编辑器的自动完成余下的部分，但默认值只是针对具有.com、.org、.edu扩展名的网址。修改注册表可以使其同样适用于.gov和.net结尾的网址。运行“RegEdit.exe”，找到“HKEY-LOCAL-MACHINE＼SOFTWARE＼Microsoft＼Internet Explorer＼Main＼Url Template”，将会发展下面六个字符串：
    1“www.%s.com”
    2“www.%s.edu”
    3“www.%s.org”
    4“%s.com”
    5“%s.edu”
    6“%s.org”
    选择“编辑/新建/字符串值”，添加下面的内容：
    7“www.%s.net”
    8“www.%s.gov”
    9“%s.net”
    10“%s.gov”
    2.加入自己的频道
    要在频道栏中添加HTML文件，打开“C:Windows＼FAVORITES＼CHANIVELS”，建立指向该HTML文件的快捷键。
    3.改变搜索引擎
    在“开始”菜单中选择“查找”网络时，可以改变默认的网络搜索引擎。
    运行“Regedit.exe”找到“HKEY-CURRENT-USER＼Software＼Microsoft”，双击“查找页”，在数据项中输入需要的搜索引擎。
    4.建立更多的搜索快捷键
    可以选择搜索浏览栏中的搜索引擎，单击IE 4.0中的“搜索”按钮，在显示出的左边窗口中单击下拉菜单中选择“提供者”，可以选择AOL 网络搜索、激励、Infoseek、Lycos、Yahoo或Provider-of-the-day。
    5.快速安全开关    最快捷的修改安全设置的方法是双击编辑器状态栏右下方，出现Internet安全属性对话框，从中定义高、中、低和安全级别。
    6.改变“开始”菜单
    可以简单地通过向“开始”菜单中拖拉需要的任何内容就可以把它加入到“开始”菜单中，而按右键就可以做包括删除等的其它操作。
    7.改变快速运行图标
    任务栏中的快速运行工具条中的图标是由一种新的SCF格式的文件控制，这种格式其实只是一种你可以编辑的文本格式，要在SCF文件中修改某个图标，运行“记事本”程序，选择“文件/打开”，找到“C:＼Windows＼Application Data＼Microsoft＼Internet Explorer＼Quick Launch” 目录，选择所要编辑的SCF文件，将“IconFile=“指向选定的文件的文件名和路径。还可以从浏览器窗口中向工具条中拖拉文件或文件夹的图标，以便在快速运行工具条中添加图标。
    8.填充“钱包”信息
    Windows 98和IE4附带了一个INTERNET“钱包”，允许存放信用卡付款信息。选择“查看”、“INTERNET选择”、“连接”，单击“付款”按钮（如果提示要安装信用卡扩展，单击“安装”），在下面的对话框中，单击“添加”，按照提示操作即可装入用户信用卡信息。
    9.快速订阅
    订阅一批站点最快捷的方法就是访问它们，打开“历史”窗口或文件夹，将需要的站点拖到订阅文件夹即可。
    10.修饰文件夹
    可以在一个打开的文件夹中改变默认墙纸，在菜单中选择“查看”、“定制这个文件夹“，单击“选择背景图片”，在根据提示选择为墙纸设置确定的颜色和图像。
    11.最大化浏览器
    Windows 98允许在编辑时使用“全屏”。在工具条上选择”全屏“按钮（或者“查看/全屏”），右击其余的标准工具条，选择”隐藏”。于是WINDOWS的任务栏、链接和地址栏、菜单等都“不见了”。想要切换回正常的显示方式，只要把鼠标移到屏幕最顶部再选择“查看/全屏”。
    12.通过键盘显示桌面
    如果键盘上带有WinKey，按WinKey+D，会马上将所有窗口最小化，这样就能看见桌面了，再重新按WinKey+D，所有窗口又恢复原状了。
    13.消除下划线
    如果使用的浏览器是网站风格，需要消除文件和文件夹下面的下划线，可以修改设置，这样只有当移动鼠标经过图标时，才会看到下划线了。单击“开始”，选择“设置/文件夹和图标“，单击“自定义”和“设置”，选择“单击打开图标”，单击子选项“只有在被指向时给图标加下划线”。
    14.关闭自动完成
    Windows 98的自动完成功能可以在地址栏为你填入站点地址，如果不需要这个功能，可以把它关闭。选择“查看”、“INTERNET选项”，单击“高级”、选去“自动完成”选项。
    15.使用频道屏幕存储器
    可以选择屏幕存储器，用来滑动显示所选的频道内容。在桌面上右击，选择“属性”，单击“屏幕保存”、“频道屏幕存储器”，选择“设置”，所有的频道都会显示出来，选择一些喜欢的频道放入“频道屏幕存储器”。
    16.高级用户的文件夹设置
    默认的文件夹设置是为新手准备的，要打开高级用户界面，打开“我的计算机”，选择“查看”、“文件夹选项”、“查看”，选中“在标题区显示全路径”、“在工具栏显示网络驱动按钮”和“允许大写字母”，在隐含文件区选择“显示所有文件”。
    17.全“能”浏览器
    IE内置了一个应用程序允许查看JPEG和GIF格式的图形文件、运行带有WAV和AV后缀的声音文件、观看AVI、MOV和MPEG影象文件。只要把该文件拖到编辑器窗口，单击“从当前位置打开该文件”后运行它。
    18.重新打开活动桌面
    微软在它的活动桌面库中加入了许多免费的很“酷”的内容，其中包括“地址查找功能”（只要在地址栏中输入，就会出现一个地图）。要把它加入到墙纸中，在相应内容的预览页单击“添加到活动桌面”按钮。
    19.编辑快捷键
    下面是几个浏览网页时很有用的几个快捷键：
    Key Action 
    Tab移动到下一个链接 （从页的顶部到页的低部）。
    Shift+Tab  移动到下一个链接 （从页的低部到页的顶部）。
    Ctrl+Tab移动到下一个区 （跳过链接）。
    Enter  如果某个超链接为高亮度，就进入该链接；如果光标在某一个表域，就进入该表域。
    20.分离任务条和工具栏
    大多数人把任务条放在屏幕底部，另一些则喜欢放在顶部。现在可以两样都做到。将需要的个别工具条拖到顶部或其它位置，在底部留下最小化的程序，或者相反。右击在顶部的内容，选择“属性”、“总是在顶部”，以便最大化窗口不会覆盖它，还可以检查“自动隐藏”选项，这样只有当把鼠标放在相应位置时，工具条才会出现。
    21.WEB页在缓存中吗
    当鼠标经过一个既不在缓存又不是现场的链接时，会看到一个特殊的“没有链接”的符号，这是最快的确定一个链接是否能够进入下一页或能否开始拨号链接的办法。
    22.拖曳URL
    地址栏每一个URL都带有一个小图标，只要拖曳这些图标，就可以在任意位置（桌面、文件夹、收藏夹或“开始”菜单）建立一个相应URL的快捷键，如果把某个URL的图标拖到工具栏中启始页的按钮上，就等于把该URL作为起始页。
    23.修改收藏
    可以按自己的需要拖动“收藏”中的项目或文件夹来改变它们在收藏夹中的顺序或位置。
    24.改变连接图标
    可以改变IE4链接工具条上的图标，同样也可以改变快捷键图标：在某个按钮上右击，选择“属性”，单击“改变图标”，这时就会发现文件中已经包含了你指定的图标。
    25.保留你的WINDOW设置
    在关闭某个窗口时按住ALT键，则窗口的大小和位置就自动保存下来了（对保持最大化模式的应用程序不识用）
    26.按照自己的方法设置
    当用户从资源管理器的“查看”菜单中选择了“详细查看”后，可以按照名字、大小、类型和修改时间将文件排序，这里有个巧妙的排序小窍门：可以在内容窗口顶部放置名字、大小、类型和修改时间的区域中拖动这些项目来改变它们的水平顺序，这样排序更快。
    27.快速加入收藏
    在浏览器中的任何位置按右键，选择“添加到收藏夹”，随后收藏对话框就会出现。
    28.简化搜索
    IE 4.0 的搜索按钮可以在双窗口的浏览器中同时访问搜索引擎和主浏览器。在工具栏中的“搜索”按钮上单击，在主浏览器左边的窗口中带出搜索窗口，选择搜索引擎，输入搜索内容，单击“搜索”。结果会在左边的窗口中出现，点击其中一个链接，相应的页面在右边的窗口中出现。因为搜索的结果还保留在左边的窗口中，就不必在页面和链接之间来回移动了。
    关闭搜索窗口只需再次点击“搜索”图标。
    29.自动完成
    IE4.0地址栏有“自动完成”功能，但是也可以用于本地的文件夹。输入要打开的文件夹的路径名，“自动完成”功能就会猜测，一旦它猜测正确，就可以停止输入。
    30.使用“帮助”的帮助
    如果需要及时的帮助，选择“开始”菜单中的“帮助”选项不会很合适。如果已经这样做了，它会先运行INTERNET的在线向导（浪费时间）。如果要跳过WEB快速获得本地支持，打开“C:＼Windows＼Help” 文件夹，选中WINDOWS.HLP文件，并向桌面上或“开始”菜单中拖动建立一个快捷方式。
    31.移动压缩盘
    在向Windows 98升级时，如果正在升级的机器中装有用Drivespace3压缩的硬盘分区，可能需要相当长的安装时间。你需要在备分后解压缩。需要注意的是：不管用没用Windows  98，都必须有至少未压缩时磁盘容量一半的空间才能保证解压缩的顺利完成。
    32.备份网络
    Windows 98中新的备份程序BACKUP.EXE可以通过网络备份Windows 98的驱动程序，这样比其它方法更快。但需要选择存放备份的网络位置，如果选择“确认”方式可能需要多花费50%或更多的时间，因为网络具有相当的可信度，“确认”其实并不是必须的。
    33.双倍MODEM，双倍的速度
    新的拨号网络程序允许联合通讯设备以获得更快的通讯速度。首先，用其中一个设备建立拨号网络连接，然后在它的图标上右击，选择“属性”和“多连接设备”，单击“使用附加设备”按钮，然后加入计算机所能支持的最多的设备。如果所连接的主机支持“hunt group，"、“循环” 或 “旋转”连接多线访问，就可以给每个新设备使用相同的电话号码。否则需要检查每个应答设备的电话号码了。要使用多线访问，双击他的图标，就会开始第一次连接，一旦联系上，开始从附加设备上启动调用。要了解实际连接的设备数，单击“连接”。
    34.完美的打印
    在打印一个页面时，可以利用IE4.0中一些新的很“酷”的打印选项。选择“文件”、“打印”，定制页面，打印所有连接的文档或者当前页中的所有连接。
    35.把任何站点作为墙纸
    在桌面上双击选择“属性”，单击背景对话框中的“编辑”按钮，在硬盘中搜索需要的HTML和HTM文件，确定后双击。此外还可以使用INTERNET上的HTML和HTM作为墙纸，只要打开相应页面将该文件另存为本地文件。但是，如果你要显示WEB页中的图片，必须右击每一个图片，选择“将图片另存为”，把它们存入本地硬盘。然后在已存好的文件中修改<IMG>标号为本地的图片（要使用HTML文件做墙纸，必须先双击桌面打开活动桌面，选择“活动桌面”、“浏览网页”）。
    36.Windows 98的安装问题
    有时，Windows 98在安装时不会提供所需的或者是用来升级的文件。
    这时它会使用默认的VxD也就是vmm32.vxd来代替这些文件，虽然它的速度慢一些。要检查你是否丢失一些文件或使用了旧文件可以在“我的计算机”右击，选择“属性”、“驱动程序”。
    检查系统中每个设备的属性，选择“驱动程序”、“属性”，单击“驱动程序”，如果有些文件的说明里有“UNKNOW”，就需要检查一些它所使用的驱动程序“C:＼Windows＼System＼vmm32.vxd （configmg.vxd）”。
    这意味着已经丢失了括号中的文件。
    要找到这个文件以升级旧文件，找到Windows 98中的存放.cab文件的文件夹winbeta。选择“开始”、“查找”、“文件或文件夹”，在“所包含的文字”中输入文件名，选择*.cab所在的目录开始寻找，提取它们到应该在的位置。大多数丢失的文件会在Windows 98_49.cab 和 Windows 98_50.cab中。还要找到鼠标所需的VCOMM.VXD文件以修正一些小问题，相关文件为Windows 98_49.Cab
    37.移动不需要的文件
    使用Windows 98的磁盘清理工具，它在某些条件下（例如当你向一个剩余空间不足3%的硬盘拷贝文件时）可以自动运行。具体步骤如下：
    选择“开始”、“程序”、“附件”、“系统工具”、“磁盘清理”，下面出现一个对话框包含两个表项：磁盘清理和更多的选择。“磁盘清理”包含关于要移走的文件的四个类别选项：临时网络文件，下载程序文件，回收站，临时文件。选择“更多的选择”，其中有两项：Windows setup and Installed programs，作用和控制面板中的“添加/移动程序“是一样的，主要用于删除WINDOWS应用程序，如果你使用的不是FAT32，你还可以选择把它转换为FAT32。删除时如果有疑问，可以选择”查看文件”，查看文件的描述信息，确保所删除的就是不需要的文件。
    38.了解所删除的文件
    在用“磁盘清理”删除文件之前，要先确定所删除的文件的性质。在所要删除的文件列表当中，首先有两类：临时网络文件和下载的程序文件，它们都指向下载文件所在的目录，如果删除它们，下次你再参观该网页的时候，就不会又下载这些文件了。如果过于频繁地清除这些目录（一周一次），就会多次重复地下载相同的网页，可删除文件的名单中可以显示它所占用的磁盘空间，如果选择这两类文件只能使你获得很少的空间，那么最好不选择。
    第三种文件类型的选择是“回收站”，检查它是否空。临时文件选项是让你选择是否删除各种应用程序放置在硬盘上的文件，选择删除这些一般很安全。
    39.怎样转换为FAT32
    如果是在旧的系统上升级为WINDOWS 98，那么就仍然使用的是旧的FAT16，而不是新的FAT32，WINDOWS 98自带了转换程序，选择“开始”、“程序”、“附件”、“系统工具”、“驱动器转换程序（FAT32）“，出现转换程序画面，在你运行之前，先看看下面的内容。 
    40.应该转换为FAT32吗
    在转换FAT32之前，先要确定是否应该转换。FAT32的优点在于它能更有效地使用硬盘，例如在1GB的硬盘上FAT16使用的最小文件为32K，而FAT32只有4K，另一个优点是它没有每个分区不能超过2GB的限制。
    另一方面，在使用FAT32的磁盘上不能使用DriveSpace，也不能双击运行大多数早期的WINDWOS和DOS程序。此外，一些老的磁盘工具程序只能在FAT16上运行。
    如果选择转换，转换向导会提醒用户可能会出现的问题，选择时要注意，如果有疑问，最好取消转换。
    41.快速整理
    WINDOWS 98允许在运行磁盘整理程序的同时运行其它程序，因磁盘的容量、性能、数据的多少以及碎片化的程度不同，整理需要的时间也不同。
    在整理的过程中如果有任何对硬盘的写动作，整理程序都会磁盘的内容发生改变，重新开始整理，因此如果要尽快地结束整理，就不要有其它操作。
    42.保持文件及时更新
    随时获得最新的驱动程序或系统文件能获得很显著的性能提高，WINDOWS 98提供了升级向导帮助用户随时更新。
    用户可以通过网络访问的这个向导，扫描系统以确定硬件和软件的配置，它将得到的结果和网站上的资料比较后，检查是否有最新的驱动程序、DLL文件等等，如果有更新的文件它可以自动下载。要运行该向导，选择“开始”、“WINDOWS升级”，当你到达了该站点，选择升级向导，按照相应说明操作即可。
    43.自动调整
    使用WINDOWS 98的“WINDOWS维护向导”可以自动帮助用户调整系统。要运行该向导，选择“开始”、“程序”、“附件”和“系统工具”、“WINDOWS维护向导”，选择“修改我的调整设置或计划”（第一次运行时不会有该选择，直接选“下一步”），在出现的对话框中，可以选择“快速”或“自定义”安装，一般来说，选择“自定义安装”会更好一些。
    此外还可以在“WINDOWS维护向导”中选择“现在调整计算机”做及时调整。
