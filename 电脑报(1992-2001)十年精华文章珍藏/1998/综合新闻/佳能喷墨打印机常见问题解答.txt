#1　　●为什么佳能BJC－210s/sp打印机在UCDOS操作系统下打印不出中文?
　　因为UCDOS操作系统是希望公司开发研制的中文操作平台，该系统没有配备BJC－210s/sp打印机的驱动程序，所以只有选用佳能BJC－210s/sp打印机所提供的LQ仿真模式才能够在UCDOS操作系统下打印出中文。这就要使用BJC－210s/sp打印机随机所配的DOS驱动软件将打印机设置成LQ模式，具体的设置方法如下：
　　一、安装DOS驱动软件：
　　在DOS提示符下：
　　C:>a:回车
　　A:>canon回车
　　English回车
　　Install BJ Setup utility program回车
　　Drive C:回车
　　随后自动释文文件，一路回车就可以了，最后回到C:BJPRN>下，安装成功。
　　二、使用DOS驱动软件：
　　在DOS提示符下：回车
　　C:>cd_bjprn回车（“_”表示空格）
　　C:\BJPRN>BJC210回车
　　进入系统后，确认Priont Control Mode中的选项为“LQ MODE”，在连接打印机的状态下按F10键，显示屏会自动退回到C:BJPRN>下，并提示“Warning!Printer off line.Press any key to continue.”，则表示设置未传给打印机，可按下述操作重新做一次或更换打印电缆试一下（有可能是打印电缆传输数据有问题）。
　　另外，在WPS中使用时，在打印机类型中选择“Epson，Net系更，AR－3240”这一项即可。在CCED中使用时，在打印机类型中选择LQ类型的打印机即可。
　　这样，您的佳能BJC－210s/sp打印机随机所配的DOS驱动软件将打印机设国在LQ模式的具体设置方法，另外，还有一种更简便的方法可以将佳能BJC－210s/sp打印机设置成LQ模式，具体操作步骤如下；
　　1.先将打印机的电源关闭。
　　2.在打印机加电时，按下并按住POWER按钮，直到听到打印机的蜂鸣器鸣叫8声，然后松开POWER按钮。
　　这样，您的佳能BJC－210s/sp打印机就被设置成LQ模式了，您就可以仿真模式在UCDOS操作系统下用佳能BJC－210s/sp打印机打印出中文了。
#1　　●如何用佳能BJ－330打印机在UCDOS操作系统下打印出中文?
　　UCDOS操作系统没有配备BJ－330打印机的驱动程序，所以如果要选用佳能BJ－330打印机在UCDOS操作系统下打印中文，一定要将佳能BJ－330打印机设置成LQ仿真模式，通过调拨打印机背部的小开关，将第一排开关的SW－1号开关往下拨。这样就可以在LQ仿真模式下，用佳能BJ－330打印机在UCDOS系统下打印出中文。
#1　　●为什么BJC－610机在Windows英文版下不能打印?
　　在中国大陆销售的BJC－610型打印机，所配的打印机驱动程序是专为大陆用户所设计的，能够满足中国大陆计算机用户的要求，BJC－610打印机只能在Windows简体字中文版环境下使用，而不能在其他语言的Windows版本下使用。如果要在英文版的Windows环境下使用BJC－610型打印机驱动程序一套。所以如果要在英文版的Windows环境下用BJC－610打印机进行打印操作，那只有使用英文版的Windows驱动程序才行。
　　另外，因为BJC－610型打印机的Windows驱动程序受到软件版权的保护，不能非法复制。按照规定，每位佳能BJC－610打印机用户只能够拥有所有版本的BJC－610打印机驱动程序一套。所以如果要在英文版的Windows环境下用BJC－610打印机，进行打印操作，您就必须同佳能打印机北京热线中心联系，并将您随机所配的一套BJC－610打印机Windows简体字中文版环境下使用的驱动程序软盘寄往佳能打印机北京热线中心，佳能打印机北京热线中心会马上将一套Windows英文版环境下使用的驱动软件寄给您以便于使用。
#1　　●Windows95下打印不完全，如何处理?
　　1.在Windows95内更改打印接口设置：
　　开机（open）==>开始（start）==>设置（setting）==>控制面板（control panel）==>系统（system）==>设备管理（device manager）==>端口（ports）==>打印机端（print port）==>驱动程序（driver）==>更改驱动程序（chang driver）==>显示所有设备（show all devices）==>把“ECP打印端口”改成“打印机端口”后确认（change“ECP printer port”to“printer port”then press“ok”）。
　　2.更改计算机CMOS设置
　　由于计算机系统以及BIOS版本的不同，请您参考所用计算机的《系统手册》，把打印接口由“ECP模式”改成“标准并行接口”。
　　3.替换Windows系统文件
　　（1）利用硬盘或一张空盘复制系统文件“Ipt.vxd”
　　例如：copy c:\windows\system\lpt.vxd a:\或copy c:\windows\system\lpt.vxd c:\bzck up
　　（2）从软盘上复制文件“lpt.vxd”到windows系统
　　例如：copy a:\lpt.vxd c:\windows\system
　　以上三步，一般操作完第一步即可改善。
#1　　●如何使用佳能BJC－210s/sp打印机在Auto CAD应用软件下进行打印设计图纸的操作?
　　在中国大陆被广泛应用的Auto CAD应用软件的版本为Auto CAD R10 DOS版，Auto CAD R12 DOS版，Auto CAD R12 Windows版和Auto CAD R13 Windows版。佳能下的佳能打印机驱动程序。到目前为止，佳能公司还不能提供Auto CAD Windows版本下的佳能打印机驱动程序。在佳能BJC－210s/sp打印机随机提供的软盘中有一张DOS实用程序软盘，这张软盘能够在DOS环境下对您的打印机进行设置并提供了许多应用软件的驱动程序，您所要使用的Auto CAD应用软件驱动程序就在这张软盘中。具体的Auto CAD应用软件驱动程序的安装方法如下：
　　首先，一定要将佳能BJC－210s/sp打印机设置成BJ模式。然后，进入DOS实用程序软盘的驱动程序子目录中，将后缀名是EXT的文件拷贝到Auto CAD DOS版应用软件所在的子目录下的DRV子目录中去。之后，再进入Auto CAD应用软件系统，在它的PLOT CONFIG子功能下的打印绘图设备清单中就可以找到佳能BJC－210s/sp打印机了，将它选定为默认打印机就可以了。
　　在上述的操作过程完成以后，就可以使用佳能BJC－210s/sp打印机在Auto CAD应用软件下进行打印设计图纸的操作了。

#1　　●BJ－10ex/sx打印机在联机或自检时，打印头有动作，但打印不出墨，怎么办?
　　这一情况的发生，有多方面原因，包括BC－01喷头故障，清洁单元故障，电缆及电路板故障等，但在排除了喷头故障后，可以对BJ－10ex/sx打印机进行简单的维护。
　　这里先介绍一下由用户可以自己进行处理的一种方法：
　　先打一把镊子钳，将打印机的上盖打开，然后将内美国打开，打到打印机左端的清洁单元，在清洁单元上有一个小的方型的橡胶墨盒盖帽，它的功能是用来将打印头密封的，如果打印头没有被密封，在清洁单元对打印头进行清洗时，就不能够将墨水吸出，从而打印不出墨水。先看这一个墨盒盖帽有没有脱落掉下，如果掉下，用镊子钳拿起它，并将它以竖直方向装到清洁单元的吸墨口上，直到它的挂勾扣入吸墨口上；如果发现墨盒盖帽还在清洁单元的吸墨口上，但其位置不是以竖直方向而是斜向或横向摆放，就用镊子钳将墨盒盖帽的位置放成竖直位置；在此之后，请将墨盒清洗几次就可以进行正常打印操作了。
#1　　●BJC－210s/sp打印机在打印时不进纸，怎么办?
　　如果发生这一问题，很大可能是属于机械上的故障，例如送纸器故障或有东西掉入送纸器中，下面就对有异物掉入送纸器中而造成打印机不能进纸的情况的简单维护方法进行介绍：
　　由于BJC－210s/sp打印机的送纸器的开口较大并且朝上，所以在办公室或家庭使用时，象别针，橡皮筋，瓜壳等的小形的异物很容易掉进送纸器中，因而造成打印不能进纸。要取出这些掉进打印机送纸器中的异物，请按照以下的步骤进行操作：
　　如果发生不能进纸的情况，就马上停止使用打印机，不让异和进入机内更深的地方。
　　先将打印机的电源关闭，拔下AC电源电缆和打印电缆，然后，将打印机的送纸器倒置，并摇晃、拍打打印机机身，如果发现有异物从打印机的送纸器中掉出的话，就表明这一工作完毕，然后再接上AC电源电缆和打印电缆，打开打印机电源，在送纸器上放上一叠打印纸，进行送纸操作，如果打印机能够正常进纸，那么这一问题就解决了。
#1　　●BJ－330打印机不进纸，并在控制面板上报缺纸错。
　　如果发生这一问题，很大可能是属于机械上的故障，例如送纸传感器故障或打印机的摆放位置不正确，下面就对传感器故障或打印机的摆放位置不正确，而造成打印机不能进纸的情况的简单维护方法进行介绍：
　　由于BJ－330打印机的送纸是靠在打印机底部类似重垂装置的送纸传感器来进行的，如果不正确，这样就会使打印机不能正常进纸。因此，将打印机放在水平的工作台上，就可以解打印机不进纸的故障了。如果，打印机的摆放位置正确，打印机还不能进纸，那可能是因为送纸传感器的重垂杆被纸悄或脏物卡住，没有处于下垂的位置，因而造成打印机不能进纸。要解决这一问题，请按照以下的步骤进行操作：
　　如果发生不能进纸的情况，就马上停止使用打印机，先将打印机的电源关闭，拔下电源电缆和打印电缆，然后用双手提起打印机的前端，让打印机的后端放在工作台上，将打印机的前端稍重地摔在工作台上，这样做，可以使送纸传感器的重垂杆自然下垂，然后再接上电源电缆和打印电缆，打开打印机电源，送入打印纸，进行送纸操作，如果打印机能够正常进纸，那么这一问题就解决了。
#1　　●BJ－330打印机在Windows系统下打印乱码。
　　如果发生这一问题，大部分是由于打印机没有被正常设置，可以从以下两方面来进行简单的维护：
　　一、计算机系统中接入了打印机共享器（包括一对多或多对一类型的打印机共享器）：
　　由于BJ－330打印机的并行接口是标准接口，其电路性能是标准的，而市场上所供应的打印机共享器大多是无源型的，如果长期使用就会影响到打印机和并行接口的电路性能，从而使打印机所接收和发送的信号受到影响，打印出乱码。在这种情况下很可能将打印机和计算杨接口板损坏，因此就请将计算机系统中的打印机共享器去除，不要使用，然后您就可以用打印机打印出满意的页稿。
　　二、在Windows下安装的BJ－330打印机驱动程序不匹配：
　　在Windows操作系统下要用BJ－330打印机来打印文稿，就必须使用BJ－330的Windows驱动程序。请检查一下您的Windows操作系统中是否已经安装了BJ－330驱动程序。（例如：Epson LQ－1600K型打印机）
#1　　●BJ－330打印机在打印出的页稿上有横向划痕。
　　如果发生这一问题，可以从以下两方面来进行简单的维护：
　　一、在BJ－330打印机表面的前端有一块棕色的透明前盖，这块前盖是专门设计用来固定在打印机纸辊曲面上正在被打印的打印纸的。如果这一前盖没有盖好，打印纸就不会被固定在打印机纸辊曲面上，在打印机小车来回移动的时候，所产生的气流就会将正在被打印的打印纸掀起离开其在纸辊上的固定位置，从而产生横向拖线的情况。所以，在您明白上述的道理之后，请将这块棕色的透明前盖完全盖好，这样就能够打印完美的页稿了。
　　二、使用了卷曲或皱褶的打印纸：
　　在BJ－330打印机上如果使用了卷曲或皱褶的打印纸来打印文稿，就会造成在打印纸上凸出的部分被来回移动的打印机小车碰到，即使打印机上的前盖已被盖好，也会造成横向拖线的情况。所以，建议不要使用卷曲或皱褶的打印纸来进行打印操作。
#1　　●BJC－4100打印机在打印普通单页打印纸时，一次进多张打印纸?
　　如果发生这一问题，很大可能是属于机械上的故障或者打印机的纸张厚度调节杆和打印纸类选择杆所拨的位置不对。
　　由于BJC－4100打印机配有纸张厚度调节杆和打印纸类型选择杆，这些拨杆的设置组合能够适应于不同类型和不同数量的打印介质。所以，这些拨杆的设置正确与否，直接关系到打印纸能否正常地送入打印机。
　　如果BJC－4100打印机在打印普通单页打印纸时，一次进多张打印纸，就表明打印机的纸张厚度调节杆和打印纸类型选择杆所拨的位置不对。应该将纸厚度调节杆调节到向后方向的单页纸位置，并将打印机小车上的绿色打印类型选择杆拨到向左的普通打印纸位置。这样，BJC－4100打印机就不会发生在打印普通单页打印纸时，一次进入多张打印纸的情况了。
#1　　●BJC－210s/sp打印机开机后，电源指示灯闪烁，出错指示灯不亮，蜂鸣器叫一声报警，不进行打印操作?
　　如果发生这一出错现象，表示打印机没有检测到BJ墨盒，有以下这些原因，第一是BJ墨盒没有装好，请检查一下。
　　如果BJ墨盒已经安装在打印机上，但电源指示灯闪烁，出错指示灯不亮，蜂鸣器叫一声报警，那就请您检查一下您的BJ墨盒是否安装正确：打开打印机前盖，检查位于打印机右端的打印机小车上的墨盒，请从侧向进行观察，如果发现BJ墨盒的后平面与小车支架之间有一段空隙，请关闭打印机的电源，并按照要求重新将BJ墨盒安装一次，请将BJ墨盒向前推到底后再将绿色的小车拨杆往下拉，直到看不见BJ墨盒的后平面与小车支架之间的一段空隙为止。然后，再打开打印机，您就可以进行正常的打印操作了。
