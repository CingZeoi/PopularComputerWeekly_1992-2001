中文Windows 95十种最常用的磁盘操作

 
1998年 第03期 07版

#1　　预备知识：
#1　　一、访问文件系统的两种方法
#1　　Windows 95提供了两种方法访问文件系统，一是使用资源管理器，它在显示时分为左、右两个窗口，左边显示所有的文件夹（即目录），右边则显示当前文件夹中的所有文件。它的优点是可以同时看到计算机中全部文件夹和某个文件夹中的文件，这样复制文件比较方便。另一种是使用桌面，单击“我的电脑”，选择某个盘符，出现该盘中的所有文件夹，单击某个文件夹，才能显示其中的文件。这种方法经常要在桌面打开很多窗口，因此，在下面我们主要介绍“资源管理器”中的用法。
#1　　二、启动“资源管理器”的方法
#1　　启动“资源管理器”可以用下面几种方法：
#1　　1.左击“开始”、“程序”、选择“资源管理器” ；
#1　　2.右击“我的电脑”，选择“资源管理器”；
#1　　3.右击“开始”，选择“资源管理器”；
#1　　4.右击任一文件夹或驱动器，选择“资源管理器”。
#1　　在“资源管理器”右边的窗口中，如果一个文件夹中还有其他文件夹，其名称左边会显示一个“＋”。单击这个加号，便会扩展该文件夹的结构；在扩展文件夹后，左边出现一个“－”，单击减号，便可以压缩这个文件夹结构。
#1　　1.列文件目录
　　利用“资源管理器”和“我的电脑”都可以列文件目录，但通常只会列出文件的图标、文件名，如果想要更详细的资料，点击“查看”、“详细资料”，便会显示出每一个文件的图标、名称、类型、最后修改时间，在“查看”菜单中选择“选项”、“查看”，还可以设置文件的显示方式，例如可以设置隐藏某些类型的文件以及显示时是否列出其路径等，设置结束后，单击“应用”。
　　相应的DOS操作：DIR [盘符：]/S/O/A
#1　　2.修改、查看文件的属性
　　文件的属性通常有四种：只读、隐含、存档和系统。在Windows 95中，当我们要查看一个文件的属性时，右击该文件，选择“属性”，可以获得有关文件的所有信息，如大小、创建日期及最后被访问时间，此时，你就可以修改其属性了。设置某种属性时，只需用鼠标单击属性前的方框，使其中出现“√”，如^030701a^。
　　相应的DOS操作：ATTRIB
#1　　3.查找文件
　　Windows 95可以按文件的名称、建立日期、内容和其它条件查找文件，单击“开始”、“查找”，也可以右击某个文件夹或在“资源管理器”中找到“查找”选项。Windows 95除了可以查找文件以外，还可以查找文件夹、计算机。在按“名称和位置”查找时，文件名可以含有通配符“＊”和“？”；按“修改日期”查找时，可输入具体日期或某日期之前（后）。在“高级”栏，用户还可以定义所查找文件的类型、所包含的文件和文件的大小。
　　相应的DOS操作：DIR/S
#1　　4.建立、删除、重命名文件夹
　　1）创建：点击所要建立文件夹的驱动器或文件夹对象，从“文件”菜单中选择“新建”、“文件夹”，随即桌面窗口出现一个新的文件夹，其名称高亮显示。在“资源管理器”中右窗口内，可以看到新的文件夹，键入名字回车即可。
　　2）重命名有两种方法：
　　＊单击文件夹，再单击其名称，便可以修改；
　　＊右击文件夹，选择“重命名”。
　　3）删除有下面两种方法：
　　＊右击文件夹，在键盘上按“DELETE”键；
　　＊右击文件夹，从菜单上选“删除”；
　　＊右击文件夹并将其拖到回收站。
　　相应的DOS操作： MD [目录名]
                    RD [目录名]
　　　　　　　　　　REN [目录名]
#1　　5.复制、移动文件（夹）
　　在Windows 95中，复制、移动文件（夹）有两种基本方法：
　　1）拖放：右击文件或文件夹，并把它们拖到新的位置，出现菜单后，如果是复制，单击“复制到当前位置”；如果是移动，选择“移动到当前位置”。
　　2）剪取、复制和粘贴
　　＊右击要复制（移动）的文件，选择“复制”复制文件或“剪切”移动文件；
　　＊找到要复制或移动到的目标文件夹后，右击该文件夹，选择“粘贴”或“复制”；
　　＊右击一个文件，从菜单中选择“发送”；
　　在复制一批文件时，利用“查找”程序先找到这些文件可以节约很多时间。
　　相应的DOS操作：
　  COPY [路径1] [源文件名] [路径2] [目标文件名]
　　MOVE [源文件] [目标路径]
#1　　6.删除和恢复文件
　　删除：选定文件后，按“DELETE”键或从菜单上选“删除”即可。
　　恢复：双击“回收站”，单击要恢复的文件，从“文件”菜单中选择“还原”。
　　相应的DOS操作：DELETE [路径] [文件名]
　　 　　　　　　　 UNDELETE [文件名]
#1　　7.格式化
　　新盘在使用之前，一定要被格式化。格式化一个已有数据的盘会完全删除盘上的数据。Windows 95不允许格式化文件正在使用中的磁盘。
　　在“我的电脑”中用右键单击某个磁盘驱动器对象，选择“格式化”项；或单击磁盘对象后选择“文件”、“格式化。在“格式化”时有下面一些选项：
　　1）容量：输入和磁盘相对应的容量；
　　2）类型：
　　＊快速格式化：只格式化磁盘，不检查错误，仅抹除原文件分配表，相当于FORMAT A：/Q；
　　＊完全格式化：真正的格式化，可以检查磁盘是否有错并且重新形成扇区和磁道，坏的扇区会打上标记。以前没做过格式化的磁盘只能做“完全格式化”而不能做“快速格式化”。
　　＊只复制系统文件：相当于FORMAT /S，格式化后，还要将系统文件拷贝到磁盘上，该盘可以用做启动盘。
　　此外，还可以选择是否输入卷标。
　　相应的DOS操作：FORMAT [盘符：]/S/Q
#1　　8.扫描
　　磁盘用久了，常会出现一些错误，使用SCANDISK能消除这些错误，但它不能检查光驱、网络驱动器等。具体步骤是：
　　1）右击“我的电脑”中的磁盘驱动器对象，选择“属性”，单击“工具”、“开始检查”；
　　2）单击“开始”、“程序”、“附件”、“系统工具”和“磁盘扫描“。
　　如^030701b^所示，在确定要扫描的驱动器后，选择测试类型，如果是“标准”，则只检查文件和文件夹，选择“全面”、“选项”，确定扫描选项，扫描系统区和数据区。写测试是指在错误检查中磁盘的数据区被读出后，又被写回磁盘。这项测试非常耗时，但比只读检查彻底。
　　相应的DOS操作：
　　SCANDISK或CHKDSK [盘符：] [文件名]/F/V
#1　　9.整理磁盘
　　所谓“碎片”是指一个文件被分为几个部分存在磁盘的不同区域中，“碎片”现象通常是由频繁地拷贝、删除文件造成的。如果磁盘中的“碎片”过多，会使读写时间延长，降低效率。在Windows 95中可以用磁盘整理程序消除碎片。在整理过程中，碎片都被临时转移到磁盘不用的空闲空间，一旦一个文件所有的碎片都收集在一起，便将文件移到靠近磁盘引导区边缘的一片连续区域中，最终碎片都消失了。下面两种方法都可以启动磁盘整理程序：
　　1）单击“我的电脑”，右击所要整理的磁盘驱动器，选择“属性”、“工具”，单击“开始整理”；
　　2）单击“开始”、“程序”、“附件”、“系统工具”、“磁盘碎片整理程序”，选择驱动器，按“确定”。
　　如果磁盘中的碎片很少，系统会提醒用户无需整理，但如果用户坚持继续整理，点击“开始”即可。
　　相应的DOS操作：DEFRAG [盘符：]
#1　　10.备份、还原文件
　　单击“开始”、“程序”、“附件”、“系统工具”、“备份程序”；或选定某一驱动器对象后，按右键，选择“属性”、“工具”、“备份”都可以使用备份程序。备份程序中主要有三个功能：备份、还原（恢复）和比较文件。
　　相应的DOS操作：BACKUP [源文件名][目标盘]
                   RESTORE [目标路径]/S
