《北方密使》（白志超路线）简明攻略

王波
1998年 第15期 28版

　　笔者是个很懒的人，所以攻略部分只写了我打的路线，大家要看所有的路线就看后面的流程图吧。
#1第一部分
    ^152801f^
#1　　第一神剑
　　开始嘛，难度都不大，第三回合后敌方有援兵。不过，我方在第五回合也会有邢天的加入，唯一的遗憾是你不能直接领导他，这只能希望志超早日学会领导技能，以后就能控制他了。
　　（9，40） 法师帽；（11，17）隐藏　药草;（13，33）化石草；补给：金钱1500，药草X2，疗伤草，无村庄。
#1　　拦路虎
　　难度与前一关相差无几，只是武器换成了长刀，可攻击两格。第四回合敌人会有援兵出现。
　　（2，16） 绳索；（10，38）行气散；（15，18）隐藏 力量辉石；补给；金钱2000，药草，魔草，有村庄。
#1　　镜湖城
　　邢天老婆秋月以及连弩手莱迪加入，秋月是一个十分厉害的角色。注意本关老大会隐形，须贴着他才看得见。
　　（10，34）魔草；（14，10）金钢石；（16，35）隐藏 疗伤草；补给：金钱2100，有村庄。
#1　　血染哨站
　　迅速向哨站移动，并留意勿让哨站的人都死翘翘了。战斗结束后将会出现分支剧情（1，0）。
　　（7，26）上锁，智慧果；（11，52）丝衣；（14，40）隐藏解毒草。
#1　　遗迹恶鬼
　　此关的不死生物会令你头疼，不过在商店里买的张天师的灵符还蛮管用。
　　（1，12）战斗力仪；（14，7）个人传送器；补给：金钱2000，无村庄。
#1　　黄沙烈焰
　　如果你能在这关里练出一个有飘浮或传送能力的兵种，那就再好不过了。注意保护商队的安全。
　　（2，8）  巨剑；（21，34）魔法盾；补给：金钱2000，有村庄。
33　　 沙盗营地
　　杀掉全部沙盗就行，注意用商队吸引一部分火力，别忘了拿转职物品——银牙弯刀。^152801a^
　　（0，11）隐藏　象牙圣杯；（2，29）隐形药水；（3，12）兴奋剂；（4，6） 上锁，玫瑰石；（7，12）银牙弯刀；（10，5）上锁，闪动珠；补给：金钱2500，无村庄。
#1　　沙盗黑蝎子
　　最简单的方法，干掉首领黑蝎子（是不是有点废话）。
　　（7，48）生命之果；补给：金钱3500，有村庄。
#1　　罕泉哨所
　　打或不打随便你，跟剧情的发展也没有什么影响。本关结束后会出现分支剧情（2）。
　　（7，34） 能量值；（16，33）小还丹；补给：金钱3000，有村庄。^152801b^
#1第二部分
    ^152801g^
#1　　恶德商人
　　本关难度不大，只有头目祈师父实力较强，最好用催眠术之类的魔法对付他。结束后又会出现分支剧情（3）。
　　（3，22） 无极战衣；（5，16） 闪动珠；（8，12） 魔导士长袍；（17，22）爆裂斧；（21，33）充血的棺材；（22，37）隐藏　吸血鬼之齿；补给：金钱5500，有村庄。
#1　　人口贩子
　　只需注意十回合后敌援兵弓箭兵，他们占有地形优势，本关结束后将出现分支剧情（4）。
　　（0，25） 微笑的面具；（0，43） 运功散；（17，33）漂亮的洋装；补给：金钱4500，有村庄。
#1　　神秘的访客
　　吸血鬼会将对方的伤害转化来为自身疗伤，所以千万小心。
　　（5，22）精钢盾；（9，43）射手弓；补给：金钱6000，有村庄。
#1　　意外的友军
　　女主角娜塔莉雅终于出现了，不过由于敌人太强了还是趁早走人才好。
　　（2，55）上锁，丑陋的面具；（6，38）上锁，骑士靴；（18，34）小还丹；（27，30）魔法之酒；补给：金钱4500，有村庄。^152801c^
#1　　沙漠名珠
　　又要保护弱角不能放手一搏，唉……小丑有“灵击”的能力，不管你防御力有多高都会失血，更别说他们还会远程攻击了，所以本关要加以小心。只要坚持十个回合后小丑们就会退却。
　　（4，29）仙丹；（13，23）铁面具；（16，33）光之矛；（27，51）龙筋战衣；补给：金钱4000，有村庄。
#1　　领主的晚宴
　　难度颇大的一关，志超这时如已有“领导”能力，还可以试试，如没有就快溜之大吉吧。
　　（2，18）隐藏　粉晶石；（12，28）魔王的点心；（15，43）水晶战甲；补给：金钱6000，无村庄。
#1　　奚族刺客
　　与黑蝎子那关差不多，干掉大魔道后就完事了。本关又有分支剧情（5）。
　　（6，42） 上锁，忍靴；（9，41） 通神草；（15，47）精钢盾；补给：金钱6500，无村庄。^152801d^
#1　　潜入领主府
　　守卫的能力不强，十五回合内搞定他们应不成问题。
　　（9，25）上锁，无名剑谱；（9，50）龙之麟；补给：金钱6000，无村庄。
#1　　两只老鼠
　　本关僵尸王非常难对付，在他的血很多时，千万不要近他的身，否则，他的血咒法术一次就可以让我方的一名队员阵亡。还有他与圣职的罗莎莉是互克的（如果你还有转职的话），彼此的伤害都会以三倍计算。
　　（7，51）魔法之酒；（10，39）行气散；（18，55）猫眼石；补给：金钱6500，无村庄。
#1　　携手合作
　　克雷布特里被干掉后，会得到军事计划书。
　　（14，42）魔力冠；（16，46）隐藏　诅咒的法杖；（17，49）兵法书；补给：金钱7500，有村庄。
#1　　侧门突围
　　干掉骑士团统领就行了，不过骑弓手的能力也不能小视。
　　（14，42）魔力冠；（16，46）隐藏　诅咒的法杖；（17，49）兵法书；补给：金钱8000，有村庄。
#1　　状府地牢
　　武师的“气功”威力不小，要稍加留意。干掉典狱长就会得到万能钥匙。
　　（6，14）上锁，高级战靴；（9，21）虎眼石；补给：金钱8000，有村庄。
#1　　密盗库库克
　　相当简单的一关，南巡使的离去也无关紧要，主力库库克会在战役结束后加入我方。
　　（1，55）飞燕刀；（10，49）力量辉石；（24，16）生命之果；补给：金钱3000，有村庄。
#1　　军事计划书
　　卖身契这个转职用的好东东一定不要忘了拿哟！
　　（7，42）卖身契；（10，39）隐藏　火神珠；（14，42）连弩；（17，49）智慧果；补给：金钱9000，有村庄。
#1　　秘密据点
　　敌人的实力虽不强，但占有地利优势，只要注意一下，就能轻松过关。本关又有分支剧情出现（6）。
　　（7，53）龙之麟；（19，62）魔法之盾；补给：金钱10000，有村庄。
#1第三部分
    ^152801h^
#1　　前往军港
　　本关敌人身上的道具多多，所以就先暂时留圣堂骑士小命一条，等奖品拿完再解决他。
　　补给：金钱10000，有村庄。
#1　　骷髅兵团
　　又是不死生物，这次就让罗莎莉大显神威吧。又有分支，是打隐藏关，还是多收一个人呢？
　　（8，22）丑陋的面具；补给：金钱12000，有村庄。
#1　　官道巡弋者
　　敌人实力本来就弱，再加上不断增加的强力援兵，我看还是走人的好。
　　（0，41） 隐藏，偏袒的天平；（14，63）隐藏，变身丸；补给：金钱无，有村庄。
#1　　女妖之塔
　　销魂猫具有“灵体”的特殊技能（不怕物理攻击），用法术对付它会轻松很多。头目女王尤西丝的生命值很高，且法术也很高明。所以只有坚持、坚持、再坚持。
#1　　落难的骑士
　　第四回合有敌人援兵出现，5个邪灵使，1个巫师。不过请不要在四回合前结束战斗。
　　（3，42）巫蛊覆面；（10，61）上锁，龙鳞牙；（15，55）无名剑谱；补给：金钱13500，有村庄。
#1　　港都喋血
　　记住三十回合内必须到达敌方军旗处。^152801e^
　　（1，27）隐藏，丑角面具；（2，59）护身项链；（22，53）琥珀。
#1　　追击柯莉特
　　此关的宝箱一定要取到。
　　（1，52）上锁，正气战衣；（13，49）上锁，兵法书；（14，23）隐藏，妖精药罐；补给：金钱10000，有村庄。
#1　　小村劫祸
　　解决柯莉特还是用群殴这个老方法比较管用。
　　（9，34）隐藏，龙鳞头盔；（17，18）上锁，七彩镜盾；
　　补给：金钱1200，有村庄。
#1　　击破运金团
　　（3，25）智慧果；（8，58）漂亮的洋装；补给：金钱15000，有村庄。
#1　　山巅之役
　　本关难度很低，慢慢练功吧。
　　（3，25） 隐藏，孔雀石；（16，44）幻之镜；补给：金钱12500，有村庄。
#1　　研究室
　　这一关比较具有挑战性，记住将机兵一个一个地引近我方再打，并小用敌方魔法师的魔法。（17，60）隐藏，个人传送器；补给：金钱15000，有村庄。
#1　　散财童子
　　坚持10回合，村民没有被杀光，并且，敌人也没接触到你的军旗就行了。本关敌人的数量不少，不过很容易对付。如果战斗后，村民们没有阵亡，散财童子就会要求加入队伍。
　　（11，23）隐藏，神奇闪避鞋；（12，18）祖母绿；（11，75）隐藏，充血的棺材；补给：金钱15000，有村庄。
#1　　重装机甲
　　本关又是一个练功的好地方。
　　（6，27）无极仙丹；（7，51）玄龟外军；（18，31）玄铁重剑；补给：金钱16000，有村庄。
#1　　命运分歧点
　　一定要控制进领主府的出入口，如谁控制了这里，谁就能把主力部队调到决定点上，扭转战局。
　　（13，20）力量辉石；（21，68）隐藏，黑曜石；补给：金钱18000，有村庄。
#1　　小丑献媚
　　本关夏普死后，敌方会有大批的援兵出现，且会从南北两侧出现。如果你有实力，去打败贝洛的人也可以，这样会增加额外的评价与名誉。
　　（21，72）银牙弯刀；（27，36）妖精药罐；补给：金钱20000，有村庄。
#1第四部分
    ^152801i^
#1　　蛮河水坝
　　敌方的弓箭手占有地利优势，再加上有上忍的纠缠，没有“侦察”能力，就要靠施展“真实之眼”的法术找出上忍了。
　　（2，52）魔法之酒；（7，32）小华珠；（13，48）月长石；（20，36）兵法书；补给：金钱18000，有村庄。
#1　　提那城
　　我方的脱离点跟我方位置距离不远，在这儿大可趁机捞点经验与宝物，支持不住再跑也不晚。
　　（3，49） 生命之果；（10，77）隐藏药水；（23，58）珍珠；（23，75）偏袒的天平；补给：金钱20000，有村庄。
#1　　迟滞敌军
　　本关会不断出现一些小妖来让你练功，十五个回合就慢慢练吧。
　　（3，29）智慧果；（7，28）护身项链；（12，26）隐藏，玫瑰石；补给：金钱20000，有村庄。
#1　　坚守高地
　　对于敌人法师和小丑，最好争取在第一次攻击时就打垮他们的大多数，否则我方可能会受到重创。
　　（8，63）  隐藏，猫眼石；（12，57） 上锁，仙女棒；（15，36） 兵法书；补给：金钱20000，有村庄。
#1　　提那南桥
　　占领军旗动作一定要迅速，否则在第九回合敌方会有一支援军出现，不过本关的敌人很适合用来练功。
　　（11，54）隐藏，龙之鳞；补给：金钱25000，有村庄。
#1　　侧翼的决战
　　这一关第七回合出现的援兵很强大，要好好地协同奚族作战。本关的宝物也都不错，请一定拿到。
　　（3，8）  上锁，仙丹；（10，7） 隐藏，月长石；（17，35）星象宝典；补给：金钱25000，有村庄。
#1　　力量与魔法
　　我方开始只有3人，敌人却多如牛毛。还是忍一口气守住军旗，等到八回合刑天夫妇归队后再从长计议吧。
　　补给：金钱30000，有村庄。
#1　　和平会谈
　　决定的时刻到了，是要橄榄枝，还是要血与火都在于你了。
　　（14，29）古老魔法书；（20，70）魔王的点心；（27，58）孔雀石；补给：金钱30000，有村庄。
#1　　内讧
　　（14，29）古老魔法书；（20，70）魔王的点心；（27，58）孔雀石；补给：金钱20000，有村庄。
#1　　落魄的元帅
　　什么元帅现在对我来说都能轻而易举就解决掉。
　　补给：金钱25000，有村庄。
#1　　奚族圣地
　　由于西碧拉会使用攻击面很广的魔法，所以在我方接近敌人之前，会先受到一定程度的伤害。如果你在这关还没有练出“领导”能力，你将会……
　　（1，62）长刀（银霞）；（14，33）上锁，晶碳战甲；（14，67）隐藏，铡斧；（24，48）上锁，炎魔狮子盾；（27，75）上锁，反引力滑靴；补给：金钱32000，有村庄。
#1　　返乡的游子
　　本关只能依靠传送点进行传送，但传送地点却是随机的，只能祈求上帝把你的强力伙伴一一传送到克雷布特里身边。如果你等到十五回过关就没有下一关可以玩了。
#1　　超时空传说
　　在此，如果队伍还无一人会高温气化这一法术，那就赶快去取得不明矿石，这是你制胜的法宝。除去克雷布特里的装备后，就送他到西天极乐世界去吧！
　　（3，65）不明矿石。
