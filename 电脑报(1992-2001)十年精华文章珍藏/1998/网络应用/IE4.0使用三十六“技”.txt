IE4.0使用三十六“技”

太阳雨
1998年 第17期 19版

#1　　IE4.0是目前网友用得最多的浏览器之一，已成为大家上网冲浪必不可少的重要工具。常言道，工欲善其事，必先利其器。如果你想在网上轻松冲浪，细读本文必有帮助。
#1　　一、安装符合自己需要的软件版本
　　IE 4.0和Communicator 4.0一样，都不是一个单纯的WWW浏览器，而是一套多功能的Internet应用程序。IE 4.0分为标准版（包含浏览器，负责电子邮件、网络新闻管理的Outlook Express和多媒体增强组件）和完整版（比标准版又多了网络电话功能的NetMeeting组件、编辑HTML文件的FrontPage Express组件、Web发行向导、多媒体组件NetShow和通讯组件Chat 2.0等）。
　　网友应根据实际需要来决定使用何种版本，对多数拨号上网的用户来说，建议安装标准版。
#1　　二、根据电脑的硬件配置来确定组件的安装
　　虽然微软声称IE 4.0的最低系统要求是486 DX2－66 CPU搭配8MB RAM，不过这仅是指只安装浏览器的情况。若你还想安装Outlook Express和Active Desktop等组件，在Windows 95环境下至少也要Pentium级的CPU外加16MB RAM，如果是在NT 4.0的环境，则要24MB以上。总之，应根据自己的硬件配置来确定可安装哪些组件。
#1　　三、安装正式版本前务必先删除测试版
　　在安装IE4.0正式版之前，如已安装了测试版（Preview Releases），请务必先将其删除，再行安装。如想用新版本覆盖旧版本的方法安装，那将是自找麻烦。同样地，如果已安装好IE 4.0，就不要再试图安装任何以前的版本（如IE 3.0），如确有必要，请先将其删除，再进行安装操作。
#1　　四、与Office 97出现不兼容问题时的解决方法
　　怎么微软自己的产品会不兼容呢？别紧张！这只发生在你先安装了IE 4.0，然后又安装Office 97，最后又删除了IE 4.0的情况下。这是因为两个软件共用部分文件所致，当安装IE 4.0时它会自动将所取代的旧文件以建立backup文件方式保留，如果在安装Office 97后删除IE 4.0，则它会还原那些旧文件，而覆盖Office 97会用到的共同文件。解决方法很简单，重新安装一遍Office 97就OK了。
#1　　五、离线浏览网页
　　利用读取硬盘上的Cache文件，IE 4.0允许用户进行离线浏览，这可说是一种省钱的变通方法。请在主目录栏的“文件”中选“脱机工作”即可。此时IE 4.0将不理会是否处于连线状态，直接到硬盘上抓Cache。在离线浏览时，如果网页上某链接标记并不存在于Cache中，则鼠标旁会出现一个类似禁止通行的小图标，以便识别。若仍然选取该链接，则系统会问你是否要建立连接以取得网页或是仍保持离线。
#1　　六、重设工具栏
　　IE 4.0的画面上部分为标准按钮、地址和链接等三个工具栏。你可在主目录栏的“查看”中的“工具栏”项，选择将其消隐或呈现。还可将这些工具栏集合为一栏，请用鼠标按住其左端，拖曳它们到同一栏。如集合后要重现该工具栏，双击该栏即可。
#1　　七、全屏幕显示
　　如果嫌画面太小，请选择标准按钮工具栏上的“全屏”，IE 4.0的操作界面就不会占据网页，等于是让你以14英寸屏幕看15英寸屏幕的效果。重复选取则回复原状。
#1　　八、建立快速链接
　　你不用每次都进入“收藏夹”里，选择去处。请用鼠标拖曳网页上的链接标记（文字或图形皆可）或是“收藏夹”中的书签、“记录”里的浏览记录到链接工具栏上，然后只需鼠标轻轻一点这些图标即可直接连上。如果链接工具栏上放了太多东西，无法同时呈现，请按其末端的箭头卷动。
#1　　九、改变快速链接的图标
　　快速链接固然好用，但统一的图标却嫌单调，也无法代表各网页的特性。如想改变，请在其上单击鼠标右键，在出现的菜单“属性”中Internet快捷键方式页中选择“更改图标”。如果还是不够看，请选其上的“浏览”，将路径指向C:\windows\system\shell32.dll，即可发现更多的图标。
#1　　十、加强网络安全
　　在IE 3.0中，你可设置接受Cookie和建立ActiveX时的警告功能。但是设置好该选项后，常常在浏览网页时不断出现警告提示；若不设置，又有网络安全的顾虑。然而IE 4.0提供一个好办法，让你可兼顾浏览时的安全性和方便性。请到主目录栏中的“查看”选“Internet选项”中的“安全”，将各网站依其可信程度分类到不同的组中，再设置不同的安全标准。网站组可分为“本地企业网区域”（指自己公司Intranet上的网站）、“可信站点区域”（指那些你信得过可上传和下载任何文件的网站）、“Internet区域”（指一般或不熟悉的网站）和“受限制的站点区域”（指那些臭名昭著或相当可疑的网站）等四类。像在“可信站点区域”的安全标准就可设低些，不用凡事都警告；而在“受限制的站点区域”则须把安全标准设到最高，以高度警戒其可能的破坏行为。
#1　　十一、善用浏览栏
　　IE 4.0新增的浏览栏功能，可让你点选标准按钮工具栏上的“搜索”、“收藏”、“历史”和“频道”等四项，让它以独立的目录栏方式出现在画面左方的小窗口中，若选取其中的链接，则目标网页会出现在右方的主窗口内。在连到所要网页后，可再次到标准按钮栏上点选该项，则分割画面消失，回复原状。
#1　　十二、巧用搜索引擎
　　搜索引擎是你上网查找信息很重要的工具，但是使用某搜索引擎时，你必须先连上某搜索引擎，输入关键字进行检索，再从一堆检索结果中做进一步连接。如果不是你想要的，那又得回到前一画面，重复前面的过程。不过，IE 4.0的检索程序可就方便多了。请到标准按钮工具栏上选“搜索”，即可在左方发现预设的搜索引擎小窗口，直接在其上进行检索，检索结果会出现在该小窗口中，并不会影响到原有的主窗口。在检索结果中选择欲前往的网页，则该目标网页会显示在右方的主窗口中，并不会盖住检索结果画面。如此可避免以往来来回回检索的不便。
#1　　十三、网络搜索小技巧
　　当搜索引擎报告检索结果后，你可以不必连接到目标网页即可获得摘要信息。请将鼠标置于左方检索结果窗口的各条目上，即可出现其相关信息（注意，并非所有的搜索引擎都支持该功能）。
#1　　十四、增强的打印功能
　　IE 4.0的打印功能已经增强，如需打印网页，不用直接到标准按钮工具栏上选“打印”，而是到主目录栏“文件”中选“打印”（或是按Ctrl＋P）。在调出的打印窗口中，“按屏幕所列布局打印”是指打印全部画面，“仅打印选定框架”是指在frame构件的分割窗口中，可指定单一窗口来打印（不过得先用鼠标单击该窗口加以识别），而“逐个打印所有框架”则是指逐一打印出frame中的每一个窗口。
#1　　十五、直接组织“收藏夹”
　　以前组织“收藏夹”可说是件苦差事，因为非得展开一个新窗口来执行。但在IE4.0上，你可在“收藏夹”菜单中直接用鼠标拖曳各网页书签来重组其次序，或是用鼠标右键单击它们来组织。
#1　　十六、快速前进和后退
　　在浏览网页过程中，如果一时想跳到前面、后面好几页的画面，不要只是猛按标准按钮工具栏上的“前进”、“后退”钮，请使用“前进”、“后退”钮旁边的“”钮。
#1　　十七、发送整个网页画面
　　点选标准按钮工具栏上的“邮件”项，在菜单上除阅读邮件、新建邮件、发送链接外，值得注意的是新增的“发送网页”功能。它可将整个网页附带的图文文件一并寄给对方。如对方的E－mail软件可支持HTML格式邮件，那他不必连往该站也可浏览其网页。
#1　　十八、编辑HTML格式的邮件
　　Outlook Express允许你编写和查看HTML格式的邮件。请点选标准按钮工具栏上“邮件”中的“新邮件”，再到新邮件窗口主目录栏中的“格式”中选取“HTML"项，一个编辑HTML邮件的新工具栏即出现。
#1　　十九、混合存放电子邮件与网络新闻
　　Outlook Express将浏览电子邮件与网络新闻的界面集合于一起，因此不必打开新窗口或调出应用程序即可同时读取。亦可将这两种文件混合存放，只要用鼠标直接拖曳窗口右方的邮件和新闻文件到左方的收件箱中即可。
#1　　二十、即时增加新收件箱
　　只要用鼠标右键单击Outlook Express左方的收件箱目录，选取菜单中的“新文件夹”，即可建立新收件箱。你还可在同一收件箱中再建立多个子收件箱。
#1　　二十一、筛选网络新闻
　　网络新闻组的内容庞杂，通常远超过个人的实际需求，该功能就是为了解决这个问题而设。请在Outlook Express的主目录栏上“工具”的“收件箱助理”中选“添加”，设置筛选器内容。首先选定想筛选的新闻组，再设置过滤标准，可包含你不欢迎的寄件者E－mail地址、新闻主题、文件长度、公布日期等。
#1　　二十二、使用多重账号来收发电子邮件和网络新闻
　　现在，一人有多个E－mail账号已不稀奇。而当你在安装IE 4.0时，就已先设好了一个账号。如想新增，请在Outlook Express主目录栏上“工具”的“账号”中选“添加”，再选择“邮件”或“新闻”，然后依连线向导的指示进行即可。要查看新邮件，请到主目录栏上“工具”的“发送及接收”中，可选择查看单一账号，或同时查看多个账号。
#1　　二十三、自动筛选和分类邮件
　　IE 3.0已有该功能，但IE 4.0则更上一层楼。请在Outlook Express的主目录栏上“工具”的“收件箱助理”中选“添加”，设置分类标准。除原有的收件人、副本、寄件人、信件标题等标准外，IE 4.0还多了自动回复和转寄、直接在邮件服务器上删除信件等选项，可协助你筛选和分类新邮件。
#1　　二十四、从Netscape导入通讯簿和邮件
　　刚安装好IE 4.0不要就忙着重建通讯簿，如你原先是使用Netscape的E－mail组件，其实可将其通讯簿和邮件导入IE 4.0中使用。在Outlook Express的主目录栏上“文件”的“导入”中选择要导入何种资料，再依提示进行。而Netscape就无此功能。
#1　　二十五、将新的E－mail地址添加到通讯簿中
　　如果接到一封来自新朋友的信，可直接将其E－mail地址加入通讯簿中，而不必重打一遍。请先选取工具栏上的“回复作者”，在展开的回信窗口上用鼠标右键单击该E－mail地址，然后在菜单中选“添加到通讯簿”即可。
#1　　二十六、快速搜索亲友的E－mail地址
　　IE 4.0可以帮助你在整个Internet上搜索亲友的E－mail地址，当然，其前提是他们得先在网络人名录上注册过。请在主目录栏上“编辑”的“查找用户”中输入查询资料，那里有Bigfoot、WhoWhere、和Four11等几个主要的人名录可供检索。
#1　　二十七、在发送邮件中附上个人电子名片
　　除了典型的签名文件外，IE 4.0还可让你附上如同电子名片般的个人名片（V－Card）。请到主目录栏上“工具”的“信纸”中选“签名”，然后编辑“个人名片”。
#1　　二十八、安装Windows桌面更新组件
　　IE 4.0内含有一个名为“Windows桌面更新”的额外组件，可让用户将该软件真正集合到Windows操作系统中。如果你想享用Web Active Desktop，请在安装时选择接受此项。
#1　　二十九、以Web方式显示电脑的文件夹与文件
　　经过集合后的操作环境，会让电脑中的文件夹与文件外观面目一新，也可让你浏览它们就和Web一样容易。此时Windows资源管理器就好像浏览器一样，做任何动作都只要鼠标单击，而非以往的连续双击。如果你在某文件夹中再打开另一文件夹，并不会有个新窗口出来，而是直接显示在原有窗口上。若想回到前画面，只须选取工具栏上的“上一页”按钮。
#1　　三十、启用Active Desktop功能
　　IE4.0的Active Desktop可将股票行情、天气预报等即时信息放在桌面上。其内容是以主动播放频道为基础而设计的，微软为了让该功能充分发挥，而与其他业者合作，为国内网友提供实用的中文信息。目前可接收到人民日报、中国国际互联网络、经济信息网络、上海热线等频道的信息，还可通过频道指南找到更多精彩的选择。若要改变接收内容，请用鼠标右键单击桌面空白处，调出菜单直接修改。
#1　　三十一、关闭Active Desktop功能
　　刚启用Active Desktop功能的网友会觉得它很“炫“，不过它也是不折不扣的系统资源杀手。若因其耗损资源而影响其他程序运行时，你可随时在桌面上单击鼠标右键调出菜单，选择暂时关闭它。还可再次用此法重新启动该功能。
#1　　三十二、使用新的任务栏
　　Web与窗口系统集合后，桌面的任务栏也变得不一样了。若有必要，你可把一些组件（如浏览器的地址栏、快速链接栏的网页标记或其他应用程序等）加到里头。请用鼠标右键单击任务栏的空白处，在弹出菜单的“任务栏”项中选择之。如果加了太多东西，任务栏过于拥挤的话，则可调整各组件间距，或是拉高任务栏，以双栏方式显示。
#1　　三十三、在桌面上建立网页链接
　　如果已启动Active Desktop，就可在桌面上建立网页链接。用鼠标把正在浏览的网页上的重要链接标记拖曳到桌面，以后点选该链接，即可方便地直接前往该处。
#1　　三十四、快速查找文件夹
　　经Web集合后的Windows资源管理器里有个地址栏，你可在其中输入要前往的文件夹路径和名称，而Auto Complete即会自动地协助你完成后续部分。如果出现的并不是你想要的，请用鼠标右键单击该地址栏，在随后出现的菜单中点选正确去处，再按Enter即可。
#1　　三十五、DOS不死？
　　谁说DOS已被扫进历史的垃圾堆里了？经Web集合后的Windows资源管理器可接受DOS指令，如在文件夹的地址栏中键入“..”，即可回到上层目录。当然，最便捷的方法是直接选取工具栏的“回到上一层”按钮。
#1　　三十六、让文件夹更美观
　　经Web集合后，你可运用HTML指令让文件夹看起来像网页一样美观。请先由Windows资源管理器进到该文件夹窗口，在空白处单击鼠标右键，在弹出菜单中选“自定义文件夹”，然后依照向导的指示进行。最简单的是只选一个图形文件来当成文件夹背景。若要以HTML指令布置文件夹，则向导会调用预设的HTML编辑器来执行。同样地，如想恢复旧观，也得通过该向导。
