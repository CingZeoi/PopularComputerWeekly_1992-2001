共享文本编辑器评测报告之档案篇

歪歪
1998年 第38期 11版

　　各位在看完共享文本编辑器评测报告后，可能对编辑器又有了一个新的认识。的确，歪歪在参加完这个测试后，发现它们的功能真是多多。在评测中，一些无法用数据来表达的观点和歪歪的一些看法便构成了这篇文章。 
#1EditPlus 1.1
     EditPlus在一些无法用评测数据体现的方面，具有良好的创意，最具潜力。
　　文件大小:914K；类型：共享软件；费用:＄20.00
　　下载地址：http://www.editplus.com
　　插件地址：http://www.www.editplus.com/files.html
　　评述：EditPlus在评测中的各项数据并不突出，只能说是具备了一般的特性。其查找和替换的搜索算法还有一定的问题。但EditPlus在一些无法用评测数据体现的方面具有良好的创意，夺得最具潜力奖。首先，它采用了一个开放的开发方式，允许使用者开发相关的插件和模版，（单击可出现文本快捷粘贴栏）在它的主页上很多使用者提供了插件和模板（最新的插件发布日期为9月24日），下载后解包拷入EditPlus的目录就可以给文本快捷粘贴栏和模板增加很多内容。其次，它在评测软件中唯一实现了同浏览器的无缝连接，它调用IE3以上版本的核心，可以马上把EditPlus变成一个小型的浏览器。监视剪贴版的功能也非常好用。当然，该软件由一个韩国人开发，其双字节的支持自然是非常优秀的。同EmEditor一样我们在网上也看到了一个简体中文的汉化版，希望作者取得合法授权。
　　推荐使用对象：全国网民^381101a^
#1Cobpad Plus 2.1
　　可以对文件加密是选择这个软件的主要原因，当然用它替代Windows 95/98的NotePad 记事本也是绰绰有余。
　　文件大小：268K；类型：免费软件
　　下载地址：http://www.lh.umu.se/～cobian/
　　评述：Cobpad Plus 2.1也是一个免费软件，其编辑功能与其他评测软件相比相对较弱，就其编辑功能来看我们推荐使用EmEditor，但在这个免费的Cobpad Plus 2.1中有几项独特的功能：可以对文本进行加密；可以无需开启任何外部电子邮件系统发送邮件；在程序内打开注册表，系统配置文件，控制面板等。可以对文件加密是选择这个软件的主要原因，当然用它替代Windows 95/98的NotePad 记事本是绰绰有余。
　　推荐使用对象：希望完全保护个人隐私的用户，分居两地的情侣。^381101g^
#1Super Note Tab 4.01
　　真是一个可口的鸡肋，如果你有5美元闲钱，不妨一试。
　　文件大小:1340K；类型：共享软件；费用:＄5
　　下载网址：http://www.notetab.ch/
　　评述：尽管Zdnet的对它评价很高，但在测试中我们发现它的速度和流畅性与其他共享软件非但无出色之处，而且还很慢。内建的文本快捷粘贴栏（可以进行简单的数学运算），自动滚屏，自动监视剪贴板和＄5美元的费用是它的卖点。当然与免费软件相比它的功能还是强了很多，但我们不认为它是一个必需的软件。真是一个十分可口的鸡肋，如果你有5美元闲钱，不妨一试。
　　推荐使用对象：有5美元闲钱的用户。^381101h^
#1EmEditor 1.27
     我们认为它完全可取代 Windows 95 或 NT内建的NotePad 记事本。
　　文件大小：264K；类型：免费软件
　　下载地址：http://www.emsoft.co.jp/ftp_pub/EmEditor/English/eme127ei.exe
　　繁体中文补丁：http://www.emsoft.co.jp/ftp_pub/EmEditor/contribute/cemed127.zip
　　评述：EmEditor的最高版本为2.0 Beta3，但这是一个共享软件，需要注册。他的1.27版是一个完全免费的软件。它的最大特点可直接在文件中打开连接网址 URL（支持 HTML 超链接）。 他支持双字节字集不再有乱码，且打开文件大小不受限制，撤消及重做次数无限，更也可自定义热键、字型、色彩、工具栏、醒目字句有分割视窗等功能!再加上它完全免费，我们认为它可完全取代 Windows 95 或 NT内建的NotePad 记事本功能。在它的主页上更可看到一个繁体中文的补丁，这是台湾的一个爱好者汉化该软件后得到日本作者的同意放在上面的。在网上我们已经看到了一个简体中文的汉化版，我们希望该软件的汉化者同作者联系，在得到同意后放在EmEditor网站上，以造福广大中国电脑爱好者。
　　推荐使用对象：一切电脑用户都可以用它完全取代 Windows 95 或 NT内建的NotePad 记事本。^381101c^
#1UltraEdit32 5.21
　　它强大的功能和流畅的速度，使它完全有资格成为本次评测的冠军。
　　文件大小:1，073K；类型：共享软件；费用:＄30.00；试用周期:45天
　　主页:http://www.idmcomp.com
　　评述：UltraEdit32是编辑器里的长青树，中国很多用户以前不用它很可能是因为它以前的版本不支持双字节，这个版本可以完全支持双字节，再也不会出现删除半个汉字的尴尬了。16进制的编辑和列编辑是它最大的特色。它完全的功能和流畅的速度完全有资格成为本次评测的冠军。如果还要我们挑一些毛病的话：＄30美元的注册费太贵了。列编辑要求等宽字体和HTML语言的支持不够是它还需要改进的地方。^381101f^
　　推荐使用对象：游戏爱好者（修改游戏用），电脑高级用户。
#1GWDTextEditor2.0
     一个充满激情和超乎想象力的作品，每一个程序员都该看看，可惜它不支持双字节。 
　　文件大小:1690K；类型：共享软件；费用:＄20
　　http://www.iridis.com/gwd/index.htm
　　评述：这是一个一般用户会觉得很普通的编辑器软件。但如果你会C语言的话，你很可能会被它强大的内建脚本功能所震撼。这是唯一令我们的评测人员感到震惊的编辑器，（尽管它的这些功能已超出了编辑器的范围），对C语言的全面支持加上自身函数和数十个WindowsAPI的调用。在上面它完全可以写出一些功能强大的脚本（甚至可媲美word的宏）。无需编译，马上运行，就这么简单。这真是一个程序员为程序员写的编辑器，一个充满激情和超乎想象力的作品，每一个程序员都该看看，可惜它不支持双字节。^381101d^
　　推荐使用对象：每一个程序员。
#1Text Pad 3.2.4
    这个软件的最大特点就是没有特点。
　　文件大小:1200K；类型：共享软件；费用:＄27
　　下载网址：http://www.textpad.com/
　　评述：这个软件的最大特点就是没有特点。它中规中矩的实现了文本编辑器的绝大部分功能。如果没有UltraEdit32出色的16进制的编辑功能，没有EditPlus全面的创新精神，我也许会选择它，不过＄27美元的费用和不支持双字节的特性会让我犹豫一下。^381101e^
　　推荐使用对象：不予推荐！
#1发展与遗憾：
　　在完成这个评测后，我们发现编辑器的发展方向有如下几个方面：一、与网络的联系日渐紧密，支持HTML语言的关键字彩色显示，HTML标记的快捷生成，超链接的点击打开，同浏览器无缝连接和内置的邮件系统都在评测软件中得到了不同程度的体现。二、 强大的软件可扩展性。在评测中，以GWD TextEditor2.0为代表的编辑器（包括EditPlus， Super Note Tab， UltraEdit32）提供了强大的二次开发和扩充的能力，使软件充满了活力，我们认为这符合网络开发的精神。
　　在这次评测中，我们比较遗憾的是，没有找到一款和评测软件匹敌的国产共享编辑器。我们看到日本的EmEditor和韩国的EditPlus都有了一定的水平，心里感到比较难受。希望我们的评测会给国内的共享软件开发者一丝启迪！
