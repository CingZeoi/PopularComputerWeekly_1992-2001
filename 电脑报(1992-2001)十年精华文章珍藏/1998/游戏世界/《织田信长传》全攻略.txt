《织田信长传》全攻略

火舞流星
1998年 第31期 29版

　　 织田信长，这个名字相信广大玩家都不会陌生；然而，一代枭雄传奇式的一生，我们又了解多少呢？现在，就让我们一起来进入信长时代吧。
#1　　一、吉良海滨之战
　　天文16年，信长14岁便披挂出征，初试锋芒。本关的敌兵很少也很弱，况且还有平手政秀这个军师相助（他可以帮你回复耐久力），简直是练功的好机会。先由两名步兵挡住左边的两个海贼，别让他们占领你的本阵。现在你可以放心去消灭下边的敌兵了。^312901a^
　　本关宝物：回复豆 ×2 ，500金。
#1　　二、萱津之战
　　信长的父亲信秀病死，信长即位的第一件事就是镇压家臣坂井大膳的叛乱。本关将有平手政秀、织田信光、织田信行及猛将柴田胜家等援军相助。敌兵不多，但要小心敌军的四个枪骑兵。先让友军们当一下“人肉挡战牌”吧，你就从后突进，坐收渔人之利。
　　本关宝物：回复豆 ×2 ，800金。
#1　　三、镇压海贼
　　平手政秀的死谏令信长恍然大悟，使其踏出了称霸天下的第一步。本关的对手是一群海贼，乌合之众，何足道哉！多赚点经验值吧 。
　　本关宝物：回复麦 ，回复豆 ×2。 
#1　　四、清洲城攻略战
　　织田信友暗杀信长的阴谋败露后，信长决定将他铲除。本关有佐佐成政的加入，又有叔叔信光相助，取胜不难。但要注意千万不要孤军深入，否则只有用SL大法了。
　　本关宝物：回复豆、革张盾、600金。
#1　　五、长良川之战
　　斋藤家的义龙谋反，身为斋藤道三的女婿，信长当然不能就袖手旁观。面对强大的斋藤义龙部队，取胜与否，关键在于斋藤道三那边所能支撑时间的长短。若能支撑三至四个回合（这里要多试试SL大法），则可望创造全歼敌人的胜利条件。否则只能逃到右下角的村庄，达到第二个过关条件。注意，第二回合在地图下方会出现四个敌兵，要及早派兵拦截，否则本阵一失，只有仰天长叹了。
　　本关宝物：革张盾（打倒斋藤义龙后取得） ，500金（具备第一胜利条件过关才有的）。
#1　　六、稻生原之战
　　织田家重臣林秀贞拥立信长之弟信行为主，信长唯有出兵平叛（又是一场兄弟相争的悲剧）。本关敌兵不少，更有柴田胜家这员猛将在阵，不能强攻，只能诱敌深入，逐步蚕食敌人。
　　本关宝物：回复豆、回复麦、回复米，500金。
#1　　七、浮野之战
　　要称霸天下，就要先统一尾张。而要统一尾张，就要先打倒郡内另一股势力——织田信贤。如果你刚才没有派人杀掉信行的话，那么现在他又再背叛了你。你可以先埋伏几个人在右下角的树丛附近，兵力不能太少，他们的实力可不弱哦！
　　本关宝物：粟毛马、回复酒、梅，800金。
#1　　八、桶狭间袭击战
　　此战是战国时代的一次著名的袭击战，亦是信长一生最重要的一次战役。本关敌兵很强大，但不要太担心，他们是不会那么快攻过来的。发挥骑兵队的速度优势，先歼灭敌军先头部队，再对付今川义元。由于本关是在大雨下进行的（这是历史上的自然环境，不能用SL大法改变），在战前装备武器时要注意把“得意技”中没有“雨避”一项的铁炮队改装长枪，否则该部队大部分时间将会无法攻击。^312901b^
　　本关宝物：风神像、粟毛马（打倒松平元康后得到），黑毛马（打倒朝比奈泰朝后得到），藤弓（打倒今川义元后得到），1000金。
#1  　九、清洲城防御战
　　面对今川义元强大的军力，信长准备以清洲城为依托，打一场防御战。本关只要兵分两路，把守在城门周围，就可以逐一消灭进城的敌军，简直容易得像吃豆腐一样。不过，我还是极力推荐选择打“桶狭间袭击战”，毕竟此战是信长传奇一生的最光辉的一页，又是战国时期的一场重要战役，我们又怎能错过呢！
#1　　十、美浓的暗云
　　本关的盗贼绝对不是你的对手，尽情练功吧！噢，对了，还有木下藤吉郎（后来的丰臣秀吉）加入队伍。
　　本关的宝物：桶侧胴执，回复米×2，1500金。
#1　　十一、稻叶山城攻略战 
　　信长平息了美浓的暴乱，成功阻止了武田信玄的阴谋。下一步，是攻占稻叶山城，信长终于迈出了上京之路的第一步。本关的敌人很容易对付，而且还会有三个从属国的部队相助。
　　本关宝物：2000金。
#1　　十二、铃鹿的山神
　　哈，又是一帮土匪……大错特错了！
　　这次的土匪与以前的不同，不仅作战能力强，而且还会从村庄里涌出援兵。总而言之，本关切忌轻敌，一定要步步为营，逐一消灭敌人。本关除了村庄有宝物以外，土匪们的身上也有好东西哦（打倒他们吧，我的勇士们！把好东西都夺过来）。
　　本关宝物：名品太刀、名品打刀、藤弓、中回复药、小回复药，10500金。
#1　　十三、六角攻击战
　　信长受前将军义辉之弟义昭之托，出兵攻打六角家，三河的德川家康亦带兵参战。本关我方有明智光秀和不破光治加入，实力虽然不算太强，但对付这几个敌兵还是绰绰有余。先集中兵力消灭左下角的蒲生贺秀部队，然后再北上，讨六角义贤。如果你是一个练功狂，那么你就不要占领下面的箕作城，否则上面的六角义贤会率部逃跑的。
　　本关宝物：1000金。
#1　　十四、山贼之村
　　扫除了六角义贤这个障碍，信长终于顺利入京。信长入京后，虽然军纪严明，但由于有山贼隐患，治安很不稳定。于是，信长决定出兵剿灭山贼。本关的敌人很强，而且会从村庄中涌出援军，因此要加倍小心，步步为营，分批消灭敌军。
　　本关宝物：名品太刀、南蛮胴执 ×2 ，绀丝缄胴执（买不到的极品），1000金。
#1　　十五、三好的进攻
　　第十四代将军义荣的支持者三好三人对信长的进京极为不满，并且联合发动了攻势。本关的敌兵不难对付，只是要小心敌军的埋伏，以免被敌伏兵偷袭本阵。伏兵共有四处：第一处，三好长逸身后的树林。第二处，中央山坡右边的山脚。第三处，左下角村庄旁边的树林。第四处，三好政康的左下方。前三处伏兵只会在某几处特定地点触发，可打亦可不打。第四处伏兵在你接近三好政康部队时就会出现，而且有几个铁炮兵，因此要特别小心。最好先让有自动回复特性的信长或柴田胜家惊动伏兵，再集中火力消灭敌军。
　　本关宝物：白盾（打倒三好长逸后得到），名品长枪（打倒三好政康后得到） 回复米，1000金。
#1　　十六、伊势平定战
　　伊势的北富具教叛乱，率兵七万进攻信长，信长决定出兵镇压。本关将会有大炮兵九鬼嘉隆的加入。以你现在的实力，本关的敌军只是菜鸟一群，你完全可以在五个回合内摆平他们。
　　本关宝物：回复米 ×4 、回复麦 ×3、中回复药，1000金。
#1　　十七、金崎撤退战
　　面对有上京野心的朝仓义景，信长决定出兵讨伐。本关虽然有德川军的相助，但一看游戏给出的“99”回合的限制就知道本关的难度非同一般。当你到达地图右上角的时候（大约在 7－8 回合），敌大批援军会在地图中部及右下角出现。数量之多，吓你一跳吧！此时，你只有两个选择：一、消灭所有敌人；二、信长从地图右下角的村庄逃出。^312901c^
　　本关我用的战术是：先集中兵力歼灭海滩的敌兵，再北上。当出现敌援军时，分兵两路，消灭北部和中部敌军。当我把这两股敌人歼灭时，左下角的六角义贤部队已经到达中部山脉。哈哈，正好是我的饭后甜品！要注意的是右上角和左下角的两座城池不要进入，否则，后果可能令你很痛苦 …… 进入城池的我方部队“耐久力”降到1 ，更有可能会有大批伏兵出现。
　　本关宝物：回复米 ×5、中回复药、回复麦、黑毛马（打倒朝仓义景后得到），1000金。
#1　　十八、美浓迎击战
　　“甲斐之虎”武田信玄接到将军足利义昭的密诏，上京驱逐日渐强大的信长。信长联合德川迎击武田。本关的敌人是武田信玄——我的偶像，虽然于心不忍，但毕竟这个游戏是《织田信长传》而不是《武田信玄传》，我只好把心一横！一开始，山县昌景的先锋部队就会渡河进攻我军，而武田本队则不动。第三回合，武田胜赖的预备队会出现在地图的右上角。先集中击溃山县昌景，再回马收拾武田胜赖，最后就只剩下武田信玄了。第七回合德川援军将会在左下角的树林旁边出现，不过那时战斗已经快要结束了（你为什么不等明天才来？）。
　　本关宝物：全回复药 ×2，1000金。
#1　　十九、海的略夺者
　　终于阻止了武田的入侵，现在该收拾浅井和朝仓了。但在此之前，先要剿平抢略渔村的海贼。乌合之众，何足道哉！尽情捞经验值吧。留意一下海盗船被击中的动画，好Cool哟。
　　本关宝物：名品太刀、桶侧胴执、南蛮十匆筒（买不到的好东东，消灭所有海贼后得到）， 3000金。
#1　　二十、小谷城攻击战
　　浅井长政背弃了当初的誓盟，现在是报复的时候了。本关的敌军分为三股，分别在三座山寨中驻扎。他们既不主动出击，又不会相互支援，而且还不会走位攻击，只要你站到他们的攻击死角，就万无一失了。（这不是打活靶吗？）不过，在第八回合或你击退赤尾清纲后，敌援军会从左下角出现。要留下部分兵力抵御敌军，否则，本阵被占，就会GAME OVER 。当敌援军出现后两个回合，德川的援军也会加入战场助战，不过对于胜券在握的你，只是多此一举而已。
　　本关宝物：重藤弓，回复酒 水，回复米，鬼包丁，桶侧胴执，白盾（打倒浅井长政后得到），1000金
#1　　二十一、沛川之战
　　织田、德川联军与浅井、朝仓联军在沛川两岸集结，进行决战。本关一开始双方已经在河两岸摆开阵势，双方力量均衡，这样必然会导致我方一定的伤亡，而且本关宝物不多，似乎选择柴田胜家的建议更好！
　　本关宝物：白盾（打倒浅井长政后得到），1000 金
#1　　二十二、琵琶湖的湖贼
　　琵琶湖的湖贼又出来作恶，信长当然不会袖手旁观。本关的湖贼只是小菜一碟，又有经验值可赚了。
　　本关宝物：名品长枪，重藤弓，南蛮胴具足 X 3，鬼包丁，大竹弓，大身太刀， 2000金^312901d^
#1　　二十三、野田福岛攻略
　　好了伤疤忘了痛。三好三人众再度兴兵入侵，信长在野田福岛迎击敌军。战斗开始前，重臣森可成得疾病死去，但不要担心，其子森长可继承其位，而且各项技能数值不变。本关敌军的伏兵很多，而且作战能力很强，有一定难度。首先，第一批敌军……三好家的部队会在左下角驻扎，他们是不会主动进攻的。当你接近他们时，下间赖照的僧兵部队和铃木佐太夫的铁炮兵部队会出现在他们身边，他们的战斗力很强。因此，你不能一路狂冲，要注意各部队的协调，采用引蛇出洞，用大范围杀伤武器集中歼灭敌人的战术。另处，你要预留一半或至少三分之一的兵力（以攻击力为单位）在本阵附近，因为当你把下边的敌军收拾得差不多（大约是剩下七八个）的时候，浅井长政及朝仓义景的部队会突然出现在你本阵的左侧。他们不但人多，而且攻击力也很强。你最好留几个大炮兵在那里，先杀伤一部分敌兵，否则你将难以抵挡他们的猛烈攻击。本关的取胜要诀是战术与运气，祝你好运吧！
　　本关宝物：阿修罗像（打倒三好长逸后得到），大回复药，回复米，1000金
#1　　二十四、三方原会战
　　甲斐之虎武田信玄终于发兵上京了。德川家康不仅未能阻止武田的前进，反而节节败退，最后被迫退守居城滨松城，并向信长求助。而信长亦想藉此机会击溃自己一生所遇最强的敌人。就是这样，战国的将星们在滨松城以北的三方原展开激烈的大会战。本关敌军在中下部展开，兵力强大。德川的部队则困守滨松城，他们是不会出城进攻的，别对他们寄予厚望，还是靠自己吧。先集中所有兵力击溃敌先头部队，再对付武田信玄。要小心他的骑兵队，过于接近他们只会令你损失更多的兵力。应集中火力一次性消灭敌兵，否则他们可怕的攻击将会令你损兵折将。
　　本关宝物：黑毛马，回复酒 竹， 1000金
#1　　二十五、义昭追放战
　　最大的敌人武田信玄病逝，阻碍信长称霸之路的大山已经不复存在。只要铲除将军足利义昭，信长就可以名正言顺地成为日本的霸主了。本关的敌军主要分布为城外、城内两股，只要先消灭城外之敌，剩下的就不足为患了。
　　本关宝物：木盾，回复米，1000金
#1　　二十六、一兼谷小谷攻略战
　　信玄病死，将军义昭被逐，眼下就只有朝仓和浅井这两个敌人了。信长率兵进攻浅井的居城小谷城，朝仓亦发兵援助。两军在小谷城前开战。敌军集中在中央山脉的左侧，一开始敌军就会主动进攻，因此已出阵的先头部队不可贸然前进，一定要和主力会合，再迎击敌军，否则只会损兵折将。另外，当你击溃敌先头部队时，朝仓义景会向后方撤退。这时你不要急于追赶，因为只要你的部队踩到中央山脉左侧某特定点时，浅井长政的援军就会出现在峡谷的出口处。如果你全军追击的话，恐怕现在会被前后夹击，甚至被敌军偷袭了本阵。因此，应先在路口处集结兵力，再派人去踩踩点，先击溃浅井援军，再追歼朝仓余部。
　　本关宝物：回复米 X 2 ，1000金
#1　　二十七、长岛的一向宗门徒
　　甲斐的武田胜赖为再次西进开始了准备工作，信长为了专心迎击武田，必须消除背后的隐患——长岛的一向宗门徒。信长之子织田信忠亦参加了本次战役，开始了他的戎马生涯。本关开始只见零星的敌人，但你别开心得太早，当你接近他们时，周围就会出现伏兵。还是那句话，在主力未到之前不可冒进。（本关好像没有什么好东东，不必再到处搜索了。）
　　本关宝物：1000金
#1　　二十八、长条之战
　　武田胜赖继承信玄的遗志，再次西进。武田大军兵临长条城下，城主奥平信昌向信长求助。信长联合德川支援，在长条城下双方展开了著名的长条之战。战前建议选择“通常进攻”，不要选择“诱出作战”，以免造成兵力的分散。敌军主力全部集中在右侧的小岛上，用你的火枪大炮轰他们吧，胜利是属于你的。
　　本关宝物：1000金
#1　　二十九、滨松笼城战
　　如果你先前选择不向长条城发援军的话，本关你将被迫与德川军退守滨松城。敌军在左上角和右上角分两路进攻，我军主力居中。应先集中全力歼灭较接近我方的左上角的两股敌军，只要火力运用适当，应该不成问题。再回师截杀小山田信茂部队，大局已定，取胜不成问题。
　　本关宝物：1000金
#1　　三十、落泊武士
　　朝仓被灭，但其残余势力仍滋扰附近的村庄，信长决定出兵扫荡。开始敌军很少，且集中于围栏内，很容易对付。但要小心，当你接近敌军时，敌援军会出现在你后面。不过，面对强大的信长军势，他们的命运注定是失败。（本关的村庄是陷井，不要进去）
　　本关宝物：大竹弓，1000金
#1　　三十一、进攻本愿寺
　　武田死后，义昭又找到了一个能抗衡信长的强大靠山，他就是安芸的毛利隆元。为了铲除毛利的东进踏脚石，信长决定出兵围剿本愿寺及他的门徒。本关我军分布很散，不能形成有效的战斗力。各自为阵，又面对数目众多的敌人，我军处于明显的劣势；加之以柴田胜家、前田利家等多名主力缺阵，本关有一定难度。建议采用如下战术：各部队尽量牵制敌军，羽柴秀吉部队把中央小岛左侧的敌人尽快歼灭，继而会合右面的信长，击退本愿寺显如，速战速决。（显如就在中央小岛上，切忌恋战）
　　本关宝物：回复米 X2 ，1000金
#1　　三十二、松永久秀讨伐战
　　信长得知松永久秀秘密勾结本愿寺，决定出兵讨伐。本关敌军全数集结在城内，火力分配均匀，再加上三个大炮兵，实力非同小可。左边的三个仓库没有宝物，而且有伏兵，还是不进为妙。本关应集中兵力进攻正门，不要分散兵力攻侧门，分散兵力只会对敌人有利。只要接近松永久秀，他会自杀的，然后就顺利过关。
　　本关宝物：2000 金
#1　　三十三、一国一城
　　发现有来历不明的武装集团盘踞，信长决定出兵围剿。本关敌兵很多，但都密集于围栏内，因此，只要集中大炮火枪等到重火力，歼之并不是难事。
　　本关宝物：菊池枪，千鸟枪，1000金
#1　　三十四、荒木村重讨伐战
　　荒木村重、本愿寺、毛利、足利四家联合谋反，以图打倒信长。信长决定各个击破，先灭较弱的荒木村重。本关开始前，所有离开的部队将会归队，且物品及各项技能数值不变，大大增强了战斗力。本关的敌军虽多，但战斗力不强，可以集中火枪队一举歼灭。
　　进入本关的村庄，村民会告诉你附近有匹名马，千万别错过哦！
　　本关宝物：大炮，江（刀），名马（加三格的移动力），2000金
#1　　三十五、伊贺攻势
　　面对北伊贺恶党的威胁，信长决定出兵伊贺，平定乱党。本关你会有数名主力离队，但对方人马亦不算多，因此取胜的问题不大。敌人分三股，每股九人，可以各个击破，也可以来个一举歼灭。
　　本关宝物：铁盾，影杀，全回复药，2000金
#1　　三十六、本能寺兵变
　　如果先前的选择中你两次都打了明智光秀，就会引发出著名的本能寺兵变。此时如果你选择了“自杀”，就会引发出游戏的第一个结局。如果你选择了“逃跑”，那么本关你只管拼命往右上角的“藏”逃跑，成功率会很高的。你的兵力微弱，要想达到第二个胜利条件——击退明智光秀，简直是不可能的。不过，最明智的选择还是“以德服人”，别那么臭脾气，既可以改写历史，又不致于失去明智光秀这员猛将，何乐而不为呢？
#1　　三十七、愚蠢的门徒
　　伊贺的忍者残部想趁机溜走，信长当然不会让他们得逞。敌军人数很少，别上当，只要你一接近，他们的伏兵就会出现。先派人去接近敌军，主力集结在下面的树林中，待敌伏兵一出，先击溃下面的一股，再歼灭左上角的敌人，最后再回马收拾右上角的伏兵。要特别注意那些可怕的忍者，他们的集群攻击会损耗你有限的兵力，一定要尽早铲除这些祸根。
　　本关宝物：伊贺十字手里剑，村雨
#1　　三十八、四国平定战
　　信长接到四国的长宗我部元亲造反起兵的情报，马上出兵四国，平定四国之乱。本关敌兵不多，两股分列桥头左右，长宗我部亲兵在下面的小岛上。不要过早接近长宗我部的本队。否则他一旦逃跑，你就少挣很多经验值了。集中火力，一次打掉他。
　　本关宝物：大铁扇，5000金
#1　　三十九、关东平定战
　　最后一战了，武田被灭，上杉降伏，全日本就只剩下北条一个敌人了。兵分三路，右上方部队由柴田胜家率领，其任务是要缠住并消灭北条氏邦部队，不让其南下支援北条氏政。左下方是德川家康的援军，北条氏规的部队就交给他们好了。中央是信长的本队，实力雄厚，专门对付北条氏政。要特别注意，北条氏政身边有风魔小太郎率领的忍者队，他们的杀伤力相信大家在先前的几关已经领教过了，一不留神就会栽在他们手上。取胜的关键是如何把握移动方位与火力的最佳配合。既要躲避风魔小太郎的可怕攻击（要设法站到他们的攻击死角），又要得到火力齐射，一击必杀的攻击效率（各部队不能站得太远），似乎是比较困难的。唯有以最小的伤亡代价，换取较大的攻击效率。即是先由级数较低的部队在前面充当挡箭牌，待敌军攻击完后，再由后面的重火力部队齐射击杀忍者队，最好能一个回合全歼。没有了忍者队的守护，北条氏政就只剩下死路一条了。只要北条氏政一死，战斗就会结束，战争亦随之而告终，然后就会引发出游戏的第二个结局。
#1　　结局一：由于你的暴政逼反了大将明智光秀，引发了“本能寺兵变”（条件是在此之前的选择中你多次打了明智光秀），并且在兵变时选择“自杀”，那么就会出现历史中的那个结局：战国时代叱咤风云的织田信长被叛将明智光秀烧死于本能寺，正值盛年的他带着未竟的伟业含恨而终。
#1　　结局二：你改变了历史，信长终于统一了全日本，实现了战国群雄为之奋斗终生的梦想，成为日本的一代霸主！
