一步一步学WinAmp

 
1999年 第2期 10版

#1    安  装
    1.运行WinAmp的ZIP压缩格式自解安装程序WinAmp 205.exe（^021001a^1）。
    贝贝附注：
    WinAmp作为共享软件，却无任何试用限制，虽然如此，如果注册方便，大家更应该自觉注册（注册参见图3，个人注册费10美元）。WinAmp还有个与众不同的特点，即升级特别快，可以说每月都有升级，有时升级的时间间隔只有一两天。目前WinAmp的最新版是2.08版，这里介绍的是2.05版，在使用上与前者无甚区别，其下载网址为http:∥www.winamp.com/，文件大小为500KB。
    2.输入安装路径（^021001b^2）。按“Install”按钮开始安装。
    3.安装拷贝文件后出现连接提示窗（Dynamic Links），询问是否马上访问更新软件的网页http:∥update.winamp.com。按“No”继续。
    4.随后弹出WinAmp的关于窗口（^021001c^3），可以查看有关WinAmp的信息。按“确定”继续。
    5.WinAmp弹出设置窗口（参见图11），按“确定”退出。
    贝贝附注：
    安装退出后，WinAmp会自动启动，并演示播放一段4秒钟的MP3声音。WinAmp的安装主要是一个文件解压拷贝过程，并未设置开始菜单和桌面快捷方式，但大家可以自己设置桌面或菜单快捷方式（参见图11）。程序的图标如图4等也可以自己更改（参见图11）。
#1    播  放
#1    1.运行程序：
    进入安装WinAmp的目录，双击程序图标（^021001d^4）。
    贝贝推荐：
    如果设置了文件关联（参见图11），就可双击关联的声音文件直接启动WinAmp来播放了。也可以在图11中设置关联的声音文件图标。
#1    2.界面：
    WinAmp的界面模仿CD/VCD播放机的控制面板（^021001e^5），只要了解各个功能按钮，就可方便地播放音乐了。图5中包含了MP3播放器（含条状播放窗口）、均衡器、编辑播放序列等几个界面。
    贝贝附注：
    运行程序，一般只出现标准播放器界面。如要打开均衡器、播放序列或条状播放器界面，可点击图5所示的功能按钮或打开菜单选择（参见图10）。启动WinAmp后的界面将与上次关闭时的界面相同。另外，在界面的快捷工具栏中有五个字母，点击字母可以快捷地实现某项功能：“O”弹出“Option”设置菜单；“A”播放器界面显示在其它应用窗口的最前面；“I”显示当前播放的音乐文件信息；“D”将播放器界面放大一倍；“V”弹出“Visualization”界面外观设置菜单。
#1    3.选择音乐文件：
    点击播放器界面的打开文件按钮（参见图5、图7，也可选择菜单“Play File”项打开），从弹出的对话窗口中选定要播放的音乐文件（^021001f^6），如要选择多个文件可先按住“Ctrl”键然后用鼠标一一点选）。
    贝贝附注：
    WinAmp不但可以播放MP3音乐文件，还可以播放其它多种格式音乐文件，如MIDI、WAV、MOD等十多种格式的声音文件。
#1    4.播放：
    点击界面的播放按钮（^021001g^7），WinAmp就解压播放刚才选中的音乐文件，同时在时间窗口显示播放时间、播放的声音频普以及在播放文件信息窗口显示正在播放的文件名称。
#1    5.声音调节：
    可以在界面上直接调节声音的总音量、左右声道的音量比例，以及打开混响功能。播放时可以暂停、停止播放或跳转到下一首音乐播放（参见图5、图7）。
#1    6.音乐的均衡调节：
    参见图5打开均衡设置界面（^021001h^8），拖动界面中的滑条就可以调节播放的音乐频率均衡。
    贝贝建议：
    一般情况下，播放系统对低音和高音的播放声效赶不上中音，因此可以参见图8的均衡调节来设置，即加大对高低音的输出。当然，可以根据自己的电脑音响系统、爱好和音乐的不同来设置高中低音的输出比例。前置放大设置按钮可以统一调节音乐频率的均衡基准。另外，预设置按钮可以储存、读取调好的均衡值，以方便在播放不同的音乐时使用。
#1    7.序列播放：
    序列播放也称列表播放，打开序列播放窗口（^021001i^9，可参见图5），按“＋FILE”钮可以打开图6所示的打开文件窗口，然后选择音乐文件将之添加到列表窗口，按“MiSC OPTS”按钮，可以设置音乐文件的排序，然后就可以用排好的序列来播放这些音乐文件了。
    贝贝推荐：
    按“LOAD LIST”钮可以将编辑好的序列储存起来，以后就可以随时读取某项序列并按序列播放音乐，以免临时费劲地一首首地选择。
#1    8.播放网络音乐：
    打开菜单选择“Play Location”项，输入音乐网址，然后就可联网播放其中的音乐文件。
    贝贝附注：
    目前MP3站点很多，如著名的http:∥www.mp3.com，当然也可以将音乐文件下载来播放。
#1    9.菜单：
    点击播放器界面左上的按钮（参见图5），可以打开WinAmp的菜单来实现各种功能的操作（^021001j^10）。
#1    重要设置
    点选菜单项“Option|Preferences...”，弹出WinAmp的参数设置窗口（^021001k^11）。
#1    1.初始设置：
    见图11，可以设置文件关联、快捷方式，以及更换图标。
#1    2.分析器的视觉外观：
    分析器也称声音频谱显示窗，选择菜单项“Visualization”可以设置分析器窗口的声音频谱显示外观（参见图10）。
    贝贝附注：
    可以在图11所示的窗口选择设置各种插件，以及对WinAmp进行各种高级设置（Options标签项）。
#1    使用插件（Plug_In）
    WinAmp的插件可分为外观类“Visualization”、音频处理类“Audio Processing”、一般类“General Plug－Ins”、音频输入类“Aaudio Input”、音频输出类“Audio Ouput”。这些插件可以增强WinAmp对声音文件的处理、表现。
#1    1.下载插件：
    可从http:∥www.winamp.com/plug_ins下载插件文件。
#1    2.解压：
    用WinZip等软件解压插件文件到WinAmp插件目录Plugins。
    贝贝附注：
    WinAmp安装时在程序安装目录下建立了一个专放插件的目录Plugins，必须将解压的插件文件放在该目录下。
#1    3.选择插件：
    点选菜单“Visualization|Select Plug_In”选择插件（^021001l^12）。
    贝贝附注：
    图12的界面是选择外观视觉插件，实际上是图11的设置窗口的一个子标签项，点击标签“Setup”即是图11的界面。可以点击标签项“Audio”选择音频输入输出插件或在“Misc Plug－Ins”项选择其它类的插件。
#1    4.运行插件：
    选择菜单“Visualization|Start/Stop Plug_In”（参见图10），运行当前选中的插件。这样当播放音乐时，WinAmp就会运用插件程序提供的功能。例如使用WinAmp自带的一个外观插件，当WinAmp播放时，就可在播放器界面下随着音乐的节拍展现出绚丽的视觉图画（^021001m^13）。
#1    5.插件设置：
    打开参数设置窗口，如果要设置音频输入/输出插件的参数，可选择“Audio I/O”标签栏（^021001n^14）。然后选中某个插件，按属性“Configure”按钮，即可设置该插件的各种属性。
    贝贝附注：
    如果添加了下载的插件，WinAmp可以将之分类到不同的插件设置标签栏进行设置。如果是外观视觉插件，还可以在插件运行时右击其显示界面，弹出快捷菜单进行设置。WinAmp的插件众多，其设置和功能朋友们可以自己研究。
#1    使用外壳（Skin）
    WinAmp可以使用许多专门设计的外壳来改变自己的界面形象，使之更符合使用者的个性。
#1    1.下载外壳：
    从网址http:∥www.winamp.com/skins下载，该站点下共有938个外壳供选择。
#1    2.加入：
    将ZIP包拷贝到WinAmp的Skin目录下。
#1    3.选择：
    在菜单的“Option”下选“Skin Brower”项，出现外壳选择窗口（Skin  Brower）。选定一个外壳，确定后WinAmp播放器界面就改变成新的样式。图示的Sconicated和Gold Fnsion外壳都曾被评为五星级外壳。
    贝贝附注：
    还有很多很好的插件和外壳可供选择，也可从http:∥www.winamp.com/skins下载多达1175种WinAmp外壳和74种WinAmp插件。不过朋友们可以自己设计插件和外壳来相互交流，设计方法可访问http:∥www.winamp.com/plug－ins/dev.html和http:∥www.winamp.com/skins/dev.html。
