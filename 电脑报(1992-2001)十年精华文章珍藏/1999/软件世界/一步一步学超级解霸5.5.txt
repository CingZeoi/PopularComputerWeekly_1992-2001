一步一步学超级解霸5.5

 
1999年 第9期 10版

#1    安装和卸载
#1    1.启动安装程序
    插入超级解霸5.5的光盘，系统自动启动光盘上的安装程序（^091001a^1），也可以直接运行光盘根目录下的安装程序Setup.exe。然后选择“安装”按钮，程序弹出“欢迎安装”对话框，在对话框下面选择是否安装超级解霸的面版图案，如果勾掉，可以节省10MB左右的空间。按“下一步”进行安装目录设置。
    贝贝附注：超级解霸是豪杰公司出品的视频、音频解压软件，作者梁肇新。如果已经安装了超级解霸的老版本，安装新版本前必须先卸载老版本（参见图1）。
#1    2.设置安装目录
    在弹出的“选择安装目的地”对话框中（^091001b^2），按“浏览”按钮，输入安装目录，然后单击“下一步”继续。也可使用默认目录安装。
#1    3.安装超级VCD软解压测试版
    接下来，安装程序开始复制文件，然后自动进行必要的设置。完成后，安装程序弹出对话框，询问是否继续安装豪杰超级VCD的测试版，如选择“完成”按钮可结束安装。
#1    4.超级解霸的卸载
    超级解霸提供了自卸载程序，可在开始菜单的超级解霸程序组中选择“卸载超级解霸5.5”项，即可自动完成程序卸载。也可打开“开始\控制面板”中的“添加/删除程序”项，选择“豪杰超级解霸5.5”，单击“添加/删除”按钮完成卸载。
#1    手动启动与自动伺服器启动
#1    1.启动超级解霸
    超级解霸安装后，将在“开始”菜单中建立启动程序组（^091001c^3），可以选择启动超级解霸的不同程序。如从“开始”菜单中点击超级解霸程序组里的“超级解霸MMX5.5”，就将启动“超级解霸 MMX 5.5”程序来播放VCD。
    贝贝附注：超级解霸使用多个程序来播放VCD、CD、MP3等数字影音文件，是为了更好地完成不同文件的播放任务——虽然显得口罗嗦了一些。
#1    2.超级解霸MMX 5.5控制面版
    超级解霸播放VCD有两个不同的程序，一个是超级解霸5.5，一个就是超级解霸MMX 5.5，后一个程序支持带MMX多媒体指令的CPU。两个程序的控制面板基本相同（^091001d^4）。关于播放CD、MP3音乐的程序界面参见图11。
#1    3.自动伺服VCD影碟播放
    超级解霸安装后，将启动伺服VCD影碟自动播放的程序，并在Win95的状态栏中显示一个光盘图标（^091001e^5）。该程序可以在VCD影碟插入后自动探测识别，并开始播放。如果没有启动伺服器程序，可在开始菜单中的超级解霸程序组中运行“自动播放伺服器”。
#1    4.自动伺服的设置
    鼠标点击图5中所示的自动伺服图标，弹出自动伺服对话窗口，选择“设置”菜单，可以在菜单中选择需要设置的项目（大多数的项目可以参见图5所示的右键菜单），如任意勾选是否自动播放VCD、CD、MP3；控制弹出、关闭光驱托盘等。如果要关闭自动伺服程序，选择该设置菜单的“关闭探测器”项，程序会询问下次启动电脑时是否启动自动伺服器程序。
    贝贝建议：超级解霸的自动播放伺服器的功能是很有特色的，如果不是经常播放VCD，会耗费系统资源，可以关闭。但超级解霸5.5在伺服器程序中增加了一项“空闲时让CPU节能”的功能（可使CPU降温）。
#1    播放VCD
#1    1.播放VCD
    插入VCD光盘，点击图6所示的按钮“A”，超级解霸即可开始播放该VCD光盘。也可以选择菜单“文件\播放（P）VCD影碟”，如果启动了自动播放伺服器，则插入VCD影碟后就可以坐等超级解霸播放。注意：以下提到的按钮A、按钮B等等均是指^091001f^6所示的按钮。
#1    2.画面调节
    超级解霸默认的VCD播放画面是Windows窗口画面，如^091001g^7中所示的VCD默认播放画面大小为382×288像素（点击图6中按钮“T”可以查看播放信息）。可以拖动窗口的边框来改变画面的大小。如果要满屏显示，可以选择右上的窗口放大按钮，即可使显示器如电视机似的全屏播放。使用鼠标右键菜单的“满屏显示”或者点击图6中的按钮“J”也可以实现该功能。
    贝贝附注：当使用VCD满屏播放时，鼠标指针可以自动隐藏。另外据称，超级解霸以窗口方式播放VCD时，使用了一种直接写屏技术，播放画面速度赶上满屏显示方式。
#1    3.亮度调节
    选择控制面板上的亮度调节滑杆，上下拖动即可调节画面的亮度。如果是窗口画面，调节的亮度只是窗口中VCD的画面亮度，并未有改变整个屏幕的亮度。其它VCD播放软件很少有画面亮度调节的功能。点击按钮“D”，还可以设置显示画面为黑白，并且可以在面板中进行单色调的调节，如调节黑白画面略带红色。
#1    4.进度调节
    超级解霸的播放进度调节功能虽然还赶不上VCD机，但也算是基本齐全。点击按钮“R”是快进（参见图6），一次前进15秒左右。按钮“P”是到段头，按钮“S”是到段尾。按钮“L”是暂停，如要继续播放点击按钮“K”。按钮“N”是向前搜索，即向前播放约30帧的画面。
    当然，最方便的进度调节，是直接拖动进度滑杆到要播放的位置，即滑杆下的时间刻度指示。直接双击画面，弹出^091001h^8所示的进度调节对话窗口；也可以拖动其中的滑杆调节播放进度（该窗口中还可以进行段落选择）。不过，超级解霸警告，对于烂碟，并不提倡此种拖动方法。
#1    5.声音调节
    ●音量：选择控制界面窗口中的音量滑杆，上下拖动即可调节音量。
    ●左右声道的音量：选择音量调节上面的滑杆，左右拖动。
    ●声音开关：选择按钮“E”。
    ●选择声道：选择菜单“音频\左声道”，即可使超级解霸只播放左声道声音。当然也可以选择右声道。这项功能为使用双语配音的VCD提供了方便。
    ●卡拉OK：超级解霸支持话筒声音输入，也就是支持卡拉OK功能。选择菜单“音频\话筒声音输入”，开启话筒。然后同样在“音频”菜单中选择话筒输入的声音在左声道播放还是在右声道播放。在控制面板上可以控制输入话筒声音的音量。
    ●均衡调节：选择菜单“音频\设置频率均衡器”，打开均衡器调节窗口，根据需要调节。
    ●混音和回音：超级解霸自称播放VCD有类似影院的效果，因为它模仿了影院音响效果的混音或回音。可在面板上调节混音大小，在菜单“音频”中调节回音大小。
    ●采样频率：声音的采样频率决定了音质的好坏，可在菜单“音频”中选择声音播放的采样频率。建议对于非MMX的机器，采样频率最多设定在“正常44100Hz”。
#1    6.视频调节
    使用超级解霸，可以根据机器的情况，对播放的画面视频提供多种优化调节。如修改画质，设置颜色显示方式等。
#1    7.单帧与循环播放
    选择按钮“O”，可以单帧播放（快捷键Ctrl＋T），也就是一幅一幅画面播放。这就可以方便地查找需要的画面。点击按钮“C”，则可以将该帧画面以BMP的形式保存。
    如要设置循环播放，可以点击按钮“F”，拖动进度滑杆（^091001i^9），到要循环的开始位置。点击按钮“G”，设置循环开始位置。同样拖动滑杆到循环结束位置，点击按钮“H”设定循环结束。要取消循环播放，可再次点击按钮“F”。
#1    8.其它重要控制
    选择菜单“控制\控制设置”，打开控制设置窗口，在该窗口中可以选择多项功能设置，如^091001j^10。
    ●支持3D Now!：超级解霸5.5支持对带3D Now!指令的CPU优化。如果电脑中的CPU是K6－2，请一定要选择该项，据贝贝测试，可以增加播放速度15％左右。
    ●预读取：预读取功能可以加快播放速度。
    ●DirectCDROM直接读取：这是超级解霸引以为自豪的防读死功能，对播放质量不好的光碟很有效，是超级解霸向VCD影碟机看齐的重要功能。
    ●设置背景：可以设置超级解霸的控制面板的背景图案。但必须要安装背景图案。选择菜单“控制\选择背景”，在弹出的图案选择窗口中选择喜欢的背景。超级解霸提供了77种BMP格式的背景图片。
#1    9.结束播放
    点击按钮“B”，即可结束VCD的播放。另外超级解霸在文件菜单中还提供了开、关光驱托盘的命令。
#1    播放其它格式影音文件
#1    1.播放各种视频文件
    现在电脑里后缀为MPG/DAT/MPV/M1V/VBS的影视文件应用比较广泛，超级解霸提供了对这些文件格式的支持。播放这些文件只要在“文件”菜单下选“播放多个文件”或选择“打开一个文件”，再按面板上的“播放键”即可。如果要播放无文件的怪VCD影碟，可以选择菜单“文件\无文件影碟”。万能轨道播放方式可以播放非标准的VCD影碟，在播放不出图像时不妨一试。
#1    2.播放MP3等音频文件
    近年来，随着Internet的盛行，以MP1/MP2/MP3/MPA/ABS为后缀的音频文件也逐渐流行起来，超级解霸充分考虑到了这一点。在“开始”菜单超级解霸的程序组里选中“音频解霸5.5”或“音频解霸MMX 5.5”， 在启动程序后会出现相应的播放界面（^091001k^11）。播放的方法与播放影视文件大同小异。
#1    3.手动播放CD
    从“开始”菜单超级解霸的程序项里，选择“CD解霸5.5”便可以播放CD了。播放器的界面与音频播放器相似。
#1    4.播放超级VCD
    超级解霸5.5提供了播放超级VCD的测试版，该测试版可以播放超级VCD、SVCD、CVD等影碟。使用方法与播放VCD相似。但该程序对系统的要求相当高。
#1    其它功能
#1    1.设备查看
    超级解霸提供了程序来查看计算机硬件系统所拥有的功能，像DirectDraw和音频设备功能的查询（选择开始菜单中的相关程序选项）。另外超级解霸5.5还提供了对8位老声卡的支持。
#1    2.速度测试
    插入VCD光盘，选择菜单“控制\速度测试”，即可开始对系统播放VCD的速度进行测试。这种测试常常被使用来了解电脑的整体性能。
