一步一步学WinZip 7.0

 
1999年 第1期 10版

    WinZip是Nico Mak  Computing公司开发的著名ZIP压缩文件管理器，该工具有操作简便、压缩运行速度快等显著优点，能与网络浏览器Internet  Explorer和Netscape Communication实现无缝连接，可以大大方便Internet用户进行网上软件的下载、解压。下面我们贝贝学苑将给大家详细分解WinZip7.0各个功能的使用。
#1    安装
#1    1.运行安装文件：
    WinZip 7.0是在Win95/98环境下安装使用的工具，运行其安装文件WinZip70SR.exe（^011001a^1），弹出Setup对话框，单击框中的“Setup”按钮继续。
    贝贝附注:
   最新的WinZip版本是SR－1，可创建跨多个软盘的ZIP文件。WinZip最权威的下载网址是http:∥www.winzip.com。下载文件大小为900KB。
#1    2.设置安装路径：
    安装程序弹出WinZip  Setup对话框，在其中输入程序安装目录的名称（^011001b^2）。输入好后，单击“OK”按钮，程序弹出WinZip的功能说明框，再单击“Next”按钮继续。
#1    3.界面风格设置：
    程序弹出授权说明框，单击“Yes”按钮弹出如^011001c^3对话框，在此设置WinZip将以何种界面风格启动，选择使用标准窗口方式设置，单击“Next”按钮继续。贝贝附注：如果选择向导方式的设置，WinZip会以另一种方式安装，这种方式将查找硬盘中所有ZIP等格式的压缩文件，并记录在数据库中。
#1    4.安装设置：
    如^011001d^4设置，单击“Next”按钮继续。
#1    5.结束：
    安装完成后，单击界面中的“Finish”按钮结束安装。
     贝贝附注：
    WinZip安装后将会在桌面、开始菜单、文件右键快捷菜单设置多种快捷启动方式，并与压缩文件建立的关联（压缩文件的图标将替换成与图1相似的图标），以后双击压缩文件就可方便地调用WinZip程序打开。注意，未登记注册的WinZip启动后会有一注册提示框，可按“I Agree”继续。
#1    向导解压文件
#1    1.启动WinZip向导：
    如果在安装时已经设置了程序以向导界面方式启动，直接单击开始菜单或系统桌面上的WinZip快捷启动方式启动（^011001e^5）；如果已经处于WinZip的标准窗口界面，单击按钮条中的Wizard按钮启动（参见图9）。
#1    2.选择解压的文件：
    单击图5“Next”按钮，弹出^011001f^6所示界面，界面的列表框中列出程序以前搜索到的压缩文件，如果要解压的文件不在列表中，单击该界面中的“Search”按钮，程序弹出^011001g^7所示对话框，在其中选择需要扫描查找的对象，然后单击“OK”按钮开始查找，找完后自动回到列表对话框（即图6），在框中就列出了找到的压缩文件，用鼠标选择列表中需要解压的ZIP文件。注意，WinZip除ZIP格式外，还可以搜索解压LZH、ARJ、ARC、TAR、TGZ、GZ、Z和CAB等格式的压缩文件。
     贝贝推荐：
    如果在资源管理器中或文件夹窗口双击压缩文件，直接启动关联的WinZip程序，就可以不进行此步的文件扫描选择，而直接进行第三步开始文件解压。
#1    3.解压文件：
    单击图6“Next”按钮，WinZip会询问是否将要解压的文件目录记录入“最爱的ZIP文件”数据库，以便以后使用。选择回答后，程序弹出^011001h^8所示界面。单击“Unzip Now”按钮，程序开始解压文件，解压后弹出解压文件所在的文件窗口和解压结束提示界面。单击“Next”按钮可重回图6界面，单击“Close”按钮可退出程序，但在退出WinZip程序时，WinZip会询问下次以何种界面方式启动。     
#1    压缩文件
#1    1.启动标准界面窗口：
    如果程序启动的是向导界面，可参见图5跳转到标准界面窗口（^011001i^9）。
#1    2.新建：
    鼠标单击程序界面按钮工具条中的第一个按钮“New”（图9），程序弹出“New  Archive”对话框。
#1    3.设置名称：
    在弹出对话框中设置新建压缩文件的名称、压缩的格式等（^011001j^10）。
     贝贝提示：
    WinZip是先建立一个空的压缩文件（一般通称压缩包），以后再将要压缩的文件“打包”添加压缩进这个压缩包。WinZip可以创建ZIP等多种压缩格式的压缩包。
#1    4.添加压缩文件：
    选择图10中的“Add Dialog”复选框，单击OK按钮，程序给出关联的“Add”设置对话框（^011001k^11）。在“Add from”列表框中选择需进行压缩打包的文件和文件夹，按“Add”按钮将其添加压缩到刚才建立的空压缩文件中。
     贝贝附注：
    如果要压缩当前目录下列出的全部文件，点击“Add with wildcards”按钮；如果要保留压缩对象的目录结构，选择使用“Folders”项中的“Save extra folder info”复选框；如果要压缩文件夹及其下的子文件夹，选择“Include subfolders”复选框；要压缩文件夹中隐藏的文件，选择Attributes项中的“Include system and hidden files”复选框。
#1    5.完成：
    程序完成压缩后回到主窗口界面（图9）。
#1    查看和试用压缩包中文件
#1    1.打开压缩文件包：
    进入标准窗口界面方式，单击“Open”按钮，选择需打开的压缩文件包，单击“OK”，WinZip自动解压文件，并将解压的文件存入临时文件夹，然后在标准的界面窗口列出压缩包里的文件（参见图9），现在可以像一般的Windows文件窗口一样，查看列出的压缩包里的文件了。
     贝贝提示：
    熟悉WinZip的朋友应充分使用这项功能，它可以在不解压的情况下了解压缩文件包里的文件内容，如果有需要可再进行解压操作。当然在Windows窗口中直接双击压缩文件打开查看压缩包会更方便，也可以将压缩文件从别的窗口中直接拖动到WinZip的标准界面窗口来打开。这些功能都非常适合上网的朋友使用。另外，选择菜单“Actions|Virus Scan”项还可以调用第三方防毒软件检测压缩包中是否有病毒。
#1    2.查看设置：
    单击WinZip窗口界面（图9）中的“CheckOut”按钮，可以修改查看打开压缩包文件时用的临时文件夹和文件名称。
#1    3.试用压缩包中的程序：
    可以在WinZip标准窗口中直接运行压缩包中的应用程序。如果打开的文件压缩文件包中有安装程序（Setup.exe或者Install.exe文件），WinZip界面中的“CheckOut”按钮会自动变成“Install”按钮，单击该按钮，WinZip弹出^011001l^12所示对话框，单击“OK”按钮，就可进行程序的安装。
     贝贝推荐：
    试用压缩包中的程序功能非常适合网民下载试用软件。但如果要使用图12中的反安装功能，需要有WinZip的一个插件程序WinZip  Install/Try Uninstall  Component，这个插件有100余KB，提供的功能很好，可以用它删除安装程序在系统中安装的所有东西。
#1    4.解压文件：
    在如图9的压缩包文件列表窗口中选择需进行解压的文件，点击界面中的“Extract”按钮，弹出解压设置对话框，具体设置操作见^011001m^13。
#1    5.关闭WinZip：
    关闭时程序会弹出一对话框，询问用户是否保留临时解压用的文件夹。
#1   快捷压缩文件
    WinZip的压缩过程实际上是先创建一个空压缩包，然后就可以不断地向其中添加文件进行压缩打包。WinZip程序提供了多种快捷添加方式，这里简要列出各种操作方法：
    1.在资源管理器窗口中（或者文件窗口），使用鼠标右键单击需进行压缩的文件夹（如Files文件夹），程序会给出系统鼠标右键关联菜单（^011001n^14），选择其中的“Add to Zip”命令，程序会弹出“Add”对话框（参见图10），即可进行压缩包的创建和向压缩包中添加压缩文件的操作。而选择“Add to Files.zip”菜单命令，程序可在文件当前目录下创建一个名为“Files.zip”的压缩文件。
    2.用WinZip打开一个文件压缩包，使用鼠标直接将文件窗口中需要压缩打包的文件和文件夹拖曳到打开的WinZip窗口中即可（参见图9）。
    3.启动系统的资源管理器，找到需进行压缩打包的文件和已建好的一个文件压缩包，然后拖曳需压缩的文件图标到相应的压缩包文件图标上，压缩过程就可以自动完成了！当然也可以在桌面上打开文件窗口来操作（^011001o^15）。
     贝贝推荐：
    使用以上快捷压缩操作方式，朋友们就可以方便地备份数据和暂时不用的程序了。
#1    创建分卷压缩文件
#1    1.创建：
    启动WinZip到标准窗口界面方式（图9），单击按钮条中的“New”按钮，程序弹出“New  Archive”对话框（图10）；
#1    2.设置文件名：
    在“File Name”文本框中输入需创建的文件压缩包名称和软盘盘符；
#1    3.压缩打包文件：
    单击“OK”按钮，程序弹出“Add”设置对话框（图11）。在“Multiple Disk Spanning”下拉文本框中设置分卷压缩方式。其中“Automatic”项自动检测软盘可用空间，并进行分卷压缩文件的创建，一般选择此项。“Automatic  ＋ wipe first disk prompt”项自动检测软盘的可用空间，并进行分卷压缩文件的创建。在进行压缩时，提示用户是否删除软盘中已有的数据。“No spanning”项不进行分卷文件创建，只向软盘中压缩数据。
#1    4.压缩：
    单击“Add with  wildcard”按钮，程序开始压缩，当软盘满了时，WinZip会提示插入下一张软盘。
     贝贝附注：
使用WinZip进行文件的分卷压缩，只能在软盘驱动器中完成，如果其中一个软盘坏了，则要完全重新制作。
#1   创建自解压文件压缩包
    使用WinZip创建自解压文件压缩包要首先创建一个ZIP压缩格式的文件压缩包，然后用WinZip的压缩包格式转换功能将其转换为具有自解压能力的EXE可执行文件。
#1    1.创建Zip文件：
    使用WinZip创建一个ZIP压缩文件。如果ZIP文件已经存在，鼠标双击该文件图标，用程序标准窗口界面打开。
#1    2.转换：
    单击程序菜单条中的“Actions |Make .Exe file”命令。
#1    3.解压方式设置：
    在弹出的对话框中的“Default ‘Unzip To’ folder”文本框中设置自解压的默认解压目录；在“Self Extracting  Type”项中选择生成的EXE文件类型，如可在Windows 3.2下使用的16位应用程序和在Win95/98/NT下使用的32位应用程序。
     贝贝附注：
    如果要创建自解压的分卷压缩文件，应在“Spanning Support”项中设置，该项中提供三个复选设置框，其中“No spanning”表示不创建分卷文件、“Safe  spanning method”表示使用安全拆分方式创建EXE文件、“Old spanning  method”表示使用旧的方式拆分文件。建议选择“Safe spanning method”。
#1    4.转换：
    单击“OK”按钮确定，按程序要求提示操作即可。
     贝贝提示：
    可以在资源管理器窗口中使用鼠标右键单击需进行转换的ZIP文件，然后从关联菜单中使用“Create  Self－Extractor（.EXE）”命令（参见图14）即可。另外，为将ZIP文件转换为自解压文件，WinZip的开发公司提供了一个名为WinZip Self－Extractor的专业工具，使用向导方式操作，可以制作出很好的应用程序安装文件。
