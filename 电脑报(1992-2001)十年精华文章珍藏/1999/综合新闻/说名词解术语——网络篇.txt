说名词解术语——网络篇

何嘉、徐虹
1999年 第9期 07版

 　　相信上网是许多人心中向往的事情，如果你也是其中的一员，不妨跟我们一起先熟悉一下网络世界的常见术语。
#1　　1.因特网（Internet） ：
    是一个全球的、开放的计算机互连网络。也是一个用户自行管理的特殊网络。它可以为网络上的用户提供丰富的信息资源和各种网络服务。Internet 基本功能有：发送邮件、发布新闻、远程登录、文件传输和信息检索等等。它目前已成为世界上规模最大、用户最多、影响最广的国际互连网络系统。
#1　　2.冲浪：
    如果把 Internet 比作海洋，你的计算机就是一叶小舟，在 Internet 上遨游就是冲浪。
#1　　3.调制解调器（MODEM）：
    戏称“猫”，是上网必需的设备。它可以将计算机的数字信号转换（调制— Modulation）成模拟信号，通过电话线路传送到另一台计算机（比如服务器）上，接受端用它的 Modem 将其还原（解调—Demodulation）为计算机能识别的数字信号，从而实现双方的信息交换。 Modem 一般分为内置式和外置式两种。常用的速率有：14.4 Kbps，28.8 Kbps，33.6 Kbps  和56 Kbps 几种。 bps:比特／秒。
#1　　4.因特网服务提供商（ISP： Internet  Service Provider ）：
    用户只有通过ISP才能连接到Internet 网。也就是：在你上网之前，需要从ISP 那里得到登录网络的账号及密码。而且你还应该事先从ISP那里了解到使用网络的各种费用等。
#1　　5.超文本标识语言（HTML——HyperText Markup Language）：
    它是 Internet 的标准文本格式。也就是说，WWW上的信息是以HTML 格式来进行组织的。例如：你在网络上所见到的画面都称作网页，它们包括一般意义上的文本、图片、声音、视频和动画等等。另外，可以通过 HTML 格式上允许定义的超级链接（HyperLink）访问网页。也就是，当你将鼠标在屏幕画面上移动时，在某些地方鼠标指针变成了一只手，单击鼠标左键，你就可以进入其它的网点。另外一个典型的应用是，如果在电子邮件中使用 HTML 格式，你还可以在邮件中添加图形和指向 Web 站点的链接。
#1　　6.TCP／IP（Transmission  Control  Protocol  ／  Internet  Protocol）：
    传输控制协议与互连网协议。它是国际互连网上的所有网络进行交流的共同“语言”，是  Internet 上使用的一组完整的标准网络连接协议。TCP 对应 七层模型的传输层，IP 对应网络层。TCP／IP除了本身的两个协议（TCP 和 IP）之外，还包括其它多种协议。例如：FTP（文件传输协议）、SMTP（电子邮件协议）、TELNET（远程登录协议）等等。
#1　　7.服务器（Server）：
    一台功能强大的计算机，有较高的速度、较大的存储空间以及断电保护措施等。它能够在网络中为用户提供通讯、文件共享、网络打印等功能。
#1　　8.主页（Home Page）：
    它的全称应该是WWW主页。它指的是WWW服务的起始装入信息页，是WWW用户（比如你）可以看见的第一信息界面。你连接到主页后，除了可以直接在主页了解到主页制作者的一般信息外，还可以在主页上移动鼠标，当鼠标的形状变成一只手的时候，单击鼠标的左键，就可以又进入到另外的一个画面再进一步地获取到更多的信息。也就是通过主页上定义的超级链接（HyperLink）访问网页。
#1　　9.WWW（World Wide  Web）：
    简单地说，WWW是漫游Internet 网的工具。它是一个基于超文本方式的信息检索工具，它为用户提供了一种友好的信息查询接口，是 Internet 上的信息服务系统。它把 Internet上不同地点的相关信息聚集起来，通过 WWW 浏览器（又叫做 Web浏览器，如：Netscape、IE 等）检索它们，无论用户所需的信息在什么地方，只要浏览器为用户检索到之后，就可以将这些信息（文字、图片、动画、声音等）“提取”到用户的计算机屏幕上。由于WWW 采用了超文本链接，你只需轻轻单击鼠标，就可以很方便地从一个信息页转移到另一个信息页。WWW是目前Internet 上最让人倾心的系统。
#1　　10.电子邮件（E－MAIL）：
    这是一种最常用的Internet 功能，也是一种最便捷的利用计算机和通信网络传递信息的现代化手段。电子邮件的传递由SMTP 协议来完成。网络用户可以通过 Internet 与全世界的 Internet 用户收发信件。电子邮件的内容不仅仅包含文字，还可以包含图像、声音、动画等多媒体信息。你可以通过申请获得自己的信箱，使用计算机撰写邮件，通过网络送往对方信箱，对方从自己的信箱中读取信件。电子邮件因其快速、价廉而被广泛采用。常用的电子邮件软件有 Eudora、Foxmail 、Outlook Express等。
#1　　11.电子邮件地址（E－MAIL地址）：
    电子邮件地址和收发信件人的地址意义相同。只不过它必须以特殊的格式来标识。其一般形式为：用户名@主机域名。例如，本报综合报道版块的电子邮件地址为 :  report@cpcw.com 。
#1　　12.POP3 和SMTP ：
    POP3 ——邮局协议3，是 Post Office Protocol 3 的缩写。它是一种接收邮件的协议。说明怎样将你的计算机与Internet 上的邮件服务器相连接，从而将邮件下载到你的计算机上。这样，你就可以脱机（即不在网上）阅读和撰写邮件。支持POP3 的服务器是一种能保存外界发送给你的电子邮件的服务器。SMTP——简单邮件传送协议，是Simple Message Transfer Protocol 的缩写。SMTP服务器将把你的邮件发送出去。
#1　　13.http（HyperText Transfer Protocol）：
    超文本传输协议。该协议经常用来在网络上传送Web 页。当用户以 http:／／ 开始一个链接的名字时，是告诉浏览器去访问使用 http 的Web 页。
#1　　14.URL（Uniform Resource Locator）：
    统一资源定位格式。是Web 的地址编码。Web 上所能访问的资源都有一个唯一的URL。URL 包括：所用的传输协议、服务器名称、文件的完整路径。例如：在浏览器URL处键入：www.cpcw.com 就可以访问《电脑报》的主页。
#1　　15.IP地址（ Internet  Protocol ）：
    互连网络协议地址。Internet 上每台主机都有一个唯一的地址（或者叫名字），它是数字形式的一组地址，例如：202.98.96.68。它是计算机在网络上的数字标识，一般分为两部分：网络地址（NETID）、主机地址（HOSTID）。IP 地址分为五类：A、B、C、D、E，常用的是B、C两类。上面的地址是C类地址。
#1　　16.域名：
    也是 Internet 上主机的名字。它是一种有意义名词的缩写，用英文句点分开的一组地址。一般主机域名可以以机构或者地域作区分。以机构区分的域名有：.com（商业机构）、.edu（教育机构）、.net （网络机构）、.info（信息服务机构）等等；以地域区分的域名有：.cn（中国）、.us（美国）、.uk（英国）、.au（澳大利亚）等等。域名采用层次结构，从左至右，从小范围到大范围，表示主机所属的层次关系。例如：lib.swnu.edu.cn 。“lib”是图书馆的缩写，“swnu”是西南师范大学的缩写，“edu”是教育机构的缩写，“cn”是中国的缩写。这样，从域名地址就可以知道这是中国教育网上的西南师范大学校园网上图书馆的某一台机器。
#1　　17.DNS服务器（Domain Name System Server）：
    域名服务器。它的作用是把用户使用WWW 浏览器时输入的好记的域名地址翻译成难记的IP 地址。
#1　　18.Telnet：
    远程登录。它是Internet 上最早的应用程序之一，也是Internet 上最早提供的常用服务。Telnet 是Internet 上的远程访问工具。利用它，在网络通信协议的支持下，用户计算机上暂时成为远程计算机终端。这样，用户计算机就可以使用对方远程计算机对外开放的全部资源。
#1　　19.FTP（File Transfer Protocol）：
    文件传输协议。它是Internet 上提供的常用服务之一。FTP实际上是一个程序，是Internet 上的文件传递工具。网络上的用户可以通过 FTP 功能登录到远程计算机，从其它计算系统中下载需要的文件，也可以将自己的文件上传到网络上。用FTP 登录到服务器的方式有两种：一种是以该系统的正式用户登录，另一种是以匿名（anonymous）用户登录。后者不需要账号和密码就可以登录到远程计算机上获取大量免费信息。
#1　　20.新闻组（ Newsgroup）：
    新闻组是Internet 上提供的服务之一。是个人向新闻服务器所投递邮件的集合。你可以查找任何特殊主题的新闻组，而且访问过该新闻组的任何人都可以免费加入：可以投递或阅读邮件。但是 ISP 必须预先为你建立一条与新闻服务器的链接，以便像 Microsoft Outlook Express  这样的新闻阅读软件能在该新闻服务器上设置账号。设置账号后，可以在该新闻服务器所存储的新闻组中随意阅读和投递邮件。新闻服务器由公司、群组或个人负责维护，并且可以管理成千上万个新闻组。
#1　　21.BBS（Bulletion Board  System）:
     电子公告栏。是一种交互性强、内容丰富而及时的 Internet 电子信息服务系统。用户可以通过 Modem 和电话线登录 BBS 站点 ，也可以通过Internet 登录。用户在 BBS 站点上可以获得各种信息服务：下载软件、发布信息、进行讨论、聊天等。BBS 站点的日常维护由 BBS 站长负责。
#1　　22.搜索引擎（Search Engineer）: 
    在Internet 网上使用WWW浏览器作为查询工具查询信息时，不可避免地会用到搜索引擎。搜索引擎是一个用来查询搜索世界各地Internet 网络资源的Web 服务器。它就象一本书的目录，Internet 上各个站点的网址就像是页码。你可以通过关键字（Keyword）或者主题分类的方式查找感兴趣的信息所在的Web 页面。著名的搜索引擎有：
YAHOO！  http:／／www.yahoo.com ；   
INFOSEEK GUIDE    http:／／www.guide.infoseek.com ；
搜狐    http:／／www.sohoo.com.cn  等。
#1　　23.下载（DownLoad）：
    通过网络从其他计算机中拷贝软件到自己计算机的过程，是一种获取软件、图片等最便捷的途径。
#1　　24.上传：
    将自己计算机中的文件拷贝到网络中其他计算机的过程。
#1　　25.（网络寻呼机）ICQ：
    它是“I Seek You”的连续读音缩写，含义是“我找你”。是目前世界上最流行的网络寻呼机，用户数量已超过一千万。网络寻呼机和人们日常使用的寻呼机相似，便于你在网络上与其他人快速建立联系。首先，你需要连线申请一个ICQ号码（UIN—Uninersal Internet Number），这就是你的网络 BP 机号码。以后你上网运行ICQ软件，就能够即时了解到和你同在网上的朋友，也能够让网上其他的人知道你也在网上。如果你需要和他人进行联系，通过ICQ 就可以马上让他们知道。这样你们就可以进行相互间的邮件传送（E－mail）、文件传送（File）、交谈（Chat）、打电话（Iphone）等。
#1　　26.网管（Administrator）：
    网络管理员。他们一般负责对网络服务器进行管理和维护。
#1　　27.资源和资源共享：
    资源是指在计算机系统中，处理一个任务或者作业时所需要的硬件设备和软件设备的总称。在硬件设备方面，包括：主存储器、输入输出设备、中央处理部件等；在软件设备方面，包括：数据库、数据集、控制程序和加工程序等。资源共享是指多个用户共用计算机系统中的资源，主要是CPU时间和内存空间的共享。在网络盛行的今天，还可以在网络上共享其它的外部设备，比如硬盘、打印机等。
