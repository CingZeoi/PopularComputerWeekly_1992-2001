PC Tools 9.0使用指南


1994-01-07

        PC Tools最早出现于1985年(第1版)，它仅仅是一个带单个菜单的DOS外壳。85年推出的第2版，将磁盘功能从文件功能中分离出来，放在了一个单独的菜单中，从而使该软件成为真正的PC Tools。
        第2版的PC Tools能以常驻内存形式运行，可方便地管理文件、日期和时间等属性；具有了“修剪和移植”的功能。还有可以编辑任何文件的任何字节或磁盘的任何部分的一个工具。
        1986年推出第3版，第一次具有了外部程序特性。该版本增加了镜像(Mirror)和重建(Rebuild-Unformat的前身)、压缩(Compress)、PC备份(PC Backup)和PC转储(PC Restore)等功能。
        1987年推出的第4版，精简了PC Backup，使得通过单个程序就能备份和转储，并增加了PC-Cache和PC Format功能。随后在88年推出的4.XX_5.0版，使用更为方便灵活，致使许多用户爱不释手，不愿更换。
        1989年，先是推出5.1版。这是一个大的改变，它使用了新的界面，用PC Shell代替了原来的PC Tools。它支持鼠标器操作，采用流行的下拉式菜单，增加了Desktop Manager和PC Secure等功能。
        同年6月，又推出功能更强的5.5版。在许多程序中增加了快速菜单，并初次使该软件能用在网络上。精简了许多激活命令所需的击键序列。
        1990年推出的第6版，作了一些微小的改进。主要是增加了磁盘维护(DiskFix)功能。
        1991年出现7.0版，它综合了各种PC实用程序，可保护数据、简化操作，增强系统，并支持Windows操作环境。
        同年推出的7.X版，使PC Tools的各种功能更日趋完善，除了能进行文件与程序管理、磁盘维修、数据保护等之外，还有笔记本、数据库、剪贴版、宏编辑、发送和接收电子邮件以及传真等功能，尤其是恢复数据、网络支持、系统检测、文件修补，更为有效，增加的紧急恢复磁盘，可在硬盘无法启动时用于启动计算机，它还增加了对500多种病毒的防范程序。
        1992年推出现在的8.0版。除了完全具有以前版本的特点和功能外，将过渡版7.X发展为使用方便、高效、设计合理、功能强大、完备的PC Tools，它将以前的PC Shell改为PC Desktop，PC Desktop改为Desktop Accessories，增加了防病毒程序CPAV，可检查和删除1300多种病毒。此外，还增加了可恢复磁盘信息的应急磁盘程序Edisk，查看和编辑磁盘扇区的DiskEdit等程序或功能。
        二、PC Tools V9.0新增加的内容
        1.系统状态恢复程序CP Recuperator
        Central Point的Recuperator程序是一个全新的实用工具，是PC Tools V9.0的重要特色之一。该程序可以间隔一定时间后自动地将用户PC机中的CPU和内存状态存储起来，以便在系统出现崩溃、死锁或断电时，将事故之前的状态恢复。发生事故时，用户可以重新启动微机。CP Recuperator可以根据用户的意图，将用户的系统(包括完整的数据文件)恢复到最后一次被存储的状态。CPR所作的存储总是间隔一定的时间段，并且只有在系统“空闲”期间进行以减少对用户工作的干扰。
        2.支持Stacker、DoubleSpace驱动器以及大的Volumes
        Optimizer(在以前的版本中叫做Compress)，DiskFix和Backup在新版的PC Tools中可支持Stacker和DoubleSpace压缩驱动器。DiskFix除了能在DoubleSpace上查找压缩错误以外，还可以分析并修理这些Volumes。磁盘优化程序Optimizer和磁盘修复程序DiskFix现在可支持2GB的驱动器。
        3.最新防病毒软件CPAV 2.0
        PC Tools V9.0的一个重要特色是在CPAV2.0中使用了很高级的防病毒技术。这个新的病毒分析程序除了能检测和清除的已知病毒数目由原来的1500种增加到2000多种外，更为重要的是使用了一种专门检测系统文件和引导扇区中的未知病毒而不用要求任何以前的感染资料。没有改进的病毒特征资料，CPAV2.0可使用SmartChecks检测和清除未知病毒。除此之外，特征文件可更新更多的文件引导扇区以及内存病毒信息，并可由BOOTSAFE，CPAV.EXE以及VSAFE来使用。新版的CPAV2.0可扫描由自我释放和一般压缩程序(包括PKZIP，PKLITE，PLZEXE以及ARJ)压缩的文件。扫描速度大大加快，并减少了对TSR内存的要求。
        4.新的集成硬件诊断程序SIPRO
        PC Tools V9.0中的系统信息程序SI得到了很大的改进，名称由SI改为SI PRO(System Information Professional)，它不但可为用户的PC机提供广泛的资料，还可以对CPU，监视器，内存，CMOS，硬盘和软盘检测，声音卡和其它很多设备进行诊断检测。
        5.全新的高速软盘备份程序FastCopy
        FastCopy也是PC Tools V9.0新增加的程序。它可以快速进行软盘的多个备份。还可以用FastCopy存储软盘在硬盘上的映像，以便以后快速备份该盘。另一个特点是介质转换，即将备份盘从一种格式转换到另一种格式(假设目标盘小于源盘)。
        6.任何转换程序Task Switcher的改进
        新的Task Switcher可激发预约事件(以前是用CPSCHED处理的)，并允许指定在计算机空闲时运行的程序。
        7.数据监视程序DataMon的改变
        新版的DataMon需要更少的TSR内存。磁盘灯(Disk Light)不再包含在DATAMON内，而是放在了DSKLIGHT.EXE程序内。恢复删除程序Undelete可提供删除保护的功能。
        三、PC Tools 9.0的安装
        将PC Tools 9.0系统的第1号盘插入A驱动器，在A驱动器上敲入：
        A〉install
        屏幕上立即出现
        Loading install…
        然后自动进入PC Tools安装窗口：
        PC Tools installation
        Scanning Memory for Virus…
        即检查系统内存有无病毒，检查完毕，进入监视器类型选择窗口。
        Color对应于彩色显示，B/W对应于黑白显示。Monochrome对应的是单色显示器，LCD选项适用于具有LCD显示器(通常laptops)的计算机。
        Graphic fonts用以显示在PC Tools程序中使用的特殊字符。
        屏幕最下边一行是功能键与提示信息：F1用于寻求帮助，F3退出。在安装过程中的任何时候，都可执行这两个功能。
        用Tab键和光标键移到你所采用的显示器选项，并用空格或回车来选定，也可用Alt加选择栏中的亮字符来选择该项。PC Tools 9.0在安装时，就可以使用鼠标器，建议你使用鼠标来操作。
        通常我们使用缺省值(Color)项即可。
        选好后，按OK确认。
        下一屏是姓名及公司名称：
        你可以在输入姓名及公司名称之后选OK，对普通用户而言，这项没多少意义。
        下一个屏幕提示：
        恢复文件	恢复被删文件	
        磁盘维修	如果你的一个硬盘出了毛病	
        格式化恢复	如果你错误地格式化了一个硬盘	
        文件维修	恢复损坏记录、数据库或文字处理文件
        
        选择安装类型
        
        -〉全部安装   此选项将把所有PC Tools程序
        最小安装      安装到你的计算机的硬盘中，
        Laptop安装    它提供PC Tools全部功能，安
        自选安装      装程序使用路径环境变量以寻
        拆除PC Tools	找Windows 3.x目录。全部安装要求9.9M的硬盘空间。
        上图右边窗口中的信息为左侧选中的安装类型的说明信息，缺省值项为全部安装，按OK键。
        指定安装目录
        敲入你要安装PC TOOLS的目录全名，确认路径名中包含驱动器。
        若接受此路径名选OK
        C：\PCTOOLS
        OK   回退   退出
        你可以使用上述缺省目录名，也可以输入一个新目录。如果指定的目录不存在，Install将提问是否为你建立其目录。如果是多级目录，只有最后一级目录才能创建。
        路径名必须包含一个有效硬盘盘符和目录名。
        选OK后，系统将快速搜索Windows，并出现如下对话框：
        如果你准备由Windows启用PC Tools应用程序，敲入你已配置过的Windows 3.X所在的目录全名(WIN.INI文件在此目录中)，确认包括驱动器的路径。按F1可以得到更多的信息。
        C：\WINDOWS
        OK  跳过  回退  退出
        屏幕提示框中显示的目录是Install程序搜索WIN.INI得到的目录，并提供四种处理选择，除非有必要，否则选择OK选项。
        如果不打算从Windows 3.X启用PC Tools应用程序，可选跳过。
        这时，屏幕显示：
        请等待
        检查硬盘
        若发现硬盘错误，会出现“硬盘遇到错误，运行DISKFIX程序”(Disk errors encountered.Run Diskfix?)窗口。
        如果硬盘无错误，屏幕随后出现：
        选择  OK
        开始拷贝文件到你的硬盘
        OK 回退 退出
        选择OK键，屏幕会出现上下两个窗口，上面窗口显示不断变化的新版PC Tools的特点及相关信息，下面窗口则带着“读磁盘”标题，并显示正在拷贝文件的过程。安装程序依次提示插入相应的软盘，直到安装完为止。随后还有建立应急磁盘，建立PC format等功能，这里不讨论，可用skip跳过。
